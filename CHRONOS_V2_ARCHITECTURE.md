## 五大核心引擎

### 1. Paper Discovery Engine（论文发现引擎）

**功能：** 自动发现最新研究

**数据源：**
| 来源 | 每日目标 | 领域过滤 |
|------|----------|----------|
| arXiv | 30 篇 | astro-ph, hep-th, cond-mat, quant-ph, q-bio |
| Semantic Scholar | 20 篇 | 全学科（AI 相关性排序） |
| Nature | 5 篇 | 物理、生物、宇宙学 |
| Science | 5 篇 | 物理、生物、宇宙学 |
| NASA Astrobiology | 5 篇 | 生命起源 |
| Santa Fe Institute | 5 篇 | 复杂系统 |
| **总计** | **70 篇/日** | - |

**输出格式：**
```json
{
  "paper_id": "arxiv:2403.xxxxx",
  "title": "...",
  "abstract": "...",
  "authors": ["..."],
  "topic": ["time", "entropy"],
  "relevance_score": 0.85,
  "citation_count": 150
}
```

### 2. Research Engine（研究引擎）⭐

**功能：** 核心推理系统

**每轮研究流程：**
```
1. 问题发现
   ↓
2. 文献搜索（加载 20 篇论文摘要）
   ↓
3. 理论分析（加载 100 张知识卡片）
   ↓
4. 矛盾识别（跨领域对比）
   ↓
5. 研究结论（综合报告）
```

**1M Token 上下文利用：**
```
研究问题 (1k)
+ 20 篇论文摘要 (400k)
+ 100 张知识卡片 (400k)
+ 历史研究日志 (100k)
+ 相关理论框架 (100k)
= 约 1M tokens 完整上下文
```

**Prompt 模板：**
```markdown
研究问题：[问题描述]

加载背景：
- 相关论文：[20 篇摘要]
- 相关理论：[10 个理论框架]
- 相关知识卡片：[100 张]

任务：
1. 总结现有理论
2. 找出理论冲突（至少 3 个）
3. 提出解释框架
4. 生成研究结论

输出：
## 理论综述
## 关键矛盾
## 新假设
## 研究结论
```

## 研究优先级算法

### 评分指标

| 指标 | 权重 | 说明 |
|------|------|------|
| **科学重要性** | 40% | 对核心问题的贡献度 |
| **知识缺口** | 30% | 当前理解的空白程度 |
| **跨学科价值** | 20% | 连接不同领域的能力 |
| **新颖性** | 10% | 创新程度 |

### 优先级计算公式

```python
Priority Score = (
    importance * 0.4 +      # 科学重要性 (0-10)
    knowledge_gap * 0.3 +   # 知识缺口 (0-10)
    cross_domain * 0.2 +    # 跨学科价值 (0-10)
    novelty * 0.1           # 新颖性 (0-10)
)

# 优先级分类
if score >= 8.0:
    priority = "critical"    # 立即研究
elif score >= 6.0:
    priority = "high"        # 高优先级
elif score >= 4.0:
    priority = "medium"      # 中等优先级
else:
    priority = "low"         # 低优先级
```

### 示例评分

| 问题 | 重要性 | 缺口 | 跨学科 | 新颖 | 总分 | 优先级 |
|------|--------|------|--------|------|------|--------|
| 时间为什么有方向？ | 10 | 8 | 9 | 7 | 8.8 | critical |
| 宇宙初始为什么低熵？ | 10 | 9 | 8 | 8 | 9.0 | critical |
| 生命如何维持低熵？ | 9 | 8 | 9 | 8 | 8.5 | critical |
| AI 是否有意识？ | 8 | 7 | 8 | 9 | 7.9 | high |

## 研究循环（v2）

### 单轮流程

```
1. 论文发现 (Paper Discovery)
   - 抓取 70 篇最新论文
   - 相关性评分
   - 高价值论文标记

2. 研究任务生成 (Task Generation)
   - 从论文中提取新问题
   - 从知识图谱识别矛盾
   - 优先级评分

3. 深度分析 (Deep Analysis)
   - 加载 1M tokens 上下文
   - 多视角分析
   - 理论对比

4. 理论推演 (Theory Reasoning)
   - 假设生成
   - 思想实验
   - 验证方法

5. 知识固化 (Knowledge Solidification)
   - 知识卡片创建
   - 知识图谱更新
   - 研究日志记录

6. 新问题生成 (New Questions)
   - 衍生问题
   - 深度探索问题
   - 加入研究队列

循环继续 →
```
