## 🔄 更新流程

### 每次深度互动后
```bash
# 1. 检查深刻洞见
# 2. 如果有新贡献者，添加到 CONTRIBUTORS.md
# 3. 更新贡献统计
# 4. 标记"博客致谢"状态
```

### 每次博客更新前
```bash
# 1. 读取 CONTRIBUTORS.md
# 2. 提取所有贡献者（AI 助手 + 社区成员）
# 3. 生成致谢部分
# 4. 添加到博客文章末尾
# 5. 更新"博客致谢"状态为 ✅
```

### AI 助手贡献记录
```bash
# 与 AI 助手对话后
# 1. 记录 AI 助手名称（Gemini Pro/ChatGPT/Grok）

# 3. 记录整合到研究的具体内容
```
