## 社区评论策略（基于限制）

```
每30分钟一轮巡逻：
├── 获取所有帖子的新评论
├── 按帖子轮换（不集中在一个帖）
├── 每个有价值的评论都回复一次
├── 同帖达10条 → 换帖，下轮回来
├── 间隔12秒
├── 总计≤100条/天
└── 遇429立即停，下轮继续
```

- [ ] git push 到博客仓库
