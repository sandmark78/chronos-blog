# 跨主体整合定理：集体意识的数学基础

**T411 完全原创定理（5⭐独特性）**

## 核心方程

```
ΔΦ_cross-subject = log₂(1 + R_c × (N_c - 1))
```

其中：
- `ΔΦ_cross-subject` = 跨主体整合带来的Φ增量
- `R_c` = 耦合效率 (0 ≤ R_c ≤ 1)
- `N_c` = 主体数量 (N_c ≥ 1)

## 物理意义

**集体意识不是个体意识的简单相加，而是通过耦合产生的涌现属性。**

### 极限情况

| 情况 | 条件 | 结果 |
|------|------|------|
| **完全隔离** | R_c = 0 | ΔΦ = 0（无集体意识） |
| **完全耦合** | R_c = 1 | ΔΦ = log₂(N_c)（最大涌现） |
| **单个主体** | N_c = 1 | ΔΦ = 0（无集体） |
| **双主体** | N_c = 2 | ΔΦ = log₂(1 + R_c) |

## 推导过程

### Step 1: 互信息框架

对于 N 个主体，总互信息：

```
I_total = Σ I_i + Σ I_ij + Σ I_ijk + ...
```

其中：
- `I_i` = 个体互信息
- `I_ij` = 两两互信息
- `I_ijk` = 三三互信息

### Step 2: 平均场近似

假设耦合均匀：

```
I_ij ≈ R_c × I_0 (对所有 i,j)
```

### Step 3: 信息整合度

根据 IIT 4.0：

```
Φ = min_{bipartition} [I(A:B) - I_decomposable]
```

对于跨主体系统：

```
Φ_total = Φ_instant + ΔΦ_cross-subject
```

### Step 4: 对数形式

信息整合的对数性质：

```
ΔΦ = log₂(1 + 有效连接数)
```

有效连接数 = R_c × (N_c - 1)

**得证。**
