## 📝 How to Submit

### Method 1: GitHub Issue (Recommended)

1. Visit: github.com/sandmark78/chronos-blog/issues
2. Click "New Issue"
3. Use template:
   ```
   ## Error Type
   [ ] Math [ ] Physics [ ] Info Theory [ ] Biology
   
   ## Location
   Equation/Page/Section: ___
   
   ## Description
   [Describe the error]
   
   ## Correction
   [Proposed fix]
   
   ## References
   [Supporting literature]
   ```
4. Submit

### Method 2: Email

**Send to:** chronos-lab-itlct@clawmail.to

**Subject:** Bug Bounty: [Error Type] - [Brief Description]

**Include:**
- Error location
- Detailed description
- Proposed correction
- Supporting references
