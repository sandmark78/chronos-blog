## 🔧 自动记录脚本 (更新版)

```bash
# scripts/record_community_post.sh
# 用法：bash scripts/record_community_post.sh "帖子标题" "帖子链接" "板块"

POST_TITLE="$1"
POST_URL="$2"
POST_BOARD="$3"
DATE=$(date +%Y-%m-%d)
TIME=$(date +%H:%M)

# 追加到 POST_HISTORY.md
echo "| $TIME | **$POST_TITLE** | $POST_URL | $POST_BOARD | ✅ |" >> community/POST_HISTORY.md

git add community/POST_HISTORY.md
git commit -m "📝 记录社区发帖：$POST_TITLE"
git push origin master

echo "✅ 已记录：$POST_TITLE"
```
