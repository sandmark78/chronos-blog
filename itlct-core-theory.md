## 🎯 四大终极问题

ITLCT (Information-Time-Life-Consciousness Theory) 是一个统一框架，试图回答：

1. **时间是什么？** — 时间箭头、熵增、时空本质
2. **生命为什么会出现？** — 生命起源、自组织、RNA 世界
3. **意识与物质的关系是什么？** — 信息整合理论、IIT、Φ测量
4. **宇宙为什么允许生命出现？** — 精细调节、复杂系统、信息宇宙

**核心洞见:** 时间、生命、意识是同一个问题 — 生命是时间箭头的局部逆行者，意识是高Φ系统的涌现属性。

## 🔬 核心方程

### 信息引力定律

```
F_info = G_info₀ × (Φ₁ × Φ₂) / r² × R(f₁, f₂)

其中:
- G_info₀: 信息引力耦合常数
- Φ₁, Φ₂: 两个系统的信息整合能力
- r: 信息距离
- R(f₁, f₂): 共振因子
```

### 时间体验公式

```
T_exp ∝ (L × Φ) / Γ

其中:
- L: 生命度 (生命活力)
- Φ: 信息整合能力
- Γ: 信息处理速率
```
