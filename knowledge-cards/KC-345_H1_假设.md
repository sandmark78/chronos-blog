## H1 假设 (修正版)

**公式:**
```
Φ_GHPII = k(N) · Φ_IIT
```

**参数定义:**
- **k(N) = k₀ / Φ_max_IIT(N)** — 系统规模依赖标度律
- **k₀ = 0.1 ± 0.05** — 无量纲基准常数
- **C = 0** — 边界条件 (Φ→0 时两者都→0)
- **Φ_max_IIT(N)** — IIT 框架下 N 量子比特系统的最大Φ容量

**适用范围:**
- 中等整合区域：0.3 < Φ_GHPII < 0.7
- 系统规模：N ≥ 2 (单量子比特无 MIP 定义)
