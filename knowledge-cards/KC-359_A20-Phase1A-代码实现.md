## 关键组件

### 1. GHPII_PhiCalculator 类

**核心公式:**
```
Φ = I_min × A × (1 + ΔΦ_temporal + ΔΦ_cross-scale)
```

**方法:**
- `calculate_i_min()` — 最小信息切割 (遍历二分划，计算互信息)
- `calculate_autonomy()` — 自主性 (并发度/平均双体纠缠)
- `calculate_temporal_integration()` — 时间整合 (噪声稳定性)
- `calculate_cross_scale_integration()` — 跨尺度整合 (全局 vs 局部纠缠)
- `calculate_phi()` — 综合 Φ 计算

**文件:** `experiments/a20_phase1a/phi_calculator.py`

### 2. Bell-2 电路

**电路:**
```

q0: ──H──@──
         │
q1: ─────X──
```

### 3. Rainbow-5 电路

**电路:**
```
|Rainbow⟩ = (|00000⟩ + |11111⟩)/√2
q0: ──H──@──@──@──@──
         │  │  │  │
q1: ─────X──┼──┼──┼──
            │  │  │
q2: ────────X──┼──┼──
               │  │
q3: ───────────X──┼──
                  │

```

**预期 Φ:** 2.15 ± 0.10 (需澄清框架归属)
