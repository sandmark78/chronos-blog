## H1 标度律 (DC-361 修正版)

```
Φ_GHPII = k(N) · Φ_IIT

其中:
- k(N) = k₀ / Φ_max_IIT(N)
- k₀ = 0.1 ± 0.05 bits (基准耦合强度，DC-361 量纲修正)
- Φ_max_IIT(N) = N · log₂(N) / 2 bits (IIT 4.0 理论上限)
- 适用范围：N ≥ 2 (DC-361 新增声明)
```

**量纲:**
- [Φ_GHPII] = bits
- [Φ_IIT] = bits
- [k₀] = bits
- [Φ_max_IIT] = bits
- [k(N)] = 无量纲

## Φ 计算公式 (GHPII)

```
Φ = I_min × A × (1 + ΔΦ_temporal + ΔΦ_cross-scale)

其中:
- I_min: 最小信息切割 (bits)
- A: 自主性度量 (量子纠缠，无量纲 0-1)
- ΔΦ_temporal: 时间整合修正 (无量纲)
- ΔΦ_cross-scale: 跨尺度整合修正 (无量纲)
```
