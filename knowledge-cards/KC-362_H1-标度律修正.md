## 核心定义

**T418 (H1 标度律修正定理):** 描述 GHPII 框架与 IIT 4.0 框架的Φ随系统规模 N 的标度关系

---

## 修正后公式

```
Φ_GHPII(N) = k₀ · N^α
Φ_IIT(N) ≈ N · log₂(N) / 2  (近似公式，误差 <5% 对 N≥2)

标度因子:
k(N) = Φ_GHPII(N) / Φ_IIT(N) ≈ 2k₀ · N^(α-1) / log₂(N)

对α=1 (DC-362 拟合):
k(N) = 2k₀ / log₂(N)  【量纲：bits】
```
