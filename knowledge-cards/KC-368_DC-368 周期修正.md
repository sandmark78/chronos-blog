## 核心产出

### T418 最终版 (修正后)

**定理 T418:** GHPII 标度律修正定理

对于 N ≥ 1 的多比特量子系统:

Φ_GHPII(N) = k₀' · r(N) · N

其中:
- k₀' = 0.0245 ± 0.0005 bits (归一化常数)
- r(N) = Φ_max_IIT(N) / (N·log₂(N)/2)  for N ≥ 2
- r(1) = 1 (离散极限定义，非 L'Hôpital 法则)
- 适用范围：N ≥ 1 (但 H1 假设仅适用于 N ≥ 2)

边界条件:
- N=1: Φ_GHPII(1) = 0.0245 bits (经典 Shannon 熵极限)
- N=2: Φ_GHPII(2) = 0.049 bits (Bell-2 测量验证)
- N=7: Φ_GHPII(7) = 0.1715 bits (GHZ-7 测量验证)

**修正说明:**
- C-368-A01: r(1) 从 2 修正为 1 (离散极限，避免 L'Hôpital 应用于离散变量)
- C-368-A02: 明确 N=1 时 GHPII 退化为经典信息，非整合信息

### H1 修正版 (适用范围澄清)

**假设 H1:** GHPII-IIT 线性对应假设

对于 N ≥ 2 的多比特系统:

Φ_GHPII(N) = k(N) · Φ_IIT(N)

其中:
- k(N) = 2k₀ / log₂(N) (量纲：bits)
- Φ_IIT(N) ≈ N·log₂(N)/2 (近似公式，误差<5%)
- k₀ = 0.0245 ± 0.0005 bits

**适用范围:** N ≥ 2
- N=1: H1 不适用 (Φ_IIT(1) = 0，无整合信息)
- N≥2: H1 有效 (已验证 N=2→7)

---

### T419 降格版 (从定理→假设)

**假设 H19:** 跨尺度标度假设 (原 T419)

GHPII 标度律在不同尺度呈现分段行为:

Φ_GHPII(N) = k₀ · N^s(N)

其中 s(N) 分段定义为:
- s_Q = 1.00 ± 0.05  for N ≤ 10   (量子尺度)
- s_M = 1.2 ± 0.2    for 10 < N ≤ 100  (介观尺度)
- s_L = 1.5 ± 0.3    for N > 100  (宏观尺度)

**降格说明:**
- C-368-A03: α_Q/M/L 缺乏第一性原理推导，是经验拟合
- 从"定理 T419"降格为"假设 H19"
- 符号从α_Q/M/L 改为 s_Q/M/L (避免与 T417 的α冲突)

**验证状态:**
- 已验证：N=2→7 (A20 Phase 1B), s = 1.00 ± 0.03 ✓
- 待验证：N=8→20 (A20 Phase 2), 测试 s_Q→s_M 过渡
