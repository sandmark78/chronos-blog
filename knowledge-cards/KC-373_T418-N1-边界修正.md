## 核心修正

**修正前：**
```
Φ_GHPII(N) = k₀ · N
k₀ = 0.0243 ± 0.0006 bits
```
*未明确 N=1 边界行为*

**修正后：**
```
Φ_GHPII(N) = k₀ · N,  适用范围：N ≥ 2
Φ_GHPII(1) = 0  (由 GHPII 定义给出)
k₀ = 0.0243 ± 0.0006 bits
```

---

## 修正理由

1. **GHPII 定义要求：** 单系统无整合信息，Φ(1) = 0
2. **公式外推问题：** Φ = k₀·N 在 N=1 时给出 0.0243 bits，与定义冲突
3. **解决方案：** 明确公式适用范围 N≥2，N=1 由定义单独给出
