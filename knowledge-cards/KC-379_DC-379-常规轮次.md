## 关键发现

### N=1 边界间断问题 🟡

**问题:** 公式在 N=1 时的自然值≈0.024 bits，但声明Φ(1)=0

**计算:**
- N_c = 50: Φ(1) = 0.0245/(1+1/50) ≈ 0.024 bits
- N_c = 100: Φ(1) = 0.0245/(1+1/100) ≈ 0.0243 bits
- 声明值：Φ(1) = 0
- **间断点：~0.024 bits**

**DC-380 修正方案:**

**方案 A (推荐):** 平滑边界
```
Φ_GHPII(N) = k₀·(N-1)/(1+(N-1)/N_c)  for N≥1
```

**方案 B:** 分段定义
```
Φ_GHPII(N) = 0                    for N=1
Φ_GHPII(N) = k₀·N/(1+N/N_c)      for N≥2
```
