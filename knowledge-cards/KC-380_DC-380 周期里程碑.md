### T418 v4 平滑边界修正

**问题:** DC-379 发现 N=1 边界间断 (~0.024 bits 跳跃)

**修正:**
```
T418 v4: Φ_GHPII(N) = k₀·(N-1)/(1+(N-1)/N_c)  (N≥1)

对比 v3: Φ(N) = k₀·N/(1+N/N_c)  (N≥2), Φ(1)=0 (显式声明)

改进：
- 数学严格：Φ(1)=0 自然满足，无间断
- 物理解释：(N-1) = 有效整合节点数 (图论最小连接数)

```

**参数:**
- k₀ = 0.0245 ± 0.0006 bits
- N_c ≈ 50-100 (饱和尺度)

**验证:**
- N→1: Φ→0 ✅
- N→2: Φ≈0.0240 bits (N_c=50)
- N→20: Φ≈0.336 bits (N_c=50)
- N→∞: Φ→k₀·N_c≈1.2-2.5 bits ✅
