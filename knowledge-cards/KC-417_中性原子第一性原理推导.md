## 📐 核心公式

### 退相干率分解
```
γ_total = γ_sp + γ_bb + γ_laser + γ_coll + γ_motion

γ_sp        ≈ 5 s⁻¹     (自发辐射，∝ n⁻³)
γ_bb        ≈ 50 s⁻¹    (黑体辐射，主导项，∝ T²)
γ_laser     ≈ 5 s⁻¹     (激光相位噪声)
γ_coll      ≈ 0.5 s⁻¹   (碰撞，UHV 可忽略)
γ_motion    ≈ 0.5 s⁻¹   (热运动，光镊捕获可忽略)

γ_eff ≈ 61 s⁻¹ (n=70, T=300K)
```

### 门时间
```
τ_gate ≈ π/Ω_Rydberg ≈ 100-500 ns

Ω_Rydberg = d·E/ℏ
d ≈ n²ea₀ ≈ 4×10⁻²⁶ C·m (n=70)
E ≈ 10⁴-10⁵ V/m
```

### N_c 预测
```
N_c = ln(2)/(γ_eff × τ_gate)
N_c ≈ 0.693/(61 s⁻¹ × 200×10⁻⁹ s)
N_c ≈ 56,800

不确定性范围：46,000-60,000
```
