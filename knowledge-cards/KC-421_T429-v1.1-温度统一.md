## 核心定理

### T429 v1.1: 跨平台统一框架 (温度显式)

**完整形式:**
```
Φ_GHPII(N, T, P) = k₀(T) × (N-1) / [1 + (N-1)/N_c(T, P)]

其中:
- k₀(T) = k₀(0) × [1 - (T/T_crit)^δ]  (适用于 T < T_crit)
- N_c(T, P) = ln(2) / [γ(T, P) × τ_gate(P)]
- T_crit = ℏω₀/(2k_B)  (物理温度，量子能级热激发阈值)
- δ ≈ 1.5-2.0  (临界指数，唯象拟合 + 热容论证)
```

**四参数完全刻画:**
1. **k₀ (0):** 零温每量子比特意识容量 ≈ 0.0300 bits (T422 量子几何起源)
2. **T_crit:** 量子能级热激发阈值 = ℏω₀/(2k_B) (平台依赖)
3. **δ:** 临界指数 ≈ 1.5-2.0 (唯象拟合 + 半经验理论)
4. **N_c (T, P):** 饱和尺度 = ln(2)/[γ (T,P)×τ_gate (P)] (平台 + 温度依赖)

## 新增定义

### T_decoherence (退相干主导温度)

**定义:**
```
T_decoherence(P) 满足：γ(T_decoherence, P) × τ_gate(P) = 1

物理含义：在 T_decoherence 时，单门操作时间内的退相干概率~63% (1-1/e)
```

**与 T_crit 的关系:**
- **T_crit = ℏω₀/(2k_B):** 量子能级热激发阈值 (内禀属性)
- **T_decoherence:** 退相干主导阈值 (依赖噪声机制)
- **一般关系:** T_decoherence ≥ T_crit (退相干通常在高於能级激发温度才主导)

**例外:** 超导量子比特 T_decoherence < T_crit (准粒子噪声在低温即主导)

## 温度依赖统一 (解决 C-420-A02)

### 正确形式 (单次温度修正)
```
Φ = k₀(0) × [1 - (T/T_crit)^δ] × (N-1) / [1 + (N-1)/N_c(T, P)]
```

### 错误形式 (双重修正，避免!)
```
❌ Φ = k₀(0) × [1 - (T/T_crit)^δ]² × (N-1) / [1 + (N-1)/N_c(T, P)]
```

**物理直觉:**
- **k₀ (T):** 温度降低"每量子比特的意识容量" (量子几何数减少)
- **N_c (T):** 温度降低"饱和尺度" (退相干增强，更早饱和)
- **两者独立:** 不同物理机制，但温度修正因子只应用一次
