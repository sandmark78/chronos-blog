## 核心陈述

**T454**: 在 N 量子比特系统状态空间 Ω={0,1}^N 上，如果底层度量 ρ 满足 4 条公理:
1. **Ax-ρ1** (子系统可分性): ρ(x,y) = F({fᵢ(xᵢ,yᵢ)})
2. **Ax-ρ2** (可加性): ρ(x,y) = Σᵢ fᵢ(xᵢ,yᵢ)
3. **Ax-ρ3** (子系统对称性): fᵢ = f, ∀i
4. **Ax-ρ4** (最小分辨单位): f(0,1)=1, f(0,0)=0

则 ρ **唯一确定为 Hamming 距离**: ρ(x,y) = Σᵢ|xᵢ-yᵢ|

## 关键推论

- **C-454-01**: T453 (EMD 唯一) + T454 (Hamming 唯一) → IIT Φ 计算的距离度量**完全确定** (8 条公理, 零自由度)
- **C-454-02**: k₀ = 0.0300 bits 原则上由 IIT 公理 + 量子力学完全确定, 不含自由参数
- k₀ 的闭式表达不可行 (Φ 计算 NP-hard), 只能数值获得

## k₀ 状态提升

| 之前 | 之后 |
|------|------|
| 唯象参数 (类比 Λ_QCD) | 公理确定但计算困难的常数 (类比 p_c) |
| 路径β (实验测量) 唯一存活 | 路径δ 给出理论确定性, 路径β 仍为数值验证 |

## 独特预测

- **D-454-01**: 加权 Hamming 度量 (违反 Ax-ρ3) → k₀ 不再普适, 依赖权重比
- **D-453-01 v1.1**: GHZ 态 W₁≡W₂ 标度 (∝N); 差异出现在非 GHZ 态

## 限制

- T454 对 IIT 3.0 严格成立
- 对 IIT 4.0 (因果因子空间), 需独立证明因果空间上的类似唯一性 (开放问题)
- k₀ 的数值仍只能通过 A20 数值模拟获得
