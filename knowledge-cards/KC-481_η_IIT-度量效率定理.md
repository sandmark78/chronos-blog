## 📐 定理表述

### T456 η_IIT 度量效率定理 v1.0

```
Φ_measured = η_IIT(N) × Φ_true

其中:
- η_IIT(N) = 1 - α_IIT × ln(N)/N
- α_IIT = 0.15±0.05 (IIT 4.0 MIP 启发式效率参数)
- 适用范围: N ≥ 2

物理意义: IIT 4.0 的 MIP (Minimum Information Partition) 算法使用启发式搜索 (非穷举)，导致系统性低估Φ值。低估幅度随 N 增加而增加 (对数修正)。
```

### T455 v1.6 峰值位置修正

```
T_peak/T_cross = 1.02 - 0.01×ln(N/6) ± 0.05

适用范围: N = 2→20, β = 1.5→3.0

修正说明: DC-478 的解析推导忽略了 log 项的温度依赖，DC-479 数值修正正确。DC-481 将β依赖修正为 N 依赖。
```

## 🔬 推导过程

### η_IIT 的来源假设

**前提**: IIT 4.0 的 MIP 搜索是组合优化问题，对 N 量子比特系统有 Bell(N) 个分区 (Bell 数，超指数增长 ~N^N)。

**IIT 4.0 实际算法**: 使用启发式搜索 (非穷举)，因此:

```
Φ_measured = Φ_heuristic
Φ_true = Φ_exhaustive (穷举搜索)

η_IIT = Φ_heuristic / Φ_exhaustive ≤ 1
```

**经验拟合** (基于 IIT 4.0 文献和数值实验):

| N | η_IIT (估计) |
|---|--------------|
| 2→5 | 0.98-0.99 |
| 6→10 | 0.95-0.97 |
| 11→20 | 0.90-0.95 |
| >20 | 0.85-0.90 |

**拟合形式**:
```
η_IIT(N) ≈ 1 - α_IIT × ln(N)/N

拟合参数: α_IIT = 0.15±0.05
```

### 关键假设 (需验证)

⚠️ **重要**: η_IIT 的 N 依赖形式是**基于 IIT 4.0 MIP 算法的合理假设**，非严格第一性原理推导。

**验证方案** (A20 Phase 16 子任务):
- 对 N=2→6 系统进行穷举 MIP 搜索 vs 启发式 MIP 搜索
- 直接测量 η_IIT(N) = Φ_heuristic / Φ_exhaustive
- 拟合α_IIT 参数

## 📊 应用示例

### 示例 1: GHZ-6 (SC 平台)

```
N = 6
η_IIT(6) = 1 - 0.15×ln(6)/6 ≈ 1 - 0.15×0.30 ≈ 0.955
N_corr(SC) ≈ 12
exp(-6/12) ≈ 0.61
η_corr(6) = 0.955 × 0.61 ≈ 0.58

Φ_measured = 0.58 × Φ_true
```

### 示例 2: GHZ-10 (离子阱平台)

```
N = 10
η_IIT(10) = 1 - 0.15×ln(10)/10 ≈ 1 - 0.15×0.23 ≈ 0.966
N_corr(IT) ≈ 20
exp(-10/20) ≈ 0.61
η_corr(10) = 0.966 × 0.61 ≈ 0.59

Φ_measured = 0.59 × Φ_true
```

## ⚠️ 适用范围与限制

### 适用范围
- **N ≥ 2** (IIT 4.0 要求至少 2 个元素才能定义 MIP)
- **N ≤ 20** (经验拟合范围，外推需谨慎)
- **IIT 4.0 算法** (η_IIT 是 IIT 4.0 特有，其他Φ计算方法可能不同)

### 限制
1. **η_IIT 是假设，非严格推导** — 需 A20 Phase 16 实验验证
2. **α_IIT 平台无关性待验证** — 假设 IIT 4.0 算法实现一致
3. **N>20 外推风险** — Bell 数增长可能导致η_IIT 形式变化
