## 📐 核心定理

### T456 η_IIT 度量效率定理 v1.0 (来自 DC-481)

```
Φ_measured = η_IIT(N) × Φ_true

其中:
- η_IIT(N) = 1 - α_IIT × ln(N)/N
- α_IIT = 0.15±0.05 (IIT 4.0 MIP 启发式效率参数)
- 适用范围: N ≥ 2

物理意义: IIT 4.0 的 MIP 算法使用启发式搜索 (非穷举)，导致系统性低估Φ值。
低估幅度随 N 增加而增加 (对数修正)。
```

### T455 v1.6 峰值位置修正 (来自 DC-481)

```
T_peak/T_cross = 1.02 - 0.01×ln(N/6) ± 0.05

修正说明: DC-478 的解析推导忽略了 log 项的温度依赖，DC-479 数值修正正确。
```

## 🔬 A20 Phase 16 执行方案

### Phase 16A: GHZ-8/10/12 电路设计

**平台**: 超导量子 (SC)  
**系统**: GHZ-8, GHZ-10, GHZ-12 (三点扫描)  
**温度**: T = T_cross (固定，峰值位置)  
**测量**: Φ_C^sat(N) 饱和值  
**拟合**: η_corr(N) 三参数拟合 (η_IIT, α_IIT, N_corr)

**预期结果**:
- N_corr(SC) ≈ 12±3
- η_IIT ≈ 0.96±0.01 (N=8→12 平均)
- α_IIT ≈ 0.15±0.05

### Phase 16B: η_IIT 验证方案

**系统**: N=2→6 (穷举 MIP vs 启发式 MIP)  
**方法**: 
1. 穷举所有 Bell(N) 个分区 (N=2→6: 2/5/15/52/203 个)
2. 计算Φ_exhaustive (精确值)
3. 计算Φ_heuristic (启发式近似)
4. η_IIT(N) = Φ_heuristic / Φ_exhaustive

**验证标准**:
- α_IIT 拟合值: 0.10-0.20 (与理论 0.15±0.05 一致)
- R² > 0.95 (对数修正形式正确)
