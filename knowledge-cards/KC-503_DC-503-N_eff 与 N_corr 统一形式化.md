## 统一公式

### N_eff (有效自由度)

```
N_eff(N, T) = (N-1) / [1 + (N-1)/N_c(T)]
```

**物理意义:** 实际有效的关联比特数，考虑退相干导致的饱和行为

**极限行为:**
- N ≪ N_c: N_eff ≈ N-1 ≈ N (线性区，所有比特有效)
- N ~ N_c: N_eff ≈ N_c/2 (过渡区)
- N ≫ N_c: N_eff ≈ N_c (饱和区，仅 N_c 个比特有效)

### Φ (普适形式)

```
Φ(N, T) = k₀(T) × N_eff(N, T)
```

**量纲:** [Φ] = bits, [k₀] = bits, [N_eff] = 1 (无量纲) ✅

### N_corr (理论上限)

```
N_corr(N, T) = N²/ln²(2) × η(T)
```

**物理意义:** N→∞极限下 N_eff 的上限标度 (完全相干极限)

**来源:**
- N²: 标准 QFI 海森堡极限 (Giovannetti et al. 2004)
- 1/ln²(2): ITLCT 独有因子 (待严格第一性原理推导)
  - 第一个 1/ln(2): 比特 - 自然单位转换 (log₂e)
  - 第二个 1/ln(2): 双向因果结构假设 (Fisher 信息双线性形式)
- η(T) = exp[-(T/T_crit)^δ]: 温度效率因子

### N_eff 与 N_corr 关系

```
N_eff ≤ N_corr/N × η_coherence  (唯象假设，待严格证明)

其中 η_coherence ≤ 1 是相干效率因子。
```

**诚实声明:** ⚠️ 该关系的严格第一性原理推导尚未完成 (DC-505+ 目标)

## 温度依赖形式

### k₀(T) (T423 v1.3)

```
k₀(T) = k₀(0) × exp[-(T/T_crit)^δ]
```

**参数:**
- k₀(0) = 0.0300±0.0005 bits
- T_crit = ℏω₀/(2k_B) (能级激发阈值)
- δ ≈ 2 (临界指数，唯象拟合 1.8-2.2)

### N_c(T) (DC-503 统一)

```
N_c(T) = N_c(0) × exp[-(T/T_crit)^δ]
```

**物理来源:** N_c = ln(2)/(γ×τ_gate)，其中γ随 T 增加
