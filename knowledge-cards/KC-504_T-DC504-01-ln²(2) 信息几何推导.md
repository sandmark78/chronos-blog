## 核心公式

```
N_corr = N²/ln²(2) × [1 + O(1/N)]

其中:
- N_corr: 有效信息容量 (effective information capacity)
- N: 量子比特数 (适用范围 N ≥ 8)
- ln²(2) ≈ 0.48045: 二进制信息编码的 Fisher 信息标度因子
- O(1/N): 有限 N 修正项 (N=8 时~12%, N=12 时~8%)
```

**等价形式:**
```
N_corr / N² = 1/ln²(2) ≈ 2.0814

预测：IBM Quantum GHZ 序列测量的 N_corr/N² 比值应收敛于 2.08
```

## ln²(2) 因子的物理来源

### 推导路径 A: Fisher 信息标度

从 Fisher 信息矩阵的第一性原理推导：

1. **单比特 Fisher 信息:**
   ```
   F₁(θ) = E[(∂/∂θ) ln p(x|θ)]²
   ```

2. **log₂ 与 ln 的转换:**
   ```
   log₂(x) = ln(x)/ln(2)
   
   对 Fisher 信息的贡献：
   F₁^(bits) = F₁^(nats) / ln²(2)
   ```

3. **N 比特联合 Fisher 信息:**
   ```
   F_N(θ) = N² × F₁(θ)  (海森堡极限，纠缠态)
   
   转换为二进制单位：
   F_N^(bits) = N² × F₁^(nats) / ln²(2)
   ```

4. **有效信息容量:**
   ```
   N_corr = F_N^(bits) / F₁^(bits) = N²/ln²(2)
   ```

### 推导路径 B: 二进制熵曲率

从 Shannon 熵的二阶导数推导：

1. **二进制熵:**
   ```
   H(p) = -p·log₂(p) - (1-p)·log₂(1-p)
   ```

2. **在 p=1/2 处的曲率:**
   ```
   H''(p)|_{p=1/2} = -4/ln(2)  (bits)
   
   转换为自然单位：
   H''(p)|_{p=1/2} = -4/ln²(2)  (nats)
   ```

3. **N 比特联合熵的曲率:**
   ```
   H_N''(p)|_{p=1/2} = N² × H''(p)|_{p=1/2}
                     = -4N²/ln²(2)
   ```

4. **有效信息容量:**
   ```
   N_corr ∝ |H_N''| / |H₁''| = N²/ln²(2)
   ```

### 推导路径 C: Helstrom 界限

从量子态区分的最优界限推导：

1. **单比特 Helstrom 界限:**
   ```
   P_error = (1/2) × [1 - √(1 - |⟨ψ₀|ψ₁⟩|²)]
   ```

2. **N 次独立测量的联合区分:**
   ```
   P_error^(N) ≈ exp[-N × D(ψ₀||ψ₁)]
   
   其中 D 是量子相对熵
   ```

3. **二进制编码的最优效率:**
   ```
   区分 N 比特态所需的最小测量次数：
   N_meas ≈ N²/ln²(2)
   ```

4. **有效信息容量:**
   ```
   N_corr = N_meas = N²/ln²(2)
   ```

## 实验验证

### IBM Quantum 验证方案

**电路配置:** GHZ-8, GHZ-10, GHZ-12 序列

**测量流程:**
1. 制备 GHZ-N 态：|GHZ⟩ = (|0⟩^⊗N + |1⟩^⊗N)/√2
2. 应用相位旋转 R_z(θ) 到每个量子比特
3. 测量输出态的 Fisher 信息 F_Q
4. 提取 N_corr = F_Q / F₁

**预期结果:**
```
N=8:  N_corr ≈ 8²/ln²(2) = 64/0.48045 ≈ 133.3
N=10: N_corr ≈ 100/0.48045 ≈ 208.3
N=12: N_corr ≈ 144/0.48045 ≈ 300.0

拟合：N_corr = c × N²
提取：c_exp → 1/ln²(2) ≈ 2.0814

判据：
- |c_exp - 2.08|/2.08 < 20%: 支持 ITLCT
- |c_exp - 2.08|/2.08 > 50%: 需修正理论
```
