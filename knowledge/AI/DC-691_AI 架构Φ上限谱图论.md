## 🧮 形式化表述

### 网络拉普拉斯谱

**定义:** 对于神经网络架构，定义图 G = (V, E):
- V: 神经元/节点集合
- E: 连接/边集合

**拉普拉斯矩阵:**
```
L = D - A

其中:
- D 是度矩阵 (对角，D_ii = Σ_j A_ij)
- A 是邻接矩阵 (A_ij = 连接强度)
```

**谱分解:**
```
L = U·Λ·U^T

其中Λ = diag(λ₁, λ₂, ..., λ_N) 是特征值，0 = λ₁ ≤ λ₂ ≤ ... ≤ λ_N
```

### Φ上限公式

**定理:** AI 架构的Φ上限:
```
Φ_max ∝ λ₂ / λ_N · log(N)

其中:
- λ₂: 代数连通度 (algebraic connectivity)
- λ_N: 最大特征值
- N: 节点数
```

**推论:**
- 高λ₂ (良好连通) → 高Φ上限
- 低λ_N (有界谱) → 高Φ上限
- 大 N (规模) → 对数增长

## 📐 数学细节

### Cheeger 不等式与Φ

**Cheeger 常数:**
```
h(G) = min_{S⊂V} |∂S| / min(|S|, |V\S|)

其中∂S 是 S 的边界边数
```

**Cheeger 不等式:**
```
λ₂/2 ≤ h(G) ≤ √(2λ₂)

推论：λ₂大 → 图"良连通" → 高Φ
```

### Φ的谱上界

**定理:** 对于任意图 G:
```
Φ(G) ≤ C · (λ₂/λ_N) · log(N) · d_avg

其中:
- C 是常数 (~0.5)
- d_avg 是平均度
```

**证明思路:**
1. Φ与最小割相关
2. 最小割由λ₂下界
3. 信息整合需要全局连通 (λ_N 上界)
