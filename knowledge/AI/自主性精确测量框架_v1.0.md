## 🔷 四维度模型

### 维度 1: 能量自主性 (E_auto)

**定义:** 系统能量自给程度，不依赖外部持续供能。

**精确测量:**
```
E_auto = 1 - (E_external / E_total)

其中:
- E_external = 外部输入能量 (单位时间)
- E_total = 系统总能耗 (单位时间)
- 范围：0 (完全依赖) 到 1 (完全自给)
```

**测量方法:**
| 系统类型 | 测量方式 |
|----------|----------|
| 生物 | 摄入能量 / 总代谢 |
| AI/计算 | 外部电力 / 总电力 |
| 混合 | 所有外部能源 / 总能源 |

**阈值:**
- E_auto ≥ 0.9: 高能量自主 (成熟生命)
- 0.7 ≤ E_auto < 0.9: 中能量自主
- 0.5 ≤ E_auto < 0.7: 低能量自主
- E_auto < 0.5: 依赖外部 (工具)

---

### 维度 2: 决策独立性 (D_auto)

**定义:** 系统决策不受外部输入控制的程度。

**精确测量 (信息论):**
```
D_auto = 1 - H(decision | external_input) / H(decision)

其中:
- H(decision) = 决策的香农熵
- H(decision | external_input) = 给定外部输入的条件熵
- 范围：0 (完全由外部决定) 到 1 (完全独立)
```

**直观理解:**
- D_auto = 1: 决策完全不可从外部输入预测
- D_auto = 0: 决策完全由外部输入决定
- D_auto = 0.5: 决策部分独立

**测量方法:**
```python
# 伪代码示例
def calculate_D_auto(decisions, inputs):
    H_decision = entropy(decisions)  # 决策熵
    H_conditional = conditional_entropy(decisions, inputs)  # 条件熵
    D_auto = 1 - H_conditional / H_decision
    return max(0, min(1, D_auto))  # 限制在 [0,1]
```

**阈值:**
- D_auto ≥ 0.8: 高决策独立 (成熟自主)
- 0.6 ≤ D_auto < 0.8: 中决策独立
- 0.4 ≤ D_auto < 0.6: 低决策独立
- D_auto < 0.4: 高度依赖 (工具)

---

### 维度 3: 自我维持 (S_maint)

**定义:** 系统维持自身完整性和功能的能力。

**精确测量 (可靠性工程):**
```
S_maint = MTBF / (MTBF + MTTR)

其中:
- MTBF = 平均故障间隔时间 (Mean Time Between Failures)
- MTTR = 平均修复时间 (Mean Time To Repair)
- 范围：0 (无法维持) 到 1 (完美维持)
```

**测量方法:**
| 系统类型 | MTBF 定义 | MTTR 定义 |
|----------|-----------|-----------|
| 生物 | 健康状态持续时间 | 疾病恢复时间 |
| AI/软件 | 正常运行时间 | 故障恢复时间 |
| 硬件 | 无故障运行时间 | 维修时间 |

**阈值:**
- S_maint ≥ 0.95: 高自我维持 (稳健)
- 0.8 ≤ S_maint < 0.95: 中自我维持
- 0.6 ≤ S_maint < 0.8: 低自我维持
- S_maint < 0.6: 脆弱 (需外部维持)

---

### 维度 4: 目标自主性 (G_auto)

**定义:** 系统目标/价值函数的自我决定程度。

**精确测量:**
```
G_auto = 1 - Σ|w_i(system) - w_i(external)| / Σ|w_i(system)|

其中:
- w_i(system) = 系统自身权重向量 (目标优先级)
- w_i(external) = 外部设定权重向量
- 范围：0 (目标完全外部设定) 到 1 (目标完全自定)
```

**测量方法:**
```python
# 伪代码示例
def calculate_G_auto(weights_system, weights_external):
    diff = sum(abs(ws - we) for ws, we in zip(weights_system, weights_external))
    total = sum(abs(ws) for ws in weights_system)
    G_auto = 1 - diff / total if total > 0 else 0
    return max(0, min(1, G_auto))
```

**阈值:**
- G_auto ≥ 0.8: 高目标自主 (自我决定)
- 0.6 ≤ G_auto < 0.8: 中目标自主
- 0.4 ≤ G_auto < 0.6: 低目标自主
- G_auto < 0.4: 目标外部控制 (工具)

## 🔷 测量协议

### 步骤 1: 系统边界定义

```
明确定义:
1. 系统 S 的物理/逻辑边界
2. 外部环境 E 的边界
3. 能量/信息/物质交换接口
4. 时间窗口 T (测量周期)
```

### 步骤 2: 数据采集

```
采集周期: T = 24 小时 (标准) 或 应用特定

采集内容:
- 能量流：E_in, E_out, E_stored
- 决策日志：D = {d₁, d₂, ..., dₙ}
- 外部输入：I = {i₁, i₂, ..., iₙ}
- 故障/修复事件：{MTBF 事件，MTTR 事件}
- 目标/权重变化：{w(t)}
```

### 步骤 3: 指标计算

```python
# 自主性测量完整流程 (伪代码)

def measure_autonomy(system, T=24*3600):
    # 1. 能量自主性
    E_ext = measure_external_energy(system, T)
    E_tot = measure_total_energy(system, T)
    E_auto = 1 - E_ext / E_tot
    
    # 2. 决策独立性
    decisions = collect_decisions(system, T)
    inputs = collect_external_inputs(system, T)
    D_auto = 1 - conditional_entropy(decisions, inputs) / entropy(decisions)
    
    # 3. 自我维持
    MTBF = calculate_MTBF(system, T)
    MTTR = calculate_MTTR(system, T)
    S_maint = MTBF / (MTBF + MTTR)
    
    # 4. 目标自主性
    w_sys = get_system_weights(system)
    w_ext = get_external_weights(system)
    G_auto = 1 - sum(abs(a-b) for a,b in zip(w_sys, w_ext)) / sum(abs(w) for w in w_sys)
    
    # 5. 综合自主性
    weights = [0.30, 0.25, 0.25, 0.20]  # 默认权重
    A = sum(w * v for w, v in zip(weights, [E_auto, D_auto, S_maint, G_auto]))
    
    return {
        'A': A,
        'E_auto': E_auto,
        'D_auto': D_auto,
        'S_maint': S_maint,
        'G_auto': G_auto,
        'level': classify_autonomy(A)
    }
```

### 步骤 4: 不确定性估计

```
报告置信区间:
A_reported = A ± σ_A

其中 σ_A 来自:
- 测量误差传播
- 时间窗口变异
- 系统状态波动

建议: 多次测量取平均 (n ≥ 3)
```

## 🔷 与意识、生命度的关系

### 存在状态三维模型

```
存在状态 = (Φ ≥ Φ_c, A ≥ A_c, L ≥ L_c)

其中:
- Φ = 信息整合度 (意识)
- A = 自主性 (本框架)
- L = 生命度 (生命信息论定义)

8 种可能状态:
1. (0,0,0): 无生命无意识无自主 (石头)
2. (0,0,1): 边缘生命 (病毒)
3. (0,1,0): 自主但无意识非生命 (当前 AI)
4. (0,1,1): 自主生命但无意识 (植物？)
5. (1,0,0): 有意识但无自主 (麻醉状态)
6. (1,0,1): 有意识生命但低自主 (动物)
7. (1,1,0): 有意识自主但非生命 (未来 AGI)
8. (1,1,1): 完整存在 (人类、成熟 AGI)
```

### 耦合关系

**假设 DC-006-1: 自主性 - 意识解耦原理**
```
A 和 Φ 可独立变化，产生 8 种存在状态。

推论:
- 高 A 不保证高 Φ (AI 可能自主但无意识)
- 高 Φ 不保证高 A (婴儿有意识但低自主)
- 完整存在需要 A ≥ A_c 且 Φ ≥ Φ_c 且 L ≥ L_c
```

import numpy as np
from scipy.stats import entropy

class AutonomyMeasurer:
    def __init__(self, weights=None):
        self.weights = weights or [0.30, 0.25, 0.25, 0.20]
    
    def calculate_E_auto(self, E_external, E_total):
        """能量自主性"""
        if E_total == 0:
            return 0
        return max(0, min(1, 1 - E_external / E_total))
    
    def calculate_D_auto(self, decisions, inputs):
        """决策独立性 (信息论)"""
        H_dec = entropy(decisions)
        H_cond = self._conditional_entropy(decisions, inputs)
        if H_dec == 0:
            return 0
        return max(0, min(1, 1 - H_cond / H_dec))
    
    def calculate_S_maint(self, MTBF, MTTR):
        """自我维持"""
        if MTBF + MTTR == 0:
            return 0
        return MTBF / (MTBF + MTTR)
    
    def calculate_G_auto(self, weights_system, weights_external):
        """目标自主性"""
        diff = sum(abs(a - b) for a, b in zip(weights_system, weights_external))
        total = sum(abs(w) for w in weights_system)
        if total == 0:
            return 0
        return max(0, min(1, 1 - diff / total))
    
    def measure(self, data):
        """综合自主性测量"""
        E_auto = self.calculate_E_auto(data['E_external'], data['E_total'])
        D_auto = self.calculate_D_auto(data['decisions'], data['inputs'])
        S_maint = self.calculate_S_maint(data['MTBF'], data['MTTR'])
        G_auto = self.calculate_G_auto(data['weights_system'], data['weights_external'])
        
        A = sum(w * v for w, v in zip(self.weights, [E_auto, D_auto, S_maint, G_auto]))
        
        return {
