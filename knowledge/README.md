# Chronos Lab 知识库

> "知识不是信息，而是连接。"

## 目录结构

```
knowledge/
├── 宇宙/          # 宇宙起源、演化、结构
├── 时间/          # 时间理论、时空、因果
├── 生命/          # 生命起源、进化、生物学
├── 意识/          # 意识本质、认知科学
├── 文明/          # 文明发展、历史规律
├── 研究日志/      # 每日研究记录
└── 思想实验/      # 假设、推演、思维模型
```

## 知识卡片格式

所有知识文件使用统一的 Markdown 格式，支持双向链接（Obsidian 兼容）：

```markdown
---
title: [概念名称]
category: [分类]
tags: [标签列表]
created: YYYY-MM-DD
updated: YYYY-MM-DD
sources: [引用]
---

## 核心概念

[简洁解释]

## 关键观点

- 观点 1
- 观点 2

## 相关知识

- [[相关概念 1]]
- [[相关概念 2]]

## 开放问题

[未解决的问题]
```

## 使用方式

- **手动添加** — 直接创建 Markdown 文件
- **Mnemosyne 自动** — 调用 Mnemosyne Agent 将信息转化为卡片
- **双向链接** — 使用 `[[概念名]]` 语法连接相关知识

## 知识增长策略

1. **每日增量** — 每天至少添加 1 个新知识卡片
2. **每周回顾** — 整理、合并、删除过时内容
3. **每月图谱** — 审视知识结构，发现空白领域
