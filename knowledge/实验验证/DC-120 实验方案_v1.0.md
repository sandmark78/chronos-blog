## 📋 实验概述

**理论预测:**
```
dS/dt ∝ L(S) × Φ

其中:
- dS/dt = 熵产生率 (代谢率代理)
- L(S) = 生命度
- Φ = 信息整合度 (意识)
```

**核心假设 (DC-120):**
> 意识系统的熵产生率与生命度和意识强度的乘积成正比。

---

## 🎯 实验目标

### 主要目标
1. 验证 dS/dt 与 L×Φ的正相关关系
2. 估计耦合系数 σ₃ (dS/dt = σ₀ + σ₁·L + σ₂·Φ + σ₃·L×Φ)
3. 确定预测效力 (R²)

### 次要目标
1. 跨物种类群验证 (啮齿类、灵长类、人类)
2. 意识状态操纵验证 (清醒、睡眠、麻醉、冥想)
3. 建立Φ-代谢率基线数据库

## 🔬 实验设计

### 研究 1: 跨物种代谢率 - Φ关联

**设计:** 横断面比较研究

**被试:**
| 物种 | N | 估计 L | 估计Φ范围 |
|------|---|--------|-----------|
| 小鼠 | 20 | 0.75-0.80 | 0.2-0.3 |
| 猕猴 | 10 | 0.85-0.90 | 0.5-0.6 |
| 人类 | 30 | 0.90-0.95 | 0.6-0.8 |

**测量:**
1. **代谢率 (熵产生代理)**
   - 间接热量测定 (VO₂, VCO₂)
   - 静止代谢率 (RMR)
   - 单位：kcal/kg/day

2. **生命度 L(S)**
   - 信息密度：脑质量/体重比
   - 信息处理速率：神经传导速度估计
   - 自主性：行为复杂度评分
   - 能量利用效率：代谢效率

3. **意识Φ**
   - EEG 记录 (32-64 通道)
   - Φ近似计算 (DC-138)
   - 替代指标：Lempel-Ziv 复杂度、PCI 指数

**分析:**
```
回归模型:
代谢率 = β₀ + β₁·L + β₂·Φ + β₃·L×Φ + ε

预测:
- β₃ > 0 (耦合项显著)
- R² > 0.6 (中等效应)
- 跨物种一致性
```

**时间:** 6-9 个月

---

### 研究 2: 意识状态操纵 (人类)

**设计:** 被试内设计，4 条件

**被试:** N = 30 健康成人

**条件:**
1. **清醒基线** (正常警觉)
2. **睡眠** (NREM 阶段 2-3)
3. **麻醉** (丙泊酚，亚麻醉剂量)
4. **冥想** (有经验冥想者，专注状态)

**测量:**
1. **代谢率**
   - fMRI BOLD (全脑代谢代理)
   - PET (绝对代谢率，子样本 N=10)

2. **Φ值**
   - EEG (64 通道)
   - TMS-EEG (PCI 指数)
   - 近似Φ计算

3. **意识报告**
   - 主观意识评分 (0-10)
   - 内容报告 (如有)

**预测:**
| 条件 | 预期Φ | 预期代谢率 |
|------|-------|------------|
| 清醒 | 0.6-0.7 | 基线 100% |
| 睡眠 | 0.3-0.4 | 80-85% |
| 麻醉 | 0.1-0.2 | 70-75% |
| 冥想 | 0.7-0.8 | 95-100% |

**关键预测:**
- Φ × 代谢率 相关性 > 0.7
- 冥想状态：Φ高但代谢率不升高 (效率提高)

**时间:** 9-12 个月

### 研究 3: 冥想训练干预 (纵向)

**设计:** RCT，8 周干预

**被试:** N = 60 冥想新手
- 干预组：n = 30 (8 周 MBSR)
- 对照组：n = 30 (等待列表)

**测量时间点:**
- 基线 (T0)
- 4 周 (T1)
- 8 周 (T2)
- 随访 3 月 (T3)

**测量:**
1. **代谢率**: 间接热量测定
2. **Φ**: EEG 近似Φ
3. **F 值**: 自由意志量表 (新开发)
4. **脑结构**: MRI (皮层厚度，子样本)

**预测:**
- 干预组Φ提高 10-20%
- 代谢率效率提高 (相同Φ，更低代谢)
- F 值提高与Φ提高相关

**时间:** 12-15 个月

## 📊 测量协议

### 代谢率测量

**方案 A: 间接热量测定 (推荐)**
```
设备：代谢车 (Cosmed Quark, Jaeger Oxycon)
程序:
1. 空腹 12 小时
2. 静卧 30 分钟适应
3. 测量 30 分钟 (面罩)
4. 计算 RMR (Weir 方程)

指标:
- VO₂ (ml/kg/min)
- VCO₂ (ml/kg/min)
- RMR (kcal/day)
- 呼吸商 (RQ)
```

**方案 B: fMRI BOLD (相对代谢)**
```
设备：3T MRI
序列：静息态 EPI (10 分钟)
分析:
- 全脑平均 BOLD 信号
- 葡萄糖代谢估计 (校准 PET 子样本)
```

### Φ值测量

**方案 A: EEG 近似Φ (推荐)**
```
设备：64 通道 EEG (Neuroscan, Brain Products)
记录：
- 静息态 10 分钟 (睁眼 + 闭眼)
- 采样率：1000 Hz
- 参考：乳突

近似Φ计算 (DC-138):
Φ_approx = α·I_local + β·C_global + γ·R_feedback

I_local: 局部同步 (α/γ波段)
C_global: 半球间相干
R_feedback: 前 - 后反馈连接

软件：自定义 MATLAB/Python 脚本
```

**方案 B: TMS-EEG PCI 指数**
```
设备：TMS + EEG
程序:
- 单脉冲 TMS (运动皮层)
- 记录 TEPs (TMS 诱发电位)
- 计算 PCI (Perturbational Complexity Index)

优势：与意识状态强相关
局限：设备昂贵，需专门训练
```

### 生命度 L(S) 评估

**跨物种方案:**
```
L(S) = 0.3·log₂(I/M) + 0.2·log₂(R) + 0.3·A + 0.2·E

I/M (信息密度):
- 脑质量/体重比
- 神经元密度估计

R (信息处理速率):
- 神经传导速度
- 反应时任务

A (自主性):
- 行为灵活性评分
- 问题解决任务

E (能量效率):
- 代谢效率 (脑代谢/认知表现)
```

## 📈 统计分析计划

### 主要分析

**多层次回归:**
```R
# 模型 1: 主效应
model1 <- lmer(代谢率 ~ L + Φ + (1|物种), data)

# 模型 2: 耦合效应
model2 <- lmer(代谢率 ~ L * Φ + (1|物种), data)

# 模型比较
anova(model1, model2)

# 预测: model2 显著优于 model1 (p < 0.01)
```

**效应量估计:**
```R
# 耦合项效应量
β₃ = 回归系数 (L×Φ)
R²_change = model2 R² - model1 R²

# 预测: R²_change > 0.15 (中等效应)
```

### 次要分析

**跨物种比较:**
```R
# 物种调节效应
model3 <- lmer(代谢率 ~ L * Φ * 物种 + (1|被试), data)

# 预测：耦合效应在高 L 物种更强
```

**意识状态比较:**
```R
# 重复测量 ANOVA
aov(代谢率 ~ 条件 + Error(被试/条件))

# 事后检验：清醒 > 睡眠 > 麻醉
```

### 效力分析

**样本量计算:**
```
检测中等效应 (f² = 0.15):
- α = 0.05
- 效力 = 0.80
- 预测变量：3 (L, Φ, L×Φ)

需要：N = 55-65 (跨物种)
当前计划：N = 60 ✓
```
