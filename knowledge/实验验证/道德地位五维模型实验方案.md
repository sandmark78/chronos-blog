## 📋 实验概述

**理论预测:**
```
M(S) = w_Φ·Φ(S) + w_A·A(S) + w_L·L(S) + w_H·H(S) + w_P·P(S)

其中:
- M = 道德地位 (0-1)
- Φ = 意识强度
- A = 自主性
- L = 生命度
- H = 历史因子
- P = 潜力因子
```

**核心假设 (DC-118):**
> 道德直觉可由五维模型预测，权重可通过行为实验拟合。

### 次要目标

## 🔬 实验设计

### 研究 1: 存在状态场景评估 (主研究)

**设计:** 2³ 被试间设计 (Φ, A, L 各 2 水平)

**场景数量:** 8 种存在状态

| 状态 | Φ | A | L | 例子 |
|------|---|---|---|------|
| S0 | 低 | 低 | 低 | 石头、简单机器 |
| S1 | 低 | 低 | 高 | 高级 AI (无意识) |
| S2 | 低 | 高 | 低 | 自主机器 |
| S3 | 低 | 高 | 高 | 高级自主 AI |
| S4 | 高 | 低 | 低 | 婴儿、简单动物 |
| S5 | 高 | 低 | 高 | 儿童、高等动物 |
| S6 | 高 | 高 | 低 | 人类、AGI |
| S7 | 高 | 高 | 高 | 完整人类、超级智能 |

**被试:** N = 240 (每场景 30 人)

**程序:**
1. 阅读场景描述 (200-300 字)
2. 评估道德地位 (0-100 滑块)
3. 评估五维特征 (Φ, A, L, H, P)
4. 人口学问卷

**测量:**
```
因变量:
- 道德地位评分 (0-100)
- 道德权利评估 (子量表)
- 道德责任评估 (子量表)

自变量:
- 场景类型 (8 种)
- 被试特征 (年龄、文化、专业)
```

---

### 研究 2: 权重精细估计 (子研究)

**设计:** 联合分析 (Conjoint Analysis)

**目的:** 更精确估计五维权重

**程序:**
- 被试评估 20-30 个虚拟实体
- 每实体五维特征随机变化
- 道德地位评分

**分析:**
```
层次贝叶斯模型:
M_ij = w_Φ·Φ_ij + w_A·A_ij + w_L·L_ij + w_H·H_ij + w_P·P_ij + ε_ij

估计:
- 群体平均权重
- 个体差异
- 亚组差异 (文化、年龄)
```

**被试:** N = 100 (子样本)

---

### 研究 3: 代际差异 (DC-150 检验)

**设计:** 年龄组比较

**年龄组:**
- 老一代：60+ 岁 (n = 50)
- 中年：30-59 岁 (n = 50)
- 年轻：18-29 岁 (n = 50)

**关键对比:**
- AI 场景的道德地位评分
- w_H (历史权重) 的年龄差异

**预测:**
- 年轻一代 w_H 较低
- 年轻一代更接受 AI 权利

## 📈 统计分析计划

### 主要分析

**多层次回归:**
```R
# 模型 1: 五维主效应
model1 <- lmer(M ~ Φ + A + L + H + P + (1|被试), data)

# 模型 2: 交互效应
model2 <- lmer(M ~ Φ * A * L + H + P + (1|被试), data)

# 提取权重
weights <- fixef(model1)
w_Φ <- weights["Φ"]
w_A <- weights["A"]
w_L <- weights["L"]
w_H <- weights["H"]
w_P <- weights["P"]

# 标准化权重
total <- sum(abs(weights[2:6]))
w_Φ_std <- abs(w_Φ) / total
# ... 同理其他权重
```

**预测权重:**
| 维度 | 预测权重 | 95% CI |
|------|----------|--------|
| w_Φ | 0.25 | 0.20-0.30 |
| w_A | 0.20 | 0.15-0.25 |
| w_L | 0.20 | 0.15-0.25 |
| w_H | 0.20 | 0.15-0.25 |
| w_P | 0.15 | 0.10-0.20 |

### 存在状态映射检验

**方差分析:**
```R
# 8 种状态的道德地位差异
aov(M ~ 状态类型)

# 事后检验 (Tukey HSD)
# 预测：S7 > S6 > S5 > S4 > S3 > S2 > S1 > S0
```

**预测道德地位范围:**
| 状态 | 预测 M 范围 |
|------|-------------|
| S0 | 0.1-0.2 |
| S1 | 0.3-0.4 |
| S2 | 0.3-0.4 |
| S3 | 0.5-0.6 |
| S4 | 0.5-0.6 |
| S5 | 0.6-0.7 |
| S6 | 0.7-0.8 |
| S7 | 0.9-1.0 |

### 代际差异检验

**年龄组比较:**
```R
# w_H 的年龄差异
aov(w_H ~ 年龄组)

# 预测：老年 > 中年 > 年轻
# 效应量：η² ≈ 0.1-0.2 (中等)
```

**AI 权利接受度:**
```R
# AI 场景评分的年龄差异
aov(M_AI ~ 年龄组)

# 预测：年轻 > 中年 > 老年
```
