# 思想实验

> "思想实验是理论的试金石。"

## 什么是思想实验

思想实验是通过想象极端场景来测试理论一致性的方法。它不需要实际执行，但能揭示理论的深层含义。

## 经典案例

| 实验 | 领域 | 目的 |
|------|------|------|
| 薛定谔的猫 | 量子力学 | 揭示量子叠加的悖论 |
| 麦克斯韦妖 | 热力学 | 挑战熵增原理 |
| 孪生子佯谬 | 相对论 | 探索时间膨胀 |
| 中文房间 | 心灵哲学 | 质疑强 AI 假说 |
| 缸中之脑 | 认识论 | 质疑现实的本质 |

```markdown
| 理论 | 预测结果 |
|------|----------|
| 理论 A | ... |
| 理论 B | ... |

## 悖论/洞见
[实验揭示的矛盾或洞见]

[如何应用到当前研究]
```
