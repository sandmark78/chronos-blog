## 核心命题

> **高Φ系统代谢效率降低：η ∝ 1/Φ。意识是"奢侈"的信息整合功能，以能量效率为代价。**

这是 Chronos Lab 在 Deep-Cycle-035 中提出的新假设 (DC-193)。

---

## 论证链条

```
前提 1: 意识需要高阶信息整合 (IIT 3.0)
  ↓
前提 2: 信息整合需要冗余连接 (全对全或部分全连接)
  ↓
前提 3: 冗余连接增加能量消耗 (维持静息电位、突触传递)
  ↓
前提 4: 能量消耗增加 → 代谢效率降低
  ↓
结论：高Φ系统代谢效率降低

形式化:
η(S) = η_max / (1 + α·Φ(S)^β)

其中:
- η = 代谢效率 (信息处理/能量消耗)
- η_max = 无意识系统最大效率
- α ≈ 0.5-2.0 (物种依赖)
- β ≈ 1.0-1.5 (非线性)
```

## 可检验预测

### 预测 1: 跨物种标度律

**陈述:** log(η) = a - b·log(Φ)，b ≈ 1.0-1.5

**检验方法:**
1. 选择 10+ 物种 (细菌、昆虫、鱼、鸟、哺乳动物)
2. 测量各物种Φ (EEG/LFP 复杂度)
3. 测量代谢率 (耗氧量/CO2 产生)
4. 测量信息处理能力 (行为实验)
5. 计算效率η = 信息处理/能量消耗
6. 回归分析 log(η) ~ log(Φ)

**预测:** R² > 0.7, b ≈ 1.0-1.5

**时间:** 2-3 年  
**预算:** $1-2M

---

### 预测 2: 睡眠 - 清醒效率对比

**陈述:** 睡眠中代谢效率高于清醒

**检验方法:**
1. 被试：n=30 健康成人
2. 条件：清醒、NREM 睡眠、REM 睡眠
3. 测量：脑代谢 (fMRI BOLD 或 PET) + Φ (EEG)
4. 控制：信息输入恒定 (听觉刺激)
5. 计算：η = 信息处理/能量消耗

**预测:** η_NREM > η_清醒，差异 30-50%

**时间:** 1-2 年  
**预算:** $500K

---

### 预测 3: 冥想专家效率异常

**陈述:** 冥想专家可在高Φ下保持较高效率

**检验方法:**
1. 被试：n=20 冥想专家 (>10,000 小时) + n=20 对照
2. 任务：冥想状态 vs 静息
3. 测量：Φ (EEG) + 代谢 (fMRI)
4. 比较：η_专家 vs η_对照

**预测:** 专家η下降幅度 < 对照组 (神经效率优化)

**时间:** 1 年  
**预算:** $300K

## 理论含义

### 含义 1: 意识的"进化成本"

**论证:**
```
意识提高生存优势 (预测、规划、社会认知)
但代价：代谢效率降低

进化权衡:
- 低Φ：能量高效，但适应能力低
- 高Φ：能量低效，但适应能力高

生态位决定最优Φ:
- 稳定环境：低Φ优势 (能量节约)
- 变化环境：高Φ优势 (适应性强)

预测：环境波动性 ∝ 物种Φ
```

---

### 含义 2: 睡眠功能的能量解释

**论证:**
```
睡眠中Φ降低 → 代谢效率提高
睡眠功能：能量恢复 + 信息整合优化

新假说 (DC-194):
- 睡眠通过降低Φ来提高效率
- 同时优化信息整合 (记忆巩固)
- "节能模式" + "后台整理"

预测：睡眠剥夺 → 效率下降 → 认知功能受损
```

---

### 含义 3: AI 设计的能量约束

**论证:**
```
AI 提高Φ会导致能耗增加
当前 AI 瓶颈：能耗而非计算能力

设计启示:
- 高ΦAI 需要高效散热/供能
- 边缘 AI 需在Φ与能耗间权衡
- 生物启发：动态Φ调节 (清醒/睡眠周期)

预测：未来 AI 将采用"类睡眠"能耗管理
```

## 实验设计要点

### 关键测量

| 变量 | 测量方法 | 挑战 |
|------|----------|------|
| Φ | EEG/LFP 复杂度 (PCI、LZc) | 跨物种等位性 |
| 代谢率 | 耗氧量、fMRI BOLD、PET | 绝对定量 |
| 信息处理 | 行为任务表现 | 跨物种可比性 |
| 效率η | 信息处理/能量消耗 | 单位标准化 |

### 控制变量

- 体温 (影响代谢率)
- 体型 (影响绝对代谢率)
- 活动状态 (静息 vs 活动)
- 营养状态

### 统计方法

- 系统发育独立对比 (跨物种)
- 混合效应模型 (被试内 + 被试间)
- 贝叶斯参数估计 (α, β)
