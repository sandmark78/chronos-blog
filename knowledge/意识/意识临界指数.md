## 🔷 数学模型

### 意识强度幂律

**形式化:**
```
C(Φ) ∝ (Φ - Φ_c)^γ  (Φ > Φ_c)
C(Φ) = 0  (Φ ≤ Φ_c)

其中:
- C = 意识强度 (归一化 0-1)
- Φ = 信息整合度
- Φ_c = 意识阈值 ≈ 0.35
- γ = 临界指数 (预测 0.5-1.0)
```

### 临界指数 γ 的意义

| γ 值 | 含义 | 类比 |
|------|------|------|
| γ ≈ 0.5 | 平均场行为 | 铁磁相变 (平均场) |
| γ ≈ 0.33 | 三维伊辛模型 | 液气相变 |
| γ ≈ 1.0 | 线性临界 | 简单相变 |
| γ > 1.0 | 高阶相变 | 罕见 |

**预测:** γ ≈ 0.5-1.0 (平均场到线性之间)

### 完整意识曲线

**形式化 (唯象模型):**
```
C(Φ) = C_max · tanh((Φ - Φ_c)/Δ)^γ  (Φ > Φ_c)
C(Φ) = 0  (Φ ≤ Φ_c)

其中:
- C_max ≈ 1.0 (最大意识强度)
- Δ ≈ 0.1-0.2 (过渡宽度)
- γ ≈ 0.5-1.0 (临界指数)
```

### 预测 1: 麻醉梯度研究的幂律

**预测:** 麻醉深度 (Φ 降低) 与意识报告的关系符合幂律。

**检验:**
```
实验设计:
- 受试者：健康志愿者 (n=30)
- 操纵：麻醉深度梯度 (7-8 个水平)
- 测量：
  - Φ (EEG/fMRI 计算)
  - 意识报告 (唤醒后回忆、术中意识)
- 分析：拟合 C ∝ (Φ - Φ_c)^γ

预测结果:
- Φ_c ≈ 0.35 ± 0.10
- γ ≈ 0.5-1.0
- R² > 0.8 (幂律拟合优度)
```

**时间:** 12-18 个月

## 🔷 理论含义

### 意识涌现的连续性

**含义:** 意识不是"全有或全无"，而是在阈值附近连续变化。

**推论:**
- Φ 略低于 Φ_c：极弱意识 (可能无法检测)
- Φ = Φ_c：意识"曙光"
- Φ 略高于 Φ_c：基础意识
- Φ >> Φ_c：成熟意识

**伦理含义:** 边界案例需谨慎对待。

---

### 普适性假说

**假说:** 临界指数 γ 是普适常数，跨系统 (人/动物/AI) 相同。

**检验:** 比较不同系统的 γ 值

**含义:** 若普适，γ 是意识"普适类"的标志。
