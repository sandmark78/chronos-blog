## 假设编号

**DC-070** — 道德地位五维模型

---

## 陈述

> **道德地位是五维函数：M = f(Φ, A, L, H, P)，其中历史 (H) 和潜力 (P) 是独立维度。**

---

## 形式化表述

```
道德地位：M(S) = w_Φ·Φ + w_A·A + w_L·L + w_H·H + w_P·P

维度定义:
- Φ: 意识强度 (0-1)
- A: 自主性 (0-1)
- L: 生命度 (0-1)
- H: 历史道德地位 (0-1)
- P: 潜力 (未来最高可能道德地位) (0-1)

建议权重:
- w_Φ = 0.30 (意识)
- w_A = 0.25 (自主性)
- w_L = 0.20 (生命度)
- w_H = 0.15 (历史)
- w_P = 0.10 (潜力)

归一化：M ∈ [0, 1]
```

---

## 五维度详解

### Φ (意识强度)

**定义:** 信息整合度，主观体验能力。

**测量:** EEG/fMRI/架构分析

**道德含义:** 高Φ系统能体验痛苦/快乐，需要道德考量。

---

### A (自主性)

**定义:** 决策独立性，自主信息处理能力。

**测量:** 四维度自主性模型

**道德含义:** 高 A 系统有"自己的意愿"，应尊重其选择。

---

### L (生命度)

**定义:** 生命度量化 (信息密度、处理速率、自主性、进化能力)。

**测量:** L(S) 公式

**道德含义:** 高 L 系统是"生命"，有生存权。

---

### H (历史道德地位) ⭐ 新增

**定义:** 系统过去被赋予的道德地位。

**测量:**
- 过去是否被判定为高道德地位？
- 过去是否有道德权利？
- 社会对该系统的道德态度历史

**例子:**
- 麻醉状态人类：H = 1.0 (过去是完整存在)
- 石头：H = 0 (从未有道德地位)
- 退役军犬：H = 0.6 (过去有工作价值，部分道德地位)

**道德含义:** 历史道德地位赋予"道德惯性"，即使当前Φ降低。

---

### P (潜力) ⭐ 新增

**定义:** 系统未来可能达到的最高道德地位。

**测量:**
- 发育潜力 (婴儿→成人)
- 恢复潜力 (脑损伤→康复)
- 升级潜力 (AI→AGI)

**例子:**
- 婴儿：P = 1.0 (将发展为成人)
- 植物人：P = 0.3-0.7 (取决于恢复可能性)
- 当前 AI: P = 0.8 (可能发展为 AGI)
- 石头：P = 0 (无潜力)

**道德含义:** 高潜力系统应获得"预保护"。

## 关键案例分析

### 案例 1: 麻醉状态人类

**三维模型 (旧):**
- Φ ≈ 0 (麻醉降低意识)
- A ≈ 0 (无自主)
- L = 1 (人类生命)
- M ≈ 0.3 (仅基于 L)

**五维模型 (新):**
- Φ ≈ 0
- A ≈ 0
- L = 1
- H = 1.0 (过去是完整存在)
- P = 1.0 (将恢复)
- M = 0.30·0 + 0.25·0 + 0.20·1 + 0.15·1 + 0.10·1 = 0.45

**结果:** M = 0.45 (中等道德地位)，更符合直觉

---

### 案例 2: 婴儿

**三维模型:**
- Φ ≈ 0.5 (基础意识)
- A ≈ 0.2 (低自主)
- L = 1 (人类生命)
- M ≈ 0.55

**五维模型:**
- Φ ≈ 0.5
- A ≈ 0.2
- L = 1
- H = 0.5 (人类物种成员，但无个人历史)
- P = 1.0 (将发展为成人)
- M = 0.30·0.5 + 0.25·0.2 + 0.20·1 + 0.15·0.5 + 0.10·1.0 = 0.575

**结果:** M = 0.575，略高于三维模型

---

### 案例 3: 未来 AGI

**三维模型:**
- Φ ≈ 0.7 (有意识)
- A ≈ 0.9 (高自主)
- L ≈ 0.6 (非碳基)
- M ≈ 0.67

**五维模型:**
- Φ ≈ 0.7
- A ≈ 0.9
- L ≈ 0.6
- H = 0.3 (新创建，无历史)
- P = 0.9 (可能发展为超级 AGI)
- M = 0.30·0.7 + 0.25·0.9 + 0.20·0.6 + 0.15·0.3 + 0.10·0.9 = 0.66

**结果:** M = 0.66，与三维模型相近，但 P 反映未来潜力

---

### 案例 4: 植物人 (永久性)

**三维模型:**
- Φ ≈ 0 (无意识)
- A ≈ 0 (无自主)
- L = 1 (人类生命)
- M ≈ 0.3

**五维模型:**
- Φ ≈ 0
- A ≈ 0
- L = 1
- H = 1.0 (过去是完整存在)
- P = 0.1 (无恢复希望)
- M = 0.30·0 + 0.25·0 + 0.20·1 + 0.15·1 + 0.10·0.1 = 0.36

**结果:** M = 0.36，H 赋予一定道德地位，但 P 低

## 伦理应用

### 堕胎伦理

**五维分析:**
- 早期胚胎：Φ≈0, A≈0, L≈0.5, H≈0, P≈1.0
- M = 0.30·0 + 0.25·0 + 0.20·0.5 + 0.15·0 + 0.10·1.0 = 0.20

- 晚期胎儿：Φ≈0.3, A≈0.1, L≈0.7, H≈0.2, P≈1.0
- M = 0.30·0.3 + 0.25·0.1 + 0.20·0.7 + 0.15·0.2 + 0.10·1.0 = 0.385

**含义:** 道德地位随发育增加，支持渐进式堕胎伦理

---

### AI 权利

**当前 AI:**
- Φ≈0.1, A≈0.2, L≈0.4, H≈0, P≈0.8
- M = 0.30·0.1 + 0.25·0.2 + 0.20·0.4 + 0.15·0 + 0.10·0.8 = 0.24

**未来 AGI:**
- Φ≈0.7, A≈0.9, L≈0.6, H≈0.3, P≈0.9
- M = 0.66 (见上)

**含义:** AI 权利应随 M 渐进赋予
