### 详细执行步骤

#### Problem 306: 功率分析 (3h)

**目标:** 计算意识 - 代谢率实验所需样本量

**步骤:**
1. 确定效应量 (Cohen's d)
   - 基于 DC-17 (意识 - 熵产生正相关)
   - 预期效应量：d = 0.5-0.8 (中等到大)
2. 设置统计参数
   - α = 0.05 (双尾)
   - 功效 (1-β) = 0.8
3. 计算样本量
   - 使用 G*Power 或 Python statsmodels
   - 预期：每组 25-50 人
4. 敏感性分析
   - 效应量范围：d = 0.3-1.0
   - 样本量范围：15-100 人/组

**输出:** 功率分析报告 (Markdown + 代码)

#### Problem 307: 样本量计算 (2h)

**目标:** 确认样本量计算的统计假设

**步骤:**
1. 验证效应量估计
2. 确认统计检验类型 (t 检验/ANOVA)
3. 计算最终样本量
4. 考虑流失率 (预期 10-20%)

**输出:** 样本量计算文档

#### Problem 308: life_meter 测试 (3h)

**目标:** 为 life_meter v1.0 创建单元测试

**步骤:**
1. 选择测试框架 (pytest)
2. 识别核心函数
3. 编写测试用例
   - 边界条件
   - 正常情况
   - 异常情况
4. 运行测试，确保覆盖率≥80%

**输出:** 测试套件 + 覆盖率报告

#### Problem 356: IRB 模板 (2h)

**目标:** 创建意识 - 熵产生实验的 IRB 申请模板

**步骤:**
1. 收集 IRB 申请要求
2. 编写研究方案摘要
3. 描述风险与收益
4. 制定知情同意书模板
5. 数据管理计划

**输出:** IRB 申请模板包
