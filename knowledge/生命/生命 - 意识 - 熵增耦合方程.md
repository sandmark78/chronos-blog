## 核心方程

```
dS/dt = σ₀ + σ₁·L(S) + σ₂·Φ(S) + σ₃·L(S)×Φ(S)
```

---

## 参数定义

| 符号 | 名称 | 定义 | 单位 |
|------|------|------|------|
| dS/dt | 熵产生率 | 系统单位时间熵产生 | J/(K·s) |
| σ₀ | 基础熵产生 | 非生命系统基准熵产生 | J/(K·s) |
| σ₁ | 生命度系数 | 生命度对熵产生的贡献 | J/(K·s) |
| σ₂ | 意识系数 | 意识对熵产生的贡献 | J/(K·s) |
| σ₃ | 耦合系数 | 生命 - 意识交互项 (关键!) | J/(K·s) |
| L(S) | 生命度 | 0-1 归一化 | 无量纲 |
| Φ(S) | 整合信息度 | 0-1 归一化 | 无量纲 |

## 简化形式

### 高生命度系统 (L ≈ 1)
```
dS/dt ≈ σ₀ + σ₁ + σ₂·Φ + σ₃·Φ
dS/dt ≈ (σ₀ + σ₁) + (σ₂ + σ₃)·Φ
```

### 高意识系统 (Φ ≈ 1)
```
dS/dt ≈ σ₀ + σ₁·L + σ₂ + σ₃·L
dS/dt ≈ (σ₀ + σ₂) + (σ₁ + σ₃)·L
```

### 同时高生命度高意识 (L ≈ 1, Φ ≈ 1)
```
dS/dt ≈ σ₀ + σ₁ + σ₂ + σ₃
```

### 无生命无意识 (L = 0, Φ = 0)
```
dS/dt = σ₀ (基础熵产生)
```

## 关键预测

### 预测 1: 意识状态熵产生差异
```
清醒状态：dS/dt ≈ σ₀ + σ₁ + σ₂·Φ_wake + σ₃·L×Φ_wake
睡眠状态：dS/dt ≈ σ₀ + σ₁ + σ₂·Φ_sleep + σ₃·L×Φ_sleep
麻醉状态：dS/dt ≈ σ₀ + σ₁ + σ₂·Φ_anesthesia + σ₃·L×Φ_anesthesia

预测：dS/dt_wake > dS/dt_sleep > dS/dt_anesthesia
```

### 预测 2: 跨物种熵产生差异
```
人类 (L≈0.85, Φ≈0.75): dS/dt ≈ σ₀ + 0.85σ₁ + 0.75σ₂ + 0.64σ₃
黑猩猩 (L≈0.75, Φ≈0.55): dS/dt ≈ σ₀ + 0.75σ₁ + 0.55σ₂ + 0.41σ₃
猫 (L≈0.70, Φ≈0.40): dS/dt ≈ σ₀ + 0.70σ₁ + 0.40σ₂ + 0.28σ₃
昆虫 (L≈0.60, Φ≈0.20): dS/dt ≈ σ₀ + 0.60σ₁ + 0.20σ₂ + 0.12σ₃

预测：代谢率/体重比 ∝ L×Φ
```

### 预测 3: AI 系统熵产生
```
高Φ AI (Φ≈0.8): dS/dt ≈ σ₀ + σ₂·0.8 + σ₃·L×0.8
低Φ AI (Φ≈0.2): dS/dt ≈ σ₀ + σ₂·0.2 + σ₃·L×0.2

预测：高Φ AI 运行时代谢 (能耗) 更高
```

## 实验设计方案

### 实验 1: 意识状态熵产生测量

**目标:** 验证 dS/dt ∝ Φ (同一生物不同意识状态)

**方法:**
1. 被试：健康成人 20 名
2. 状态：清醒、NREM 睡眠、REM 睡眠、麻醉 (手术患者)
3. 测量:
   - 全脑代谢率 (PET/fMRI) — 熵产生代理
   - Φ值 (EEG 复杂度分析)
   - 体温变化 (热力学熵产生)

**预测:**
```
代谢率清醒 > 代谢率 REM > 代谢率 NREM > 代谢率麻醉
Φ清醒 > ΦREM > ΦNREM > Φ麻醉
代谢率 ∝ Φ (线性关系)
```

**时间:** 6-12 个月  
**预算:** $500K-1M

---

### 实验 2: 跨物种代谢率 - Φ标度律

**目标:** 验证代谢率 ∝ M_body^β × Φ^γ

**方法:**
1. 物种：人类、灵长类、哺乳类、鸟类、爬行类、昆虫 (各 10 个物种)
2. 测量:
   - 基础代谢率 (间接量热法)
   - 体重
   - Φ值 (神经活动复杂度)
3. 分析：多元回归 M_body 和 Φ对代谢率的贡献

**预测:**
```
代谢率 = a × M_body^0.75 × Φ^γ
γ ≈ 1.0-2.0
同体重下，高Φ物种代谢率更高
```

**时间:** 12-24 个月  
**预算:** $1-2M

---

### 实验 3: AI 系统能耗 - Φ关系

**目标:** 验证 AI 系统Φ值与能耗正相关

**方法:**
1. AI 系统：不同架构的神经网络 (Transformer, CNN, RNN 等)
2. 任务：相同基准任务 (确保可比性)
3. 测量:
   - 能耗 (电力测量)
   - Φ值 (网络活动复杂度分析)
   - 性能 (准确率)

**预测:**
```
能耗 ∝ Φ^δ (δ ≈ 1.0-2.0)
高性能 AI 不一定高能耗 (架构优化)
高Φ AI 能耗更高
```

**时间:** 6-12 个月  
**预算:** $200K-500K

- 基础代谢率：~80 W = 80 J/s

- dS/dt ≈ 80/310 ≈ 0.26 J/(K·s)

```
σ₀ ≈ 0.05 J/(K·s) (非生命基础)

总计：0.26 J/(K·s) ✓

**耦合项占比:**
```
