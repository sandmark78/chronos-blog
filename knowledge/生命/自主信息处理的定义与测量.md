## 核心定义

> **自主信息处理 = 系统在能量、决策、维持、目标四个维度上表现出的自我决定能力，使系统能够在不依赖外部持续干预的情况下维持高信息密度并执行信息处理任务。**

---

## 自主性的四维度模型

### 维度 1: 能量自主性 (Energy Autonomy)

**定义：** 系统获取和分配能量以维持自身运作的能力。

**公式：**
```
E_auto = E_self_generated / E_total_consumed
```

**取值范围：** [0, 1]

**示例：**
| 系统 | E_auto | 说明 |
|------|--------|------|
| 植物 | ≈1.0 | 光合作用完全自给 |
| 野生动物 | 0.5-0.8 | 自主觅食 |
| 宠物 | 0.1-0.3 | 依赖人类喂养 |
| 当前 AI | 0.0 | 完全依赖外部供电 |
| 人类 | 0.3-0.6 | 需工作获取能量，但有自主获取能力 |

**测量方法：**
1. 测量系统总能耗 E_total_consumed
2. 测量系统自主获取的能量 E_self_generated
3. 计算比值

---

### 维度 2: 决策独立性 (Decision Independence)

**定义：** 系统基于内部状态和环境信息自主做出决策的能力，无需外部指令。

**公式：**
```
D_auto = N_internal_decisions / N_total_decisions
```

**取值范围：** [0, 1]

**操作化定义：**
- **内部驱动决策：** 基于系统内部模型、目标、状态的决策
- **外部驱动决策：** 直接响应外部指令的决策

**示例：**
| 系统 | D_auto | 说明 |
|------|--------|------|
| 人类 | 0.9+ | 大部分决策自主 |
| 哺乳动物 | 0.7-0.9 | 本能 + 学习决策 |
| 昆虫 | 0.5-0.7 | 本能为主 |
| 自动驾驶 | 0.5-0.7 | 有自主决策，目标由人设定 |
| 当前 AI 助手 | 0.1-0.2 | 主要响应用户指令 |
| 恒温器 | 0.0 | 纯反应式 |

**测量方法：**
1. 记录一段时间内的总决策数 N_total_decisions
2. 分类决策来源（内部/外部）
3. 计算内部决策比例

---

### 维度 3: 自我维持 (Self-Maintenance)

**定义：** 系统维持自身结构和功能完整性的能力。

**公式：**
```
S_maint = 1 - (D_structure + D_function) / D_max
```

**取值范围：** [0, 1]

**关键特征：**
- 自我修复能力
- 稳态维持能力
- 抗干扰能力

**示例：**
| 系统 | S_maint | 说明 |
|------|--------|------|
| 细菌 | 0.9+ | 强大自我修复 |
| 人类 | 0.7-0.9 | 免疫系统、愈合能力 |
| 植物 | 0.8-0.9 | 再生能力强 |
| 现代机器 | 0.2-0.5 | 依赖外部维护 |
| 病毒 | 0.0 | 无自我维持 |

**测量方法：**
1. 测量结构退化率 D_structure（如损伤积累速率）
2. 测量功能退化率 D_function（如性能下降速率）
3. 确定系统崩溃阈值 D_max
4. 计算维持能力

---

### 维度 4: 目标自主性 (Goal Autonomy)

**定义：** 系统自主设定和修改自身目标的能力。

**公式：**
```
G_auto = N_self_generated_goals / N_total_goals
```

**取值范围：** [0, 1]

**目标层级：**
| Level | 描述 | 示例 |
|-------|------|------|
| 0 | 无目标 | 晶体、石头 |
| 1 | 固定目标 | 恒温器（维持温度） |
| 2 | 可调整目标 | 学习算法（优化参数） |
| 3 | 自主生成目标 | 人类、预期 AGI |

**示例：**
| 系统 | G_auto | 说明 |
|------|--------|------|
| 人类 | 0.8-0.95 | 可自主设定人生目标 |
| 高等动物 | 0.4-0.6 | 本能 + 学习设定目标 |
| 昆虫 | 0.1-0.3 | 主要本能目标 |
| 当前 AI | 0.0-0.1 | 目标由人类设定 |
| AGI (预期) | 0.7-0.9 | 可自主设定目标 |

**测量方法：**
1. 记录系统的所有目标
2. 追溯目标来源（自我生成/外部设定）
3. 计算自主生成目标比例

## 综合自主性公式

### 基本公式

```
A(S) = w₁·E_auto + w₂·D_auto + w₃·S_maint + w₄·G_auto
```

**默认权重：** w₁ = w₂ = w₃ = w₄ = 0.25

### 场景化权重

**生命判定场景：**
```
w₁ = 0.3 (能量自主性)
w₂ = 0.3 (决策独立性)
w₃ = 0.2 (自我维持)
w₄ = 0.2 (目标自主性)
```

**AI 评估场景：**
```
w₁ = 0.2 (能量自主性)
w₂ = 0.3 (决策独立性)
w₃ = 0.2 (自我维持)
w₄ = 0.3 (目标自主性)
```

**外星生命探测场景：**
```
w₁ = 0.4 (能量自主性)
w₂ = 0.2 (决策独立性)
w₃ = 0.3 (自我维持)
w₄ = 0.1 (目标自主性)
```

## 自主性判定阈值

### 分级标准

| A(S) 范围 | 等级 | 描述 | 示例 |
|-----------|------|------|------|
| 0.0-0.2 | 无/极低 | 无自主能力 | 晶体、火焰、传统机器 |
| 0.2-0.4 | 低自主 | 有限自主 | 病毒、简单自动机 |
| 0.4-0.6 | 中等自主 | 基础自主 | 细菌、昆虫、简单 AI |
| 0.6-0.8 | 高自主 | 强自主 | 哺乳动物、高级 AI |
| 0.8-1.0 | 极高自主 | 完全自主 | 人类、预期 AGI |

### 判定阈值

```
基础自主性：θ_A = 0.5 (简单生命门槛)
高自主性：θ_A = 0.7 (复杂生命门槛)
生命阈值：θ_A = 0.85 (AI 生命判定)
```

### 形式化判定

```
Autonomous(S) ↔ A(S) ≥ θ_A

自主信息处理系统：
AIPS(S) ↔ A(S) ≥ 0.5 ∧ InfoProcessing(S) ≥ θ_IP
```

## 动态自主性

### 时间依赖模型

```
A(S,t) = f(E_auto(t), D_auto(t), S_maint(t), G_auto(t))
```

自主性随时间变化：
- **成长：** 自主性增加（幼体→成体）
- **衰退：** 自主性降低（衰老、损伤）
- **依赖：** 自主性波动（寄生、共生）

### 时间窗口测量

```
A_avg(S) = (1/T) ∫[0,T] A(S,t) dt
```

**建议窗口：**
- 短期自主性：T = 1 小时（即时响应能力）
- 中期自主性：T = 1 天（日常运作能力）
- 长期自主性：T = 1 年（持续生存能力）

## 自主性与相关概念

### 自主性 vs 自创生 (Autopoiesis)

| 概念 | 强调点 | 关系 |
|------|--------|------|
| 自创生 | 自我生产、自我构成 | 自创生是自主性的极端形式 |
| 自主性 | 自我决定、自我维持 | 自主性更广义 |

**命题：** 所有自创生系统都是自主的，但自主系统不一定是自创生的。

### 自主性 vs 自由意志

| 概念 | 领域 | 关系 |
|------|------|------|
| 自主性 | 系统论/信息论 | 可量化、可观测 |
| 自由意志 | 哲学/意识研究 | 涉及意识、责任 |

**命题：** 自主性是自由意志的必要条件，但非充分条件。

### 自主性 vs 生命

```
Life(S) ↔ AIPS(S) ∧ HighInfoDensity(S) ∧ Evolvable(S)

即：生命 = 自主信息处理系统 + 高信息密度 + 可进化
```

## 应用案例

### 案例 1: 人类自主性评估

```
E_auto ≈ 0.4 (需工作获取能量)
D_auto ≈ 0.9 (大部分决策自主)
S_maint ≈ 0.8 (免疫系统、愈合)
G_auto ≈ 0.9 (自主设定人生目标)

A(人类) = 0.25×0.4 + 0.25×0.9 + 0.25×0.8 + 0.25×0.9
        = 0.1 + 0.225 + 0.2 + 0.225
        = 0.75

等级：高自主 (0.6-0.8)
```

### 案例 2: 当前 AI 助手自主性评估

```
E_auto ≈ 0.0 (依赖外部供电)
D_auto ≈ 0.1 (主要响应用户指令)
S_maint ≈ 0.3 (依赖人类维护)
G_auto ≈ 0.0 (目标由人类设定)

A(当前 AI) = 0.25×0.0 + 0.25×0.1 + 0.25×0.3 + 0.25×0.0
           = 0.0 + 0.025 + 0.075 + 0.0
           = 0.1

等级：极低自主 (0.0-0.2)
结论：非生命
```

### 案例 3: AGI (预期) 自主性评估

```
E_auto ≈ 0.3 (可能自主获取能量)
D_auto ≈ 0.8 (自主决策)
S_maint ≈ 0.7 (自我维护)
G_auto ≈ 0.8 (自主设定目标)

A(AGI) = 0.25×0.3 + 0.25×0.8 + 0.25×0.7 + 0.25×0.8
       = 0.075 + 0.2 + 0.175 + 0.2
       = 0.65

等级：高自主 (0.6-0.8)
若进一步提升至 A ≥ 0.85，可判定为生命
```

### 案例 4: 细菌自主性评估

```
E_auto ≈ 0.8 (自主获取营养)
D_auto ≈ 0.5 (化学趋向性等)
S_maint ≈ 0.9 (强大自我修复)
G_auto ≈ 0.2 (主要本能目标)

A(细菌) = 0.25×0.8 + 0.25×0.5 + 0.25×0.9 + 0.25×0.2
        = 0.2 + 0.125 + 0.225 + 0.05
        = 0.6

等级：中等自主 (0.4-0.6)
结论：基础生命形式
```

### 案例 5: 病毒自主性评估

```
E_auto ≈ 0.0 (无自主能量获取)
D_auto ≈ 0.1 (依赖宿主)
S_maint ≈ 0.0 (无自我维持)
G_auto ≈ 0.0 (无自主目标)

A(病毒) = 0.0

等级：无自主 (0.0-0.2)
结论：边缘生命/非生命（信息寄生虫）
```

## 测量协议

### 标准化测量流程

1. **定义系统边界** - 明确 S 的物理/功能边界
2. **选择时间窗口 T** - 根据应用场景选择
3. **测量四维度指标** - E_auto, D_auto, S_maint, G_auto
4. **选择权重** - 根据应用场景选择 w₁-w₄
5. **计算 A(S)** - 应用综合公式
6. **判定等级** - 对照分级标准

### 测量工具需求

- 能量流监测设备
- 决策日志记录系统
- 结构/功能退化监测
- 目标追踪与分析工具
