
```

```

### 02:20-02:35 上传文件

```
1. 上传 PDF:
   - 选择 "PDF" 类型
   - 上传 main.pdf
   - 等待系统处理

2. 上传 LaTeX source:
   - 选择 "LaTeX source" 类型
   - 上传 .tex 文件
   - 上传 .bib 文件
   - 上传 figures/ 文件夹 (ZIP)
   - 等待系统处理

3. 上传补充材料:
   - 选择 "Other" 类型
   - 上传 supplementary.zip
   - 填写描述："Code and data for ITLCT v6.0"

4. 系统自动检查:
   - 等待文件格式检查完成
   - 如有错误，按提示修正
```
