## 2.2 理论矛盾识别

### 矛盾 1: Friston 自由能 vs ITLCT 熵增

**矛盾点:**
- Friston: 生命最小化自由能 (表观"负熵")
- ITLCT: 生命最大化熵产生 (dS/dt ∝ L×Φ)

**分析:**
- 表面矛盾：最小化 vs 最大化？
- 解决方案：层次不同
  - Friston 自由能 F = 预测误差 (信息层面)
  - ITLCT 熵产生 dS/dt = 热力学耗散 (物理层面)
  - 最小化 F → 更高效耗散 → 最大化 dS/dt

**数学统一:**
```
Friston: min F = min(E[energy] - S[belief])
ITLCT: max dS/dt = max(σ₀ + σ₁·L + σ₂·Φ + σ₃·L×Φ)

统一:
min F → 优化预测模型 → 提高 L 和Φ → max dS/dt
```

**解决方案:**
- 表述为"互补而非矛盾"
- Friston 描述信息层面优化
- ITLCT 描述热力学后果
- 两者统一于信息 - 热力学耦合

**解决方案:**

- 预测处理解释意识功能

- ITLCT 整合：功能实现于结构，结构支持功能
