## A10 降格：量子-经典过渡

### 原公理陈述
> A10: 存在一个特征退相干时间 τ_dec，当 t >> τ_dec 时系统表现为经典。

### T381 推导路径
**退相干时间尺度分离定理**

对于宏观系统（N >> 1），退相干时间：
```
τ_dec ≈ ħ / (k_B T · √N)
```

对于 N ~ 10²³（宏观物体）：
```
τ_dec ~ 10⁻²³ s << τ_obs ~ 1 s
```

因此"经典行为"不是独立假设，而是：
- 量子力学（A1）
- 统计力学（大 N 极限）
- 信息整合动力学（GHPII）

的**必然推论**。

### 置信度评估
- 路径 A（尺度分离）：85% ✅
- 路径 B（量子达尔文主义）：60%
- 联合：85%

**降格类型：** 条件性（依赖 A1 + 统计力学）

## A13 降格：集体意识

### 原公理陈述
> A13: 多体系统的意识整合度 Φ_total 不等于个体 Φ 之和。

### T382 推导路径
**IIT 多体推广定理**

根据 IIT 3.0/4.0 的整合信息定义：
```
Φ_total = I(A:B) - I_decomposable
```

对于 N 个体系统，若存在跨个体信息整合（语言/文化/技术）：
```
Φ_collective > Σ Φ_individual
```

这直接从互信息的非加和性推出，不需要独立公理。

## A17 降格：文明信息整合

### 原公理陈述
> A17: 文明的信息整合能力存在上限 Ψ_max。

### T383 推导路径
**信息论上限定理**

根据 Bekenstein  bound 和 Bremermann 极限：
```
I_max ≤ 2πRE/(ħc)  (Bekenstein)
R_info ≤ mc²/(ħ·ln2)  (Bremermann)
```

对于行星级文明：
```
Ψ_max ≈ 10⁴² bits/s
```

对于银河级文明：
```
Ψ_max ≈ 10⁶⁷ bits/s
```

上限来自基础物理，不是独立假设。

### 置信度评估
- Bekenstein bound 推导：95%
- 文明尺度外推：70%
- 联合：80%

**降格类型：** 条件性（依赖广义相对论 + 信息论）
