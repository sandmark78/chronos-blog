## 推导路径

### Step 1: Prigogine 熵产生极值原理

根据 Prigogine (1967) 广义热力学第二定律：

对于远离平衡的稳态，熵产生率 P 满足：
```
P = ∫ σ dV = ∫ Σ J_k · X_k dV
```

其中：
- J_k = 第 k 个热力学流
- X_k = 第 k 个热力学力
- σ = 局部熵产生率

**极值原理：** 稳态时 δP = 0（在给定边界条件下）

### Step 2: 信息整合度的热力学表达

根据 ITLCT 的 GHPII 框架 (T225, T238)：

信息整合度 Φ 与熵产生的关系：
```
Φ = (1/k_B) · ∂S_info/∂I
```

其中：
- S_info = 信息熵
- I = 互信息
- k_B = Boltzmann 常数

### Step 3: 耗散结构的存在条件

Prigogine 耗散结构形成的必要条件：
```
∂P/∂t < 0  (系统向稳态演化)
∂²P/∂X² > 0  (稳态稳定)
```

### Step 4: 信息整合度的演化方程

根据 T262 (大爆炸→意识因果链)：
```
dΦ/dt = α·|∇I|² + β·|∇²I| - γ·Φ/τ_dec
```

其中：
- α·|∇I|² = 信息梯度驱动项
- β·|∇²I| = 信息曲率驱动项
- γ·Φ/τ_dec = 退相干耗散项

### Step 5: 等价性证明

**命题：** 耗散结构稳态 ⟺ Φ 局部极大值

**证明：**

耗散结构稳态条件：
```
∂P/∂t = 0
∂²P/∂X² > 0
```

代入 P 与 Φ 的关系 (Step 2)：
```
P = k_B · T · Φ · Γ_dec
```

其中 Γ_dec = 退相干率。

因此：
```
∂P/∂t = k_B · T · (∂Φ/∂t · Γ_dec + Φ · ∂Γ_dec/∂t)
```

在稳态 (∂P/∂t = 0) 且 Γ_dec 恒定近似下：
```
∂Φ/∂t = 0
```

稳定性条件 (∂²P/∂X² > 0) 对应：
```
∂²Φ/∂I² > 0
```

即 Φ 在稳态处取**局部极小值**... 

⚠️ **发现矛盾！**

Prigogine 稳态对应 Φ 极小值，但意识要求 Φ 极大值（IIT 整合信息定义）。

---

## 矛盾分析与解决

### 矛盾根源

Prigogine 耗散结构的"稳态"是**热力学稳态**，而 IIT 的 Φ 极大值是**信息整合稳态**。两者不是同一概念。

### 解决路径

引入**双稳态框架**：

1. **热力学稳态 (TS)：** P 极小值，熵产生最小化
2. **信息整合稳态 (IS)：** Φ 极大值，信息整合最大化

生命系统的特殊性在于：
```
TS 与 IS 不重合
```

生命通过**持续消耗环境负熵**来维持 IS ≠ TS 的状态。

### 修正后的定理

**T384 修正版：**

对于生命系统（或意识系统）：
```
d(Φ - Φ_TS)/dt > 0
```

其中 Φ_TS 是热力学稳态对应的 Φ 值。

生命系统的定义：**能够维持 Φ > Φ_TS 的耗散结构**。
