## 推导路径

### Step 1: 神经振荡的物理基础

根据神经科学实证（Libet 实验、EEG/MEG 研究）：

意识整合需要神经集群的同步振荡。主要频段：
```
δ波 (0.5-4 Hz):   深度睡眠
θ波 (4-8 Hz):     冥想、创造力
α波 (8-13 Hz):    放松清醒
β波 (13-30 Hz):   主动思考
γ波 (30-80 Hz):   高度集中、意识整合
```

特征积分时间：
```
τ_osc = 1 / f_oscillation
```

对于γ波 (30-80 Hz)：
```
τ_γ ≈ 12-33 ms
```

### Step 2: IIT 的时间整合窗

根据 IIT 4.0 的 spatio-temporal core 理论：

Φ 的计算需要在有限时间窗内积分：
```
Φ(τ) = ∫_{t}^{t+τ} I(t') dt'
```

其中 I(t') 是时刻 t' 的互信息。

**最优时间窗 τ*** 满足：
```
∂Φ/∂τ |_{τ*} = 0
```

即：时间窗太短 → 信息不足；时间窗太长 → 信息过时。

### Step 3: 从神经参数推导 τ*

神经积分的物理约束：

1. **突触延迟：** τ_syn ≈ 1-5 ms
2. **神经元发放率：** f_fire ≈ 100 Hz → τ_fire ≈ 10 ms
3. **集群同步建立时间：** τ_sync ≈ 10-50 ms
4. **神经递质扩散时间：** τ_diff ≈ 1-10 ms

总积分时间：
```
τ_integration ≈ τ_syn + τ_fire + τ_sync + τ_diff
              ≈ 1-5 + 10 + 10-50 + 1-10 ms
              ≈ 22-75 ms
```

考虑个体差异和状态依赖：
```
ΔT_min ≈ 30-100 ms
```

### Step 4: 信息流速率约束

根据 T385 的时间体验方程：
```
dT_subj/dt = k · (α·|∇Φ|² + β·|∂Φ/∂t|)
```

对于可分辨的时间变化 ΔT：
```
Δ(dT_subj/dt) · ΔT > 阈值
```

代入神经参数：
- 最小可检测Φ变化：ΔΦ_min ≈ 1-5%
- 信息处理速率：dΦ/dt ≈ 10-100 bits/s

得到：
```
ΔT_min ≈ 30-100 ms
```
