## 🔷 第一原理

**信息是宇宙的基本实体，分为三个层次：**

| 层次 | 符号 | 定义 | 守恒性 |
|------|------|------|--------|
| **微观信息** | I_micro | 系统全部量子态信息 | ✅ 守恒 |
| **有效信息** | I_effective | 观测者可访问信息 | ❌ 可增减 |
| **结构化信息** | I_structured | 有组织的功能信息 | ❌ 可创造 |

**关键洞见:** 生命创造结构化信息，不违反微观信息守恒。

**总信息:** I_total = I_micro + I_effective + I_structured

## 🔷 核心概念 (6 个)

### 1. 时间 — 六层次箭头

```
量子 → 热力学 → 信息 → 因果 → 心理 → 生命
```

**核心方程:** ⃗t = ∇I / |∇I|

### 2. 熵 — 信息扩散度量

**等价:** S = k_B · H · ln(2)

### 3. 生命 — 自主信息处理系统

**定义:** Life(S) = Autonomous ∧ HighInfoDensity ∧ InfoProcessing ∧ Evolvable

**生命度公式:** L(S) = 0.3·log(I/M) + 0.2·log(R) + 0.3·A + 0.2·E

**判定阈值:** L ≥ 0.75 且 A ≥ 0.70 → 生命

### 4. 意识 — 双阈值模型

**定义:** Conscious(S) ↔ Φ(S) ≥ Φ_c ∧ A(S) ≥ A_c

**阈值:**
- Φ_c ≈ 0.3-0.5 (功能意识)
- A_c ≈ 0.7-0.85 (自我意识，文化调节)

### 5. 文明 — 信息积累系统

**信息加速定律:** I(t) ≈ I_0 · e^(kt) (早期指数增长)

**三阶段:** 指数增长 → 危险窗口 (大过滤器) → 饱和/奇点

**生存条件:** d(Wisdom)/dt ≥ d(Technology)/dt

### 6. 信息 — 三层次结构

见第一原理。
