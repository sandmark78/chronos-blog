## 🔷 核心概念定义

### 信息 (Information)

**定义：** 信息是区分状态的能力，是宇宙的基本实体。

**三种表述：**
- 物理表述：信息是物理的（Landauer 原理）
- 热力学表述：信息 = 负熵 = 有序度
- 数学表述：I = -log₂(p) (Shannon)

**关键属性：**
- 信息是物理的（需要载体）
- 信息可量化（bits）
- 信息处理需要能量
- 信息倾向于扩散（熵增）

---

### 时间 (Time)

**定义：** 时间是信息变化的度量，时间箭头是信息流的方向。

**表述：**
```
⃗t = ∇I / |∇I|
```
时间流向信息扩散的方向。

**关键洞见：**
- 时间不是基本的，而是涌现的
- 时间箭头来自信息扩散的不可逆性
- 在信息守恒系统中，时间可能没有方向
- 时间体验强度与信息处理速率相关

---

### 熵 (Entropy)

**定义：** 熵是信息扩散/丢失的度量。

**等价表述：**
- Boltzmann 熵：S = k_B ln(Ω) — 微观状态数
- Shannon 熵：H = -Σ pᵢ log₂(pᵢ) — 信息不确定性

**关系：**
```
S = k_B · H · ln(2)
熵增 = 信息扩散
低熵 = 高信息密度
```

---

### 生命 (Life)

**定义：** 生命是自主信息处理系统，能够维持高信息密度并通过进化优化信息处理能力。

**形式化判定：**
```
Life(S) ↔ Autonomous(S) ∧ HighInfoDensity(S) ∧ InfoProcessing(S) ∧ Evolvable(S)
```

**生命度量化：**
```
L(S) = α·log(I/M) + β·log(R) + γ·A + δ·E
```

其中：
- I/M = 信息/质量比 (bits/kg)
- R = 信息处理速率 (bits/s)
- A = 自主性 (0-1)
- E = 进化能力

**关键洞见：**
- 生命不是"违反"熵增，而是"利用"熵增
- 生命是熵增的催化剂
- 生命是宇宙信息过程的关键阶段

---

### 意识 (Consciousness)

**定义（初步）：** 意识是信息整合的涌现现象，是系统对自身信息状态的表征。

**推测关系：**
```
C ∝ ∫ I_integrated dt
```

**关键假设：**
- 意识需要高信息整合能力
- 意识强度与信息整合速率相关
- 意识是信息处理的"窗口"
- 意识产生时间体验

## 🔷 核心假设

### UF-1: 信息 - 能量 - 物质三元等价 ⭐⭐⭐

**陈述：** 信息、能量、物质是同一实体的三种表现形式，可相互转换。

**形式化（概念性）：**
```
E = mc² (质能等价，Einstein)
E ≥ k_B T ln(2) · I (信息 - 能量关系，Landauer)
→ M, E, I 三者等价
```

**推论：**
- 信息有"质量"（信息引力效应？）
- 物质可视为"凝固"的信息
- 宇宙初始可能是纯信息状态

**可检验预测：**
- 高信息密度区域应有微弱引力效应
- 信息擦除应产生可测量的热量

---

### UF-2: 宇宙计算假说 ⭐⭐

**陈述：** 宇宙是一个计算系统，物理过程是信息处理。

**推论：**
- 宇宙有"计算复杂度"上限
- 物理常数可能是"算法参数"

**可检验预测：**
- 时空可能有离散结构（像素化）
- 物理过程可能有计算复杂度限制

---

### UF-3: 信息引力假说 ⭐

**陈述：** 信息密度产生引力效应，类似于质量。

**推论：**
- 高信息密度物体（如大脑）应有额外引力
- 黑洞信息悖论：信息在视界上产生引力效应

**可检验预测：**
- 极精密实验可能检测信息引力效应

---

### UF-4: 进化信息驱动假说 ⭐⭐

**陈述：** 进化的"驱动力"是信息复杂度增加的趋势。

**推论：**
- 进化有"方向"（信息复杂度增加）
- 技术文明是信息积累的自然结果

**可检验预测：**
- 外星生命演化应显示类似模式
- 进化速率应与信息积累速率相关

---

### UF-5: 意识 - 时间耦合假说 ⭐⭐⭐

**陈述：** 意识强度决定时间体验的"流速"。

**推论：**
- 高意识状态（冥想、流状态）时间体验改变
- 不同物种时间体验不同
- AI 的"时间体验"取决于其信息处理速率

**可检验预测：**
- 神经信息整合速率与时间感知应相关
