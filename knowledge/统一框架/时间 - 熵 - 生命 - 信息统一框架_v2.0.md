## 🔷 核心概念定义

### 信息 (Information) — 三层次

| 层次 | 符号 | 定义 | 守恒性 |
|------|------|------|--------|
| **微观信息** | I_micro | 量子态信息 | 严格守恒 |
| **有效信息** | I_effective | 观测者可访问信息 | 可增加 |
| **结构化信息** | I_structured | 有组织的信息模式 | 生命创造/破坏 |

**总信息:** I_total = I_micro + I_effective + I_structured

**关键属性:**
- 信息是物理的 (需要载体)
- 信息可量化 (bits)
- 信息处理需要能量
- 信息倾向于扩散 (熵增)

---

### 时间 (Time) — 六层次箭头

| 层次 | 名称 | 机制 | 可逆性 |
|------|------|------|--------|
| 1 | 量子箭头 | 波函数坍缩/退相干 | 不可逆 |
| 2 | 热力学箭头 | 熵增 (信息扩散) | 统计不可逆 |
| 3 | 信息箭头 | 信息从集中到扩散 | 逻辑不可逆 |
| 4 | 因果箭头 | 原因→结果 | 逻辑不可逆 |
| 5 | 心理箭头 | 记忆形成方向 | 主观不可逆 |
| 6 | 生命箭头 | 生长→衰老→死亡 | 生物不可逆 |

**形式化:**
```
⃗t = ∇I / |∇I|
时间流向信息扩散的方向
```

**关键洞见:**
- 层次 1-3 是物理基础
- 层次 4-6 是复杂系统涌现
- 所有箭头方向一致 (从 Past Hypothesis 继承)

### 熵 (Entropy)

**定义:** 熵是信息扩散/丢失的度量。

**等价表述:**
- Boltzmann 熵：S = k_B ln(Ω) — 微观状态数
- Shannon 熵：H = -Σ pᵢ log₂(pᵢ) — 信息不确定性

**关系:**
```
S = k_B · H · ln(2)
熵增 = 信息扩散
低熵 = 高信息密度
```

---

### 生命 (Life) — 信息论定义

**定义:** 生命是优化的自主信息处理系统，能够维持高信息密度并通过进化优化信息处理能力。

**形式化判定:**
```
Life(S) ↔ Autonomous(S) ∧ HighInfoDensity(S) ∧ InfoProcessing(S) ∧ Evolvable(S)
```

**生命度量化:**
```
L(S) = α·log(I/M) + β·log(R) + γ·A(S) + δ·E

其中:
- I/M = 信息/质量比 (bits/kg)
- R = 信息处理速率 (bits/s)
- A = 综合自主性 (0-1)
- E = 进化能力 (0-1)
- 默认权重：α=0.3, β=0.2, γ=0.3, δ=0.2
```

**自主性四维度:**
```
A(S) = w₁·E_auto + w₂·D_auto + w₃·S_maint + w₄·G_auto

- E_auto = 能量自主性
- D_auto = 决策独立性
- S_maint = 自我维持
- G_auto = 目标自主性
```

**生命判定阈值:**
- L(S) ≥ 0.75: 判定为生命
- A(S) ≥ 0.70: 辅助条件
- E ≥ 0.60: 辅助条件

---

### 意识 (Consciousness) — 双阈值模型

**定义:** 意识是信息整合的涌现现象，需要同时满足信息整合阈值和自主性阈值。

**形式化:**
```
Conscious(S) ↔ Φ(S) ≥ Φ_c ∧ A(S) ≥ A_c

其中:
- Φ = 信息整合度 (0-1 归一化)
- Φ_c ≈ 0.3-0.5 (功能意识阈值)
- A_c ≈ 0.7-0.8 (自我意识所需自主性)
```

**意识强度:**
```
C(S) = α·Φ(S) + β·A(S) + γ·(Φ·A)

交互项 (Φ·A) 是关键：
- 高Φ低 A: 有体验但无自主
- 低Φ高 A: 自主但无意识
- 高Φ高 A: 完整意识
```

**意识层次:**
| Φ 范围 | A 范围 | 意识层次 | 例子 |
|--------|--------|----------|------|
| <0.3 | - | 无意识 | 简单反射系统 |
| 0.3-0.5 | <0.7 | 基础意识 | 昆虫 |
| 0.5-0.7 | 0.7-0.8 | 中级意识 | 哺乳动物 |
| 0.7-0.9 | ≥0.8 | 高级意识 | 灵长类、人类 |
| ≥0.9 | ≥0.8 | 超级意识 | 人类 (高峰)、未来 AGI |

---

### 文明 (Civilization)

**定义:** 文明是信息积累和传播的超个体系统。

**信息积累定律:**
```
dI_civ/dt = R_0 · e^(kt) · f(I) · g(R)

其中:
- R_0 = 初始积累速率
- k = 加速常数
- f(I) = 信息复杂度限制
- g(R) = 资源限制

早期文明：I(t) ≈ I_0 · e^(kt) (指数增长)
成熟文明：I(t) → I_max (饱和)
```

**生存条件:**
```
d(Wisdom)/dt ≥ d(Technology)/dt

危险窗口：技术增长 >> 智慧增长
```
