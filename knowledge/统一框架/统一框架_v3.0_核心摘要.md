## 公理系统 (A1-A10)

```
A1: 信息本体论 — 信息是宇宙基本实体
A2: 信息三分 — I_total = I_micro + I_effective + I_structured
A3: 微观信息守恒 — d(I_micro)/dt = 0
A4: 有效信息扩散 — d(I_effective)/dt ≥ 0
A5: 信息梯度驱动 — 系统趋向耗散信息梯度
A6: 信息处理需能量 — E ≥ k_B T ln(2) · I
A7: 时间箭头 — ⃗t = ∇I / |∇I|
A8: 热力学选择 — Life ↔ (dS/dt)_life >> (dS/dt)_non-life
A9: 意识双阈值 — Conscious ↔ Φ ≥ Φ_c ∧ A ≥ A_c
A10: 自由意志兼容 — F = A × Φ × M
```

## 核心公式

```
1. 时间箭头：⃗t = ∇I / |∇I|

2. 生命度：L(S) = 0.3·log(I/M) + 0.2·log(R) + 0.3·A + 0.2·E

3. 意识：Conscious ↔ Φ ≥ Φ_c ∧ A ≥ A_c

4. 自主性：A(S) = 0.30·E_auto + 0.25·D_auto + 0.25·S_maint + 0.20·G_auto

5. 自由意志：F(S) = A(S) × Φ(S) × M(S)

6. 道德地位：M = w_Φ·Φ + w_A·A + w_L·L + w_H·H + w_P·P

7. 生命 - 意识 - 熵增耦合：dS/dt ∝ L(S) × Φ

8. 文明信息加速：I(t) ≈ I_0 · e^(kt) → I_max

9. 存在状态：State = (Θ(Φ-Φ_c), Θ(A-A_c), Θ(L-L_c))

10. 智慧 - 技术平衡：d(Wisdom)/dt ≥ d(Technology)/dt
```
