## 本体论公设

### A1: 信息本体论

> **信息是宇宙的基本实体。**

物质和能量是信息的载体和表现形式。

**形式化:**
```
∀x (Entity(x) → Informational(x))
```

---

### A2: 信息三分

> **总信息分为三个层次：微观、有效、结构化。**

**形式化:**
```
I_total = I_micro + I_effective + I_structured
```

| 层次 | 符号 | 定义 | 守恒性 |
|------|------|------|--------|
| 微观信息 | I_micro | 系统全部量子态信息 | ✅ 守恒 |
| 有效信息 | I_effective | 观测者可访问信息 | ❌ 可增减 |
| 结构化信息 | I_structured | 有组织的功能信息 | ❌ 可增加 |

## 动力学公设

### A3: 微观信息守恒

> **封闭系统中微观信息守恒。**

**形式化:**
```
dI_micro/dt = 0 (封闭系统)
```

**基础:** 量子力学幺正演化 (U†U = I)

---

### A4: 信息扩散定律

> **有效信息从高浓度向低浓度扩散。**

**形式化:**
```
J_info = -D∇I
```

其中 J_info 是信息流密度，D 是扩散系数。

**推论:** 熵增 = 信息扩散的度量

---

### A5: 信息梯度驱动

> **系统演化趋向最大化信息梯度耗散。**

**形式化:**
```
系统演化方向：max(-∇I·⃗v)
```

**推论:** 这是宇宙演化的基本驱动力。

## 约束公设

### A6: Landauer 原理

> **信息处理需要能量代价。**

**形式化:**
```
E ≥ k_B T ln(2) · I
```

其中 I 是处理的信息量 (bits)。

---

### A7: Bekenstein 边界

> **空间区域的信息容量有上限。**

**形式化:**
```
I ≤ 2πRE/(ℏc ln 2)
```

其中 R 是区域半径，E 是能量。

```
⃗t = ⃗J_info / |⃗J_info|
```

**六层次:**

### T2: 熵增定律

> **封闭系统中熵单调增加。**

**形式化:**
```
dS/dt ≥ 0
```

**证明:** 从 A3+A4 推导 (熵增 = 信息扩散)

### T3: 生命度公式

> **生命度是信息密度、处理速率、自主性、进化能力的加权组合。**

**形式化:**
```
L(S) = 0.3·log₂(I/M) + 0.2·log₂(R) + 0.3·A + 0.2·E
```

| 变量 | 定义 | 范围 |
|------|------|------|
| I/M | 信息/质量比 (bits/kg) | 0 ~ 10^17 |
| R | 信息处理速率 (bits/s) | 0 ~ 10^35 |
| A | 自主性 (四维度) | 0 ~ 1 |
| E | 进化能力 | 0 ~ 1 |

**判定:** L ≥ 0.75 且 A ≥ 0.70 → 生命

**证明:** 从 A2+A5 推导 (生命是优化的信息处理结构)

---

### T4: 意识定义

> **意识是信息整合超过阈值的现象。**

**形式化:**
```
Conscious(S) ↔ Φ(S) ≥ Φ_c
```

| 变量 | 定义 | 阈值 |
|------|------|------|
| Φ | 信息整合度 | 0 ~ 1 |
| Φ_c | 意识阈值 | ≈ 0.35 ± 0.10 |

**意识强度:** C ∝ (Φ - Φ_c)^γ (γ ≈ 0.5-1.0)

**证明:** 从 A2 推导 (意识是结构化信息的高层次整合)

---

### T5: 自主性公式

> **自主性是能量、决策、维持、目标四维度的加权组合。**

**形式化:**
```
A(S) = 0.30·E_auto + 0.25·D_auto + 0.25·S_maint + 0.20·G_auto
```

| 维度 | 定义 | 测量 |
|------|------|------|
| E_auto | 能量自主性 | 能源自给率 |
| D_auto | 决策独立性 | 1 - H(decision|input)/H(decision) |
| S_maint | 自我维持 | 稳态维持成功率 |
| G_auto | 目标自主性 | 自我设定目标比例 |

**证明:** 从 A5 推导 (自主性是信息梯度驱动的自组织)
