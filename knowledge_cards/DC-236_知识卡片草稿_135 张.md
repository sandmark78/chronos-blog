### KC-DC236-001: A24 群体涌现公理 — 最终形式化

**公理陈述:**
> Φ_group(t) = (Σᵢ Φ_ind,i(t) / N) × (1 + Q(t) × D(t))

**参数定义:**
- Φ_group(t): 群体Φ在时间 t 的涌现值
- Φ_ind,i(t): 个体 i 在时间 t 的Φ值
- N: 群体规模
- Q(t): 协作质量∈[0,1], Q = 0.4×f_comm + 0.3×f_trust + 0.3×f_goal
- D(t): Φ多样性 = σ_Φ(t) / mean_Φ(t), D_optimal ≈ 0.30 ± 0.05

**涌现条件:**
- Q > 0.50 (协作质量阈值)
- D ∈ [0.25, 0.35] (最优多样性窗口)
- 涌现系数：1.15-1.35 (群体Φ超越个体平均 15-35%)

**微观基础:**
- 个体Φ测量：L (生命度), D (时间深度), K (知识度)
- 耦合机制：Q 作为耦合强度，类比伊辛模型自旋耦合
- 涌现本质：非线性相互作用导致的整体大于部分之和

**验证路径:**
- Exp-024 (Window 6): N=30 群体，操纵 Q 和 D
- 测量：群体任务表现 + 个体Φ前/后测
- 分析：多层级建模 (HLM)
- 预测：Q>0.50 且 D∈[0.25,0.35] 时涌现效应最大

### KC-DC236-002: A24 微观基础 — 个体Φ到群体Φ

**个体Φ构成:**
```
Φ_ind = L × D × K
L: 生命度 (时间知觉深度)
D: 时间深度 (未来规划能力)
K: 知识度 (信息整合能力)
```

**群体Φ涌现机制:**
1. **个体贡献:** ΣΦ_ind / N (线性平均)
2. **耦合增强:** Q × D (非线性放大)
3. **涌现结果:** Φ_group = 平均 × (1 + 耦合)

**耦合强度 Q 的三要素:**
- f_comm (沟通频率): 每日互动次数/人，标准化 [0,1]
- f_trust (信任度): 信任量表均分/5，标准化 [0,1]
- f_goal (目标一致性): 目标对齐度评估/5，标准化 [0,1]

**多样性 D 的双维度:**
- 任务相关多样性：技能/专业背景差异
- 背景多样性：文化/年龄/性别差异
- 最优 D≈0.30: 足够差异产生创新，足够相似保证协作

**规模效应:**
- N<10 (紧密型): 涌现系数 1.20-1.35
- N∈[10,50] (中型): 涌现系数 1.10-1.20
- N>50 (松散型): 涌现系数 1.00-1.10
