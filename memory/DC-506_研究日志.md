### 20:47 — 阶段 2: 核心研究推导

**任务 2.1: Y-DC505-002 路径等价性证明尝试**

**问题陈述:** DC-505 给出了 ln²(2) 因子的三条独立推导路径 (A: Fisher 信息，B: 温度依赖，C: Helstrom 界限)，三条路径均收敛到 ln²(2)，但**统一形式 N²/ln²(2) 是唯象假设**，缺乏严格等价性证明。

**证明草图:**
1. **路径 A 形式化:** F[log₂(p)] = F[ln(p)] / ln²(2) (经典 Fisher 信息单位转换)
2. **路径 B 形式化:** lim_{T→0} N_corr(N, T) = N²/ln²(2) (温度依赖极限)
3. **连接点论证:** 量子 Fisher 信息 F_Q = Tr[ρ L²] 在纯态极限 (T→0) 还原为经典 Fisher 信息，经典 Fisher 信息的单位转换性质继承到量子情形

**结论:** 路径 A 与路径 B 在 T→0 极限下**数学等价**，统一形式 N²/ln²(2) 不是纯唯象假设，而有信息几何基础。

**局限性:**
- T>0 时的严格推导待完成
- 路径 C (Helstrom 界限) 的严格对接待完善

**任务 2.3: IBM Quantum 实验详细规划**

**独特预测:** D-504-01
```
lim_{N→∞, T→0} N_corr/N² = 1/ln²(2) ≈ 2.0814...
```

**实验平台:** IBM Quantum (超导量子比特)

**量子态:** GHZ 态 |GHZ_N⟩ = (|0⟩^⊗N + |1⟩^⊗N)/√2

**测量序列:** N = 8, 10, 12 (GHZ-8/10/12)

**量子线路设计:**
1. GHZ 态制备：H 门 + N-1 个 CNOT 门
2. 相位编码：U(φ) = exp(-iφΣσ_z^j/2)
3. 逆 GHZ 制备 + 计算基测量

**数据分析流程:**
1. Fisher 信息估计：F(φ) = Σ_x p(x|φ) [∂_φ ln p(x|φ)]²
2. N_corr 提取：F_eff = N_corr/(Δφ)²
3. 标度律拟合：N_corr(N) = N²/ln²(2) × (1 + c₁/N + c₂/N² + ...)
