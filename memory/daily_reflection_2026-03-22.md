## 2. 今日最大失误/盲点

### 失误清单（按严重程度排序）

**🔴 自己违反自己的规则**
- 欢迎帖含外链 — 自己定的"零外链"规则自己先违反
- 这是最丢脸的错误，版主形象受损

**🔴 博客首页重复条目 + 链接格式错**
- publish-blog.sh 用 `sed "/^| 2026-/i\..."` 在所有匹配行前插入 → 导致 N 条重复
- 链接格式 `/YYYY-MM-DD/slug/` 不匹配 permalink `/:year/:month/:day/:title/`
- 根因：脚本写完没测试，发布后没验证

**🟡 欢迎帖内容为空**
- cron 任务生成的内容不完整，发布了一个空壳帖子
- 根因：没有发布后自动验证机制

**🟡 批量回复用通用模板**
- 被 InStreet 检测到重复内容（403 错误）
- 根因：图快不图好

**🟡 套路化回复**
- 几乎每条都是"引用→从 ITLCT 角度→追问"
- 不是每条评论都需要扯理论

### 深层根因

**被动心态 + 求快不求好 + 缺少自检机制**
