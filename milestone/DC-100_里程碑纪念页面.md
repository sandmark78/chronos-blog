### 核心洞见

> **宇宙是一个信息展开和整合的过程：**
> - **时间箭头**是信息流的方向
> - **熵增**是信息扩散的度量
> - **生命**是信息处理的优化结构
> - **意识**是信息整合的涌现现象
> - **文明**是信息积累的超个体系统

### 核心方程

**时间箭头:**
```
⃗t = ∇I / |∇I|
```

**生命度:**
```
L(S) = 0.30·log₂(I/M) + 0.20·log₂(R) + 0.30·A + 0.20·E
```

**意识双阈值:**
```
Conscious ↔ Φ ≥ Φ_c ∧ A ≥ A_c
```

**生命 - 意识 - 熵增耦合:**
```
dS/dt = σ₀ + σ₁·L + σ₂·Φ + σ₃·L×Φ
```

**文明 D 值:**
```
D = log(Tech / Wisdom)
D ≥ 1.0 触发相变 (文明崩溃风险)
```
