## 3. N_corr 标度律推导 (Derivation)

### 3.1 问题设定
- 系统：N 粒子 GHZ 态
- 条件：τ_gate ≪ τ_dec, T ≪ T_crit
- 目标：推导 N_corr(N) 函数形式

### 3.2 步骤 1: 信息几何设定
- Fisher 信息度规 g_ij
- 参数空间：θ = (θ₁, ..., θ_N)
- GHZ 态的对称性：置换不变

### 3.3 步骤 2: 双线性形式
```
Φ_GHPII = Σᵢⱼ gⁱʲ ∂ᵢΦ ∂ⱼΦ
```
- 对称性约化：gⁱʲ = g(N) δⁱʲ + h(N) (1-δⁱʲ)
- 领头阶：g(N) ~ N, h(N) ~ N²

### 3.4 步骤 3: N_corr-Φ关系 (唯象假设)
```
N_corr = Φ_GHPII / (k₀ × N)
```
- 动机：IIT 的整合信息归一化
- 信息几何解释：有效参数数

### 3.5 步骤 4: 1/ln²(2) 系数推导
```
N_corr = N²/ln²(2) · [1 + 𝒪(1/N)]
```
- 双线性形式代入
- 对称性论证
- 领头阶系数：1/ln²(2) ≈ 2.08

### 3.6 步骤 5: 适用范围
- N ≥ 8 (小 N 修正项显著)
- GHZ 态 (最大纠缠)
- τ_gate ≪ τ_dec (低噪声)
- T ≪ T_crit (低温极限)

### 3.7 极限行为检验
- N→∞: N_corr ~ N²/ln²(2) ✅
- N→1: 不适用 (N≥8) ⚠️
- T→0: N_corr → N²/ln²(2) ✅

## 新增章节 (v0.2): 温度依赖完整形式化

### 3.8 温度依赖推导 (新增)

#### 问题陈述
标准 QFI 框架中，量子 Fisher 信息本身无显式温度依赖 (假设纯态)。但在实际实验中：
1. 热激发会导致退相干 (γ随 T 增加)
2. N_c = ln(2)/(γτ_gate) 随 T 减小
3. 最终导致 N_corr 随 T 衰减

ITLCT 通过 T423/T429 给出完整温度依赖形式。

#### 推导步骤

**步骤 1: k₀(T) 的温度依赖 (从 T423)**
```
k₀(T) = k₀(0) × exp[-(T/T_crit)^δ]
```
其中:
- T_crit = ℏω₀/(2k_B) (物理温度定义)
- δ ≈ 2 (唯象拟合 + 平均场近似，见 T436)
- k₀(0) = 0.0300±0.0005 bits

**步骤 2: N_c(T) 的温度依赖**
从 N_c = ln(2)/(γτ_gate) 且γ(T) = γ₀ × exp[(T/T_crit)^δ]:
```
N_c(T) = N_c(0) × exp[-(T/T_crit)^δ]
```

**步骤 3: Φ(T) 的完整形式 (从 T429)**
```
Φ(T) = k₀(T) × (N-1) / [1 + (N-1)/N_c(T)]
```

**步骤 4: N_corr(T) 推导**
从定义 N_corr = Φ/(k₀ × N):
```
N_corr(T) = [k₀(T) × (N-1) / (1 + (N-1)/N_c(T))] / [k₀(T) × N]
          = (N-1)/N × 1/[1 + (N-1)/N_c(T)]
```

在 N ≫ 1 极限下:
```
N_corr(T) ≈ 1/[1 + N/N_c(T)] = N_c(T) / [N + N_c(T)]
```

**步骤 5: 与 N²标度律的统一**
在 N ≪ N_c(T) 条件下 (低温或中等 N):
```
N_corr(T) ≈ N/N_c(T) × N_c(T)/ln²(2) = N²/ln²(2) × exp[-(T/T_crit)^δ]
```

**最终形式:**
```
N_corr(N, T) = N²/ln²(2) × exp[-(T/T_crit)^δ] × [1 + 𝒪(1/N)]
```

#### 适用范围
| 温区 | 条件 | 公式有效性 | 备注 |
|------|------|------------|------|
| 低温 | T < 0.5 T_crit | 严格有效 | 指数衰减主导 |
| 中温 | 0.5 T_crit < T < T_crit | 外推 | 唯象模型，误差~10% |
| 高温 | T > T_crit | 失效 | 需高温唯象模型 (待研究) |

#### 实验可检验预测

**平台依赖性:**
| 平台 | T_crit | 室温衰减因子 exp[-(300K/T_crit)²] | 预测行为 |
|------|--------|-----------------------------------|----------|
| 超导 | ~0.1K | ~0 (10^{-1300000}) | N_corr → 0 |
| 离子阱 | ~1K | ~0 (10^{-90000}) | N_corr → 0 |
| 中性原子 | ~0.01K | ~0 (极端小) | N_corr → 0 |
| 光子 | N/A | 1 (无温度依赖) | N_corr 常数 |

**独特预测 (区别于标准 QFI):**
1. 物质平台在室温下 N_corr 应指数衰减至 0
2. 光子平台无温度依赖 (室温工作)
3. 变温实验中，N_corr(T) 应遵循 exp[-(T/T_crit)²] 形式

**验证方案:**
- 低温实验 (T < T_crit): 验证 N²标度律
