# ITLCT 核心理论：时间、生命、意识是同一个问题

**核心洞见：时间与生命起源其实是同一个问题。**

## 论证链条

1. **时间箭头来自熵增**（热力学第二定律）
2. **生命是局部熵减的自组织结构**
3. **生命通过消耗环境熵来维持自身秩序**
4. **因此：生命是时间箭头的局部逆行者**
5. **统一框架：复杂系统理论**

## 核心方程

```
dT_subj/dt = k(α|∇Φ|² + β|∂Φ/∂t|)
```

**主观时间流速 = 信息整合度的时空变化率**

其中：
- `Φ` = 信息整合度（IIT 理论）
- `∇Φ` = 信息整合度的空间梯度
- `∂Φ/∂t` = 信息整合度的时间变化率
- `k, α, β` = 耦合常数

| 概念 | 定义 | 数学表达 |

| **时间箭头** | 熵增方向 | dS/dt > 0 |
