### 推导 2.1: MinVP-1 实际数值执行 (N=64 DMRG)

**目标:** 使用 TeNPy 库执行 MinVP-1 Core 的 DMRG 数值计算，验证 GHPII 预言

**理论预期 (from DC-311 T391 + DC-312 T393):**
```
R(64,32)_theory = S_EE(χ=64) / S_EE(χ=32) = 1.02 ± 0.02
```

**数值执行方案:**
1. **平台:** Python + TeNPy (Tensor Network Python)
2. **系统:** 1D 横场 Ising 链 (GHPII 最小模型)
3. **参数:**
   - N = 64 (格点数)
   - χ = 32, 64, 128 (键维度，用于收敛性检查)
   - 边界条件: 开放边界 (OBC)
   - 哈密顿量: H = -Σᵢ σᵢˣσᵢ₊₁ˣ - hΣᵢ σᵢᶻ
4. **测量:** 冯·诺依曼纠缠熵 S_EE = -Tr(ρ_A log ρ_A)
5. **计算量:** ~2-4 小时 (单 GPU)

**执行脚本框架:**
```python
import tenpy
from tenpy.models.spins import SpinChain
from tenpy.algorithms import dmrg

# 模型参数
L = 64  # 格点数
chi_list = [32, 64, 128]  # 键维度

# 初始化模型
model_params = {
    'L': L,
    'Jxx': 1.0,
    'Jz': 1.0,
    'hz': 1.0,  # 临界点
    'bc_MPS': 'finite'
}
model = SpinChain(model_params)

# DMRG 优化
psi = MPS.from_product_state(model.lat.mps_sites(), [0]*L)
dmrg_params = {
    'chi_max': max(chi_list),
    'trunc_params': {'svd_min': 1e-10},
    'max_E_err': 1e-8,
    'N_sweeps_check': 4
}

# 执行 DMRG
for chi in chi_list:
    dmrg_params['chi_max'] = chi
    engine = dmrg.TwoSiteDMRGEngine(psi, model, dmrg_params)
    E, psi = engine.run()
    
    # 计算纠缠熵 (中心键)
    S_EE = psi.entropy_chi(L//2)
    print(f"χ={chi}: S_EE = {S_EE:.6f}")
    
    # 计算比率 R
    if chi == 32:
        S_32 = S_EE
    elif chi == 64:
        S_64 = S_EE
        R_64_32 = S_64 / S_32
        print(f"R(64,32) = {R_64_32:.6f}")
```

**预期结果分析:**
- **R ∈ [1.00, 1.04]:** GHPII 验证通过 ✅
- **R < 1.00 或 R > 1.04:** 需修正 GHPII ⚠️
- **收敛性:** |R(χ) - R(χ/2)| / R(χ) < 0.01 → 收敛 ✅
