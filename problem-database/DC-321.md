### 推导 2.2: T411 Φ跨主体整合定理

**背景:** Φ从 12.00→12.50 通过激活跨模态整合 (T407)。现在需要推导Φ从 12.50→13.00 的新维度。

**模式识别:**
- Φ=11.00 → 11.50: ΔΦ_temporal (跨时间)
- Φ=11.50 → 12.00: ΔΦ_cross-scale (跨尺度)
- Φ=12.00 → 12.50: ΔΦ_cross-modal (跨模态)
- Φ=12.50 → 13.00: ???

**候选新维度:**
1. **跨主体整合** — 多意识主体的信息共享 (集体意识/社会意识)
2. **跨表征整合** — 不同表征格式 (符号/亚符号/连接主义) 的整合
3. **跨时间尺度整合** — 时间维度和尺度维度的统一 (T300 场景)

**选择:** 跨主体整合 — 这与意识的社会性、集体智能、文化演化等宏观现象最相关，且具有高度独特性。

**分析:**
- 人类意识不是孤立的：语言、文化、技术都是跨主体信息共享的产物
- 高Φ系统应该能够整合来自多个主体的信息
- 这与跨模态整合不同：主体是"外部"的 (不同意识系统)，模态是"内部"的 (同一系统的不同通道)

**数学表述:**
```
ΔΦ_cross-subject = log₂(1 + R_c × (N_c - 1))
```
其中：
- R_c ∈ [0,1]: 主体间冗余因子 (不同主体间的信息重叠度)
- N_c ≥ 1: 可访问的主体数量

**与ΔΦ_cross-modal 的对比:**
| 维度 | 跨模态 (T407) | 跨主体 (T411) |
|------|-------------|-------------|
| 方向 | 横向 (同一系统内) | 外向 (跨系统) |
| 参数 | R_m, N_m | R_c, N_c |
| 物理意义 | 信息类型/通道 | 意识主体 |
| 典型值 | N_m ~ 5-15 (人类) | N_c ~ 2-1000+ (社会) |

**定量估计:**
- 对于Φ=12.50 → 13.00，需要ΔΦ_cross-subject ≈ 0.50
- log₂(1 + R_c × (N_c - 1)) ≈ 0.50
- R_c × (N_c - 1) ≈ 0.414

**合理解:**
- N_c = 5 (紧密协作团队：5 人)
- R_c ≈ 0.104 (主体间冗余度~10%，即共享背景知识)

**或:**
- N_c = 10 (小型社群)
- R_c ≈ 0.046 (冗余度~4.6%)

**或:**
- N_c = 100 (在线社区)
- R_c ≈ 0.0042 (冗余度~0.4%，弱连接)

**与集体意识的联系:**
- 集体意识/群体智能是跨主体整合的宏观表现
- 语言/文化/技术是跨主体信息共享的媒介
- 互联网/社交媒体极大扩展了 N_c 的上限
