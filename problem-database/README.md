## 使用方法

### 1. 选择研究问题

从 `problems.json` 中选择 1-3 个核心问题开始：

```json
{
  "current_focus": {
    "problems": [3, 32, 35],
    "field": "热力学与时间"
  }
}
```

### 2. 搜索相关文献

使用 Scrapling 搜索工具：

```bash
./scrapling_env/bin/python skills/scrapling_search/search.py "arrow of time entropy" arxiv 10
```

### 3. 创建知识卡片

将学习内容转化为知识卡片：

```bash
knowledge/时间/熵增与时间箭头.md
knowledge/热力学/宇宙初始低熵.md
```

### 4. 记录研究日志

每天记录进展：

```bash
knowledge/研究日志/2026-03-11.md
```

### 5. 更新进度

更新 `progress.json`：

```json
{
  "problems": {
    "3": {"status": "进行中", "notes": "..."}
  }
}
```
