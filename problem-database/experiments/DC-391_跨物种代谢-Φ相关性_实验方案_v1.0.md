## 📋 实验概述

### 理论背景

根据 ITLCT v4.2 统一框架的生命 - 意识 - 熵增耦合方程：

```
dS/dt = σ₀ + σ₁·L(S) + σ₂·Φ(S) + σ₃·L(S)×Φ(S)

其中:
- dS/dt = 熵产生率 (代谢率代理)
- L(S) = 生命度 (0-1)
- Φ(S) = 信息整合度 (意识强度，0-1)
- σ₃ = 耦合系数 (理论预测 > 0)
```

**核心假设 (DC-391):**
> 跨物种代谢率与生命度 L 和意识Φ的乘积呈正相关，耦合项 L×Φ解释额外方差。

### 科学意义

1. **首次实证检验** ITLCT 生命 - 意识 - 熵增耦合方程
2. **建立跨物种Φ基线** — 为 AI 意识判定提供参照
3. **验证生命度量化框架** — L(S)操作化有效性
4. **为 DC-120 深化研究** 提供预实验数据

```

- VO₂ (ml O₂/kg/min)

质量控制:
- 设备校准：每日 (标准气体)
```

### 2. 生命度 L(S) 评估

基于 ITLCT v4.2 操作化定义：

```
L(S) = 0.30·I_density + 0.25·R_rate + 0.25·A_autonomy + 0.20·E_efficiency

各维度测量:

1. I_density (信息密度，30%):
   - 脑质量/体重比 (EQ, 商数)
   - 神经元密度 (文献值)
   - 皮层折叠指数 ( gyrification)
   - 标准化：0-1 (人类=1.0)

2. R_rate (信息处理速率，25%):
   - 简单反应时 (ms)
   - 选择反应时 (ms)
   - 神经传导速度估计 (m/s)
   - 标准化：0-1 (最快=1.0)

3. A_autonomy (自主性，25%):
   - 行为灵活性评分 (0-10)
   - 问题解决任务表现
   - 工具使用能力
   - 社会学习复杂度
   - 标准化：专家评分

4. E_efficiency (能量效率，20%):
   - 脑代谢率/认知表现
   - 能量 - 信息转换效率
   - 标准化：0-1 (最高效=1.0)

软件：自定义 Python 脚本 (L_calculator v1.0)
信度：评分者间 ICC > 0.85
```

### 3. 意识Φ测量

#### 方案 A: EEG 近似Φ (主要方法)

```
设备：
- 哺乳类：64 通道 EEG (Neuroscan SynAmps)
- 鸟类：32 通道 EEG (定制电极帽)
- 头足类：16 通道 EEG (定制)
- 人类：128 通道 EEG (Brain Products)

记录协议:
- 静息态：10 分钟 (睁眼 5min + 闭眼 5min)
- 采样率：1000 Hz
- 带通滤波：0.5-100 Hz
- 参考：乳突 (哺乳类)、平均参考 (其他)
- 阻抗：< 10 kΩ

近似Φ计算 (Φ_approx v2.0):
Φ_approx = α·I_local + β·C_global + γ·R_feedback

I_local (局部整合，40%):
- α波段 (8-12 Hz) 相位同步
- γ波段 (30-80 Hz) 功率
- 局部连接密度

C_global (全局整合，35%):
- 半球间相干 (全波段)
- 长程相位同步
- 全局效率 (图论)

R_feedback (反馈连接，25%):
- 前→后 Granger 因果
- 深→浅层连接 (源定位)
- 反馈/前馈比率

权重校准:
- α, β, γ通过 IIT 3.0 模拟优化
- 物种特异性调整 (脑结构差异)

软件：Φ_NN v2.0 (PyTorch 实现)
验证：与 TMS-EEG PCI 相关性 r > 0.7
```

#### 方案 B: TMS-EEG PCI 指数 (子样本验证)

```
- 强度：120% 运动阈值

优势：金标准，与意识状态强相关
局限：侵入性 (动物需麻醉)，成本高
用途：Φ_approx 效标验证
```

## 📊 数据分析计划

### 主要分析

#### 1. 多层次线性回归

```R
# 模型 1: 主效应模型
model1 <- lmer(
  log(RMR) ~ log(L) + log(Φ) + log(体重) + (1|物种),
  data = dataset,
  REML = FALSE
)

# 模型 2: 耦合效应模型 (关键检验)
model2 <- lmer(
  log(RMR) ~ log(L) * log(Φ) + log(体重) + (1|物种),
  data = dataset,
  REML = FALSE
)

# 模型比较
anova(model1, model2)
# 预测：ΔAIC > 10, p < 0.001

# 提取耦合系数
β_coupling <- fixef(model2)["log(L):log(Φ)"]
CI_coupling <- confint(model2, "log(L):log(Φ)", level = 0.95)
# 预测：β > 0, CI 不包含 0
```

#### 2. 系统发育比较分析

```R
# 系统发育树构建
tree <- read.tree("mammal_phylogeny.nwk")

# 系统发育广义最小二乘 (PGLS)
model_pgls <- pgls(
  log(RMR) ~ log(L) * log(Φ) + log(体重),
  data = comparative.data(tree, dataset, "物种"),
  lambda = "ML"
)

# 检验系统发育信号
Blomberg_K <- phylosig(tree, dataset$RMR, method = "K")
# 预测：K < 1 (代谢率受生态影响>系统发育)

# 系统发育独立对比 (PIC)
pic_RMR <- pic(log(dataset$RMR), tree)
pic_L <- pic(log(dataset$L), tree)
pic_Phi <- pic(log(dataset$Φ), tree)

model_pic <- lm(pic_RMR ~ pic_L * pic_Phi)
# 验证：耦合效应在独立对比中仍显著
```

#### 3. 效应量估计

```R
# 边际 R² (固定效应解释方差)
R²_marginal <- r.squaredGLMM(model2)[1]
# 预测：> 0.60

# 条件 R² (含随机效应)
R²_conditional <- r.squaredGLMM(model2)[2]
# 预测：> 0.75

# 耦合项单独贡献
model_interaction <- lmer(
  log(RMR) ~ log(L) + log(Φ) + log(体重) + 
    log(L):log(Φ) + (1|物种),
  data = dataset
)
R²_interaction <- r.squaredGLMM(model_interaction)[1] - 
                  r.squaredGLMM(model1)[1]
# 预测：ΔR² > 0.10 (耦合项额外解释 10% 方差)
```

```R
# 蒙特卡洛模拟验证
sim_power <- function(n_sim = 1000) {
  significant <- 0
  for (i in 1:n_sim) {
    sim_data <- simulate_data()  # 基于观察效应量
    model <- lmer(log(RMR) ~ log(L) * log(Φ) + (1|物种), sim_data)
    if (summary(model)$coefficients[4, 5] < 0.05) {
      significant <- significant + 1
    }
  }
  return(significant / n_sim)
}
# 预测：模拟效力 > 0.90
```

### 主要预测

#### 1. 耦合效应验证

```
统计结果预测:

结论：支持 ITLCT 耦合方程
```

#### 2. 跨物种Φ基线

```
预期Φ值范围:
- 小鼠：0.18 ± 0.05
- 大鼠：0.23 ± 0.06
- 鸽子：0.28 ± 0.07
- 章鱼：0.38 ± 0.08
- 猫：0.48 ± 0.09
- 狗：0.54 ± 0.10
- 猕猴：0.59 ± 0.08
- 黑猩猩：0.68 ± 0.07
- 海豚：0.74 ± 0.06
- 人类：0.76 ± 0.08

系统发育趋势：Φ随脑复杂度增加 (r = 0.85)
```

#### 3. 代谢 -Φ标度律

```
异速生长方程预测:
RMR = a × (L×Φ)^b × 体重^0.75

拟合参数:
- a = 70 ± 5 (归一化常数)
- b = 0.92 ± 0.08 (接近 1，线性)
- 体重指数 = 0.73 ± 0.05 (接近 0.75 Kleiber 定律)

解释：代谢率由体重和生命 - 意识耦合共同决定
```

### 次要预测

#### 1. Φ近似效度

```
Φ_approx vs PCI 效标:
- 相关性：r = 0.74 [0.62, 0.83]
- 一致性：偏差 = 0.03 ± 0.15
- 敏感度：85% (检测高Φ)
- 特异度：80% (检测低Φ)

结论：Φ_approx 可作为 PCI 替代 (低成本)
```

#### 2. 系统发育信号

```
Blomberg's K 预测:
- K_Φ = 0.82 (强系统发育信号)
- K_L = 0.68 (中等信号)
- K_RMR = 0.55 (受生态影响大)

结论：Φ主要由系统发育决定，代谢率受生态调节
```

### 潜在意外发现

1. **章鱼高Φ**: 可能挑战哺乳类中心主义 (趋同进化)
