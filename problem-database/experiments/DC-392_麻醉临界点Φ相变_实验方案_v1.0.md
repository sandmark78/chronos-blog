## 📋 实验概述

### 理论背景

根据 ITLCT v4.2 统一框架的意识双阈值理论：

```
意识状态转换 = 相变过程

临界现象预测:
1. 临界慢化 (Critical Slowing Down)
   - 自相关 AC(1) 升高
   - 恢复率降低
   
2. 临界波动增加
   - 方差增大
   - 偏度变化

3. Φ值不连续跳跃
   - Φ在阈值处突变
   - 滞后效应 (hysteresis)

数学形式化:
Φ(t) = Φ₀ + ΔΦ·Θ(T - T_c) + ξ(t)

其中:
- T = 麻醉深度 (效应室浓度)
- T_c = 临界阈值
- Θ = 阶跃函数
- ξ = 临界涨落
```

**核心假设 (DC-392):**
> 麻醉诱导和苏醒过程中，意识恢复表现出临界相变特征：自相关升高、方差增大、Φ值不连续跳跃。

### 科学意义

1. **首次实证检验** ITLCT 意识相变预测
2. **验证临界现象** — 为意识阈值提供动力学证据
3. **临床转化潜力** — 改进麻醉深度监测
4. **理论约束** — 确定Φ_c 临界值范围

### 主要目标
1. 检测麻醉苏醒过程中自相关 AC(1) 升高 (临界慢化)
2. 测量Φ值在意识恢复时的不连续跳跃
3. 估计临界阈值 T_c 及其置信区间

### 次要目标

4. 验证Φ_c 与行为意识阈值的一致性

## 🔬 实验设计

### 研究类型
**前瞻性、观察性、被试内设计**

### 被试选择

#### 纳入标准
- 年龄 18-65 岁
- ASA 分级 I-II (健康或轻度系统疾病)
- 择期手术 (非紧急)
- 书面知情同意

#### 排除标准
- 神经系统疾病史
- 精神疾病史
- 药物/酒精滥用
- 怀孕/哺乳
- 麻醉过敏史
- EEG 异常 (预筛查)

#### 样本量计算

```
检测临界慢化效应 (中等 - 大效应):
- 预期 AC(1) 变化：d = 0.6 (Cohen's d)
- α = 0.05 (双侧)
- 效力 = 0.90
- 重复测量：5 时间点

使用 G*Power 3.1:
- 检验：重复测量 ANOVA (被试内)
- 需要：N = 42

考虑 15% 脱落率:
- 招募：N = 50
```

**最终样本:** N = 50 健康成人

### 麻醉方案

#### 麻醉剂选择

**组 1: 丙泊酚 (Propofol)**
- N = 25
- 理由：最常用，机制清楚 (GABA_A 激动)
- 目标：建立基准临界特征

**组 2: 七氟醚 (Sevoflurane)**
- N = 25
- 理由：吸入麻醉，不同机制
- 目标：检验麻醉剂通用性

#### 麻醉深度监测

```
标准监测:
1. BIS (Bispectral Index)
   - 范围：0 (无意识) - 100 (完全清醒)
   - 意识阈值：BIS ≈ 60-70
   - 采样率：1 Hz

2. MAC (Minimum Alveolar Concentration)
   - 仅吸入组
   - 目标：0.5-1.5 MAC

3. 效应室浓度 (Ce)
   - 药代动力学模型 (Schnider 模型)
   - 丙泊酚：Ce 目标 0.5-6 μg/mL
```

#### 麻醉流程

```
诱导期 (Induction):
1. 基线记录 (5 分钟，清醒)
2. 丙泊酚靶控输注 (TCI)
   - 起始：1 μg/mL
   - 阶梯增加：每 3 分钟 +0.5 μg/mL
   - 目标：无意识 (BIS < 60)
   - 平台：维持 5 分钟

3. 气管插管 (临床需要)
4. 手术期 (不记录，临床优先)

苏醒期 (Emergence) - 关键记录期:
1. 停止麻醉输注
2. 自然苏醒
3. 连续记录 EEG + 行为反应
4. 时间点：
   - T0: 停药即刻
   - T1: 呼气末
   - T2: 睁眼
   - T3: 遵指令
   - T4: 定向力恢复
   - T5: 完全清醒 (30 分钟后)
```

### 3. Φ值计算

#### 方案 A: Φ_approx (主要方法)

```
计算流程:
1. 预处理:
   - 带通滤波：1-45 Hz
   - 陷波：50/60 Hz
   - ICA 去眼电/肌电
   - 坏道插值

2. 时间窗口:
   - 窗长：2 秒
   - 步长：0.5 秒 (75% 重叠)
   - 总计：~100 窗口/被试/期

3. Φ_approx 计算:
   Φ_approx = α·I_local + β·C_global + γ·R_feedback
   
   权重 (麻醉优化版):
   - α = 0.35 (局部整合)
   - β = 0.40 (全局整合)
   - γ = 0.25 (反馈连接)
   
   理由：麻醉主要影响长程连接

4. 输出:
   - Φ时间序列 (0.5 秒分辨率)
   - 各成分分解
   - 质量控制指标
```

#### 方案 B: PCI (子样本验证)

```
适用：N = 15 (子样本)
设备：Magstim Rapid² TMS + EEG

程序:
- 单脉冲 TMS (顶叶 P3/P4)
- 强度：110% 运动阈值
- 时间点：
  - 清醒基线 (50 trials)
  - 麻醉深度 (20 trials, 如安全)
  - 苏醒早期 (30 trials)
  - 完全恢复 (30 trials)

PCI 计算:

- 标准化：0-1

用途：Φ_approx 效标验证
```

### 4. 临界现象检测指标

#### 临界慢化 (Critical Slowing Down)

```
1. 自相关 AC(1):
   AC(1) = Corr[Φ(t), Φ(t-1)]
   
   计算:
   - 滑动窗口：30 秒
   - 步长：5 秒
   - 预期：苏醒前 AC(1) 升高至 > 0.8

2. 恢复率 (Recovery Rate):
   λ = -ln[AC(1)] / Δt
   
   预期：λ降低 (恢复变慢)

3. 功率谱密度:
   - 低频功率增加 (1/f 斜率变平)
   - 特征频率红移
```

#### 临界波动 (Critical Fluctuations)

```
1. 方差:
   Var[Φ] 在临界点前升高
   
   计算:
   - 滑动窗口方差
   - 预期：峰值在 T_c 前 1-2 分钟

2. 偏度 (Skewness):
   - 分布不对称性
   - 预期：正偏度 (向上波动更多)

3. 峰度 (Kurtosis):
   - 重尾分布
   - 预期：峰度 > 3 (高斯=3)
```

#### 不连续跳跃 (Discontinuous Jump)

```
1. Φ跳跃幅度:
   ΔΦ = Φ_post - Φ_pre
   
   定义:
   - Φ_pre: T_c 前 30 秒平均
   - Φ_post: T_c 后 30 秒平均
   
   预期：ΔΦ > 0.2 (大跳跃)

2. 跳跃检测:
   - 断点回归 (Breakpoint Regression)
   - 贝叶斯变点检测
   - 预期：单一显著变点

3. 滞后效应:
   T_c(induction) ≠ T_c(emergence)
   
   预期：T_c(emergence) < T_c(induction)
   滞后量：ΔT_c ≈ 0.3-0.5 μg/mL (丙泊酚)
```

## 📈 预期结果

### 主要预测

#### 1. 临界慢化检测

```
自相关 AC(1) 预测:
- 清醒基线：AC(1) = 0.45 ± 0.10
- 麻醉深度：AC(1) = 0.50 ± 0.12 (无显著变化)
- 苏醒前期：AC(1) = 0.78 ± 0.08 (显著升高)
- 清醒恢复：AC(1) = 0.50 ± 0.10

统计检验:
- F(3, 147) = 28.5, p < 0.001
- 苏醒前期 vs 基线：t(49) = 8.2, p < 0.001, d = 1.16

结论：支持临界慢化预测
```

#### 2. Φ跳跃幅度

```
Φ值时间进程预测:
- 清醒基线：Φ = 0.72 ± 0.08
- 麻醉深度：Φ = 0.15 ± 0.05
- 苏醒前期 (T_c 前 30s)：Φ = 0.20 ± 0.06
- 苏醒后期 (T_c 后 30s)：Φ = 0.55 ± 0.10
- 清醒恢复：Φ = 0.70 ± 0.08

跳跃幅度:
- ΔΦ = 0.35 ± 0.08
- 95% CI: [0.33, 0.37]
- 变点检测后验概率：> 0.98

结论：支持不连续相变预测
```

#### 3. 临界阈值估计

```
丙泊酚 T_c 预测:
- 群体平均：T_c = 1.82 ± 0.38 μg/mL
- 95% CI: [1.71, 1.93]
- 个体变异：CV = 21%

七氟醚 T_c 预测:
- 群体平均：T_c = 0.68 ± 0.18 MAC
- 95% CI: [0.63, 0.73]

与 BIS 阈值对比:
- BIS = 60 对应：Ce = 1.75 μg/mL
- 一致性：r = 0.82
```

### 次要预测

#### 1. 滞后效应

```
T_c 差异预测:
- T_c(induction) = 2.05 ± 0.42 μg/mL
- T_c(emergence) = 1.65 ± 0.35 μg/mL
- 滞后量：ΔT_c = 0.40 ± 0.15 μg/mL

统计检验:
- t(49) = 4.8, p < 0.001
- 效应量：d = 0.68 (中等 - 大)

结论：支持滞后效应 (一级相变特征)
```

#### 2. 临界指数β

```
幂律拟合预测:
- β = 0.52 ± 0.08
- 95% CI: [0.50, 0.54]
- 拟合优度：R² = 0.88

理论对比:
- 平均场理论：β = 0.5
- 3D Ising 模型：β = 0.33
- 结果：接近平均场 (长程相互作用)
```

#### 3. Φ_approx 效度

```
Φ_approx vs PCI:
- 相关性：r = 0.79 [0.68, 0.87]
- 一致性偏差：0.02 ± 0.12
- 敏感度 (检测意识)：88%
- 特异度 (检测无意识)：85%

结论：Φ_approx 可作为麻醉监测工具
```

### 潜在意外发现

1. **多相变**: 可能存在多个变点 (浅麻醉→深麻醉)
2. **个体差异**: T_c 变异大于预期 (遗传/生理因素)
3. **麻醉剂差异**: 丙泊酚 vs 七氟醚临界特征不同
4. **前驱信号**: 临界前兆可能提前 5-10 分钟
