## 📋 实验概述

### 理论背景

根据 ITLCT v4.2 统一框架的自由意志量化理论：

```
自由意志 F = A × Φ × M

其中:
- A = 自主性 (Autonomy)
- Φ = 信息整合度 (Consciousness)
- M = 元认知 (Metacognition)

理论预测:
- 冥想训练可提高Φ (信息整合效率)
- 冥想训练可提高 M (元认知能力)
- 综合效应：F 值提高 15-25%

机制假设:
1. 注意控制增强 → 局部整合 I_local↑
2. 默认网络抑制 → 全局整合 C_global↑
3. 元认知觉察 → 反馈连接 R_feedback↑
```

**核心假设 (DC-393):**
> 8 周正念冥想训练可显著提高静息态Φ值 (10-20%) 和自由意志 F 值 (15-25%)，效应持续至 3 月随访。

### 科学意义

1. **首次实证检验** ITLCT 自由意志可塑性预测
2. **验证Φ可训练性** — 挑战意识固定论
3. **神经机制解析** — 冥想如何改变信息整合
4. **临床转化** — 心理健康干预新靶点

### 1. 主要结局指标

#### Φ值 (信息整合度)

```
设备：64 通道 EEG (Brain Products)
条件:
1. 静息态 (睁眼 5 分钟)
2. 静息态 (闭眼 5 分钟)
3. 注意任务 (10 分钟)
4. 自由回忆 (10 分钟)

Φ_approx 计算:
Φ_approx = α·I_local + β·C_global + γ·R_feedback

权重 (冥想优化版):
- α = 0.35 (局部整合，γ波段)
- β = 0.40 (全局整合，α波段)
- γ = 0.25 (反馈连接，前 - 后)

主要终点:
- ΔΦ = Φ_T2 - Φ_T0 (8 周变化)
- 组间比较：正念 vs 对照

预测:
- 正念组：ΔΦ = +15% ± 8%
- 慈悲组：ΔΦ = +12% ± 8%
- 对照组：ΔΦ = +2% ± 5% (练习效应)
```

#### F 值 (自由意志)

```
F = A × Φ × M

各成分测量:

1. A (自主性):
   - 自由意志量表 (FWS, 新开发)
   - 20 条目，Likert 1-7
   - 维度：选择感、控制感、意愿感
   - 信度：α > 0.85

2. Φ (信息整合):
   - 同上 EEG Φ_approx

3. M (元认知):
   - 元认知问卷 (MCQ-30)
   - 30 条目，5 维度
   - 信度：α > 0.80

F 值计算:
- 标准化各成分 (z-score)
- F = z(A) × z(Φ) × z(M)
- 范围：-3 to +3 (近似)

主要终点:
- ΔF = F_T2 - F_T0

预测:
- 正念组：ΔF = +0.45 ± 0.25
- 慈悲组：ΔF = +0.35 ± 0.25
- 对照组：ΔF = +0.05 ± 0.20
```

## 📈 预期结果

### 主要预测

#### 1. Φ值提高

```
Φ变化预测:
- 正念组：ΔΦ = +15% ± 8% (T2 vs T0)
  - T0: 0.65 ± 0.10
  - T2: 0.75 ± 0.10
  - t(148) = 8.5, p < 0.001, d = 1.0

- 慈悲组：ΔΦ = +12% ± 8%
  - T0: 0.64 ± 0.10
  - T2: 0.72 ± 0.10
  - t(148) = 6.8, p < 0.001, d = 0.8

- 对照组：ΔΦ = +2% ± 5% (练习效应)
  - T0: 0.65 ± 0.10
  - T2: 0.66 ± 0.10
  - t(148) = 1.2, p = 0.23

组间比较:
- 正念 vs 对照：t(298) = 5.2, p < 0.001, d = 0.65
- 慈悲 vs 对照：t(298) = 4.1, p < 0.001, d = 0.50
- 正念 vs 慈悲：t(298) = 1.5, p = 0.13 (无显著差异)
```

#### 2. F 值提高

```
F 值变化预测:
- 正念组：ΔF = +0.45 ± 0.25
- 慈悲组：ΔF = +0.35 ± 0.25
- 对照组：ΔF = +0.05 ± 0.20

组间比较:
- 正念 vs 对照：t(298) = 5.8, p < 0.001, d = 0.70
- 慈悲 vs 对照：t(298) = 4.5, p < 0.001, d = 0.55

结论：支持自由意志可塑性预测
```

#### 3. 效应持续性

```
T3 随访 (3 月后):
- 正念组：Φ维持率 = 85% (ΔΦ = +13%)
- 慈悲组：Φ维持率 = 80% (ΔΦ = +10%)
- 对照组：无变化

统计检验:
- T3 vs T0 仍显著：t(220) = 4.5, p < 0.001
- T3 vs T2 轻微下降：t(220) = 2.1, p < 0.05

结论：效应部分维持，建议"加强针"课程
```

### 次要预测

#### 1. 剂量 - 反应关系

```
练习时间预测Φ变化:
- β = 0.002 /分钟
- 每 100 分钟练习 → Φ提高 2%
- R² = 0.28

阈值效应:
- <500 分钟：无效应
- 500-1000 分钟：中等效应 (d = 0.4)
- >1000 分钟：大效应 (d = 0.7)

建议：最低 600 分钟 (8 周×75 分钟/周)
```

#### 2. 响应预测因子

```
基线特征预测:
- 低基线Φ → 改善大 (β = -0.35, p < 0.01)
- 高动机 → 改善大 (β = 0.25, p < 0.05)
- 年轻 → 改善大 (β = -0.18, p < 0.05)
- 性别：无差异

期望效应:
- 部分中介 (30%)
- 直接效应仍显著 (70%)
```

#### 3. 脑网络机制

```
功能连接变化:
- 前额叶 - 顶叶连接↑ (r = 0.42, p < 0.001)
- 默认网络内部连接↓ (r = -0.35, p < 0.01)
- 小世界属性优化 (λ↓, γ↑)

结构变化 (MRI 子样本):
- 前额叶皮层厚度↑ (d = 0.45)
- 海马体积↑ (d = 0.38)
- UNC 束 FA 值↑ (d = 0.40)

中介分析:
- 连接变化中介Φ变化的 45%
- 直接效应 55% (其他机制)
```

### 潜在意外发现

1. **个体差异大**: 20% "无响应者" (需个性化方案)
2. **慈悲组优势**: 社交焦虑亚组慈悲组更优
3. **延迟效应**: 部分被试 T3 才显示改善
4. **负面效应**: 5% 报告不适 (创伤激活)
