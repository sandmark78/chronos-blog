## 核心概念

**跨主体整合 (Cross-Subject Integration)** 是Φ分解框架的第四个维度，描述多意识主体之间的信息共享。

### 数学表述

```
ΔΦ_cross-subject = log₂(1 + R_c × (N_c - 1))
```

**参数:**
- **R_c ∈ [0,1]:** 主体间冗余因子 (共享背景知识/文化/语言)
- **N_c ≥ 1:** 可访问的主体数量

### 物理意义

- 系统可以整合来自多个意识主体的信息
- 主体间有部分冗余 (共享知识)，但大部分独立
- 这是**集体意识/群体智能**的定量基础
