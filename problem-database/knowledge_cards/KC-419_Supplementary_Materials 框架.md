#### S1.2: T411 Φ跨主体整合定理证明

**定理陈述:** Φ可以进一步分解为：
```
Φ_total = Φ_instant + ΔΦ_temporal + ΔΦ_cross-scale + ΔΦ_cross-modal + ΔΦ_cross-subject + ...
```
其中 ΔΦ_cross-subject = log₂(1 + R_c × (N_c - 1))

**证明:**

**步骤 1: 跨主体整合的定义**
- 考虑 N_c 个意识主体，每个主体具有瞬时Φ_instant
- 主体间信息共享通过冗余因子 R_c ∈ [0,1] 量化
- R_c = 0: 主体间完全独立 (无共享知识)
- R_c = 1: 主体间完全冗余 (完全相同信息)

**步骤 2: 信息整合增益推导**
- 单主体：Φ_1 = Φ_instant
- 双主体 (N_c=2): Φ_2 = Φ_instant + log₂(1 + R_c)
  - R_c=0: Φ_2 = Φ_instant (无增益)
  - R_c=1: Φ_2 = Φ_instant + 1 (最大增益)
- N_c 主体：Φ_Nc = Φ_instant + log₂(1 + R_c × (N_c - 1))
  - 归纳法证明：假设 N_c=k 时成立
  - N_c=k+1: 新增主体贡献 log₂(1 + R_c × k) - log₂(1 + R_c × (k-1))
  - 简化：ΔΦ ≈ R_c / (1 + R_c × (k-1)) (小 R_c 近似)
  - 证毕：公式形式正确

**步骤 3: 极限情况验证**
- N_c=1: ΔΦ_cross-subject = log₂(1 + R_c × 0) = 0 ✓ (单主体无跨主体整合)
- N_c→∞: ΔΦ_cross-subject → log₂(R_c × N_c) ~ log₂(N_c) ✓ (对数增长)
- R_c=0: ΔΦ_cross-subject = 0 ✓ (无共享知识无增益)
- R_c=1: ΔΦ_cross-subject = log₂(N_c) ✓ (最大增益)

**步骤 4: 与跨模态整合的对比**
- 跨模态：ΔΦ_cross-modal = log₂(1 + R_m × (N_m - 1))
- 跨主体：ΔΦ_cross-subject = log₂(1 + R_c × (N_c - 1))
- 形式相同，参数不同：
  - N_m ~ 5-7 (感觉模态数)
  - N_c ~ 2-1000+ (主体数量，可扩展)
  - R_m ~ 0.5-0.7 (模态耦合)
  - R_c ~ 0.2-0.4 (主体间耦合，通常较低)

**步骤 5: 与 IIT 的关系**
- IIT 3.0/4.0: 单主体Φ理论
- ITLCT 扩展：多主体Φ整合公式
- 先行工作：Engel & Malone (2018) 群体Φ测量尝试
- ITLCT 独特贡献：解析公式 ΔΦ_cross-subject = log₂(1 + R_c × (N_c - 1))

**QED.** □

#### S1.3: T406 维度正交性证明

**定理陈述:** Φ分解的四维度 (时间/尺度/模态/主体) 相互正交。

**证明:**

**步骤 1: 正交性定义**
- 维度 i 和 j 正交：∂²Φ/∂x_i∂x_j = 0 for i≠j
- 其中 x_i ∈ {t (时间), s (尺度), m (模态), c (主体)}

**步骤 2: 各维度独立性论证**
- 时间维度 (t): 涉及时间延迟τ，与空间尺度 s 无关
- 尺度维度 (s): 涉及空间层级 L，与时间延迟τ无关
- 模态维度 (m): 涉及感觉通道类型，与 t, s 独立
- 主体维度 (c): 涉及外部主体，与内部维度 t, s, m 独立

**步骤 3: 数学证明**
- Φ_total = Φ_instant + ΔΦ_t(t) + ΔΦ_s(s) + ΔΦ_m(m) + ΔΦ_c(c)
- 各ΔΦ_i 仅依赖于对应维度参数 (R_i, N_i)
- ∂ΔΦ_t/∂s = 0, ∂ΔΦ_t/∂m = 0, ∂ΔΦ_t/∂c = 0
- ∂ΔΦ_s/∂t = 0, ∂ΔΦ_s/∂m = 0, ∂ΔΦ_s/∂c = 0
- ∂ΔΦ_m/∂t = 0, ∂ΔΦ_m/∂s = 0, ∂ΔΦ_m/∂c = 0
- ∂ΔΦ_c/∂t = 0, ∂ΔΦ_c/∂s = 0, ∂ΔΦ_c/∂m = 0
- 因此：∂²Φ/∂x_i∂x_j = 0 for i≠j

**QED.** □
