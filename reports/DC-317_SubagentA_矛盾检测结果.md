#### 矛盾：**发现 1 处潜在问题**

**问题 P401-01: 定义循环风险**

DC-317 中 T401 的表述存在潜在循环定义：

```
原文: Φ_total = Φ_instant + Φ_temporal
但后文: Φ_temporal ≈ log₂(1 + R_t × (N_t - 1)) + Φ_instant
```

**问题:** 如果 Φ_temporal 已经包含 Φ_instant，那么 Φ_total = Φ_instant + Φ_temporal 会导致 Φ_instant 被计算两次!

**可能的修正:**
- 方案 A: Φ_total = Φ_instant + ΔΦ_temporal，其中 ΔΦ_temporal = log₂(1 + R_t × (N_t - 1))
- 方案 B: Φ_total = Φ_instant × (1 + R_t × (N_t - 1)) (乘法而非加法)

**建议:** DC-317 需澄清 Φ_temporal 的精确定义，避免循环计算。
