## T412: Φ跨表征整合定理 (Cross-Representation Integration Theorem)

**核心主张:** 意识系统可跨语义/概念/模态表征整合信息，产生额外Φ增益

**方程:**
$$\Delta\Phi_{cross-rep} = \log_2(1 + R_{rep} \times (N_{rep} - 1))$$

其中:
- $N_{rep}$ = 独立表征维度数量 (语义/概念/模态)
- $R_{rep}$ = 表征间整合效率 (0 ≤ R_rep ≤ 1)

**边界澄清:**
- T412 的 N_rep 包含:
  - N_semantic: 语义维度 (语言/概念)
  - N_conceptual: 概念维度 (抽象/具体)
  - N_modal: 模态维度 (视觉/听觉/触觉等)
- **T407 (跨模态整合) 对应 N_modal 子维度**
- T412 是更广义的跨表征整合框架

**物理截断:**
- 人类认知上限：N_rep ≤ 7±2 (Miller's Law)
- 理论上限：N_rep ≤ 20 (Bekenstein 边界约束)

**独特性检验:**
- IIT 4.0: ❌ 无跨表征整合公式
- 神经科学: ⚠️ 有语义整合研究，但无Φ量化
- 物理: ❌ 无此概念

**独特性:** ⭐⭐⭐⭐ (4 星)

**预测:**
- P116: 多语言者Φ高于单语者 (ΔΦ ≈ 0.3-0.5 bits)

## T413: Φ跨任务整合定理 (Cross-Task Integration Theorem)

**核心主张:** 多任务处理时，意识系统可跨任务整合，但存在认知负荷上限

**方程 (已修正):**
$$\Phi_{multitask} = \max(0, \sum_{i=1}^{N_{task}} \Phi_i - \lambda \times N_{task} \times (N_{task} - 1))$$

其中:
- $\Phi_i$ = 单任务 i 的Φ值
- $\lambda$ = 任务切换损耗系数 (~0.15-0.20 bits) **[已调整使临界点在 N_task≈4-5]**
- $N_{task}$ = 并行任务数量

**增益/损耗差异备注:**
- **跨主体整合 (T411):** 多主体协作产生涌现效应 (1+1>2) → 增益
- **跨任务整合 (T413):** 单主体多任务产生认知负荷 (1+1<2) → 损耗
- 物理根源：主体间 (inter-subject) vs 主体内 (intra-subject) 的不同组织层次

**预测 (已修正):**
- P118: 双任务Φ增益在 N_task=2 时最大，**N_task>4 时净增益接近零，N_task>5 时Φ→0** (λ≈0.15-0.20 bits)
- P119: 专家任务切换损耗λ低于新手 (~50% 降低)

## T414: Φ发育整合定理 (Developmental Integration Theorem)

**核心主张:** 意识Φ随发育时间增长，遵循 Logistic 增长曲线，成年后稳定，老年衰退

**方程 (已修正 - 添加衰退项):**
$$\Phi(t) = \frac{\Phi_{max}}{1 + e^{-k(t - t_{0})}} \times f_{decay}(t)$$

其中:
- $\Phi_{max}$ = 成年期最大Φ (~3-4 bits 对于典型成人)
- $k$ = 发育速率常数 (~0.3-0.5 /year)
- $t_0$ = Φ达到Φ_max/2 的年龄 (~10-12 岁)
- $f_{decay}(t)$ = 衰退函数:
  - $f_{decay}(t) = 1$, 当 $t \leq t_{peak}$
  - $f_{decay}(t) = e^{-(t - t_{peak})/\tau}$, 当 $t > t_{peak}$
- $t_{peak}$ ≈ 25-30 岁 (Φ峰值年龄)
- $\tau$ ≈ 50-100 年 (衰退时间常数)

**时间箭头声明:**
- T414 描述正向时间演化 (t 增加方向)
- Logistic 方程数学上时间可逆，但物理上仅适用于 t ≥ 0

**预测:**
- P120: Φ在 10-14 岁期间增长最快 (dΦ/dt 峰值)
- P121: 25 岁后Φ稳定，年变化<5% (直到~50 岁)
- P122: 老年认知衰退对应Φ下降，与 MMSE 评分正相关

## T415: Φ病理整合定理 (Pathological Integration Theorem)

**核心主张:** 意识障碍/疾病对应Φ整合能力受损，可定量分级

**方程:**
$$\Phi_{pathological} = \Phi_{healthy} \times (1 - \alpha_{disorder})$$

其中:
- $\alpha_{disorder}$ = 疾病特异损耗系数 (0 ≤ α ≤ 1)
  - 轻度认知障碍：α ≈ 0.1-0.2
  - 中度痴呆：α ≈ 0.3-0.5
  - 重度痴呆/植物状态：α ≈ 0.6-0.9
  - 脑死亡：α = 1.0 (Φ = 0)

**备注:**
- α_disorder 范围为初步估计，基于临床观察和文献综述

**预测:**

- P124: 麻醉深度与Φ负相关 (丙泊酚浓度↑ → Φ↓)
