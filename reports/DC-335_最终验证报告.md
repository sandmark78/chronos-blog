## 必须修正清单 (阻塞发布)

### 修正 1: T414 方程添加老年衰退项

**当前方程:**
```
Φ(t) = Φ_max / (1 + e^{-k(t - t₀)})
```

**修正后方程 (方案 A - 推荐):**
```
Φ(t) = Φ_max / (1 + e^{-k(t - t₀)}) × f_decay(t)

其中:
f_decay(t) = 1, 当 t ≤ t_peak
f_decay(t) = e^{-(t - t_peak)/τ}, 当 t > t_peak

参数:
t_peak ≈ 25-30 岁 (Φ峰值年龄)
τ ≈ 50-100 年 (衰退时间常数)
```

**或 (方案 B - 简化):**
```
声明 T414 仅适用于发育期 (t ≤ 25 岁)
老年衰退由新定理 T416 描述 (未来扩展)
```

---

### 修正 2: T413 方程添加Φ≥0 截断

**当前方程:**
```
Φ_multitask = Σᵢ Φ_i - λ × N_task × (N_task - 1)
```

**修正后方程:**
```
Φ_multitask = max(0, Σᵢ Φ_i - λ × N_task × (N_task - 1))
```

**同时修正预测 P118:**
```
原预测：P118: 双任务Φ增益在 N_task=2 时最大，N_task>4 时净增益为负

修正后：P118: 双任务Φ增益在 N_task=2 时最大，N_task>10 时净增益接近零，N_task>11 时Φ→0

或调整λ值：λ ≈ 0.15-0.2 bits (使临界点在 N_task≈4-5)
```

---

### 修正 3: T412 与 T407 边界澄清

**建议在 T412 中添加:**
```
T412 的 N_rep (独立表征维度) 包含:
- N_semantic: 语义维度 (语言/概念)
- N_conceptual: 概念维度 (抽象/具体)

T407 (跨模态整合) 对应 N_modal 子维度

```
