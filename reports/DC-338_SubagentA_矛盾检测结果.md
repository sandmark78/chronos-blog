**问题 T416-D01: 量纲不自洽 (阻塞级)**

T416 公式左侧 (Φ_threshold) 是无量纲的 bits，但右侧计算结果是 [能量]⁻¹。

**根本原因:** κ_SK² 的量纲是能量 (T388 确认)，因此 1/κ_SK² 的量纲是 [能量]⁻¹，无法与无量纲的Φ匹配。

---

**修正方案:**

**方案 A: 引入能量标度因子 (推荐)**
```
Φ_threshold = (E_0/κ_SK²) × ln(Γ_dec / Γ_sys)
```
其中 E_0 是特征能量标度 (系统依赖)，使得 E_0/κ_SK² 无量纲。

**但:** 根据 T388，κ_SK² ≡ κ/E_0，因此 κ_SK² 已经包含 E_0。需要重新审视定义。

**方案 B: 修正 κ_SK² 定义**
```
如果要求Φ_threshold 无量纲，则需要 [κ_SK²] = 无量纲
但这与 T388 冲突 ([κ_SK²] = 能量)
→ 不可行，会破坏 T373/T378/T385/T388 的量纲自洽性
```

**方案 C: 修改 T416 公式结构 (推荐)**
```
Φ_threshold = α × ln(Γ_dec / Γ_sys)
```
其中α是无量纲耦合常数 (由实验拟合，~0.1-1.0)。

**物理解释:** Φ_threshold 由系统 - 环境耦合的对数比值决定，比例系数α是唯象参数。

**方案 D: 使用 κ_SK 而非 κ_SK²**
```
Φ_threshold = (1/κ_SK) × ln(Γ_dec / Γ_sys)
```
量纲检查：[1/κ_SK] = [能量]⁻¹/² → 仍不是无量纲 ❌

---

**推荐修正: 方案 C**
```
T416 修正公式:
Φ_threshold = α_coupling × ln(Γ_dec / Γ_sys + C)

其中:
- α_coupling: 无量纲耦合常数 (~0.1-1.0，系统依赖)
- C: 正则化常数 (~1，防止Γ_dec→0 发散)

```

```
Φ_threshold = α_coupling × ln(Γ_dec / Γ_sys + C)
```

其中 C ~ 1 是正则化常数，防止对数发散。

**或:**
```
Φ_threshold = α_coupling × ln(max(Γ_dec/Γ_sys, 1))
```
确保Φ_threshold ≥ 0。
