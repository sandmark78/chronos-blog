## 二、A20 Phase 1B: IIT 4.0 兼容Φ计算

### 问题陈述

DC-342 Phase 1A 使用了**简化版Φ计算** (基于纠缠熵):
```python
entropy = -Σ(p_i * log2(p_i))
phi = entropy  # 简化近似
```

**问题:** 这与 IIT 4.0 的严格Φ定义不一致。

IIT 4.0 的Φ定义 (Tononi et al., 2016):
$$\Phi = \min_{\text{partitions}} \frac{\text{AE}(\text{system})}{\text{normalization}}$$

其中 AE (Actual Effectiveness) 需要计算：
1. 系统所有子系统的因果库 (cause-effect repertoires)
2. 最小信息划分 (MIP)
3. 不可简化性度量

**目标:** Phase 1B 实现 IIT 4.0 兼容的Φ计算，用于与简化版对比验证。

---

### IIT 4.0 Φ计算核心算法

#### Step 1: 因果库构建

对于量子态$|\psi\rangle$，定义因果库：

$$P_{cause}(X|\psi) = \text{Tr}(\Pi_X \rho \Pi_X^\dagger)$$
$$P_{effect}(Y|\psi) = \text{Tr}(\Pi_Y U \rho U^\dagger \Pi_Y^\dagger)$$

其中:
- $\Pi_X, \Pi_Y$ 是投影算符
- $U$ 是系统演化算符
- $\rho = |\psi\rangle\langle\psi|$

#### Step 2: 信息有效性 (IE)

$$IE = D_{KL}(P_{cause} || P_{cause}^{independent}) + D_{KL}(P_{effect} || P_{effect}^{independent})$$

其中$D_{KL}$是 KL 散度。

#### Step 3: 最小信息划分 (MIP)

$$\Phi = \min_{M \in \text{partitions}} \frac{IE(M)}{N_{bits}(M)}$$

#### Step 4: 量子扩展 (IIT-Q)

对于量子系统，需要考虑：
- 纠缠导致的非定域因果
- 测量导致的波函数坍缩
- 退相干时间尺度

### Phase 1B 实现方案

#### 模块 1: 量子因果库计算

```python
"""
IIT 4.0 兼容Φ计算 — Phase 1B
Chronos Lab | DC-343 | 2026-03-20
"""

import numpy as np
from scipy.linalg import svd
from scipy.stats import entropy as kl_divergence
from qiskit import QuantumCircuit, Aer, transpile
from qiskit.quantum_info import Statevector, DensityMatrix, partial_trace


def compute_cause_repertoire(statevector, subsystem_indices):
    """
    计算因果库 (Cause Repertoire)
    
    参数:
        statevector: Statevector 对象
        subsystem_indices: 子系统量子比特索引
    
    返回:
        dict: 因果库概率分布
    """
    n_qubits = statevector.num_qubits
    dim = 2 ** len(subsystem_indices)
    
    # 计算子系统的约化密度矩阵
    all_indices = list(range(n_qubits))
    trace_out = [i for i in all_indices if i not in subsystem_indices]
    
    if trace_out:
        rho_reduced = partial_trace(statevector, trace_out)
    else:
        rho_reduced = DensityMatrix(statevector)
    
    # 计算计算基下的概率分布
    probs = []
    for i in range(dim):
        basis_state = format(i, f'0{len(subsystem_indices)}b')
        # 投影到计算基
        proj = np.zeros(dim)
        proj[i] = 1.0
        prob = np.real(np.dot(proj, np.dot(rho_reduced.data, proj)))
        probs.append(prob)
    
    probs = np.array(probs)
    probs = probs / (np.sum(probs) + 1e-10)  # 归一化
    
    return {"probabilities": probs, "dimension": dim}


def compute_effect_repertoire(circuit, subsystem_indices, shots=1024):
    """
    计算效应库 (Effect Repertoire)
    
    参数:
        circuit: QuantumCircuit
        subsystem_indices: 子系统索引
        shots: 测量次数
    
    返回:
        dict: 效应库概率分布
    """
    simulator = Aer.get_backend('qasm_simulator')
    transpiled = transpile(circuit, simulator)
    
    # 只测量目标子系统
    meas_circuit = circuit.copy()
    meas_circuit.measure(subsystem_indices, range(len(subsystem_indices)))
    
    result = simulator.run(meas_circuit, shots=shots).result()
    counts = result.get_counts()
    
    dim = 2 ** len(subsystem_indices)
    probs = np.zeros(dim)
    
    for state, count in counts.items():
        # 提取目标子系统的状态
        subsystem_state = state.zfill(circuit.num_qubits)
        target_bits = ''.join([subsystem_state[circuit.num_qubits - 1 - i] 
                               for i in subsystem_indices])
        idx = int(target_bits, 2)
        probs[idx] += count
    
    probs = probs / shots
    
    return {"probabilities": probs, "dimension": dim}


def compute_information_effectiveness(cause, effect):
    """
    计算信息有效性 (IE)
    
    IE = D_KL(P_cause || P_cause_indep) + D_KL(P_effect || P_effect_indep)
    """
    p_cause = cause["probabilities"]
    p_effect = effect["probabilities"]
    
    # 独立分布 (均匀分布)
    p_cause_indep = np.ones_like(p_cause) / len(p_cause)
    p_effect_indep = np.ones_like(p_effect) / len(p_effect)
    
    # KL 散度
    kl_cause = kl_divergence(p_cause, p_cause_indep, base=2)
    kl_effect = kl_divergence(p_effect, p_effect_indep, base=2)
    
    ie = kl_cause + kl_effect
    
    return ie


def compute_mip_phi(circuit, statevector, n_partitions=10):
    """
    计算最小信息划分Φ (简化版 MIP)
    
    参数:
        circuit: QuantumCircuit
        statevector: Statevector
        n_partitions: 采样划分数
    
    返回:
        float: Φ值 (bits)
    """
    n_qubits = circuit.num_qubits
    
    phi_values = []
    
    # 采样不同的子系统划分
    for _ in range(n_partitions):
        # 随机划分量子比特为两个子系统
        indices = np.random.permutation(n_qubits)
        split_point = np.random.randint(1, n_qubits)
        subsystem_a = indices[:split_point].tolist()
        subsystem_b = indices[split_point:].tolist()
        
        # 计算因果库和效应库
        cause_a = compute_cause_repertoire(statevector, subsystem_a)
        effect_a = compute_effect_repertoire(circuit, subsystem_a)
        
        cause_b = compute_cause_repertoire(statevector, subsystem_b)
        effect_b = compute_effect_repertoire(circuit, subsystem_b)
        
        # 计算 IE
        ie_a = compute_information_effectiveness(cause_a, effect_a)
        ie_b = compute_information_effectiveness(cause_b, effect_b)
        
        # 归一化 (按子系统大小)
        n_bits = len(subsystem_a) + len(subsystem_b)
        phi_partition = (ie_a + ie_b) / n_bits
        
        phi_values.append(phi_partition)
    
    # MIP: 取最小值
    phi_mip = min(phi_values)
    
    return phi_mip


def compute_iit_phi(circuit, statevector, n_mip_samples=20):
    """
    计算 IIT 4.0 兼容Φ (主函数)
    
    参数:
        circuit: QuantumCircuit
        statevector: Statevector
        n_mip_samples: MIP 采样数
    
    返回:
        dict: Φ计算结果
    """
    # 多次 MIP 采样取平均 (提高稳定性)
    phi_samples = []
    
    for _ in range(n_mip_samples):
        phi = compute_mip_phi(circuit, statevector, n_partitions=5)
        phi_samples.append(phi)
    
    phi_mean = np.mean(phi_samples)
    phi_std = np.std(phi_samples)
    
    return {
        "phi": phi_mean,
        "phi_std": phi_std,
        "samples": phi_samples,
        "n_qubits": circuit.num_qubits
    }
```

    """
    对比简化版Φ (Phase 1A) 与 IIT 4.0Φ (Phase 1B)

            "diff_rel": diff_rel,
            "consistency": "✅" if abs(diff_rel) < 0.3 else "⚠️"
        })
    
    return results

### Phase 1B 实验协议

#### 实验 1: Φ计算方法对比

**目标:** 验证简化版Φ与 IIT 4.0Φ的相关性

**假设:** 
- $H_0$: $\Phi_{simple}$ 与 $\Phi_{IIT}$ 无线性相关 (r < 0.7)
- $H_1$: $\Phi_{simple}$ 与 $\Phi_{IIT}$ 强相关 (r > 0.7)

**成功标准:** r > 0.85, p < 0.01

#### 实验 2: α指数鲁棒性检验

**目标:** 检验使用不同Φ计算方法时，α指数的稳定性

**方法:**
1. 用$\Phi_{simple}$拟合$R \propto \Phi^\alpha$ → 得到$\alpha_{simple}$
2. 用$\Phi_{IIT}$拟合$R \propto \Phi^\alpha$ → 得到$\alpha_{IIT}$
3. 比较：$|\alpha_{simple} - \alpha_{IIT}| / \alpha_{simple} < 0.2$

**成功标准:** 相对差异 < 20%

#### 实验 3: 噪声敏感度分析

**目标:** 检验 IIT 4.0Φ对噪声的敏感度

**方法:**
1. 添加不同水平的退相干噪声 (0%, 1%, 5%, 10%)
2. 测量$\Phi_{IIT}$随噪声的变化
3. 计算$d\Phi/d(\text{noise})$

**预测:** IIT 4.0Φ比简化版Φ对噪声更敏感 (因考虑更多因果结构)
