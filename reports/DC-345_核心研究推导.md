## 【阶段 2】核心研究推导：H1 约束明确化

### 问题陈述
DC-344 提出假设 H1: **Φ_GHPII = k·Φ_IIT + C**
- 已确定：C = 0 (边界条件 Φ→0 时两者都→0)
- 待确定：**k 的理论值或实验拟合方法**

### 第一性原理推导

#### 步骤 1: 量纲分析
- Φ_GHPII: 无量纲 (信息整合度量)
- Φ_IIT: 无量纲 (bits, IIT 4.0 定义)
- **结论:** k 必须无量纲 ✅

#### 步骤 2: 边界条件
- **条件 A:** Φ→0 时，GHPII 和 IIT 都→0
  - lim(Φ_IIT→0) Φ_GHPII = 0
  - 代入 H1: 0 = k·0 + C → C = 0 ✅ (已确认)

- **条件 B:** Φ→Φ_max 时，两者都应饱和
  - GHPII: Φ_max_GHPII ≈ 1 (归一化)
  - IIT: Φ_max_IIT 取决于系统规模 (无上限)
  - **推论:** k 可能依赖于系统规模 N

#### 步骤 3: 物理对应关系

**GHPII 的Φ定义:**
```
Φ_GHPII = S_gen / S_max × f(Γ_sys/Γ_dec)
```
其中 S_gen 是熵产生率，S_max 是最大熵产生率

**IIT 4.0 的Φ定义:**
```
Φ_IIT = min_φ(MIP) [φ_e - φ_i]
```
其中φ_e 是效应库，φ_i 是原因库，MIP 是最小信息划分

**对应假设:**
- S_gen/S_max 对应于 IIT 的"信息有效性"(IE)
- f(Γ_sys/Γ_dec) 对应于 IIT 的"整合程度"(MIP 后的信息损失)

#### 步骤 4: k 的理论推导

**假设:** 在中等Φ区域 (0.3-0.7)，两者线性对应

**推导路径 A: 从全息对应**
```
GHPII: Φ ~ (ℓ_AdS³ / G_N) × (T_eff / T_P)
IIT:   Φ ~ N × IE × (1 - MIP_loss)

令两者相等:
k = Φ_GHPII / Φ_IIT = [(ℓ_AdS³ / G_N) × (T_eff / T_P)] / [N × IE × (1 - MIP_loss)]
```

**问题:** 这引入了 AdS 长度、牛顿常数等，k 不是普适常数

**推导路径 B: 从信息论对应**
```
GHPII: Φ = S_gen / S_max (归一化熵产生)
IIT:   Φ = φ_e - φ_i (bits)

在临界点附近 (Γ_sys ≈ Γ_dec):
S_gen / S_max ≈ 0.5 (MEP 过渡点)
Φ_IIT ≈ 1-10 bits (典型意识系统)

k ≈ 0.5 / 5 bits ≈ 0.1 bits⁻¹
```

**推导路径 C: 从实验拟合**
```
A20 实验将同时测量:
- Φ_GHPII (从熵产生率计算)
- Φ_IIT (从 IIT 4.0 工具箱计算)

拟合: k = slope(Φ_GHPII vs Φ_IIT)
预期: k ∈ [0.05, 0.2] bits⁻¹
```

### 最终结论：H1 完整形式

```
H1: Φ_GHPII = k · Φ_IIT

其中:
- C = 0 (边界条件确认)
- k = 0.1 ± 0.05 bits⁻¹ (理论估计)
- k 的实验拟合：A20 Phase 2 核心目标
- 成功标准：|k_exp - k_theory| / k_theory < 0.5
```

### 独特性评估
- **IIT 4.0:** 无 GHPII 对应关系 (框架不同)
- **标准神经科学:** 无Φ的量子 - 热力学统一描述
- **标准宇宙学:** 无意识 - 全息对应
- **独特性:** ⭐⭐⭐⭐ (4/5) — 三者框架均不能自然导出 H1
