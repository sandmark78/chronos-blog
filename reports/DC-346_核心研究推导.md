### A20 Phase 1A 简化设计

#### 实验 A: Bell-2 量子优势验证

**电路设计:**
```python
# Bell-2 电路 (2 qubits)
|0⟩ ── H ── @ ── M
            |
|0⟩ ─────── X ── M

# 预期: 最大纠缠态 |Φ+⟩ = (|00⟩ + |11⟩)/√2
# Φ_Bell-2 理论值: ~0.95-1.0 (高整合)
```

**测量协议:**
1. 制备 Bell 态
2. 计算约化密度矩阵ρ_A, ρ_B
3. 计算互信息 I(A:B) = S(ρ_A) + S(ρ_B) - S(ρ_AB)
4. 计算Φ = I(A:B) - I(A:B)_MIP (最小划分后的信息损失)

**预期结果:**
- 无噪声：Φ ≈ 0.95-1.0
-  depolarizing noise p=0.01: Φ ≈ 0.85-0.95
-  depolarizing noise p=0.05: Φ ≈ 0.5-0.7

#### 实验 B: Rainbow-5 尺度律验证

**电路设计:**
```python
# Rainbow-5 电路 (5 qubits, 线性链)
Q0 ── H ── @ ───────────── M
           |
Q1 ─────── X ── @ ──────── M
               |
Q2 ─────────── X ── @ ──── M
                       |
Q3 ─────────────────── X ── @ ── M
                               |
Q4 ─────────────────────────── X ── M

# 预期: 部分纠缠态，Φ随噪声衰减
```

**测量协议:**
1. 制备 Rainbow 态
2. 计算全系统密度矩阵ρ
3. 对每个划分 A|B 计算 I(A:B)
4. 找到 MIP (最小信息划分)
5. Φ = I(A:B)_full - I(A:B)_MIP

**预期结果:**
- 无噪声：Φ ≈ 2.0-3.0 (5 qubits 部分纠缠)
-  depolarizing noise p=0.01: Φ ≈ 1.5-2.5
-  depolarizing noise p=0.05: Φ ≈ 0.5-1.5

### Qiskit 实现代码 (简化版)

```python
"""
A20 Phase 1A 简化实现 - Bell-2/Rainbow-5
DC-346, 2026-03-20
"""

from qiskit import QuantumCircuit, Aer, execute
from qiskit_aer.noise import depolarizing_error, NoiseModel
import numpy as np
from scipy.linalg import svd
from scipy.stats import entropy

def create_bell_circuit():
    """创建 Bell-2 电路"""
    qc = QuantumCircuit(2, 2)
    qc.h(0)
    qc.cx(0, 1)
    qc.measure_all()
    return qc

def create_rainbow_circuit(n_qubits=5):
    """创建 Rainbow-n 电路 (线性链)"""
    qc = QuantumCircuit(n_qubits, n_qubits)
    qc.h(0)
    for i in range(n_qubits - 1):
        qc.cx(i, i + 1)
    qc.measure_all()
    return qc

def compute_phi_from_counts(counts, n_qubits):
    """
    从测量计数计算Φ (简化版)
    
    注意：这是近似计算，完整实现需要密度矩阵重构
    """
    total_shots = sum(counts.values())
    
    # 计算边缘分布 (简化：假设二分 A|B)
    n_A = n_qubits // 2
    n_B = n_qubits - n_A
    
    # 计算互信息 I(A:B) (简化估计)
    # 完整实现需要约化密度矩阵
    
    # 简化Φ估计：基于纠缠熵
    # Φ ≈ S_gen / S_max (归一化熵产生)
    
    # 对于 Bell 态，Φ ≈ 1
    # 对于乘积态，Φ ≈ 0
    
    # 从计数估计纠缠 (简化)
    bell_states = ['00', '11']  # Bell-2 预期输出
    bell_probability = sum(counts.get(s, 0) for s in bell_states) / total_shots
    
    # Φ估计 (线性映射)
    phi_estimate = bell_probability  # 简化：Φ ≈ P(Bell)
    
    return phi_estimate

def run_with_noise(circuit, noise_strength=0.01, shots=8192):
    """带噪声运行电路"""
    backend = Aer.get_backend('qasm_simulator')
    
    if noise_strength > 0:
        noise_model = NoiseModel()
        depol_error = depolarizing_error(noise_strength, 1)
        noise_model.add_all_qubit_quantum_error(depol_error, ['id', 'u1', 'u2', 'u3'])
        
        result = execute(circuit, backend, shots=shots, 
                        noise_model=noise_model).result()
    else:
        result = execute(circuit, backend, shots=shots).result()
    
    counts = result.get_counts()
    return counts

def a20_phase1a_bell2():
    """A20 Phase 1A: Bell-2 实验"""
    print("=== A20 Phase 1A: Bell-2 量子优势验证 ===\n")
    
    circuit = create_bell_circuit()
    
    noise_levels = [0.0, 0.01, 0.05]
    results = []
    
    for noise in noise_levels:
        counts = run_with_noise(circuit, noise_strength=noise)
        phi = compute_phi_from_counts(counts, n_qubits=2)
        results.append((noise, phi, counts))
        print(f"Noise p={noise:.2f}: Φ ≈ {phi:.3f}")
        print(f"  Counts: {counts}\n")
    
    return results

def a20_phase1a_rainbow5():
    """A20 Phase 1A: Rainbow-5 实验"""
    print("=== A20 Phase 1A: Rainbow-5 尺度律验证 ===\n")
    
    circuit = create_rainbow_circuit(n_qubits=5)
    
    noise_levels = [0.0, 0.01, 0.05]
    results = []
    
    for noise in noise_levels:
        counts = run_with_noise(circuit, noise_strength=noise)
        phi = compute_phi_from_counts(counts, n_qubits=5)
        results.append((noise, phi, counts))
        print(f"Noise p={noise:.2f}: Φ ≈ {phi:.3f}")
        print(f"  Counts: {counts}\n")
    
    return results

if __name__ == "__main__":
    # 运行 Bell-2 实验
    bell_results = a20_phase1a_bell2()
    
    # 运行 Rainbow-5 实验
    rainbow_results = a20_phase1a_rainbow5()
    
    # 总结
    print("=== A20 Phase 1A 完成 ===")
    print("Bell-2: 验证高纠缠态Φ计算")
    print("Rainbow-5: 验证多量子比特尺度律")
```
