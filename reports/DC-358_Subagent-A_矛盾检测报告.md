## 详细分析

### 问题 DC-358-A01: α = κ_SK²/c⁴ 量纲不匹配

**背景:**
DC-358 声称 α = κ_SK²/c⁴ = 0.01 ± 0.005，用于拟合 R_quantum ∝ Φ^α。

**量纲分析:**
```
[κ_SK²] = 能量² (from DC-308 T388 量纲统一定理)
[c] = 时间⁻¹ (退相干速率)
[c⁴] = 时间⁻⁴
∴ [κ_SK²/c⁴] = 能量² × 时间⁴
```

**问题:**
α 作为幂律指数 (R ∝ Φ^α) 必须无量纲，但κ_SK²/c⁴有量纲。

**修正方案:**

**方案 A (推荐):** 引入特征标度
```
α = (κ_SK²/c⁴) × (ℏ/E₀)²
其中 E₀ 为特征能量 (如量子比特能隙 ~5 GHz × h)
[ℏ] = 能量 × 时间
[ℏ²/E₀²] = 时间²
∴ [α] = 能量² × 时间⁴ × 时间² / 时间² = 无量纲 ✅
```

**方案 B:** 声明为经验参数
```
α ≈ 0.01 ± 0.005 (实验拟合参数)
注：κ_SK²/c⁴仅提供数量级估计，精确值由实验确定
```

**方案 C (不推荐):** 修正κ_SK²量纲
```
与 DC-308 T388 冲突，会破坏历史自洽性
```

### 问题 DC-358-A02: Φ预期值与计算公式不一致

**背景:**
DC-358 给出Φ计算公式和预期值，但两者不匹配。

**计算:**
```
公式：Φ = I_min × A × (1 + ΔΦ_temporal + ΔΦ_cross-scale)

Bell-2:
- I_min = 2 bits (2-qubit 完全关联)
- A = 1 (理想纠缠)
- ΔΦ_temporal = 0 (稳态)
- ΔΦ_cross-scale = 0 (2-qubit 无跨尺度)
- Φ_calc = 2 × 1 × (1 + 0 + 0) = 2.0 bits
- Φ_claimed = 0.92 ± 0.05
- 差异：2.0 / 0.92 ≈ 2.17x

Rainbow-5:
- I_min = 5 bits
- A = 1
- ΔΦ_temporal = 0
- ΔΦ_cross-scale = 0.5
- Φ_calc = 5 × 1 × (1 + 0 + 0.5) = 7.5 bits
- Φ_claimed = 2.15 ± 0.10
- 差异：7.5 / 2.15 ≈ 3.49x
```

**可能原因:**
1. I_min 不等于 qubit 数 (需考虑实际纠缠结构)
2. A < 1 (真实态非理想最大纠缠)
3. 存在归一化因子 (如Φ/Φ_max)
4. 公式形式需修正

**修正建议:**
```
方案 1: 明确 I_min 计算方法
I_min = 实际最小信息切割 (通过 MIP 搜索算法确定，非简单 qubit 计数)

方案 2: 添加归一化
Φ_normalized = Φ_raw / Φ_max
其中Φ_max = N_qubits (理论上限)
→ Φ_Bell-2 = 2.0 / 2 = 1.0 ≈ 0.92 ✅
→ Φ_Rainbow-5 = 7.5 / 5 = 1.5 ≠ 2.15 ⚠️

方案 3: 修正预期值
Φ_Bell-2 ≈ 2.0 ± 0.2
Φ_Rainbow-5 ≈ 7.5 ± 0.5
(需验证与 T416/T417 自洽性)
```
