### 2.2 H1 标度律量纲

```
Φ_GHPII = k(N) · Φ_IIT
k(N) = k₀ / Φ_max_IIT(N)
```

| 变量 | 量纲 | 检查 |
|------|------|------|
| Φ_GHPII | bits | ✅ |
| Φ_IIT | bits | ✅ |
| k₀ | **无量纲** | ✅ (DC-345 定义) |
| Φ_max_IIT | bits | ✅ |
| **k(N)** | **bits⁻¹** | ⚠️ **量纲问题!** |

**⚠️ 发现量纲不一致:**
- 如果 k₀ 无量纲，Φ_max_IIT 单位 bits，则 k(N) = k₀/Φ_max_IIT 的单位是 **bits⁻¹**
- 但 Φ_GHPII = k(N) × Φ_IIT 的量纲 = bits⁻¹ × bits = **无量纲** ✅
- 而 Φ_GHPII 应是 bits 单位 ❌

**解决方案:**
1. **方案 A:** k₀ 应有单位 **bits**，则 k(N) = bits/bits = 无量纲，Φ_GHPII = 无量纲 × bits = bits ✅
2. **方案 B:** 重新定义 H1 为 Φ_GHPII/Φ_max_GHPII = k(N) × Φ_IIT/Φ_max_IIT (归一化形式)

**DC-345 原文:** "k₀ = 0.1 ± 0.05 (无量纲常数)" — **这导致量纲不一致!**

        = (k₀ / Φ_max_IIT) × Φ_IIT

**✅ 自洽:** N→∞时，Φ_GHPII → k₀ × (0 到 1 之间) = 有限值，不发散!
