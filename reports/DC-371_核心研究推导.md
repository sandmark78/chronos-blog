### Qiskit 电路实现 (GHZ-N)

**GHZ 态制备电路:**
```python
def create_ghz_circuit(n_qubits):
    """创建 N 比特 GHZ 态电路"""
    qc = QuantumCircuit(n_qubits, n_qubits)
    
    # 制备 GHZ 态: |000...⟩ → (|000...⟩ + |111...⟩)/√2
    qc.h(0)  # Hadamard 门在第 0 比特
    for i in range(1, n_qubits):
        qc.cx(0, i)  # CNOT 门从第 0 比特控制到其他比特
    
    # 添加测量 (用于Φ计算)
    qc.measure_all()
    
    return qc
```

**Φ_GHPII 计算 (基于 DC-363 方法):**
```python
def calculate_phi_ghpii(circuit_results, n_qubits):
    """
    从量子电路测量结果计算Φ_GHPII
    
    基于 A20 Phase 1A/1B 验证的方法:
    Φ_GHPII = k₀ · N (T418 线性标度律)
    
    操作定义: 从测量统计中提取相干性度量
    """
    # 提取 |000...⟩ 和 |111...⟩ 的概率
    p_all_zero = circuit_results.get('0' * n_qubits, 0)
    p_all_one = circuit_results.get('1' * n_qubits, 0)
    
    # GHZ 态相干性度量
    coherence = 2 * np.sqrt(p_all_zero * p_all_one)
    
    # Φ_GHPII 与相干性成正比 (基于 A20 校准)
    phi_ghpii = k_0 * n_qubits * coherence
    
    return phi_ghpii
```
