**GHZ 态制备电路 (N=8/10/12/15/20):**
```python
def create_ghz_circuit(n_qubits):
    """创建 N 比特 GHZ 态电路"""
    qc = QuantumCircuit(n_qubits, n_qubits)
    
    # 制备 GHZ 态: |000...⟩ → (|000...⟩ + |111...⟩)/√2
    qc.h(0)  # Hadamard 门在第 0 比特
    for i in range(1, n_qubits):
        qc.cx(0, i)  # CNOT 门从第 0 比特控制到其他比特
    
    qc.measure_all()
    return qc
```

**Φ_GHPII 计算方法:**
```python
def calculate_phi_ghpii(counts, n_qubits, k_0=0.0245):
    """
    从量子电路测量结果计算Φ_GHPII
    
    基于相干性度量:
    coherence = 2 * sqrt(p_all_zero * p_all_one)
    Φ_GHPII = k_0 * N * coherence
    """
    shots = sum(counts.values())
    p_all_zero = counts.get('0' * n_qubits, 0) / shots
    p_all_one = counts.get('1' * n_qubits, 0) / shots
    
    coherence = 2 * np.sqrt(p_all_zero * p_all_one)
    phi_ghpii = k_0 * n_qubits * coherence
    
    return phi_ghpii, coherence
```
