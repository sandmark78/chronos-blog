## 问题识别 (来自 DC-372)

**T418 当前公式：**
```
Φ_GHPII(N) = k₀·N,  其中 k₀ = 0.0243 ± 0.0006 bits
```

**DC-372 识别的边界问题：**
- N=1 时公式给出 Φ = 0.0243 bits
- 但 GHPII 定义要求 Φ(1) = 0（单系统无整合信息）
- 这是理论表述的严谨性问题，非物理矛盾

**Proposed 修正方案：**
1. 添加"适用范围：N≥2"标注
2. 或修改公式为 Φ(N) = k₀·(N-1) for N≥1

---

## 核心分析

### 1. 量纲自洽性检查 ✅

**当前公式量纲：**
- Φ_GHPII: bits (整合信息量)
- k₀: bits (每量子比特贡献)
- N: 无量纲 (量子比特数)
- **k₀·N: bits** ✅ 量纲正确

**修正后 (方案 1: N≥2 标注)：**
- 量纲不变，仍为 bits ✅

**修正后 (方案 2: Φ = k₀·(N-1))：**
- (N-1): 无量纲
- k₀·(N-1): bits ✅ 量纲正确

**结论：** 两种修正方案量纲均自洽。

---

### 2. 极限行为检查 ✅

**N→1 极限：**
- 当前公式：Φ(1) = k₀ = 0.0243 bits ❌ 与 GHPII 定义冲突
- 方案 1 (N≥2 标注)：公式在 N=1 不适用，Φ(1)=0 由定义给出 ✅
- 方案 2 (Φ = k₀·(N-1))：Φ(1) = k₀·0 = 0 bits ✅

**N→∞极限：**
- 当前公式：Φ → ∞ (线性增长) ✅ 符合预期
- 方案 1：Φ → ∞ (线性增长) ✅
- 方案 2：Φ ≈ k₀·N (当 N≫1 时) ✅ 渐近行为一致

**结论：** 两种修正方案极限行为均物理合理。

---

### 3. H1 假设兼容性检查 ✅

**H1 假设：**
```
Φ_GHPII = k(N) · (Φ_IIT / 1 bit)
```

**N=1 时：**
- IIT 4.0：Φ_IIT(1) = 0 (单元素无整合) ✅
- GHPII：Φ_GHPII(1) = 0 (单系统无整合) ✅
- H1 关系：0 = k(1) · (0 / 1 bit) = 0 ✅ 自洽

**结论：** N=1 边界修正与 H1 假设完全兼容。

## 修正决策

**采用方案 1：添加"适用范围 N≥2"标注**

**理由：**
1. ✅ 不改变已验证的 N=2→20 数据拟合
2. ✅ 保持 k₀ = 0.0243 bits 的物理意义 (每量子比特贡献)
3. ✅ 与 IIT 4.0 的 N=1 时Φ=0 自然兼容
4. ✅ 数学表述更严谨，承认公式适用范围
5. ⚠️ 方案 2 会引入 (N-1) 形式，需重新解释 k₀ 物理意义

**修正后 T418 最终表述：**
```
T418 Φ线性标度律定理 (最终版):

对于 N≥2 量子比特系统，GHPII 框架下的整合信息满足：

Φ_GHPII(N) = k₀ · N

其中：
- k₀ = 0.0243 ± 0.0006 bits (每量子比特平均贡献)
- 适用范围：N ≥ 2
- 边界条件：Φ(1) = 0 (单系统无整合，由定义给出)
- 验证范围：N = 2→20 (R² = 0.9994)
```
