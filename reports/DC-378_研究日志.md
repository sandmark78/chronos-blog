## 关键决策

### 决策 1: 饱和函数形式选择

**选项:**
- A: 指数饱和 Φ = k₀·N·exp(-N/N_c)
- B: 双曲线饱和 Φ = k₀·N/(1+N/N_c) ← **选择**
- C: Logistic 饱和 Φ = Φ_max/(1+exp(-k(N-N₀)))

**理由:**
- B 形式最简洁 (奥卡姆剃刀)
- B 仅 1 个新参数 N_c
- B 极限行为合理 (N→∞时Φ→常数)
- B 类似 Michaelis-Menten 酶动力学 (物理直观)

---

### 决策 2: H1-T418 统一框架

**关键洞察:** H1 与 T418 兼容的条件是Φ_IIT 必须随 N 变化。

**推导:**
```
H1: Φ_GHPII = k(N) · (Φ_IIT / 1 bit)
    其中 k(N) = k₀ / log₂(N)

若Φ_IIT(N) = N·log₂(N)/2:

Φ_GHPII = [k₀ / log₂(N)] · [N·log₂(N)/2]
        = k₀·N / 2

→ 线性律恢复 ✅
```

**结论:** H1 与 T418 兼容，因子 1/2 差异待 A20 Phase 3 验证。

---

### 决策 3: N=1 边界处理

**选项:**
- A: 显式声明适用范围 N≥2, Φ(1)=0 由定义 ← **选择**
- B: 修正公式使Φ(1)=0 (如Φ = k₀·(N-1)/...)

**理由:**
- A 不引入额外复杂度 (奥卡姆剃刀)
- N=1 是平凡情况 (无整合信息)
- 实验关注 N≥2 (多量子比特系统)
