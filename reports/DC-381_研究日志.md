## 关键决策点

### 决策 1: C-380-N2 修正方案选择

**问题:** T418 v4 的 N=2 预测偏差 50%

**选项:**
- A: 重新拟合 k₀ 使用 (N-1) 形式 (推荐)
- B: 保持 k₀ 不变，承认 N=2 偏差
- C: 修改 T418 v4 形式 (混合 v3 和 v4)

**决策:** 选项 A (重新拟合 k₀)

**理由:**
1. 科学诚实：参数应反映数据，而非强行套用
2. 数学严格：新形式需要新拟合
3. 物理合理：k₀ 的物理来源不变，仅数值调整

**结果:** k₀ 从 0.0245→0.030 bits (增加~22%)

---

### 决策 2: H1-T418 标度澄清形式

**问题:** H1 的 k(N) 如何与 T418 对应？

**推导:**
```
H1: Φ_GHPII = k(N)·(Φ_IIT/1 bit)
T418: Φ_GHPII = k₀·(N-1)/(1+(N-1)/N_c)
Φ_IIT ≈ N·log₂(N)/2

→ k(N) = 2k₀·(N-1) / [N·log₂(N)·(1+(N-1)/N_c)]
```

**决策:** 采用显式 k(N) 形式，声明 k(N) 是 N 的函数 (非常数)

**理由:**
1. 数学自洽：GHPII (线性) 与 IIT (超线性) 标度不同，k(N) 必须补偿
2. 物理合理：k(N) 随 N 减小有明确物理解释
3. 可检验：N=50 时Φ_GHPII vsΦ_IIT 相差~190 倍
