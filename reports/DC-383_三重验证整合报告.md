- k₀ = 0.030 ± 0.005 bits (A20 Phase 3 拟合：0.0298±0.0012 bits)

### H1 v2 (GHPII-IIT 对应)
```
Φ_GHPII(N) = k(N) · [Φ_IIT(N) / 1 bit]

k(N) = 2k₀·(N-1) / [N·log₂(N)·(1+(N-1)/N_c)]
Φ_IIT(N) ≈ N·log₂(N)/2
```
