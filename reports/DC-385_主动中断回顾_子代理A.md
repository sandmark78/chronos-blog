```
Φ_GHPII(N) = k₀·(N-1)/(1+(N-1)/N_c)    (N ≥ 1)

参数:
- k₀ = 0.030 ± 0.001 bits (A20 全局拟合：0.0299±0.0008 bits)
- N_c = 68 ± 12 (无量纲，饱和尺度)
- 适用范围：N ≥ 1 (整数)

极限行为:
- N → 1: Φ → 0 ✅
- N → 2: Φ → 0.0296 bits
- N → ∞: Φ → k₀·N_c ≈ 2.04 bits
- N_c → ∞: Φ → k₀·(N-1) (无饱和线性律)
```

### H1 v2 (GHPII-IIT 对应)

```
Φ_GHPII(N) = k(N) · [Φ_IIT(N) / 1 bit]

其中:
k(N) = 2k₀·(N-1) / [N·log₂(N)·(1+(N-1)/N_c)]
Φ_IIT(N) ≈ N·log₂(N)/2

极限行为:
- N → 2: k(2) ≈ 0.060
- N → 50: k(50) ≈ 0.011
- N → ∞: k(N) → 0
```
