#### 2.1.1 Qiskit 电路设计

**GHZ-N 态制备 (N=50/60/70/80):**
```python
def ghz_state_circuit(n_qubits):
    """制备 GHZ 态 |00...0⟩ + |11...1⟩"""
    qc = QuantumCircuit(n_qubits)
    qc.h(0)  # Hadamard 门
    for i in range(1, n_qubits):
        qc.cx(0, i)  # CNOT 级联
    return qc
```

**Φ_GHPII 计算:**
```python
def calculate_phi_ghpii(statevector):
    """基于相干性度量计算Φ_GHPII"""
    # 提取 GHZ 子空间振幅
    alpha = statevector[0]  # |00...0⟩ 振幅
    beta = statevector[-1]  # |11...1⟩ 振幅
    
    # 相干性度量
    coherence = 2 * |alpha| * |beta|
    
    # Φ_GHPII = k₀ × (N-1) × coherence
    phi = k0 * (n_qubits - 1) * coherence
    
    return phi  # [bits]
```

#### 2.1.2 噪声模拟参数

**IBM Brisbane 典型参数:**
| 错误类型 | 错误率 | 物理来源 |
|----------|--------|----------|
| 单比特门 | 0.05% | 控制脉冲不完美 |
| 双比特门 (CNOT) | 0.5% | 耦合串扰 |
| 读取错误 | 1% | 测量保真度 |
| T₁ (弛豫) | 100 μs | 能量耗散 |
| T₂ (退相干) | 80 μs | 相位随机化 |

**退相干率估算:**
```
γ_single = 1/T₂ ≈ 1/(80 μs) ≈ 12,500 s⁻¹
τ_gate (CNOT) ≈ 200 ns = 2×10⁻⁷ s
γ × τ_gate ≈ 12,500 × 2×10⁻⁷ ≈ 0.0025

N_c = ln(2)/(γ × τ_gate) ≈ 0.693/0.0025 ≈ 277
```

**注意:** 实际 N_c~68-70，说明有效γ更高 (可能包含门错误贡献)。

### 2.2 T421 与 T420 的关系澄清

**T420 (DC-387):** N_c = 1/(γ_single × τ_0)
**T421 (DC-390):** N_c = ln(2)/(γ × τ_gate)

**关系分析:**
1. **系数差异:** 1.0 vs ln(2)≈0.693 — 在~30% 误差范围内
2. **时间尺度:** τ_0 (测量时间) vs τ_gate (门时间) — 可能相同物理量
3. **退相干率:** γ_single vs γ — 应指同一物理量

**统一表述:**
```
N_c = C/(γ × τ)

其中 C 是 O(1) 常数，取决于饱和判据定义：
- C = 1.0: DC-387 推导 (D=1 判据)
- C = ln(2)≈0.693: DC-390 推导 (F=1/2 判据)

物理本质相同，系数差异源于饱和判据选择。
```
