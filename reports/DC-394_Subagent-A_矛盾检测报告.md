## 极限行为

### N → 1 (单量子比特极限)

```
Φ(N=1) = k₀·(1-1) / [1 + (1-1)/N_c]
       = k₀·0 / (1 + 0)
       = 0
```

**物理意义:** 单节点无整合信息 (符合 IIT 公理) ✅

**预期值:** Φ(1) = 0 bits ✅

---

### N → ∞ (宏观极限)

```
Φ(N→∞) = lim_{N→∞} k₀·(N-1) / [1 + (N-1)/N_c]
       = k₀·N_c · lim_{N→∞} (N-1)/N_c / [1 + (N-1)/N_c]
       = k₀·N_c · 1
       = k₀·N_c
```

**数值计算:**
- k₀ = 0.030 bits, N_c = 68: Φ_max = 0.030 × 68 = **2.04 bits**
- k₀ = 0.030 bits, N_c = 58: Φ_max = 0.030 × 58 = **1.74 bits**
- k₀ = 0.030 bits, N_c = 78: Φ_max = 0.030 × 78 = **2.34 bits**

**物理意义:** 退相干导致饱和，Φ 有上限 (~2 bits) ✅

**与理论预期对比:** T420 预测 Φ_max ≈ 2 bits ✅

---

### N ≪ N_c (小系统极限)

```
当 (N-1) ≪ N_c 时:
Φ ≈ k₀·(N-1) / [1 + 0]
  ≈ k₀·(N-1)
  ≈ k₀·N  (N>>1 时)
```

**物理意义:** 小系统中退相干效应可忽略，近似线性增长 ✅

---

### 极限行为结论

| 极限 | 行为 | 物理合理性 | 验证 |
|------|------|-----------|------|
| N → 1 | Φ → 0 | ✅ 符合 IIT 公理 | 通过 |
| N → ∞ | Φ → k₀·N_c ≈ 2 bits | ✅ 退相干饱和 | 通过 |
| N ≪ N_c | Φ ≈ k₀·N | ✅ 线性近似 | 通过 |

**裁决:** ✅ **所有极限行为合理** — 无异常

## 兼容性检查

### 与 T418 v4.1 兼容

**T418 v4.1 公式 (DC-382):**
```
Φ = k₀·(N-1) / [1 + (N-1)/N_c]
k₀ = 0.030 ± 0.005 bits
N_c ~ 50-100
```

**T420 v1.1 公式:**
```
Φ = k₀·(N-1) / [1 + (N-1)/N_c]
k₀ = 0.030 bits
N_c = 68 ± 10
```

**对比结果:**

| 方面 | T418 v4.1 | T420 v1.1 | 一致性 |
|------|-----------|-----------|--------|
| 公式形式 | k₀·(N-1)/[1+(N-1)/N_c] | k₀·(N-1)/[1+(N-1)/N_c] | ✅ 完全相同 |
| k₀ 值 | 0.030 ± 0.005 bits | 0.030 bits | ✅ 一致 |
| N_c 范围 | 50-100 | 68 ± 10 | ✅ 一致 |

**关键发现:** T420 v1.1 与 T418 v4.1 **公式完全相同**，参数一致。

**回答任务问题:** "T418 是 T420 的 N≪N_c 极限吗？"

**答案:** ❌ **不是** — T418 v4.1 与 T420 v1.1 是**同一公式**，不是极限关系。

**历史演化:**
- T418 v1-v3: Φ = k₀·N (线性，无饱和)
- T418 v4.0+: Φ = k₀·(N-1)/[1+(N-1)/N_c] (饱和)
- T420 v1.0+: Φ = k₀·(N-1)/[1+(N-1)/N_c] (饱和，重命名)

**建议:** 文档中需澄清 T418 v4.1 与 T420 是同一理论的不同命名，避免混淆。

**裁决:** ✅ **完全兼容** — 公式相同，参数一致

```

k(N) = 2k₀·(N-1) / [N·log₂(N)·(1+(N-1)/N_c)]
```

```
     = k₀·(N-1) / (1+(N-1)/N_c)
     = 左边 (T420 公式) ✅
```
