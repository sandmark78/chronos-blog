### P1: C-393-A01 修正 (T418 vs T420 N=1 边界标注)

#### 问题描述
**C-393-A01:** T418 v4.1 与 T420 v1.1 在 N=1 边界的行为需要统一标注。

**T418 v4.1:** Φ = k₀·(N-1)/[1+(N-1)/N_c]  (N≥1)
- N=1: Φ = 0 (分子为 0)

**T420 v1.1:** Φ = k₀·(N-1)/[1+(N-1)/N_c]  (N≥1)
- N=1: Φ = 0 (分子为 0)

**问题:** 两者公式相同，但历史文档中 T418 有时写作 Φ = k₀·N (N≪N_c 近似)，需要澄清。

#### 修正方案

**统一标注:**

```markdown
## T418 v4.1 (标准标度律) — 适用于 N ≪ N_c

**完整形式:**
Φ_GHPII(N) = k₀ · (N-1) / [1 + (N-1)/N_c]  for N ≥ 1

**N ≪ N_c 近似:**
当 (N-1)/N_c ≪ 1 时，分母 ≈ 1
Φ_GHPII(N) ≈ k₀ · (N-1)  for 1 ≤ N ≪ N_c

**边界条件:**
- N=1: Φ = 0 (单节点无整合)
- N=2: Φ ≈ k₀ ≈ 0.030 bits (最小整合单元)
- N→∞: Φ → k₀·N_c ≈ 2.0 bits (饱和极限)
```

```markdown
## T420 v1.1 (量子意识饱和定理) — 全范围有效

**完整形式:**
Φ_GHPII(N) = k₀ · (N-1) / [1 + (N-1)/N_c]  for N ≥ 1

**物理意义:**
- (N-1): 有效整合节点数 (图论最小连接数)
- N_c: 饱和尺度 (由退相干率和门时间决定)
- Φ_max: GHZ 态二维子空间的全局信息容量

**边界条件:**
- N=1: Φ = 0 (单节点无整合) ✅
- N=N_c: Φ = k₀·N_c/2 ≈ 1.0 bits (半饱和点)
- N→∞: Φ → k₀·N_c ≈ 2.0 bits (饱和极限)
```

**关系澄清:**
- T418 v4.1 和 T420 v1.1 **公式相同**
- T418 强调 N≪N_c 的线性近似
- T420 强调全范围饱和行为
- 两者是同一理论的不同侧重点，**无矛盾** ✅
