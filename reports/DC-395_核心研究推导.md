## 2. Φ_max 理论预测细化

### 2.1 当前理论预测

**T420 预测:**
$$\Phi_{max} = \lim_{N\to\infty} \frac{k_0 \cdot (N-1)}{1 + (N-1)/N_c} = k_0 \cdot N_c$$

**代入数值:**
- k₀ = 0.0300 ± 0.0005 bits
- N_c = 68 ± 10 (Phase 6B 拟合)

**预测:**
$$\Phi_{max} = 0.0300 \times 68 = 2.04 \text{ bits}$$

**不确定性传播:**
$$\sigma_{\Phi_{max}} = \Phi_{max} \sqrt{\left(\frac{\sigma_{k_0}}{k_0}\right)^2 + \left(\frac{\sigma_{N_c}}{N_c}\right)^2}$$
$$\sigma_{\Phi_{max}} = 2.04 \sqrt{\left(\frac{0.0005}{0.0300}\right)^2 + \left(\frac{10}{68}\right)^2} = 2.04 \sqrt{0.00028 + 0.0216} \approx 2.04 \times 0.15 \approx 0.31 \text{ bits}$$

**当前理论精度:** Φ_max = 2.04 ± 0.31 bits ✅ (与 C-388-A01 一致)

### 2.2 Phase 6C 目标精度

**目标:** σ_Φ_max < 0.1 bits

**所需改进:**
- N_c 误差: 10 → 3.5 (从±10 降至±3.5, -65%)
- k₀ 误差: 0.0005 → 0.0003 (从±0.0005 降至±0.0003, -40%)

**可行性分析:**
- N_c 误差降低: Phase 6B 已从 17%→9%，Phase 6C 目标 5% 合理 ✅
- k₀ 误差降低: 增加测量点 (5 点→8 点) + 高精度拟合可实现 ✅

## 3. Qiskit 实现方案

### 3.1 GHZ 态制备电路

```python
def prepare_ghz_state(n_qubits):
    """制备 N-qubit GHZ 态"""
    qc = QuantumCircuit(n_qubits)
    qc.h(0)  # Hadamard on first qubit
    for i in range(1, n_qubits):
        qc.cx(0, i)  # CNOT cascade
    return qc
```

### 3.4 大规模模拟策略

**N ≤ 30: 精确态矢量模拟**
- 使用 Qiskit Aer 的 statevector 模拟器
- 内存需求: 2^30 × 16 bytes ≈ 16 GB

**30 < N ≤ 100: 矩阵乘积态 (MPS)**
- 使用 Qiskit 的 MPS 模拟器
- 内存需求: O(N × χ²), χ 为键维数 (~100)
- N=100, χ=100: ~100 × 10000 × 16 bytes ≈ 16 MB

**N > 100: 张量网络近似**
- 使用 ITensor 或 TeNPy 库
- 截断误差控制 < 10^-6

## 4. 预期结果与验证

### 4.1 拟合曲线预测

基于 T420 v1.1:
$$\Phi(N) = \frac{0.0300 \cdot (N-1)}{1 + (N-1)/68}$$

**计算验证:**
- Φ(100) = 0.0300 × 99 / (1 + 99/68) = 2.97 / 2.456 = **1.21 bits**
- Φ(150) = 0.0300 × 149 / (1 + 149/68) = 4.47 / 3.191 = **1.40 bits**
- Φ(200) = 0.0300 × 199 / (1 + 199/68) = 5.97 / 3.926 = **1.52 bits**
- Φ(300) = 0.0300 × 299 / (1 + 299/68) = 8.97 / 5.397 = **1.66 bits**
- Φ(500) = 0.0300 × 499 / (1 + 499/68) = 14.97 / 8.338 = **1.80 bits**

| N | 预测Φ (bits) | 预期测量Φ (bits) | 误差棒 (±) | Φ/Φ_max |
|---|--------------|------------------|------------|---------|
| 100 | 1.21 | 1.21 | 0.06 | 0.59 |
| 150 | 1.40 | 1.40 | 0.07 | 0.69 |
| 200 | 1.52 | 1.52 | 0.08 | 0.75 |
| 300 | 1.66 | 1.66 | 0.09 | 0.81 |
| 500 | 1.80 | 1.80 | 0.10 | 0.88 |

**Φ_max 拟合:** 从 5 点外推 → Φ_max = k₀ × N_c = 0.0300 × 68 = **2.04 ± 0.10 bits** ✅ (达到目标精度)

**注意:** N=500 时Φ/Φ_max = 0.88 (非 0.98)，反映饱和是渐进过程。要达到Φ/Φ_max > 0.95 需 N > 1000。
