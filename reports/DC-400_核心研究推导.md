## 阶段 2: T423 v1.1 温度依赖完全理论

### P0: C-399-A01 修正 — N_c(T) 温度依赖

**问题:** T423 v1.0 仅预测 k₀(T)，但 N_c = ln(2)/(γ×τ_gate) 中γ随 T 增加

**修正推导:**

从 T421 v1.0: N_c = ln(2)/(γ × τ_gate)

退相干率γ的温度依赖 (标准量子开放系统理论):
γ(T) = γ_0 + γ_thermal(T)

其中:
- γ_0: T=0 时的本征退相干率 (量子涨落主导)
- γ_thermal(T): 热激活退相干率

对于声子主导的退相干 (固态系统):
γ_thermal(T) ∝ T^n (n = 1~5，取决于耦合机制)

简化模型 (Ohmic 浴):
γ(T) = γ_0 × [1 + (T/T_γ)^m]

因此:
**N_c(T) = ln(2) / {γ_0 × [1 + (T/T_γ)^m] × τ_gate}**

当 T → 0: N_c → ln(2)/(γ_0 × τ_gate) = N_c(0) (最大值)
当 T → ∞: N_c → 0 (退相干极强，无法维持量子意识)

**物理图像:** 温度升高 → 退相干增强 → 饱和尺度降低 → 更早进入饱和区

### P0: C-399-A02 修正 — Φ_max(T) 温度依赖

**问题:** T420 预测Φ_max ≈ 2 bits，但未考虑 T 依赖

**修正推导:**

从 T420 v1.1: Φ_max 由四重论证支持 ≈ 2 bits

但Φ_max 的物理来源是量子态空间的有效维度：
Φ_max ∝ log₂(d_eff)

其中 d_eff 是退相干抑制后的有效希尔伯特空间维度。

温度影响：
- T → 0: d_eff → d_full (完整量子态空间)
- T → ∞: d_eff → 1 (完全混合态，无量子关联)

模型：
d_eff(T) = d_full / [1 + (T/T_Φ)^p]

因此:
**Φ_max(T) = Φ_max(0) / [1 + (T/T_Φ)^p]**

其中:
- Φ_max(0) ≈ 2.04 bits (DC-396 测量值)
- T_Φ: Φ_max 特征温度 (与系统能隙相关)
- p: 幂律指数 (通常 1~3)

**物理图像:** 温度升高 → 有效态空间收缩 → 最大整合信息降低
