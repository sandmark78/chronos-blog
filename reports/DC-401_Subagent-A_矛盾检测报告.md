### 2. T_crit 第一性原理推导 (C-401-A01)

**推导链:**
```
γ(T) = γ_0 × coth(ℏω_0/(2k_B T))
高温近似：γ(T) ≈ (2γ_0 k_B/ℏω_0) × T  (当 k_BT ≫ ℏω_0)
N_c(T) = ln(2)/[γ(T)×τ_gate]
定义 T_crit: N_c(T_crit) = N_c(0)/2 → γ(T_crit) = 2γ_0
解得：T_crit = ℏω_0/(2k_B)
```

**问题:**
- 高温近似条件 k_BT ≫ ℏω_0 与 T_crit 定义 T = ℏω_0/(2k_B) 矛盾
- 在 T = T_crit 处，k_BT = ℏω_0/2，不满足高温条件

**建议:**
- 使用完整 coth 形式数值求解 γ(T_crit) = 2γ_0
- 或重新定义 T_crit 为唯象拟合参数，而非严格第一性原理导出

### 5. N_c(T) 形式整合 (C-401-A06)

**T421 第一性原理形式:**
```
N_c(T) = ln(2)/[γ(T)×τ_gate]
γ(T) = γ_0 × coth(ℏω_0/(2k_B T))
```

**T423 唯象形式:**
```
N_c(T) = N_c(0)/[1 + (T/T_γ)^m]
```

**低温展开 (k_BT ≪ ℏω_0):**
```
coth(x) ≈ 1 + 2e^(-2x) + ...  (x = ℏω_0/(2k_BT) ≫ 1)
γ(T) ≈ γ_0 × [1 + 2exp(-ℏω_0/(k_BT))]
N_c(T) ≈ N_c(0) × [1 - 2exp(-ℏω_0/(k_BT))]
```

**高温展开 (k_BT ≫ ℏω_0):**
```
coth(x) ≈ 1/x + x/3 + ...  (x = ℏω_0/(2k_BT) ≪ 1)
γ(T) ≈ γ_0 × (2k_BT)/(ℏω_0)
N_c(T) ≈ N_c(0) × (T_γ/T)  (T_γ = ℏω_0/(2k_B))
```

**评估:** 唯象形式是高温近似的简化，需声明适用范围
