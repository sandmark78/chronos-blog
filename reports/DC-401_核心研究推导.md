### P0: T422-T423 层次关系形式化 (解决 C-400-A01)

#### 问题重述

DC-400 中 T423 引入 k₀(T) 温度依赖性，与 T422 v1.0 声称的"k₀是普适几何常数"表面冲突。

#### 解决方案：微观 vs 有效理论层次

**T422 (微观理论，T→0 极限):**
```
k₀(0) = (ln 2)/(2π × 𝒩_eff) × (1 bit)

其中:
- 𝒩_eff: T→0 时可访问的有效量子态数
- 纯几何量，由 Hilbert 空间曲率决定
- 无温度依赖性 (基态几何)
```

**T423 (有效理论，有限温度 T>0):**
```
k₀(T) = k₀(0) × f_T(T/T_crit)

其中:
- f_T(x) = 1 - x^δ (唯象温度衰减函数)
- T_crit: 量子 - 经典 crossover 温度
- 物理解释：有限温度下，热噪声限制可访问的量子几何结构
```

#### 物理图像类比

| 层次 | 量子几何 | 统计力学类比 |
|------|----------|--------------|
| **微观 (T422)** | 完整 Hilbert 空间曲率 | 纯态 Schrödinger 动力学 |
| **有效 (T423)** | 热平均可访问曲率 | 混合态热统计平均 |

**关键洞察:**
- T→0: 系统处于基态，可访问完整量子几何 → k₀ = k₀(0)
- T>0: 热激发导致态混合，有效可访问几何结构减小 → k_eff < k₀(0)
- T→T_crit: 量子相变，几何结构完全经典化 → k_eff → 0

#### 形式化声明

**定理 T422-T423 层次关系:**
```
T422 是 T423 的 T→0 极限特例。

数学表述:
lim_{T→0} k₀(T) = k₀(0) [来自 T422]
lim_{T→0} N_c(T) = N_c(0) [来自 T421]
lim_{T→0} Φ_max(T) = Φ_max(0) [来自 T420]

因此 T423 v1.1 在 T→0 极限下严格还原为 T420-T422 的零温理论。
```

### P1: T_crit 第一性原理推导 (解决 C-400-A02)

#### 问题重述

DC-400 中 T_crit 被定义为"退相干主导温度"，但缺乏从第一性原理的严格推导。

#### 推导路径：量子 Fisher 信息温度依赖性

**步骤 1: 量子 Fisher 信息的温度依赖**

对于温度 T 下的热态 ρ(T) = e^{-βH}/Z，量子 Fisher 信息 F_Q 满足:
```
F_Q(T) = F_Q(0) × g(T/T_0)

其中 g(x) 是衰减函数，T_0 是特征温度尺度。
```

**步骤 2: Bures 距离与量子 Fisher 信息的关系**

Bures 距离 D_Bures 与 F_Q 的关系:
```
D_Bures²(ρ, ρ+dρ) = (1/4) × F_Q × dρ²

在有限温度下，热涨落导致 D_Bures 减小。
```

**步骤 3: T_crit 的定义**

定义 T_crit 为量子 Fisher 信息开始显著退化的温度:
```
F_Q(T_crit) = (1/2) × F_Q(0)

即量子几何张量减小到零温值的一半。
```

**步骤 4: 从退相干率导出 T_crit**

退相干率 γ(T) 的温度依赖性 (Ohmic 浴模型):
```
γ(T) = γ_0 × coth(ℏω_0/(2k_B T))

其中:
- γ_0: T→0 极限的退相干率
- ω_0: 特征频率 (~ 量子比特频率)
- k_B: Boltzmann 常数
```

在高温极限 (k_B T ≫ ℏω_0):
```
γ(T) ≈ γ_0 × (2k_B T)/(ℏω_0) = (2γ_0 k_B / ℏω_0) × T
```

**步骤 5: T_crit 的表达式**

从 N_c(T_crit) = N_c(0)/2 (饱和尺度减小一半):
```
N_c(T) = ln(2)/[γ(T) × τ_gate]

N_c(T_crit) = N_c(0)/2
⇒ γ(T_crit) = 2γ_0

代入 γ(T) 的高温近似:
2γ_0 = (2γ_0 k_B / ℏω_0) × T_crit

解得:
T_crit = ℏω_0/(2k_B)
```

#### 最终结果

**定理 T424 (T_crit 的量子信息论起源):**
```
T_crit = ℏω_0/(2k_B)

其中:
- ω_0: 量子系统特征频率 (量子比特频率)
- ℏ: 约化 Planck 常数
- k_B: Boltzmann 常数

物理意义:
- T_crit 是量子涨落与热涨落相当的温度
- T < T_crit: 量子涨落主导 (量子几何有效)
- T > T_crit: 热涨落主导 (经典几何)

数值估计 (超导量子比特):
- ω_0/(2π) ~ 5 GHz
- T_crit = ℏω_0/(2k_B) ≈ (1.05×10⁻³⁴ × 2π × 5×10⁹)/(2 × 1.38×10⁻²³)
- T_crit ≈ 120 mK

这与超导量子比特的典型工作温度 (~10-20 mK) 一致，
说明 ITLCT 在现有量子硬件温度范围内有效。
```

#### 与 T423 的自洽性检查

T423 的唯象形式:
```
k₀(T) = k₀(0) × [1 - (T/T_crit)^δ]

在 T = T_crit 时:
k₀(T_crit) = k₀(0) × [1 - 1^δ] = 0

这与 T_crit 定义 (量子几何完全退化) 自洽。
```

### P2: 文档完善与适用范围声明

#### T423 v1.1 适用范围

**声明:**
```
T423 v1.1 适用范围: 0 ≤ T < 0.5×T_crit

理由:
- T > 0.5×T_crit: 高阶温度修正项 (T/T_crit)³ 等不可忽略
- T → T_crit: 临界涨落主导，平均场近似失效
- T > T_crit: 需要完全不同的经典极限理论
```

#### 临界指数δ的说明

**声明:**
```
δ ≈ 2 (唯象临界指数)

来源:
- 平均场理论预测 δ = 2 (Landau 二级相变)
- 精确值需从具体退相干机制计算
- 建议实验拟合：测量 k₀(T) 在 3-5 个温度点，拟合δ

预测:
- 超导量子比特：δ ~ 2 (平均场)
- 离子阱：δ ~ 1-2 (取决于囚禁势)
- 光量子：δ ~ 1 (光子数温度依赖弱)
```
