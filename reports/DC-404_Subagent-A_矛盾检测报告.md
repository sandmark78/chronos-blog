### N_c 推导

**DC-404 两条推导路径:**

**路径 A 评估:**

从量子信息论保真度推导:
```
F(N) = e^{-Nγτ}

```

**严格性:** ✅ 数学推导严格

**但物理假设:**
- 保真度指数衰减 F = e^{-Nγτ} 是唯象模型

**路径 B 评估:**

从退相干长度推导:
```
ξ_d = ℏ / √(2m k_B T) (热德布罗意波长)
N_c = ξ_d / d
```

**问题:**
- 为什么 N_c = ξ_d/d？缺乏论证
- 物理图像：退相干长度内的量子比特数？
- 但 N_c 定义是"饱和尺度"，与退相干长度的关系不清晰 ❌

**路径 A vs 路径 B 一致性:**

从路径 A:
```
N_c(T) = ln(2) / (γ_0 × coth(ℏω_0/2k_B T) × τ_gate)

高温极限 (k_B T ≫ ℏω_0):

coth(ℏω_0/2k_B T) ≈ 2k_B T / ℏω_0

```

从路径 B:
```

```

**矛盾:** 1/T vs 1/√T ❌

**评分:** 50/100
