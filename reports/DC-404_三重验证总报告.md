## 🔴 阻塞性问题 (必须修正)

### 1. T→T_crit 极限 N_c 行为矛盾

**问题:**
- DC-404 声称：T → T_crit 时 N_c → 0
- 但从 T421 定义：N_c = ln(2)/(γ×τ_gate)
- 在 T = T_crit 时，γ(T_crit)×τ_gate = 1 (由 T_crit 定义)
- 因此 N_c(T_crit) = ln(2) ≈ 0.693 ≠ 0

**修正建议:**
- 方案 A: 修改 T_crit 定义为 γ(T_crit)×τ_gate → ∞
- 方案 B: 修改 N_c 公式为 N_c = ln(2)/(γ×τ_gate - 1)
- 方案 C: 承认 N_c(T_crit) = ln(2)，修正"N_c → 0"的声称

---

### 2. 与 T418 v4.1 k₀值矛盾

**问题:**
- T418 v4.1: k₀ = 0.0245 bits (N=2→7 拟合，实验偏差 2%)
- DC-404: k₀(0) = 0.0300 bits
- N=2 预测差异：DC-404 预测 0.0296 bits vs 实验 0.048 bits (偏差 38%)

**修正建议:**
- 重新拟合 k₀(0) 使其与 T418 实验数据一致
- 或说明 DC-404 使用了新数据集，并提供拟合细节
- 或修正 T418 的 k₀值并重新评估历史数据

---

### 3. 与 H1 v2 饱和行为矛盾

**问题:**
- H1 v2: Φ ~ N^α (α≈1，无饱和，线性增长)
- DC-404: Φ → Φ_max = k₀(0)×N_c(0) (饱和到有限值)
- 根本性矛盾：H19 引入γ>1 描述加速增长，但仍未饱和

**修正建议:**
- 明确 H1 适用范围为 N ≪ N_c (此时 DC-404 ≈ 线性)
- 或修正 H19 引入饱和项：Φ = k₀×N^γ / [1 + (N/N_sat)^γ]
- 或在 DC-404 中说明 H1 是 N ≪ N_c 极限近似

## 🟡 重要问题 (建议修正)

### 4. T423-T424 N_c 温度依赖矛盾

**问题:**
- T424: N_c = ξ_d/d ∝ 1/√T (η = 0.5)
- DC-404 Section 4: N_c ~ 1/T^η (η ≈ 1-1.5)
- 两条推导路径给出不一致的温度依赖指数

**修正建议:**
- 明确 T424 适用范围 (弹道输运 vs 扩散输运)
- 或统一温度依赖形式，说明不同物理机制对应不同η值

---

### 5. T425 δ公式缺乏严格推导

**问题:**
- δ = C_V^crit/(3k_B) 公式来源不明
- 为什么是 3k_B？不是 2k_B 或 4k_B？
- 与标准临界指数关系不一致 (标准β≈0.3-0.5，DC-404δ≈1.5-2.0)

**修正建议:**
- 补充从多体退相干微观模型的第一性原理推导
- 或明确标注为唯象公式，说明拟合依据

---

### 6. δ推导链跳跃

**问题:**

**修正建议:**
- 补充多体量子系统在临界点附近的微观机制推导
