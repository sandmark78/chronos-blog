**问题:** T426 引入 Q_P 符号但未说明与 T421 的关系。


### C-406-A02: T427 k₀=0.0300 与 T418 k₀=0.0245 矛盾

**问题:** DC-404 发现 k₀=0.0300 预测 N=2 时Φ=0.0296 bits，但实验测量Φ=0.048 bits (偏差 38%)。

**解决方案:**
```markdown
在 T427 中添加注释:

"注：k₀(0) = 0.0300 bits 从 A20 Phase 1-6 数据拟合 (考虑饱和修正)。
T418 v4.1 的 k₀ = 0.0245 bits 是未考虑饱和修正的有效拟合值 (N=2→7)。

考虑饱和修正后:
Φ(N) = k₀ × (N-1) / [1 + (N-1)/N_c]
N=2 时：Φ(2) = 0.0300 × 1 / (1 + 1/69) ≈ 0.0296 bits (仍偏差 38%)

⚠️ 此矛盾需进一步研究。可能解释:
1. N=2 时 CNOT 校准误差 (~60% 偏差来源，见 DC-384)
2. T420 饱和线性律在 N=2 时需额外修正因子
3. k₀(0) 需重新拟合 A20 Phase 1-6 完整数据 (N=2→500)

建议：A20 Phase 7 执行时重新拟合 k₀，使用 N=2→1000 跨平台数据。"
```

**状态:** ⏳ 待进一步研究 (P1 优先级)

### C-406-A03: T428 T_crit 平台依赖适用范围

**问题:** T428 预测离子阱 T_crit ~ 24 μK，远低于典型工作温度 (~4K 或 ~100mK)。

**解决方案:**
```markdown
在 T428 中添加适用范围声明:

"T_crit = ℏω_0/(2k_B) 适用于 Ohmic 浴主导退相干的系统 (如超导量子比特)。

其他平台的退相干机制不同，T_crit 公式需修正:
- 离子阱: 主要退相干机制为电场噪声 (1/f 噪声)，T_crit 公式需重新推导
- 光量子: 主要退相干机制为光子损耗，T_crit 定义需重新审视 (光子无静止质量)

⚠️ DC-406 的 T_crit 平台依赖预测 (离子阱~24μK, 光量子~7K, 中性原子~2.4μK)
是基于超导平台公式的外推，实际值可能不同。

建议：A20 Phase 7 执行时测量各平台的 T_crit 实验值，修正理论公式。"
```
