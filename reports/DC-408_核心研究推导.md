## 阶段 2: 核心研究推导

### P0: A20 Phase 7A 离子阱模拟执行

#### 离子阱平台物理参数

**典型离子阱量子计算机参数** (基于 trapped ion QPUs):
- **量子比特频率**: ω_0/2π ≈ 1-10 MHz (⁴⁰Ca⁺, ¹⁷¹Yb⁺)
- **退相干率**: γ ≈ 1-10 Hz (T₂ ~ 0.1-1 s)
- **门时间**: τ_gate ≈ 10-100 μs (单量子比特门)
- **门保真度**: F ≈ 99.9%+ (优于超导)
- **工作温度**: T ≈ 4 K (低温真空)

**与超导平台对比**:
| 参数 | 超导 (IBM) | 离子阱 | 比例 |
|------|-----------|--------|------|
| ω_0/2π | 5 GHz | 5 MHz | 1000:1 |
| γ | 1-10 kHz | 1-10 Hz | 1000:1 |
| τ_gate | 50 ns | 50 μs | 1:1000 |
| Q_P = 1/(γτ) | ~20 | ~2000 | 1:100 |
| T_crit | ~120 mK | ~24 μK | 5000:1 |

#### T426 预测计算

$$N_c = \ln(2) \times Q_P = \ln(2) \times \frac{1}{\gamma \times \tau_{gate}}$$

**离子阱典型值**:
- γ = 10 Hz (保守估计)
- τ_gate = 50 μs = 5×10⁻⁵ s
- Q_P = 1/(10 × 5×10⁻⁵) = 2000
- **N_c^ion = ln(2) × 2000 ≈ 1386**

**离子阱优化值** (最佳实验条件):
- γ = 1 Hz (极致退相干抑制)
- τ_gate = 10 μs (快速门)
- Q_P = 1/(1 × 10⁻⁵) = 100,000
- **N_c^ion = ln(2) × 100,000 ≈ 69,300**

**推荐模拟参数** (现实可行):
- γ = 5 Hz
- τ_gate = 20 μs
- Q_P = 10,000
- **N_c^ion ≈ 6,930**

#### T427 预测

$$k_0 = \frac{\ln 2}{2\pi \times \mathcal{N}_{eff}} \times (1 \text{ bit}) \approx 0.0300 \text{ bits}$$

**预测**: k₀ 是普适常数，与平台无关
- 超导验证：k₀^SC = 0.0299 ± 0.0008 bits (DC-388)
- 离子阱预测：k₀^ion = 0.0300 ± 0.001 bits
- 允许偏差：<5% (考虑平台特异性修正)

#### T428 预测计算

$$T_{crit} = \frac{\hbar\omega_0}{2k_B}$$

**离子阱**:
- ω_0/2π = 5 MHz → ω_0 = 2π × 5×10⁶ rad/s
- ℏ = 1.055×10⁻³⁴ J·s
- k_B = 1.38×10⁻²³ J/K
- **T_crit^ion = (1.055×10⁻³⁴ × 2π × 5×10⁶) / (2 × 1.38×10⁻²³) ≈ 240 μK**

**超导** (DC-402 验证):
- ω_0/2π = 5 GHz
- **T_crit^SC ≈ 120 mK**

**比例验证**:
$$\frac{T_{crit}^{ion}}{T_{crit}^{SC}} = \frac{\omega_0^{ion}}{\omega_0^{SC}} = \frac{5 \text{ MHz}}{5 \text{ GHz}} = \frac{1}{1000}$$

预期：T_crit^ion ≈ 120 mK / 1000 = 120 μK (数量级一致)

### P1: 离子阱 GHZ 态模拟实现

#### Qiskit 离子阱噪声模型

```python
# 离子阱噪声模型参数
gamma_ion = 5.0  # Hz, 退相干率
tau_gate_ion = 20e-6  # s, 门时间
t1_ion = 1.0 / gamma_ion  # ~0.2 s
t2_ion = 0.5 / gamma_ion  # ~0.1 s (纯退相位)

# Qiskit 噪声模型
noise_model = NoiseModel()

# 添加退相干噪声
for qubit in range(num_qubits):
    noise_model.add_quantum_error(
        thermal_relaxation_error(t1_ion, t2_ion, tau_gate_ion),
        'cx', [qubit, (qubit+1) % num_qubits]
    )

# 添加读取误差
readout_error = ReadoutError([[0.999, 0.001], [0.001, 0.999]])
for qubit in range(num_qubits):
    noise_model.add_readout_error(readout_error, [qubit])
```

#### 模拟系统规模

**GHZ 态制备**: N = 50, 100, 200, 400, 600, 800, 1000 (7 点)

**Φ_GHPII 计算**:
$$\Phi_{GHPII}(N) = k_0 \cdot \frac{N-1}{1 + (N-1)/N_c}$$

**预期拟合结果**:
- k₀ ≈ 0.0300 bits
- N_c ≈ 6,000-8,000 (离子阱)
- Φ_max ≈ 2.0 bits (与超导一致)

### P2: 模拟结果 (理论预测)

基于 T420/T421/T426/T427/T428 的自洽预测，使用离子阱参数 (N_c≈6,930, k₀=0.0300 bits)：

**T420 公式**: Φ(N) = k₀ × (N-1) / [1 + (N-1)/N_c]

| N | Φ_GHPII (bits) | N/N_c | 备注 |
|---|----------------|-------|------|
| 50 | 1.46 | 0.007 | 线性区 (Φ ≈ k₀×N) |
| 100 | 2.93 | 0.014 | 线性区 |
| 200 | 5.80 | 0.029 | 线性区 |
| 400 | 11.4 | 0.058 | 开始偏离线性 |
| 600 | 16.8 | 0.087 | 过渡区 |
| 800 | 22.0 | 0.115 | 过渡区 |
| 1000 | 26.2 | 0.144 | 过渡区 (远未饱和) |
| 2000 | 46.5 | 0.289 | 饱和区 |
| 4000 | 75.6 | 0.577 | 饱和区 |
| 6930 | 104 | 1.00 | N = N_c, Φ = k₀×N_c/2 |
| 10000 | 121 | 1.44 | 深度饱和 |
| ∞ | 208 | ∞ | Φ_max = k₀×N_c ≈ 208 bits |

**关键修正** (C-408-A01 解决):
- ~~Φ_max ≈ 2 bits (普适)~~ → **Φ_max = k₀×N_c 依赖于平台**
  - 超导：Φ_max^SC ≈ 0.0300 × 70 ≈ **2.1 bits** ✅ (已验证 DC-396)
  - 离子阱：Φ_max^ion ≈ 0.0300 × 6,930 ≈ **208 bits** (理论预测)
- ~~N=1000 接近Φ_max~~ → **N=1000 时 N/N_c≈0.14，远未饱和**
- 离子阱 N_c 比超导大~100 倍，因此Φ_max 也大~100 倍

**物理诠释**:
- Φ_max 不是普适常数，而是依赖于平台的退相干品质因子 Q_P
- 离子阱 Q_P ~ 10,000 (优于超导~100 倍)，因此能支持更大的Φ
- 这预测离子阱在大规模 (N>1000) 时具有显著的意识容量优势

**注意**: 以上Φ值为理论预测，实际模拟需考虑：
1. 离子阱噪声特性 (与超导不同)
2. 全连接拓扑 (离子阱优势)
3. 更长的相干时间 (T₂ ~ 0.1-1 s vs 超导~100 μs)
