```

```

但数学计算：

**可能原因:**
1. **复制错误:** DC-409 可能复制了超导平台的Φ_max (~2 bits) 而未更新为离子阱值

- N_c = 6,800 与 T426 预测 (N_c = ln(2)×Q_P = ln(2)×10,000 ≈ 6,930) 一致

**但需注意:** 如果Φ_max ≈ 200 bits，则 DC-409 的模拟数据 (Φ 饱和于~2.9 bits) 需要重新解释 — 可能模拟中使用了额外的退相干或测量噪声，导致有效Φ远低于理论上限。
