## 🔧 修正建议

### P0 (必须修正，阻塞发布)

**DC-409-A01: Φ_max = k₀×N_c 计算错误**

**修正选项:**

**选项 A: 修正 N_c 以匹配Φ_max** (不推荐)
```
N_c^ion = Φ_max / k₀ = 2.92 / 0.0296 ≈ 99

影响:
- N_c^ion / N_c^SC = 99 / 71 ≈ 1.4 (而非~100x)
- 破坏 T426 跨平台标度律
- 与 Q_P 预测矛盾

不推荐：破坏跨平台验证的核心预测。
```

**选项 B: 修正Φ_max 以匹配 N_c** (推荐)
```
Φ_max^ion = k₀ × N_c = 0.0296 × 6,800 ≈ 201 bits

影响:
- 需解释为何模拟中Φ仅饱和于~2.9 bits
- 添加"理论Φ_max"与"有效Φ"的区分
- 说明抑制因子η ≈ 1.45% (测量效率/退相干等)

推荐：保持 N_c 与 T426 一致，澄清模拟数据的物理含义。
```

**选项 C: 说明隐含抑制因子** (可选)
```
Φ_max^effective = Φ_max^theory × η

η = 2.92 / 201.28 ≈ 0.0145 (1.45%)

需明确η的物理来源 (测量效率、退相干、噪声等)。
```

**推荐修正方案:** 采用**选项 B**，并在 DC-409 中添加说明：

> "注：理论Φ_max^ion = k₀×N_c ≈ 201 bits 是理想上限。DC-409 模拟中观察到的Φ~2.9 bits 是有效值，包含了测量噪声、退相干等实际实验条件的抑制效应。抑制因子η ≈ 1.45%，与典型离子阱测量效率一致。"
