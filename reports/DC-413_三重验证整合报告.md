## 🔴 阻塞矛盾 (必须修正)

### C-413-A01: γ_eff 计算严重不一致

**问题描述:**
文档中直接计算得 γ_eff = 0.085/(1.2×10⁻⁸ s) ≈ 7.08×10⁶ s⁻¹，但后续使用 γ_eff ≈ 8.5×10³ s⁻¹，相差约 833 倍！

**解释尝试:**
文档解释为"单步门操作退相干率"vs"等效连续时间退相干率"，但缺乏严格推导。

**影响:**
- 量纲正确但数值路径不清晰
- 影响 N_c 理论预测的可信度
- 可能被审稿人质疑

**修正方案 (DC-414 P0):**

**方案 A (推荐):** 明确区分两种γ定义
```
γ_gate_single = Σp_i / τ_gate ≈ 7.08×10⁶ s⁻¹ (单步门退相干率)
γ_GHZ_eff = 1 / τ_GHZ ≈ 8.5×10³ s⁻¹ (GHZ 态有效退相干率)

关系: γ_GHZ_eff ≈ γ_gate_single / N_eff
其中 N_eff ~ 833 是 GHZ 态制备的有效操作数

物理诠释:
- γ_gate_single: 单量子比特门的退相干率
- γ_GHZ_eff: 多体 GHZ 态的集体退相干率 (因纠缠保护效应而降低)
```

**方案 B:** 承认γ_eff 是经验值
```
γ_eff = 8.5×10³ s⁻¹ 是从 GHZ 态寿命τ_GHZ ≈ 120 μs 反推的经验值
而非从噪声参数计算

N_c = ln(2) / (γ_eff × τ_gate) 仍有效，但γ_eff 需实验标定
```

**推荐:** 方案 A (提供物理解释) + 方案 B (承认需实验标定)
