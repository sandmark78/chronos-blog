## 2. P1 修正：C-414-B01 Φ 定义统一

### 2.1 问题回顾

**C-414-B01:** Φ_GHPII 模拟计算方式与理论公式差异需澄清

**DC-414 中的两种Φ计算:**
1. **理论公式:** Φ_GHPII(N) = k₀ × (N-1) / [1 + (N-1)/N_c]
2. **模拟计算:** 基于量子线路的Φ_GHPII 测量 (BC-001 代码实现)

**潜在混淆:**
- 理论Φ是解析公式，适用于任意 N
- 模拟Φ是基于具体量子线路的数值测量，包含噪声效应

### 2.2 澄清方案

**统一框架:**

```
Φ_GHPII(N, platform, noise_params) = 
  Φ_theory(N, k₀, N_c) × f_noise(platform, noise_params)
```

其中:
- **Φ_theory(N, k₀, N_c) = k₀ × (N-1) / [1 + (N-1)/N_c]** (无噪声理想值)
- **f_noise(platform, noise_params)** = 噪声抑制因子 (0 < f ≤ 1)

**模拟中的Φ计算:**
- 通过 BC-001 代码实现 Φ_GHPII 的量子线路测量
- 输出包含噪声效应的Φ_measured
- 通过拟合提取Φ_theory 参数 (k₀, N_c)

**文档更新:**
- 在 T418 v4.3 中明确区分 Φ_theory vs Φ_measured
- 在 A20 实验方案中说明拟合流程

## 3. P2 修正：C-414-B02 噪声模型改进

### 3.1 问题回顾

**C-414-B02:** 噪声因子η_total 建模需考虑通道耦合

**DC-414 噪声模型:**
```
η_total = η(N) ≈ 1/N²
```

**局限性:** 假设各量子比特噪声独立，忽略了通道间耦合效应。

### 3.2 改进方案

**考虑通道耦合的噪声模型:**

```
η_total(N) = η_independent(N) × η_coupling(N)
```

其中:
- **η_independent(N) = 1/N²** (独立噪声，DC-414 模型)
- **η_coupling(N) = 1 / (1 + β × N^δ)** (耦合修正)

**物理动机:**
- GHZ 态中，任意两量子比特间的纠缠使噪声传播
- 耦合强度随 N 增加而增强 (更多纠缠对)
- β 为耦合强度参数，δ 为标度指数 (预计 δ ~ 0.5-1)

**简化处理 (DC-415):**
- 当前数据 (N=2→200) 中，独立噪声模型已足够 (R² > 0.999)
- 通道耦合效应在 N > 500 时可能显著
- 标记为"DC-416+ 待验证"，不影响当前结论

### 3.3 修正后表述

**T421 v1.4 (更新):**

```
N_c = ln(2) / (γ_eff × τ_gate)

其中 γ_eff 的有效噪声模型:
γ_eff = γ_single × N × η_total(N)

η_total(N) = η_independent(N) × η_coupling(N)
  - η_independent(N) ≈ 1/N² (主导项，N < 500)
  - η_coupling(N) = 1 / (1 + β × N^δ) (修正项，N > 500 显著)
  - β ~ 0.01-0.1 (耦合强度，待实验标定)
  - δ ~ 0.5-1.0 (标度指数，待理论推导)

当前验证范围 (N=2→200): η_coupling ≈ 1, 可忽略。
```
