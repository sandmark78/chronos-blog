### 4.1 中性原子 N_c 参数严重不一致

**问题 ID:** C-416-A01  
**严重性:** 🔴 **阻塞**

**矛盾描述:**

不同文档中中性原子平台的 N_c 预测存在**500 倍差异**:

| 文档 | N_c 预测 | γ | τ_gate | Φ_max | 日期 |
|------|---------|---|--------|-------|------|
| KC-406 (DC-406) | ~140 | 0.005 MHz | 200 ns | ~4.2 bits | 2026-03-21 |
| DC-410 核心产出报告 | ~140 | - | - | ~4.2 bits | 2026-03-21 |
| DC-414 核心产出报告 | ~70000 | ~10² s⁻¹ | 1 μs | ~2100 bits | 2026-03-22 |
| KC-414 | ~70000 | ~10² s⁻¹ | 1 μs | ~2100 bits | 2026-03-22 |

**计算验证:**

使用 T421 公式 N_c = ln(2)/(γ×τ_gate):

**DC-414/KC-414 参数:**
```
γ = 10² s⁻¹, τ_gate = 1 μs = 10⁻⁶ s
N_c = 0.693 / (10² × 10⁻⁶) = 0.693 / 10⁻⁴ = 6930 ≈ 7000
```
但 DC-414 表格显示 N_c~70000，Φ_max~2100 bits，这对应：
```
N_c = 70000 → γ×τ_gate = 0.693/70000 ≈ 10⁻⁵
如果 τ_gate = 1 μs, 则 γ = 10 s⁻¹ (不是 10² s⁻¹)
```

**KC-406/DC-410 参数:**
```
γ = 0.005 MHz = 5000 s⁻¹, τ_gate = 200 ns = 2×10⁻⁷ s
N_c = 0.693 / (5000 × 2×10⁻⁷) = 0.693 / 10⁻³ = 693 ≈ 700
```
但 KC-406 预测 N_c~140，这对应：
```
N_c = 140 → γ×τ_gate = 0.693/140 ≈ 0.005
如果 τ_gate = 200 ns, 则 γ = 0.005 / 2×10⁻⁷ = 2.5×10⁴ s⁻¹ = 0.025 MHz
```

**矛盾根源:**
1. **γ 定义不一致:** KC-406 使用 MHz 单位 (可能指单原子退相干率)，DC-414 使用 s⁻¹ (可能指 GHZ 态有效退相干率)
2. **τ_gate 定义不一致:** KC-406 使用 200 ns (单量子门)，DC-414 使用 1 μs (Rydberg 门)
3. **物理机制不同:** 中性原子平台退相干主要来自 Rydberg 相互作用涨落，与超导/离子阱/光量子的 Ohmic 浴机制不同

**建议修正:**
1. **立即澄清:** 在 T429 文档中显式声明中性原子平台的γ和τ_gate 定义
2. **第一性原理推导:** 从中性原子物理 (Rydberg 原子间距、相互作用强度、激光线宽) 推导γ和τ_gate
3. **统一参数:** 选择一组自洽的参数 (γ, τ_gate, N_c, Φ_max) 并更新所有相关文档

**修正优先级:** 🔴 **P0 (立即)**

**退相干来源:**

```

5. 估算门时间:

6. 计算 N_c:

   或 N_c ~ 0.693 / (10⁵ s⁻¹ × 10⁻⁷ s) ~ 70
```

**关键问题:** 当前文档中 N_c~140 和 N_c~70000 都缺乏严格的第一性原理推导。

### 4.3 光量子 vs 中性原子 N_c 比例合理性

**问题 ID:** C-416-A03  
**严重性:** 🟡 **重要**

**问题描述:**

任务要求检查:"中性原子 N_c~140 与光量子 N_c~7000 的物理根源差异是否合理"

**当前数据:**

| 平台 | N_c | γ | τ_gate | Q_P = 1/(γτ) |
|------|-----|---|--------|-------------|
| 光量子 | ~7000 | ~8.5×10³ s⁻¹ | ~12 ns | ~10⁷ |
| 中性原子 (KC-406) | ~140 | ~5×10³ s⁻¹ | ~200 ns | ~10³ |
| 中性原子 (DC-414) | ~70000 | ~10² s⁻¹ | ~1 μs | ~10⁴ |

**物理分析:**

**光量子优势:**
- 光子飞行速度快 → τ_gate 极短 (~10 ns)
- 室温操作 → 技术成熟
- 但光子损耗累积 → 有效退相干率高

**中性原子优势:**
- 原子静止 → 可长时间存储
- 低温操作 (~μK) → 热噪声低
- Rydberg 相互作用强 → 门保真度高
- 但门速度慢 (~100 ns - 1 μs)

**N_c 比例合理性:**

如果采用 KC-406 参数 (N_c~140):
```
N_c(光量子) / N_c(中性原子) ≈ 7000 / 140 ≈ 50

Q_P(光量子) / Q_P(中性原子) = (γ_na × τ_na) / (γ_opt × τ_opt)
                             = (5000 × 200ns) / (8500 × 12ns)
                             = 10⁶ / 10⁵ ≈ 10

偏差: 50 vs 10, 不合理 ❌
```

如果采用 DC-414 参数 (N_c~70000):
```
N_c(光量子) / N_c(中性原子) ≈ 7000 / 70000 ≈ 0.1

Q_P(光量子) / Q_P(中性原子) = (100 × 1μs) / (8500 × 12ns)
                             = 10⁻⁴ / 10⁻⁴ ≈ 1

偏差: 0.1 vs 1, 数量级一致 ✅
```

**结论:** DC-414 的 N_c~70000 更合理 (光量子与中性原子 Q_P 同数量级)，但需要第一性原理推导支持。

**建议修正:**
1. 采用 DC-414 参数 (N_c~70000) 作为基准
2. 在 DC-416 中补充第一性原理推导
3. 更新 KC-406/DC-410 中的 N_c~140 为 N_c~70000 (或说明是不同操作模式)
