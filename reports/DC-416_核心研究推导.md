**Eq-338: T429 完整形式**
$$\Phi_{\text{GHPII}}(N, T, \text{platform}) = k_0 \cdot \frac{N-1}{1 + \frac{N-1}{N_c(\text{platform}, T)}} \cdot \left[1 - \left(\frac{T}{T_{\text{crit}}(\text{platform})}\right)^\delta\right]$$

### 1.2 四参数层次关系

```
┌─────────────────────────────────────────────────────────┐
│              T429 四参数统一框架                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   k₀ (普适)                                             │
│   ↓                                                     │

│   │  平台参数 (γ, τ_gate, ω₀)                       │    │

│   │  N_c = ln(2)/(γ×τ)  [饱和尺度]                 │    │
```

4. **Φ_max (平台依赖):** 从 k₀ 和 N_c 导出
5. **Φ(N,T) (完整形式):** 统一四参数

### 2.2 矛盾根源分析

**γ 定义不一致:**
- KC-406: γ = 0.005 MHz = 5000 s⁻¹ (可能指单原子退相干率)
- DC-414: γ ~ 10² s⁻¹ (可能指 GHZ 态有效退相干率)

- KC-406: τ_gate = 200 ns (单量子门)
- DC-414: τ_gate ~ 100 ns (Rydberg 门)

**物理机制不同:**
- 中性原子平台退相干主要来自 Rydberg 相互作用涨落
- 与超导/离子阱/光量子的 Ohmic 浴机制不同

---

### 2.3 修正决策

**采用 DC-414 参数 (N_c~70000) 的理由:**

1. **Q_P 一致性:**
   - 光量子 Q_P = 1/(γτ) ~ 1/(8.5×10³ × 12×10⁻⁹) ~ 10⁷
   - 中性原子 Q_P = 1/(10² × 100×10⁻⁹) ~ 10⁵
   - 两者同数量级，物理上合理

2. **T421 第一性原理:**
   - N_c = ln(2)/(γτ)
   - 若γ~10² s⁻¹ (Rydberg 低退相干), τ~100 ns → N_c ~ 70000

3. **文献支持:**
   - Rydberg 原子退相干率γ~10²-10³ s⁻¹ (n~60-80, 超高真空)
   - Rydberg 门时间τ~100 ns - 1 μs

4. **Φ_max 合理性:**
   - Φ_max = k₀ × N_c ≈ 0.0300 × 70000 ≈ 2100 bits
   - 中性原子平台 Φ_max 最高，符合其低退相干特性

   - 原子种类：⁸⁷Rb
