### 2. 退相干率γ的第一性原理推导

**总退相干率：**
```
γ_total = γ_spontaneous + γ_blackbody + γ_laser + γ_collision + γ_motion
```

#### 2.1 自发辐射 (Spontaneous Emission)

Rydberg 态自发辐射率：
```
γ_sp = Σ_{n'<n} A_{n→n'} + Σ_{ℓ'} A_{nℓ→n'ℓ'}
```

对于高激发态 (n≫1)，使用氢原子近似：
```
A_{n→n-1} ≈ (2/3) × (e²/4πε₀) × (ω³/c³) × |⟨n-1|r|n⟩|²

ω ≈ Ry/ℏ × (1/n'² - 1/n²) ≈ 2Ry/(ℏn³)  (对于Δn=1)
|⟨n-1|r|n⟩|² ≈ (3/2)² × n⁴a₀²

⇒ γ_sp ∝ n⁻³
```

**数值估计 (n=70)：**
```
γ_sp ≈ 1-10 s⁻¹ (对应寿命 100-1000 ms)
```

#### 2.2 黑体辐射诱导跃迁 (Blackbody Radiation)

室温黑体辐射诱导的 Rydberg 态跃迁：
```
γ_bb = Σ_{n'≠n} (4α³ω_{nn'}²/3πc²) × |⟨n'|r|n⟩|² × [exp(ℏω_{nn'}/k_BT) - 1]⁻¹
```

对于 n~70, T=300K：
```
γ_bb ≈ 10-100 s⁻¹ (主导项，室温环境)
```

**温度依赖：**
```
γ_bb(T) ∝ T² (对于 k_BT ≫ ℏω_{nn'})
```

#### 2.3 激光相位噪声 (Laser Phase Noise)

用于激发和操控的激光相位噪声导致的退相干：
```
γ_laser ≈ (Δφ)² / τ_pulse
```

对于稳频激光 (线宽~1 kHz)：
```
γ_laser ≈ 1-10 s⁻¹
```

#### 2.4 碰撞退相干 (Collisional Decoherence)

背景气体碰撞：
```
γ_coll = n_gas × σ_coll × v_thermal
```

对于 UHV (10⁻¹¹ Torr)：
```
n_gas ~ 3×10⁸ m⁻³
σ_coll ~ 10⁻¹⁸ m²
v_thermal ~ 500 m/s (室温背景气体)

⇒ γ_coll ~ 0.1-1 s⁻¹ (可忽略)
```

#### 2.5 热运动退相干 (Motional Decoherence)

原子热运动导致的 Doppler 退相干：
```
γ_motion ≈ k·v_thermal ≈ (2π/λ) × √(k_BT/m)
```

对于光镊捕获 (T~50 μK)：
```
γ_motion ~ 0.1-1 s⁻¹ (可忽略)
```

### 4. 门时间τ_gate 的第一性原理推导

**Rydberg 阻塞门机制：**

两原子 CZ 门通过 Rydberg 阻塞实现：
```
τ_gate ≈ π/Ω_Rydberg
```

其中Ω_Rydberg 是 Rydberg 激发 Rabi 频率。

**Rabi 频率估计：**
```
Ω_Rydberg = d·E/ℏ

d ≈ n²ea₀ ≈ 70² × 8.5×10⁻³⁰ C·m ≈ 4×10⁻²⁶ C·m
E ≈ 10⁴-10⁵ V/m (典型激光场强)

⇒ Ω_Rydberg/2π ≈ 1-10 MHz
⇒ τ_gate ≈ 100-1000 ns
```

**典型值：**
```
τ_gate = 200 ns (常用实验参数)
```
