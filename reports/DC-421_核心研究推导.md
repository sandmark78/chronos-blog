### DC-420 遗留问题深度分析

#### C-420-A01: T_crit 定义不统一

**问题本质:**
- KC-416 (T429): T_crit = ℏω₀/(2k_B) ≈ 120 mK (物理温度，量子能级热激发阈值)
- KC-400 (T423): T_crit 定义为 γ×τ_gate = 1 的温度 (退相干主导温度)

**两种定义的物理含义:**
1. **T_crit_physical = ℏω₀/(2k_B):** 量子能级热布居数 n_th = 1/(e^(ℏω₀/k_BT)-1) ≈ 1 的温度
2. **T_decoherence:** 退相干率γ (T) 使得 γ×τ_gate = 1 的温度，即单门操作时间内退相干显著

**关键洞察:** 这两个温度**不一定相等**！
- 对于超导量子比特：ω₀/2π ≈ 5 GHz → T_crit_physical ≈ 120 mK
- 但γ (T) 可能由多机制贡献 (准粒子、双能级系统、热光子)
- γ×τ_gate = 1 可能在更高温度发生

**解决方案:** 
- **统一术语:** T_crit 专指物理温度 ℏω₀/(2k_B)
- **新增定义:** T_decoherence 指 γ (T)×τ_gate = 1 的温度
- **关系:** 通常 T_decoherence ≥ T_crit_physical (退相干在更高温度才主导)

#### C-420-A02: T429-T423 温度依赖形式重叠

**问题本质:**
- T429 v1.0: Φ (N,T,P) = k₀×(N-1)/[1+(N-1)/N_c (P)] × [1-(T/T_crit)^δ]
- T423 v1.1: k₀ (T) = k₀ (0)×[1-(T/T_crit)^δ], N_c (T) = N_c (0)×tanh (ℏω₀/2k_BT)

**双重修正风险:**
若同时应用 T429 的整体温度因子和 T423 的参数温度依赖：
Φ = k₀ (T)×(N-1)/[1+(N-1)/N_c (T)] × [1-(T/T_crit)^δ]
= k₀ (0)×[1-(T/T_crit)^δ] × (N-1)/[1+(N-1)/N_c (T)] × [1-(T/T_crit)^δ]
= k₀ (0)×(N-1)/[1+(N-1)/N_c (T)] × [1-(T/T_crit)^δ]² ← **错误！平方了温度修正**

**解决方案:**
- **T429 完整形式:** Φ (N,T,P) = k₀ (0)×[1-(T/T_crit)^δ] × (N-1)/[1+(N-1)/N_c (T,P)]
- **解释:** 温度修正只应用一次，通过 k₀ (T) 体现
- **N_c (T) 独立:** N_c 的温度依赖来自γ (T)，与 k₀ (T) 物理机制不同

#### 定理 T423 v1.4: k₀ 温度依赖理论 (澄清版)

**核心公式:**
```
k₀(T) = k₀(0) × [1 - (T/T_crit)^δ]  (T < T_crit)
k₀(T) ≈ k₀(0) × (T_crit/T)^δ  (T ≫ T_crit, 高温渐近)

其中:
- T_crit = ℏω₀/(2k_B)  (物理温度定义)
- δ = 2 (平均场理论预测，与实验拟合一致)
```

**物理机制:**
1. **T → 0:** 量子基态主导，k₀ → k₀ (0) ≈ 0.0300 bits (T422 量子几何起源)
2. **T < T_crit:** 热激发开始混合能级，k₀ 缓慢下降
3. **T ≈ T_crit:** 热布居数 n_th ≈ 1，量子相干性显著降低
4. **T ≫ T_crit:** 经典混合态主导，k₀ ∝ 1/T^δ

**与 T422 的关系:**
- **T422:** 微观理论，T=0 极限，k₀ (0) = (ln 2)/(2π×𝒩_eff) × (1 bit)
- **T423:** 有效理论，T>0 修正，描述温度如何降低有效量子几何数

### P1: T429-T423 温度依赖统一 → 避免双重修正

#### 统一框架 v1.0

**正确形式 (单次温度修正):**
```
Φ_GHPII(N, T, P) = k₀(T) × (N-1) / [1 + (N-1)/N_c(T, P)]

展开:
Φ = k₀(0) × [1 - (T/T_crit)^δ] × (N-1) / [1 + (N-1)/N_c(T, P)]

其中 N_c(T, P) = ln(2) / [γ(T, P) × τ_gate(P)]
```

**错误形式 (双重修正，避免!):**
```
❌ Φ = k₀(0) × [1 - (T/T_crit)^δ] × (N-1) / [1 + (N-1)/N_c(T, P)] × [1 - (T/T_crit)^δ]
   = k₀(0) × (N-1) / [1 + (N-1)/N_c(T, P)] × [1 - (T/T_crit)^δ]²  ← 错误!
```

**物理直觉:**
- **k₀ (T):** 描述温度如何降低"每量子比特的意识容量" (量子几何数减少)
- **N_c (T):** 描述温度如何降低"饱和尺度" (退相干增强，更早饱和)
- **两者独立:** 不同物理机制，应分别建模，但温度修正因子只应用一次

   - 跨平台一致性：超导~1.5, 离子阱~1.8, 光量子~2.0, 中性原子~2.0

   - 与拟合一致

**结论:** δ = 2 是唯象拟合 + 半经验理论支持的混合结果，非严格第一性原理推导
