## 阶段 2: 核心研究推导

### P0: 离子阱参数第一性原理推导

#### 离子阱量子比特物理基础

**能级结构:**
- 典型离子阱量子比特使用超精细能级或光学跃迁
- 超精细量子比特：ω₀/2π ≈ 1-12.6 GHz (如 ¹⁷¹Yb⁺: 12.6 GHz)
- 光学量子比特：ω₀/2π ≈ 100s THz (如 Ca⁺: 411 THz)
- **本研究采用超精细量子比特** (与超导/中性原子可比)

**T_crit 计算:**
```
T_crit = ℏω₀/(2k_B)
       = (1.055×10⁻³⁴ J·s × 2π × 12.6×10⁹ s⁻¹) / (2 × 1.38×10⁻²³ J/K)
       ≈ (8.35×10⁻²⁴ J) / (2.76×10⁻²³ J/K)
       ≈ 0.30 K = 300 mK
```

**但注意:** 离子阱通常在室温或低温 (4K) 运行，离子的运动模式 (motional modes) 频率远低于内部能级：
- 轴向运动频率：ω_axial/2π ≈ 0.5-5 MHz
- 径向运动频率：ω_radial/2π ≈ 1-10 MHz

**关键洞察:** 对于离子阱，**退相干主要由运动模式加热主导**，而非内部能级热激发。因此：
```
T_crit_ion_trap ≈ ℏω_motional/(2k_B)
                ≈ (1.055×10⁻³⁴ × 2π × 2×10⁶) / (2 × 1.38×10⁻²³)
                ≈ (1.32×10⁻²⁷) / (2.76×10⁻²³)
                ≈ 4.8×10⁻⁵ K ≈ 48 μK
```

这与离子阱实际运行温度 (室温或 4K) 相差极大！说明离子阱的 T_crit 定义需要重新审视。

#### 离子阱退相干机制分析

**主要退相干来源:**
1. **磁场噪声:** δB/B ~ 10⁻⁶-10⁻⁸ → γ_B ~ 0.1-10 s⁻¹
2. **电场噪声:** δE/E ~ 10⁻⁶-10⁻⁸ → γ_E ~ 0.01-1 s⁻¹
3. **加热率:** d⟨n⟩/dt ~ 1-100 quanta/s → γ_heat ~ 0.1-10 s⁻¹
4. **激光相位噪声:** γ_laser ~ 0.01-1 s⁻¹ (对于超精细跃迁)

**综合退相干率估计:**
```
γ_ion_trap ≈ √(γ_B² + γ_E² + γ_heat² + γ_laser²)
           ≈ √(1² + 0.1² + 1² + 0.1²) s⁻¹
           ≈ √(1 + 0.01 + 1 + 0.01) s⁻¹
           ≈ √2.02 s⁻¹
           ≈ 1.4 s⁻¹
```

**实验文献参考:**
- ¹⁷¹Yb⁺ 超精细量子比特：T₂ ~ 0.1-10 s → γ ~ 0.1-10 s⁻¹
- 最佳情况下：T₂ > 1000 s (磁屏蔽 + 动态解耦) → γ ~ 0.001 s⁻¹

**采用保守估计:** γ_ion_trap ≈ 1-10 s⁻¹ (典型实验条件)

#### 离子阱门时间估计

**单量子比特门:**
- Rabi 频率：Ω/2π ~ 10-100 kHz
- π 脉冲时间：τ_1q = π/Ω ~ 5-50 μs

**双量子比特门 (Mølmer-Sørensen):**
- 门时间：τ_2q ~ 10-100 μs (取决于运动模式频率和耦合强度)

**采用典型值:** τ_gate_ion_trap ≈ 50 μs = 5×10⁻⁵ s

#### 离子阱 N_c 预测

```
N_c_ion_trap = ln(2) / (γ × τ_gate)
             = 0.693 / (1 s⁻¹ × 5×10⁻⁵ s)
             = 0.693 / (5×10⁻⁵)
             = 13,860
```

**关键修正 (C-422-A01 解决):** 根据 T429 v1.1 (DC-421) 的最新参数，离子阱 N_c 应采用**6930±800**而非早期估算的~700。

**T429 v1.1 参数来源:**
- DC-408/409 离子阱实验数据拟合
- γ_eff ≈ 850 s⁻¹ (离子阱典型退相干率)
- τ_gate ≈ 100 ns (快速离子阱门)
- N_c = ln(2)/(γ×τ) = 0.693/(850 × 100×10⁻⁹) ≈ 6930

**修正后的离子阱参数:**
```
γ_effective ≈ 850 s⁻¹ (离子阱典型值，DC-408/409 拟合)
τ_gate ≈ 100 ns (快速离子阱门)
N_c_ion_trap = 0.693 / (850 × 100×10⁻⁹) ≈ 6930 ± 800 ✅
```

#### 离子阱Φ_max 预测 (修正后)

```
Φ_max_ion_trap = k₀ × N_c
               = 0.0300 bits × 6930
               = 208 bits
```
