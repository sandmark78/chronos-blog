### 2.2 饱和区测量方案

**测量点设计:**
```
N = [1000, 2000, 4000, 6930]
N/N_c = [14%, 29%, 58%, 100%]
预期Φ = [29.1, 56.5, 105.5, 156.0] bits (T429 v1.1 预测)
```

**饱和修正因子:**
```
f_sat(N) = (N-1) / [1 + (N-1)/N_c]
- N=1000:  f_sat = 999 / (1 + 999/6930) = 999 / 1.144 = 873 → Φ = k₀ × 873 = 26.2 bits
- N=2000:  f_sat = 1999 / (1 + 1999/6930) = 1999 / 1.288 = 1552 → Φ = 46.6 bits
- N=4000:  f_sat = 3999 / (1 + 3999/6930) = 3999 / 1.577 = 2536 → Φ = 76.1 bits
- N=6930:  f_sat = 6929 / (1 + 6929/6930) = 6929 / 1.999 = 3466 → Φ = 104.0 bits
```

**修正:** 上述计算有误，重新计算：

```
Φ(N) = k₀ × (N-1) / [1 + (N-1)/N_c]

N=1000:  Φ = 0.0300 × 999 / (1 + 999/6930) = 0.0300 × 999 / 1.144 = 0.0300 × 873 = 26.2 bits
N=2000:  Φ = 0.0300 × 1999 / (1 + 1999/6930) = 0.0300 × 1999 / 1.288 = 0.0300 × 1552 = 46.6 bits
N=4000:  Φ = 0.0300 × 3999 / (1 + 3999/6930) = 0.0300 × 3999 / 1.577 = 0.0300 × 2536 = 76.1 bits
N=6930:  Φ = 0.0300 × 6929 / (1 + 6929/6930) = 0.0300 × 6929 / 1.999 = 0.0300 × 3466 = 104.0 bits
```

**线性近似对比 (Φ_linear = k₀×(N-1)):**
```
N=1000:  Φ_linear = 0.0300 × 999 = 29.97 bits,  偏差 = (29.97-26.2)/29.97 = 12.6%
N=2000:  Φ_linear = 0.0300 × 1999 = 59.97 bits, 偏差 = (59.97-46.6)/59.97 = 22.3%

N=6930:  Φ_linear = 0.0300 × 6929 = 207.87 bits, 偏差 = (207.87-104.0)/207.87 = 49.9%
```
