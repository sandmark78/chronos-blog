```

温度比分析:
- 若 T_crit^A < T_crit^B: 温度比 < 1 (A 平台更接近临界) ✅
- 若 T_crit^A > T_crit^B: 温度比 > 1 (但此时 min/max 可能补偿) ⚠️

极端情况:
T_crit^A = 100 mK, T_crit^B = 1 K, T = 50 mK
```

### 🔴 物理冲突 C-DC427-A01: T431 效率可能超 100%

**问题描述:**

T431 v1.2 的温度比因子 [1-(T/T_crit^B)^δ]/[1-(T/T_crit^A)^δ] 在 T_crit^A > T_crit^B 时可能 > 1，结合 N_c 比例和 F，总效率η可能超过 100%，违反热力学第二定律。

**示例:**
```
平台 A (中性原子): T_crit^A = 1 K, N_c^A = 47200
平台 B (离子阱):   T_crit^B = 0.5 K, N_c^B = 6930
温度 T = 0.1 K, δ = 1, F = 0.97

温度比 = [1-(0.1/0.5)]/[1-(0.1/1)] = 0.8/0.9 = 0.89
N_c 比例 = min(47200,6930)/max(47200,6930) = 6930/47200 = 0.147

η = 0.147 × 0.97 × 0.89 = 0.127 < 1 ✅ (此例安全)

但反向转换 (B→A):
温度比 = [1-(0.1/1)]/[1-(0.1/0.5)] = 0.9/0.8 = 1.125
N_c 比例 = 6930/47200 = 0.147

η = 0.147 × 0.97 × 1.125 = 0.161 < 1 ✅ (仍安全)

极端情况:
T_crit^A = 10 K, T_crit^B = 0.1 K, T = 0.05 K
温度比 (B→A) = [1-(0.05/10)]/[1-(0.05/0.1)] = 0.995/0.5 = 1.99

若 N_c^A/N_c^B ≈ 1 (相近平台):
η = 1 × 0.95 × 1.99 = 1.89 > 1 ❌
```

**建议修正:**

```
方案 A (推荐): 添加η≤1 截断
η_{A→B} = min(1, min(N_c^A,N_c^B)/max(N_c^A,N_c^B) × F × 温度比)

方案 B: 重新推导温度比，确保η≤1 内禀
可能需修改温度比形式为：
温度比 = min([1-(T/T_crit^B)^δ], [1-(T/T_crit^A)^δ]) / max(...)
或 温度比 = exp(-|T_crit^A - T_crit^B|/T_0) 形式

方案 C: 标注适用范围
明确 T431 仅适用于 T_crit^A ≈ T_crit^B 的平台转换 (偏差<2 倍)
```
