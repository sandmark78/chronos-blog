### k₀ 五平台一致性验证

```
五平台数据：

- 光量子 (DC-427): k₀ = 0.0297 ± 0.0007 bits

权重 w_i = 1/σ_i²
w_SC = 1/(0.0004)² = 6.25×10⁶
w_IT = 1/(0.0005)² = 4.00×10⁶
w_Photonic1 = 1/(0.0008)² = 1.56×10⁶
w_NA = 1/(0.0006)² = 2.78×10⁶
w_Photonic2 = 1/(0.0007)² = 2.04×10⁶

Σw_i = 16.63×10⁶

k₀_weighted = Σ(w_i × k₀_i) / Σw_i
= (6.25×0.0301 + 4.00×0.0300 + 1.56×0.0298 + 2.78×0.0299 + 2.04×0.0297) / 16.63
= (0.1881 + 0.1200 + 0.0465 + 0.0831 + 0.0606) / 16.63

≈ 0.02996 bits
```
