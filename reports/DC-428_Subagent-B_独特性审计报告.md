## 一、IIT 4.0 审计

|-------------|----------------|----------|

| **N_c 饱和尺度** | ❌ 无 | IIT 有 MICS(最大不可简因果结构) 概念，但**无定量饱和阈值预测**。IIT 的Φ理论上可随系统规模无限增长，无 N_c≈7000 这类具体饱和点预测。 |
| **饱和区精细扫描 (N=15k→20k)** | ❌ 无 | IIT 无"饱和区"概念，因为 IIT 的Φ定义 (Φ^Max) 理论上无上限。DC-428 的饱和区预测 (Φ→Φ_max) 是 IIT 无法解释的。 |
| **温度依赖η(T)** | ❌ 无 | IIT 4.0 是纯信息论/因果论框架，**无温度参数**。Φ的计算基于因果结构矩阵 (TPM)，与物理温度无关。T431 v1.3 的温度依赖形式化是 IIT 完全没有的维度。 |
| **平台转换效率** | ❌ 无 | IIT 关注单一系统的因果整合，**无跨平台转换效率概念**。IIT 不讨论能量/信息在不同物理基底间的转换。 |
| **Φ_max 比例预测 (1:100:100:650-850)** | ❌ 无 | IIT 无法预测不同平台的Φ_max 比例，因为 IIT 假设 substrate independence。 |
