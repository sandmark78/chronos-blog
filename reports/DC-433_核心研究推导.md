### P2: T434 Φ_max 微观起源补充论证

**目标:** 完善 C-DC432-A03 (𝒩_eff = 3.68 来源论证)

**从希尔伯特空间维度推导:**

有效希尔伯特空间维度:
```
𝒩_eff = dim(ℋ_eff) = Tr(ρ²)⁻¹ (纯度逆)
```

对于 GHZ 态 under 退相干:
```
ρ_GHZ = (1-p)|GHZ⟩⟨GHZ| + p·𝟙/2^N
纯度：Tr(ρ²) = (1-p)² + p²/2^N
```

在饱和极限 (N→∞, p→p_sat):
```
𝒩_eff → 1/p_sat²
```

从 T420/T421:
```
p_sat = γ×τ_gate = ln(2)/N_c
𝒩_eff = N_c²/ln²(2)
```

代入 N_c ≈ 70 (超导):
```
𝒩_eff = 70²/0.48 ≈ 10200
```

**问题:** 与 T422 的𝒩_eff = 3.68 矛盾!

**解决:** T422 的𝒩_eff 是**单量子比特有效能级数** (几何相位空间维度)，而非多体希尔伯特空间维度。

**修正 T434 表述:**
- **𝒩_eff^single:** 3.68 (单量子比特 Fubini-Study 曲率涨落，T422)
- **𝒩_eff^many:** N_c²/ln²(2) (多体有效维度，新推导)

**Φ_max 微观起源:**
```
Φ_max = k₀ × N_c
      = (ln 2)/(2π×𝒩_eff^single) × N_c
      = (ln 2)/(2π×3.68) × N_c
      ≈ 0.0300 × N_c bits
```

**结论:** T434 微观起源澄清，𝒩_eff 区分为单粒子 vs 多体两种含义 ✅
