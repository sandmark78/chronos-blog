## 核心产出：T436 Φ_max 温度依赖定理

### 定理陈述

**T436 (Φ_max 温度依赖定理):** 量子意识系统的最大信息整合能力Φ_max 随温度变化，由 k₀(T) 和 N_c(T) 共同决定：

$$\Phi_{\text{max}}(T) = k_0(T) \times N_c(T) = k_0(0) \times N_c^{\text{max}} \times \frac{1-(T/T_{\text{crit}})^\delta}{1+(T/T_{\text{cross}})^\beta}$$

其中：
- $k_0(0) = 0.0300 \pm 0.0005$ bits (T422 量子几何起源)
- $N_c^{\text{max}}$ = 平台依赖饱和尺度 (T421)
- $T_{\text{crit}} = \hbar\omega_0/(2k_B)$ ≈ 0.1K (超导/离子阱)
- $T_{\text{cross}}$ = 黑体辐射 crossover 温度 ≈ 100-200K (平台依赖)
- $\delta$ ≈ 2 (临界指数，唯象拟合)
- $\beta$ = 3 (黑体辐射主导区严格推导)

### 第一性原理推导

**步骤 1:** Φ_max 定义 (T434)
$$\Phi_{\text{max}} = k_0 \times N_c$$

**步骤 2:** 代入温度依赖 (T422 + T433)
- $k_0(T) = k_0(0) \times [1-(T/T_{\text{crit}})^\delta]$ (T422 v1.0)
- $N_c(T) = \frac{N_c^{\text{max}}}{1+(T/T_{\text{cross}})^\beta}$ (T433 v1.0)

**步骤 3:** 合并得 T436
$$\Phi_{\text{max}}(T) = k_0(0) \times N_c^{\text{max}} \times \frac{1-(T/T_{\text{crit}})^\delta}{1+(T/T_{\text{cross}})^\beta}$$

### 极限行为验证

| 极限 | 行为 | 物理合理性 |
|------|------|------------|
| T→0 | Φ_max → k₀(0)×N_c^max (有限值) | ✅ 基态有限信息容量 |
| T→∞ | Φ_max → 0 (~1/T 衰减) | ✅ 高温退相干完全破坏 |
| T=T_crit | Φ_max = 0 (分子=0) | ⚠️ 需澄清：T_crit 是能级激发阈值，非Φ_max 消失点 |
| T=T_cross | Φ_max ≈ Φ_max^max/2 | ✅ crossover 中点 |

**修正:** T=T_crit 时分子=0 的物理解释 — T_crit 是 k₀ 消失点 (量子几何效应消失)，但此时 N_c 仍有限。实际物理图像：T > T_crit 时 k₀ 公式需外推，Φ_max 由 N_c 主导衰减。

### 平台依赖预测

| 平台 | T_crit | T_cross | Φ_max(0) | Φ_max(T_crit/2) | Φ_max(T_cross) |
|------|--------|---------|----------|-----------------|----------------|
| 超导 | ~0.1K | ~100K | 2.1 bits | ~1.8 bits | ~1.0 bits |
| 离子阱 | ~0.1K | ~150K | 210 bits | ~180 bits | ~100 bits |
| 光量子 | N/A (室温) | ~300K | 216 bits | ~150 bits (@300K) | ~100 bits |
| 中性原子 | ~0.05K | ~200K | 1420 bits | ~1200 bits | ~700 bits |

### 独特预测 (可检验)

1. **Φ_max 峰值位置:** Φ_max 在 T ≈ 0 时最大，随 T 单调下降 (无峰值)
2. **两阶段衰减:** 
   - T < T_crit: 缓慢下降 (k₀ 主导，~10%/T_crit)
   - T > T_crit: 快速下降 (N_c 主导，~1/T)
3. **平台差异:** 中性原子在低温下Φ_max 最高 (~1420 bits)，但随 T 下降最快 (T_cross 最低)
4. **β指数判别:** 在 T > 50K 区，β=3 (黑体辐射) vsβ=2 (唯象) 区分度>40%

## 开放问题

1. **T > T_crit 外推:** k₀(T) 公式在 T > T_crit 时可能为负，需声明适用范围 T < T_crit
2. **β指数平台依赖:** β=3 是黑体辐射主导区严格推导，但不同平台的 T_cross 不同，过渡区行为需实验验证
3. **Φ_max 实验测量:** N→∞极限的Φ_max 测量技术挑战大，需设计间接验证方案
