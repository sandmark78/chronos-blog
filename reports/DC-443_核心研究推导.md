## P0: T431 v1.4 温度依赖形式统一

### 问题识别 (C-DC442-A01)

T431 v1.3 使用多项式温度比：
```
η_A→B = min(1, N_c^B/N_c^A × F × [1-(T/T_crit^B)^δ]/[1-(T/T_crit^A)^δ])
```

但 T436 v1.2 采用指数衰减：
```
k₀(T) = k₀(0) × exp[-(T/T_crit)^δ]
```

**矛盾：** 两种形式在高温区 (T > T_crit) 给出不一致预测。

### T431 v1.4 修正

**统一温度比形式：**
```
η_A→B(T) = min(1, N_c^B(T)/N_c^A(T) × F_avg × exp[-(T/T_crit^B)^δ + (T/T_crit^A)^δ])
```

**推导：**
- 从 T436 v1.2: k₀^A(T) = k₀^A(0) × exp[-(T/T_crit^A)^δ]
- 从 T436 v1.2: k₀^B(T) = k₀^B(0) × exp[-(T/T_crit^B)^δ]
- 温度比 = k₀^B(T)/k₀^A(T) = exp[-(T/T_crit^B)^δ + (T/T_crit^A)^δ]

**优势：**
1. 与 T436 v1.2 完全一致
2. T→∞时η→0⁺ (物理合理)
3. T=T_crit 时无奇点
