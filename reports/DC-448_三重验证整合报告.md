### A.3 极限行为验证

#### 低温极限 (T → 0)

**预期:**
- k₀(T) → k₀(0) (最大值)
- N_c(T) → N_c^max (最大值)
- Φ(N,T) → k₀(0)×(N-1)/[1+(N-1)/N_c^max] (最大 Φ)

**实际:**
```python
k0_T(0.001, NA_PARAMS) = 0.950 × exp[-(0.001/0.120)^2] = 0.950 × 0.9999 ≈ 0.950
Nc_T(0.001, NA_PARAMS) = 70k / [1+(0.001/150)^3] = 70k / 1.000 ≈ 70k
Phi_NA(10000, 0.001, NA_PARAMS) = 0.950 × 9999 / [1+9999/70000] = 8254 bits
```

**结果:** ✅ 符合预期

#### 高温极限 (T → ∞)

**预期:**
- k₀(T) → 0 (指数衰减)
- N_c(T) → 0 (幂律衰减 ∝ T^-β)
- Φ(N,T) → 0 (双因子衰减)

**实际:**
```python
k0_T(1000, NA_PARAMS) = 0.950 × exp[-(1000/0.120)^2] ≈ 0 (数值下溢)
Nc_T(1000, NA_PARAMS) = 70k / [1+(1000/150)^3] = 70k / 297 ≈ 236 qubits
Phi_NA(10000, 1000, NA_PARAMS) ≈ 0 × 9999 / [1+9999/236] ≈ 0 bits
```

**结果:** ✅ 符合预期

#### 小 N 极限 (N → 1)

**预期:** Φ → 0 (单 qubit 无整合信息)

**实际:**
```python
Phi_NA(1, 0.01, NA_PARAMS) = k0 × 0 / [1+0/Nc] = 0 bits
```

**结果:** ✅ 符合预期

#### 大 N 极限 (N → ∞)

**预期:** Φ → k₀ × N_c (饱和值)

**实际:**
```python
# N ≫ N_c 时
Phi_NA(1000000, 0.01, NA_PARAMS) = k0 × 999999 / [1+999999/70000]
                                   = k0 × 999999 / 15.286
                                   ≈ k0 × 65420 ≈ 0.95 × 65420 ≈ 62149 bits
# 理论饱和值
Phi_max = k0 × Nc = 0.95 × 70000 = 66500 bits
# 偏差原因：N 有限，未完全饱和
```

**结果:** ✅ 符合预期趋势
