### 2.2 GHZ 态制备电路

```python
def create_ghz_circuit(n_qubits: int, measure: bool = True) -> QuantumCircuit:
    """
    创建 N-qubit GHZ 态电路
    |GHZ⟩ = (|00...0⟩ + |11...1⟩) / √2
    """
    qr = QuantumRegister(n_qubits, 'q')
    cr = ClassicalRegister(n_qubits, 'c')
    qc = QuantumCircuit(qr, cr)
    
    # 制备 GHZ 态
    qc.h(0)  # Hadamard on first qubit
    for i in range(1, n_qubits):
        qc.cx(0, i)  # CNOT from first to all others
    
    if measure:
        qc.measure(qr, cr)
    
    return qc
```

**电路深度:** O(N)  
**门数量:** 1 Hadamard + (N-1) CNOT = N 门

### 2.3 温度依赖噪声模型

```python
def create_noise_model(T: float, params: PlatformParams) -> NoiseModel:
    """
    创建温度依赖噪声模型
    
    噪声来源:
    1. 退极化噪声 (depolarizing): 模拟退相干
    2. 热弛豫噪声 (thermal_relaxation): 温度依赖的 T1/T2
    """
    noise_model = NoiseModel()
    
    # 温度依赖的 T1 (弛豫时间)
    # T1(T) = T1_0 / [1 + (T/T_cross)^β]
    T1_0 = 1.0  # 基准弛豫时间 (秒)
    T1 = T1_0 / (1.0 + np.power(T / params.T_cross, params.beta))
    
    # T2 (退相干时间) ≈ T1/2
    T2 = T1 / 2.0
    
    # 门时间 (假设)
    gate_time = 1e-6  # 1 μs
    
    # 创建热弛豫错误
    thermal_error = thermal_relaxation_error(T1, T2, gate_time)
    
    # 创建退极化错误 (强度随温度增加)
    depol_strength = 0.001 * (1.0 + T / params.T_crit)
    depol_error = depolarizing_error(depol_strength, 1)
    
    noise_model.add_all_qubit_quantum_error(
        thermal_error.compose(depol_error), 
        ['id', 'u1', 'u2', 'u3', 'cx']
    )
    
    return noise_model
```

**关键特性:**
- T1(T) 遵循与 N_c(T) 相同的温度依赖形式
- 退极化强度随 T/T_crit 线性增加
- 适用于中性原子和光量子平台

### 2.4 Φ计算公式实现

```python
def Phi_NA(N: float, T: float, params: PlatformParams) -> float:
    """
    中性原子平台 Φ计算
    Φ_NA(N,T) = k₀(T) × (N-1) / [1 + (N-1)/N_c(T)]
    """
    k0 = k0_T(T, params)
    Nc = Nc_T(T, params)
    return k0 * (N - 1) / (1.0 + (N - 1) / Nc)
```

**极限行为:**
- N ≪ N_c: Φ ≈ k₀ × (N-1) (线性区)
- N ≫ N_c: Φ ≈ k₀ × N_c (饱和区)
- T → 0: k₀ → k₀(0), N_c → N_c^max (最大 Φ)
- T → ∞: k₀ → 0, N_c → 0 (Φ → 0)

## 五、温度依赖曲线分析

### 5.1 N_c(T) 曲线

**特征:**
- T < 1 K: N_c ≈ N_c^max (平台区)
- 1 K < T < 100 K: N_c 缓慢下降
- T > 100 K: N_c 快速衰减 (∝ T^-3)
- T = 300 K: N_c ≈ 8k (室温极限)

**物理机制:**
- 低温区：热激发可忽略，退相干由量子涨落主导
- 高温区：黑体辐射光子数 ∝ T³，退相干率γ ∝ T³

### 5.2 k₀(T) 曲线

**特征:**
- T < T_crit (120 mK): k₀ ≈ k₀(0) (平台区)
- T ≈ T_crit: k₀ 开始指数衰减
- T > T_crit: k₀ → 0 (快速衰减)
- T = 1 K: k₀ < 0.01 k₀(0) (几乎完全抑制)

**物理机制:**
- 量子几何相位需要相干叠加态
- 温度升高导致能级激发，破坏叠加态纯度
- T_crit = ℏω₀/(2k_B) 为能级激发阈值

### 5.3 Φ_max(T) = k₀(T) × N_c(T) 曲线

**特征:**
- T < 100 mK: Φ_max ≈ 常数 (双平台区)
- 100 mK < T < 1 K: Φ_max 由 k₀ 主导衰减
- T > 1 K: Φ_max 由 N_c 主导衰减
- T = 300 K: Φ_max ≈ 6.5k bits (室温极限)

**最优工作点:**
- 中性原子：T < 100 mK, Φ_max ≈ 66k bits
- 光量子：室温可工作，Φ_max ≈ 5.9k bits

### 5.4 η(T) 转换效率曲线

**特征:**
- T < T_crit: η ≈ 常数 ≈ 8.05%
- T > T_crit: η 缓慢下降 (由 k₀ 比值决定)
- T = 300 K: η ≈ 7.5% (室温效率)

**物理含义:**
- 转换效率主要由 N_c 比值决定
- 温度依赖较弱 (在 T < T_cross 区域)
- 支持跨平台量子信息传输的可行性
