- T_cross ≪ T ≪ T_sat: N_c ∝ T^-β (黑体辐射区)

## 三、极限行为验证

### 3.1 T441 极限行为

#### T → 0 极限

$$\lim_{T\to 0} \eta_{A→B}(T) = \min\left(1, \frac{N_c^B(0)}{N_c^A(0)} \times F_{\text{avg}} \times \frac{k_0^B(0)}{k_0^A(0)} \times \exp[0]\right)$$

$$= \min\left(1, \frac{N_c^B(0)}{N_c^A(0)} \times F_{\text{avg}} \times \frac{k_0^B(0)}{k_0^A(0)}\right)$$

**物理含义:** 零温极限下效率由 N_c 比例和 k₀(0) 比例决定 ✅

**数值验证 (NA→Photonic):**
```
N_c^Phot(0)/N_c^NA(0) = 7k/70k = 0.1
k₀^Phot(0)/k₀^NA(0) ≈ 1.0 (假设相同)
F_avg ≈ 0.90
η(0) = 0.1 × 0.90 × 1.0 = 0.09 = 9%
```
**结果:** ✅ 物理合理

#### T → ∞ 极限

假设 T_crit^B < T_crit^A (B 平台先达到临界):

$$\lim_{T\to \infty} \exp\left[-\left(\frac{T}{T_{\text{crit}}^B}\right)^\delta + \left(\frac{T}{T_{\text{crit}}^A}\right)^\delta\right] = \lim_{T\to \infty} \exp\left[-T^\delta \left(\frac{1}{(T_{\text{crit}}^B)^\delta} - \frac{1}{(T_{\text{crit}}^A)^\delta}\right)\right]$$

由于 T_crit^B < T_crit^A:
- 1/(T_crit^B)^δ > 1/(T_crit^A)^δ
- 指数项 → exp[-∞] = 0⁺

$$\lim_{T\to \infty} \eta_{A→B}(T) = 0^+$$

**物理含义:** 高温时两平台量子几何耦合都被抑制，转换效率趋于零 ✅

**结果:** ✅ 物理合理

#### T_crit^A ≈ T_crit^B 极限 (数值稳定性)

**问题:** 当 T_crit^A ≈ T_crit^B 时，指数项可能出现数值不稳定

$$\exp\left[-\left(\frac{T}{T_{\text{crit}}^B}\right)^\delta + \left(\frac{T}{T_{\text{crit}}^A}\right)^\delta\right] = \exp\left[-T^\delta \left(\frac{1}{(T_{\text{crit}}^B)^\delta} - \frac{1}{(T_{\text{crit}}^A)^\delta}\right)\right]$$

当 T_crit^A → T_crit^B:
- 1/(T_crit^B)^δ - 1/(T_crit^A)^δ → 0
- 指数项 → exp[0] = 1

**数值稳定性分析:**

令 Δ = T_crit^A - T_crit^B (小量):

$$\frac{1}{(T_{\text{crit}}^B)^\delta} - \frac{1}{(T_{\text{crit}}^A)^\delta} \approx \frac{\delta \Delta}{(T_{\text{crit}})^{\delta+1}}$$

**条件数估计:**
- 相对误差放大因子 ≈ δ × (ΔT/T_crit)
- 对于 δ=2, T_crit=0.1K, ΔT=0.001K: 条件数 ≈ 0.02 (良好)
- 对于 δ=2, T_crit=0.1K, ΔT=0.01K: 条件数 ≈ 0.2 (可接受)
- 对于 δ=2, T_crit=0.1K, ΔT=0.1K: 条件数 ≈ 2.0 (需注意)

**🟡 重要矛盾 C-DC449-A01:**
- **问题:** T_crit^A ≈ T_crit^B 时，指数项对 T_crit 测量误差敏感
- **影响:** 当 ΔT_crit/T_crit > 10% 时，η预测误差可能超过 20%
- **建议:** 
  1. 添加数值稳定性声明：|T_crit^A - T_crit^B| > 0.1 × min(T_crit^A, T_crit^B)
  2. 或改用更稳定的形式：k₀^B(T)/k₀^A(T) 直接计算 (避免指数相减)

**结果:** 🟡 需注意数值稳定性

### 3.2 T442 极限行为

#### T → 0 极限

$$\lim_{T\to 0} f_{\text{highT}}(T) = \frac{1}{1 + 0 + 0} = 1$$

$$\lim_{T\to 0} N_c(T) = N_c^{\text{max}}$$

**结果:** ✅ 物理合理 (低温平台区)

#### T → ∞ 极限

$$\lim_{T\to \infty} f_{\text{highT}}(T) = \lim_{T\to \infty} \frac{1}{1 + (T/T_{\text{cross}})^\beta + (T/T_{\text{sat}})^\gamma}$$

主导项分析:
- 若 β > γ: (T/T_sat)^γ 主导 (高温饱和区)
- 若 β < γ: (T/T_cross)^β 主导 (黑体辐射区延续)
- 若 β = γ: 两项共同主导

**典型参数:**
- β = 3.0 (黑体辐射)
- γ = 1.5-2.0 (高温唯象)

由于 β > γ，在 T → ∞ 时：
$$(T/T_{\text{sat}})^\gamma \gg (T/T_{\text{cross}})^\beta \quad \text{(当 T 足够大)}$$

**但存在过渡区:**
- T_cross < T < T_sat: (T/T_cross)^β 主导
- T > T_sat: (T/T_sat)^γ 主导

**🔴 阻塞矛盾 C-DC449-A02:**
- **问题:** T442 的 f_highT(T) 在 T → ∞ 时的主导项不明确
- **当前形式:** f_highT = 1/[1 + (T/T_cross)^β + (T/T_sat)^γ]
- **问题所在:** 
  - 若 β > γ (如β=3, γ=1.5)，则 (T/T_sat)^γ 在 T→∞时增长更慢，不会主导
  - 实际主导项始终是 (T/T_cross)^β (因为β更大)
  - 这与"高温饱和区"的物理直觉矛盾
- **物理期望:** 高温时应趋于常数或更慢衰减 (饱和行为)
- **修正建议:** 
  - 方案 A: 改为 f_highT = 1/{1 + (T/T_cross)^β × [1 + (T/T_sat)^γ]}
  - 方案 B: 改为 f_highT = 1/{1 + min[(T/T_cross)^β, (T/T_sat)^γ]}
  - 方案 C: 改为 f_highT = exp[-(T/T_cross)^β] + f_sat(T) (双机制叠加)

**推荐修正 (方案 A):**
$$f_{\text{highT}}(T) = \frac{1}{1 + (T/T_{\text{cross}})^\beta \times \left[1 + \left(\frac{T}{T_{\text{sat}}}\right)^{\gamma-\beta}\right]}$$

或更清晰的物理形式:
$$f_{\text{highT}}(T) = \frac{1}{1 + (T/T_{\text{cross}})^\beta} \times \frac{1}{1 + (T/T_{\text{sat}})^\gamma}$$

**结果:** 🔴 需修正 (物理行为不明确)

#### T = T_cross 行为

$$f_{\text{highT}}(T_{\text{cross}}) = \frac{1}{1 + 1 + (T_{\text{cross}}/T_{\text{sat}})^\gamma} \approx \frac{1}{2 + \text{小量}} \approx 0.5$$

**结果:** ✅ 合理 (半衰减点)

#### T = T_sat 行为

$$f_{\text{highT}}(T_{\text{sat}}) = \frac{1}{1 + (T_{\text{sat}}/T_{\text{cross}})^\beta + 1} \approx \frac{1}{(T_{\text{sat}}/T_{\text{cross}})^\beta}$$

对于 T_sat/T_cross ≈ 500K/150K ≈ 3.3, β=3:
$$f_{\text{highT}}(T_{\text{sat}}) \approx 1/(3.3)^3 \approx 1/36 \approx 0.028$$

**结果:** ✅ 合理 (高温衰减)
