## 阶段 2: 核心研究推导

### 2.1 路径γ: GHZ-N 态在 CP^(2^N-1) 中的严格几何分析

#### 2.1.1 问题陈述

**目标**: 从 CP^(2^N-1) 的几何性质，独立推导 GHZ 态的整合信息 Φ 为何线性增长于 N，从而给 k₀ 一个几何解释。

**DC-464 遗产**: 7 种 CP¹ (Bloch 球) 几何量均不匹配 k₀ = 0.0300。DC-465 方向校正建议从 CP¹ → CP^(2^N-1)，看多体几何是否提供新的线性标度量。

#### 2.1.2 GHZ-N 态的 Hilbert 空间结构

**N 量子比特系统**: ℋ = (ℂ²)^⊗N，dim(ℋ) = 2^N

**GHZ 态**: |GHZ_N⟩ = (|0⟩^⊗N + |1⟩^⊗N)/√2

**射影 Hilbert 空间**: CP^(2^N - 1)，实维度 = 2(2^N - 1)

**GHZ 子流形**: |GHZ_N⟩ 属于一个 2 维子空间 Span{|0⟩^⊗N, |1⟩^⊗N}，嵌入为 CP¹ ⊂ CP^(2^N - 1)

#### 2.1.3 内蕴几何: 为什么不随 N 变化

GHZ 态族的参数化:
$$|\psi(\theta,\varphi)\rangle = \cos(\theta/2)|0\rangle^{\otimes N} + e^{i\varphi}\sin(\theta/2)|1\rangle^{\otimes N}$$

限制 Fubini-Study 度规到此子流形:
$$ds^2_{FS}\big|_{GHZ} = \frac{1}{4}d\theta^2 + \frac{1}{4}\sin^2(\theta)d\varphi^2$$

**这是半径 1/2 的标准球面 S²，与 N 无关。**

**结论**: GHZ 子流形的内蕴几何 (曲率、面积、测地线) 完全不依赖于 N。所有内蕴 Fubini-Study 量都是 N 无关的。这解释了 DC-464 中 7 种 CP¹ 几何量为何全部不匹配。

#### 2.1.4 外蕴几何: 嵌入如何依赖 N

虽然内蕴几何不变，但 CP¹ → CP^(2^N-1) 的嵌入方式随 N 剧变:

**法丛维度**: dim(normal bundle) = 2(2^N - 1) - 2 ≈ 2^{N+1} - 4 → 指数增长

**第二基本形式**: 描述嵌入弯曲程度的张量，其秩和范数可能随 N 变化。但:
- 对于线性子空间嵌入 (GHZ 就是线性子空间嵌入)，第二基本形式为零
- 因为 GHZ 子空间是 CP^(2^N-1) 中的一个完全测地子流形

**结论**: GHZ 嵌入的外蕴几何也不随 N 变化 (完全测地嵌入)。

**⚠️ 路径γ第一个障碍**: CP^(2^N-1) 的纯几何无法区分 N=2 和 N=100 的 GHZ 态，因为它们的嵌入几何同构。

#### 2.1.5 转向: 量子信息几何 vs 纯射影几何

纯射影几何行不通，因为 GHZ 态在任何 N 下都是 CP¹ 的完全测地嵌入。**关键缺失**: 射影几何不编码"张量积结构"。

**物理信息在于**: ℋ = ℋ₁ ⊗ ℋ₂ ⊗ ... ⊗ ℋ_N 的张量积分解。Φ 的定义依赖于这个分解 (谁是"部分"，谁是"整体")。

**核心洞察**: Φ 不是 CP^(2^N-1) 的纯几何量，而是 **张量积结构 + 距离度量** 的联合量。

#### 2.1.6 Total Correlation 分析: 线性标度的信息论起源

**定义**: 总关联 (total correlation / multi-information)
$$C(\rho) = \sum_{i=1}^N S(\rho_i) - S(\rho)$$

其中 S 是 von Neumann 熵。

**GHZ-N 态计算**:
- ρ = |GHZ_N⟩⟨GHZ_N| → S(ρ) = 0 (纯态)

  **修正**: S(ρ) = 0 对于纯态。但 IIT 中用的不是全局态的熵。

  让我重新计算。对于 GHZ 纯态:
  - S(ρ) = 0
  - ρ_i = Tr_{≠i}(ρ) = (|0⟩⟨0| + |1⟩⟨1|)/2 → S(ρ_i) = 1 bit (∀i)

$$C(\rho_{GHZ_N}) = N \times 1 - 0 = N \text{ bits}$$

**总关联线性标度于 N**。

#### 2.1.7 k₀ 作为"不可约分数" (Irreducibility Fraction)

**定义**: 不可约分数 (irreducibility fraction)
$$f_{irr} \equiv \frac{\Phi}{C(\rho)} = \frac{k_0 \cdot (N-1)}{N} \approx k_0 \quad (N \gg 1)$$

更精确地:
$$f_{irr}(N) = k_0 \cdot \frac{N-1}{N}$$

- N=2: f_irr = k₀/2 = 0.015
- N=10: f_irr = 0.9 × k₀ = 0.027  
- N→∞: f_irr → k₀ = 0.030

**物理意义**: GHZ-N 态的总关联为 N bits，其中只有约 3% (k₀ ≈ 0.030) 是真正不可约的 (irreducible)。其余 97% 可以被分解为较低阶的关联。

#### 2.1.8 关键定理推导

**T452: k₀-总关联分解定理 (k₀-Total Correlation Decomposition)**

**陈述**: 对于 N 量子比特 GHZ 态，整合信息 Φ 可以分解为:

$$\Phi_{GHZ}(N) = f_{irr} \times C(\rho_{GHZ}) = f_{irr} \times N$$

其中:
- C(ρ) = N bits 是总关联 (线性标度的来源)
- f_irr = k₀·(N-1)/N 是不可约分数
- k₀ ≈ 0.030 bits 编码了 IIT 距离度量中的不可约性比例

**推导**:

Step 1: 总关联提供线性标度
$$C(\rho_{GHZ_N}) = \sum_{i=1}^N S(\rho_i) - S(\rho) = N \cdot 1 - 0 = N \text{ bits}$$

Step 2: Φ 与 C 的关系
从 T420 (验证版): Φ = k₀·(N-1)/[1+(N-1)/N_c]

在 N ≪ N_c 的线性区:
$$\Phi \approx k_0 \cdot (N-1)$$

Step 3: 不可约分数
$$f_{irr} = \frac{\Phi}{C(\rho)} = \frac{k_0(N-1)}{N} = k_0\left(1 - \frac{1}{N}\right)$$

Step 4: k₀ 的信息论解读
$$k_0 = \lim_{N\to\infty} \frac{\Phi(N)}{C(\rho_N)} = \lim_{N\to\infty} \frac{\Phi(N)}{N}$$

**k₀ 是整合信息密度 (integrated information density)**: 每 bit 总关联中，有 k₀ bit 是真正不可约的。

**独特预测 D-452-01**: 如果用不同的距离度量 (例如 KL 散度替代 EMD) 重新计算 IIT 的 Φ，k₀ 的数值将改变，但线性标度 Φ ∝ N 不变。具体地:
- EMD (IIT 3.0 标准): k₀ ≈ 0.030 bits
- KL 散度: k₀_KL 预计更大 (量级 ~0.1-0.5 bits)
- Trace distance: k₀_Tr 预计介于两者之间

**独特预测 D-452-02**: 对于非 GHZ 型纠缠态 (例如 W 态、cluster 态)，f_irr 将不同:
- W-N 态: f_irr 预计更低 (W 态纠缠更"脆弱")
- Cluster-N 态: f_irr 预计更高 (cluster 态有更强的多体关联)

#### 2.1.9 路径γ的结论: 部分成功 + 重要负结果

**成功部分**:
1. ✅ 线性标度 Φ ∝ N 的来源已识别: 总关联 C(ρ) = N
2. ✅ k₀ 的信息论解读已建立: 不可约分数 f_irr
3. ✅ DC-464 的 7 种 CP¹ 几何量不匹配的原因已解释: 内蕴几何与 N 无关

**负结果 (重要!)**:
1. ❌ 纯射影几何 (Fubini-Study) 无法区分不同 N 的 GHZ 态
2. ❌ k₀ 的具体数值 0.030 无法从纯几何推导 — 它编码了 IIT 距离度量的选择
3. ❌ 路径γ在纯几何层面**被阻断**: 需要固定 IIT 度量后才能进一步

**关键结论**: **k₀ 不是纯几何常数，而是 IIT 度量-依赖常数**。
- 这解释了为什么从 CP¹/CP^(2^N-1) 几何量推导 k₀ 注定失败
- k₀ = 0.030 的值是 IIT 选择 EMD 作为距离度量的后果
- 要给 k₀ 第一性原理解释，需要先回答: "为什么 IIT 选择 EMD 而非其他度量？"
