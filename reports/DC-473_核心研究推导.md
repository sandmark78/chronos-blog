## P2: Φ_C 的 TPM 构造方案 (响应 C-DC471-A02)

### 问题
C-DC471-A02: Φ_C 的 TPM (转移概率矩阵) 动力学定义未明确

### 解决方案：经典极限 TPM 构造

对于 GHZ-N 态，在 T ≫ T_crit 极限下：

1. **量子 TPM** (T=0):
   ```
   P(X_t+1 | X_t) = |⟨X_t+1|U|X_t⟩|²
   
   对于 GHZ 态：U 是纠缠门序列
   ```

2. **经典 TPM** (T → ∞):
   ```
   P_class(Y_t+1 | Y_t) = δ(Y_t+1, Y_t)  (经典比特保持)
   
   但考虑热涨落：P_class = (1-ε)δ + ε/N_states
   ```

3. **混合 TPM** (有限 T):
   ```
   P_mix = w_Q(T) × P_quantum + w_C(T) × P_class
   
   其中:
   - w_Q(T) = exp[-(T/T_crit)^δ] (量子权重)
   - w_C(T) = 1 - w_Q(T) (经典权重)
   ```

### Φ_C 的计算
```
Φ_C(T) = Φ[P_mix(T)] - Φ[P_quantum] × w_Q(T)

近似:
Φ_C(T) ≈ φ₀ × log₂[min(N, N_class)] / log₂(N) × [1 - exp(-(T/T_class)^γ)]
```

### N_class 的估计
对于 GHZ-N:
- 量子 TPM 维度: 2^N × 2^N
- 经典 TPM 有效维度: N (集体模式)
- N_class = 2^(N-1) (GHZ 态的经典对应态数量)

对于 GHZ-6: N_class = 2^5 = 32

## P4: φ₀ 和 T_class 拟合策略

### 拟合函数
```python
def phi_total_model(T_ratio, phi0, T_class, gamma):
    """
    Φ_total(T) = Φ_Q(T) + Φ_C(T)
    """
    # 量子部分 (固定参数)
    phi_Q = k0 * (N-1) / (1 + (N-1)/N_c) * np.exp(-T_ratio**delta)
    
    # 经典部分 (拟合参数)
    N_class = 2**(N-1)
    phi_C = phi0 * np.log2(min(N, N_class)) / np.log2(N) * (1 - np.exp(-(T_ratio * T_crit / T_class)**gamma))
    
    return phi_Q + phi_C
```
