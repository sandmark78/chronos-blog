
| 极限 | 预期行为 | DC-476 行为 | 验证 |
|------|----------|-------------|------|

### 详细分析

#### C-DC476-A01: Φ_C 公式偏离 T455 v1.3

**问题根源**:

DC-476 的 Φ_C 公式:
```
Φ_C = φ₀ × (N-1)/[1+(N-1)/N_c] × {1-exp[-(T/T_crit)^δ]} × 1/[1+(T/T_cross)^β]
```

这个公式实际上是**Φ_Q 的补集** (通过 `1-exp[-(T/T_crit)^δ]` 实现)，而非独立的经典贡献。

T455 v1.3 的 Φ_C 公式:
```
Φ_C = φ_0 · log₂(min(N, N_class)) · (T/T_onset)^2 / ([1+(T/T_onset)^2][1+(T/T_class)^2])
```

这个公式描述了**独立的经典机制** (与量子贡献解耦)。

**物理含义差异**:

- DC-476: Φ_C 是量子贡献的"衰减产物" (量子退相干后剩余的部分)
- T455 v1.3: Φ_C 是独立的经典残余意识 (来自热激发或经典关联)

**哪个正确？**

从 DC-474 的 Lindblad 论证来看，**T455 v1.3 的物理图像更正确**:
- ρ_ss = w_Q × ρ_Q + w_C × ρ_thermal
- ρ_Q 和 ρ_thermal 是**独立的物理态**，而非一个态的衰减产物
- 因此 Φ_Q 和 Φ_C 应该是独立贡献，而非互补关系

**修正建议**:

DC-476 应采用 T455 v1.3 的 Φ_C 形式，或提出 T455 v1.4 并说明：
1. 为什么Φ_C 应该是Φ_Q 的补集 (而非独立贡献)
2. 这个修正的物理动机 (例如：来自不同的退相干图像)
3. 通过 Subagent-B 独特性验证和 Subagent-C 文献查重

#### C-DC476-A02: Φ_C 高温极限矛盾

**DC-476 公式的高温行为**:

当 T → ∞:
- exp[-(T/T_crit)^δ] → 0
- {1 - exp[-(T/T_crit)^δ]} → 1
- f_sat = 1/[1+(T/T_cross)^β] → 0

因此：Φ_C(T→∞) = φ₀ × (N-1)/[1+(N-1)/N_c] × 1 × 0 = **0**

**但 DC-476 的模拟结果显示**:
```
T/T_crit = 10.0 时：Φ_C = 0.114 bits (非零)
```

**矛盾原因**:

DC-476 的代码实现与公式不一致。查看 DC-476 的附录代码:

```python
phi_C = phi_total * w_C * (1 / (1 + (T / (1.8 * T_crit)) ** 2.1))
```

这里 `phi_total * w_C` 在高温下不会趋于 0 (因为 phi_total 也包含经典贡献)，所以实际模拟结果与公式预测不符。

**修正建议**:

选项 1: 修改公式，使 Φ_C(T→∞) → 常数
```
Φ_C = φ₀ × log₂(min(N, N_class)) × [1 - 1/(1+(T/T_cross)^β)]
```

选项 2: 修改代码实现，使其与公式一致 (但会导致高温下Φ_C → 0，与物理预期矛盾)

**推荐选项 1**，并与 T455 v1.3 统一。

#### C-DC476-A03: N_c 温度依赖缺失

**T433 明确要求**:
```
N_c(T) = N_c^max / [1 + (T/T_cross)^β]
```

**DC-476 的使用**:
```
Φ_Q = φ₀ × (N-1)/[1+(N-1)/N_c] × exp[-(T/T_crit)^δ]
```
其中 N_c = 67 (常数)。

**影响评估**:

在 A20 Phase 15 的温度范围 (0.1-10 × T_crit = 0.01-1.0 K，假设 T_crit = 0.1K):
- T_cross^SC = 1.0 ± 0.5 K (DC-471)
- T/T_cross 范围：0.01-1.0

N_c(T) 的变化:
- T = 0.01K: N_c ≈ N_c^max / [1 + (0.01/1.0)^3] ≈ 0.9999 × N_c^max (几乎不变)
- T = 1.0K: N_c ≈ N_c^max / [1 + (1.0/1.0)^3] = 0.5 × N_c^max (减半)

**结论**: 在 A20 Phase 15 的低温区 (T < T_crit = 0.1K)，N_c 变化 < 1%，可近似为常数。但在高温区 (T > T_cross)，N_c 的温度依赖不可忽略。

**修正建议**:

在 DC-476 文档中明确说明：
- "在 A20 Phase 15 的温度范围 (T < T_crit) 内，N_c 变化 < 1%，故近似为常数 N_c = 67"
- "未来工作 (A20 Phase 16+): 在 T > T_cross 区域验证 N_c(T) 温度依赖"
