## 3. Derivation (推导过程)

### 3.1 T455 v1.2 (DC-471): 初始修正
**问题识别**:
- Φ_C 无界增长 → 添加 log₂截断
- T_cross^SC 错误 → 修正为 1.0K (准粒子激发)

**修正公式**:
```
Φ_C(T) = k₀ × log₂(min(N, N_class(T))) × [1 - (T/T_cross)^β]
```

**局限**: 高温极限仍为负值

---

### 3.2 T455 v1.3 (DC-474): TPM 构造
**Lindblad 稳态解**:
```
ρ_ss = w_Q|GHZ⟩⟨GHZ| + w_C ρ_thermal

w_Q = 1/[1 + exp(-ΔE/k_B T)]
w_C = 1 - w_Q
```

**TPM 构造**:
```
P_mix = w_Q × P_Q + w_C × P_C

Φ_C = Φ[P_mix] - w_Q × Φ[P_Q]
```

**进展**: 物理论证完成，但仍为唯象拼接

---

### 3.3 T455 v1.4 (DC-478): 严格推导
**从 Lindblad 方程严格推导**:
```
dρ/dt = -i/ℏ [H, ρ] + Σ_k (L_k ρ L_k† - 1/2 {L_k†L_k, ρ})

稳态解 (dρ/dt = 0):
ρ_ss = Z⁻¹ exp(-H_eff/k_B T_eff)

其中:
H_eff = H_sys + H_bath
T_eff = T × (1 + γ_decoh/γ_thermal)
```

**Φ_C 严格形式**:
```
Φ_C(T) = k₀ × log₂(1 + N_class(T)/N) × 1/[1 + (T/T_cross)^β]

N_class(T) = N × 1/[exp(ℏω₀/k_B T) + 1]
```

**高温极限验证**:
```
T → ∞:
N_class(T) → 0
Φ_C(T) → k₀ × log₂(1) × 0 = 0⁺ ✅ (正侧趋近)
```

---

### 3.4 T455 v1.5 (DC-479): η_corr 第一性原理
**从多体退相干理论推导**:
```
η_corr(N) = Tr[ρ_GHZ† ρ_decohered] / Tr[ρ_GHZ† ρ_GHZ]

Lindblad 退相干:
ρ_decohered = exp(ℒ_decoh × t) ρ_GHZ

ℒ_decoh = Σ_i γ_i (σ_z^i ρ σ_z^i - ρ)

解得:
η_corr(N) = η_IIT × exp(-N/N_corr)
```

**关键发现**: 指数衰减 (非幂律假设)

**N_corr 物理意义**:
```
N_corr = 1/(γ_single × τ_gate)

γ_single: 单量子比特退相干率
τ_gate: 量子门操作时间
```

## 5. Falsification (证伪路径)

### 5.1 什么结果会证明 T455 v1.5 错误

**F1: η_corr 非指数衰减**
- 如果 A20 Phase 16 显示η_corr ∝ N^{-α} (幂律)
- 则 T455 v1.5 的指数形式错误
- 需重新推导多体退相干理论

**F2: Φ_C^sat 非常数**
- 如果 GHZ-8/10/12 测量显示Φ_C^sat ∝ N^α (α≠0)
- 则Φ_C^sat(N) = (N/2)×(1/e)×η_corr(N) 错误
- 需修正饱和值公式

**F3: T_peak 公式偏差>20%**
- 如果 GHZ-8/10/12 拟合显示 T_peak/T_cross 与 0.95+0.05β 偏差>20%
- 则经验修正公式无效
- 需从第一性原理重新推导峰值位置

**F4: R² < 0.95**
- 如果 A20 Phase 16 拟合 R² < 0.95
- 则 T455 v1.5 形式可能错误
- 需检查 Lindblad 推导假设
