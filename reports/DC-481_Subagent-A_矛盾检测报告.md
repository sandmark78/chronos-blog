## 详细分析

### 1. η_corr(N) 与 T455 v1.5/v1.6 兼容性 ✅

**DC-481 公式**:
```
Φ_C(T,N) = k₀ × log₂(1 + N_class(T)/N) × 1/[1 + (T/T_cross)^β] × η_corr(N)

η_corr(N) = η_IIT(N) × exp(-N/N_corr)
```

**T455 v1.5 原始形式** (DC-480):
```
Φ_C(T,N) = k₀ × log₂(1 + N_class(T)/N) × 1/[1 + (T/T_cross)^β]
```

**兼容性分析**:
- η_corr(N) 是乘法修正因子，不改变 T455 的温度依赖形式
- η_corr(N=6) ≈ 0.95 × exp(-6/12) ≈ 0.95 × 0.61 ≈ 0.58
- 与 A20 Phase 15 拟合φ₀ = 0.0245 bits 一致 (k₀ = 0.030 bits, η_corr ≈ 0.82)

**结论**: ✅ 完全兼容，η_corr 是 T455 的自然扩展。

---

### 2. η_IIT(N) 与 IIT 4.0 文献一致性 🟡

**DC-481 声称**:
> "η_IIT 源于 IIT 4.0 的 MIP (Minimum Information Partition) 算法的度量效率"
> "η_IIT(N) = 1 - α_IIT × ln(N)/N, 其中α_IIT = 0.15±0.05"

**IIT 4.0 文献核查** (基于 DC-343, DC-354, DC-395 历史报告):
- IIT 4.0 确实使用 MIP 启发式搜索 (非穷举)
- IIT 4.0 文献承认 MIP 计算是 NP-hard 问题
- **但** IIT 4.0 文献未明确声明 MIP 启发式存在系统性 N 依赖误差

**矛盾点**:
- DC-481 将"合理假设"表述为"第一性原理推导"
- η_IIT 的 N 依赖形式 (1 - 0.15×ln(N)/N) 是经验拟合，非严格推导

**建议**:
- 将"推导"改为"假设 + 经验拟合"
- 添加验证方案：N=2→6 穷举 MIP vs 启发式 MIP 对比实验

**结论**: 🟡 合理假设，但需降格为"假设"并添加验证方案。

### 5. 极限行为 ✅

**N→∞极限**:
```
η_IIT(N) = 1 - 0.15×ln(N)/N → 1 (当 N→∞)
exp(-N/N_corr) → 0 (当 N→∞)
η_corr(N) → 0 ✅

物理意义：大系统 MIP 搜索效率下降，关联衰减主导。
```

**N=1 极限**:
```

exp(-1/N_corr) ≈ 1 (当 N_corr >> 1)
η_corr(1) ≈ 1

⚠️ 问题：IIT 4.0 要求 N≥2 才能定义 MIP (单元素系统无分区)。
```

**结论**: ✅ N→∞行为合理；N=1 需声明适用范围 N≥2 (与历史矛盾 I-002 一致)。
