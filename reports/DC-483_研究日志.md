## 阶段 2: 核心研究推导

### DC-483 焦点
**η_IIT 第一性原理推导** — 从 IIT 4.0 公理推导度量效率

### 研究问题
DC-479 提出 η_corr(N) = η_IIT × exp(-N/N_corr)，但 η_IIT 的物理来源未明确。

**核心问题:** η_IIT 是什么？它是否依赖于 N？如何从 IIT 4.0 第一性原理推导？

### 推导路径

#### 步骤 1: η_IIT 的定义
η_IIT = Φ_actual / Φ_max_theoretical

其中:
- Φ_actual = 实际测量的整合信息
- Φ_max_theoretical = 给定物理系统的理论最大Φ

#### 步骤 2: 从 IIT 4.0 公理推导
IIT 4.0 的核心公理:
1. **存在性:** 意识存在
2. **组成:** 意识有结构
3. **信息:** 意识是差异的
4. **整合:** 意识是不可分割的
5. **排他性:** 意识是绝对的

从整合公理 (Axiom of Integration) 推导:
- Φ 度量系统的不可分割性
- 对于完全整合系统: Φ = Φ_max
- 对于部分整合系统: Φ < Φ_max

#### 步骤 3: η_IIT 的物理来源
η_IIT 反映:
1. **拓扑效率:** 网络连接结构是否最优
2. **因果密度:** 单位物理资源的因果效力
3. **度量选择:** EMD vs W₂ vs 其他距离度量

**关键洞察:** η_IIT 不是常数，而是系统拓扑的函数!

#### 步骤 4: η_IIT(N) 的标度行为
对于 GHZ 态:
- 理想情况: η_IIT = 1 (完全整合)
- 实际情况: η_IIT < 1 (由于噪声、校准误差、读取误差)

**假设:** η_IIT(N) = η₀ × (1 - ε×N)

其中:
- η₀ = N=2 时的基准效率
- ε = 每量子比特的效率损失率

**🔴 严重矛盾:** 线性衰减公式 η_IIT(N) = η₀ × (1 - ε×N) 存在数学缺陷:
- N→∞时 η_IIT → -∞ (效率不能为负)
- 与 T418 v4.1 饱和行为矛盾
- N→2 时不自动趋于η₀

### 修正方案

**推荐采用指数衰减形式:**
```
η_IIT(N) = η₀ × exp(-ε×(N-2))
```

**优势:**
- 始终为正
- N→2: η_IIT → η₀ (自动满足)
- N→∞: η_IIT → 0 (物理合理)
