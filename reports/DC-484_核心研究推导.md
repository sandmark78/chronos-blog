## 阶段 3: A20 Phase 16 方案更新

### 3.1 方案变更

| 维度 | 原方案 | 新方案 |
|------|--------|--------|
| **数据源** | GHZ-8/10/12 (超导量子) | BART fMRI (人脑) |
| **N 范围** | N=8,10,12 (3 点) | N=2→15 (14 点) |
| **被试/样本** | 1 系统 (多次测量) | 16 被试 × 3 run = 48 |
| **硬件需求** | IBM Quantum 访问 | 无 (本地计算) |
| **生态效度** | 人工量子系统 | 天然人脑整合系统 |

### 3.2 变更理由

**优势:**
1. ✅ 数据立即可用 (无需等待)
2. ✅ 人脑是天然高 N 整合系统 (N≈100-1000 脑区)
3. ✅ 统计效力高 (48 独立测量)
4. ✅ 风险决策任务激活分布式网络 (适合 IIT 分析)

**风险:**
- ⚠️ fMRI 时间分辨率较低 (TR~2s)
- ⚠️ Φ计算方法需适应连续数据
- ⚠️ 但核心η_IIT 模型检验不受影响

### 3.3 核心流程

```
fMRI 数据 (NIfTI)
    ↓
ROI 时间序列提取 (AAL 116 脑区)
    ↓
互信息矩阵计算 (MI)
    ↓
N=2→15 子系统采样 (100 次/N)
    ↓
Φ_actual 计算 (特征值方法)
    ↓
η_IIT(N) = Φ_actual / Φ_max
    ↓
拟合指数模型，提取ε
```

## 阶段 4: η_IIT 计算方法

### 4.1 从 fMRI 到Φ

**步骤 1: ROI 时间序列提取**
```python
from nilearn import datasets, input_data

atlas = datasets.fetch_atlas_aal()  # 116 脑区
masker = input_data.NiftiLabelsMasker(labels_img=atlas.maps)
time_series = masker.fit_transform(fmri_img)  # (time × 116)
```

**步骤 2: 互信息矩阵**
```python
from sklearn.metrics import mutual_info_score

def compute_mi_matrix(ts, n_bins=10):
    n_rois = ts.shape[1]
    mi = np.zeros((n_rois, n_rois))
    discretized = np.digitize(ts, bins=np.linspace(ts.min(), ts.max(), n_bins+1))
    
    for i in range(n_rois):
        for j in range(i+1, n_rois):
            mi[i,j] = mi[j,i] = mutual_info_score(discretized[:,i], discretized[:,j])
    return mi
```

**步骤 3: Φ计算 (特征值近似)**
```python
def compute_phi(mi_matrix):
    eigenvalues = np.linalg.eigvalsh(mi_matrix)
    return np.sum(eigenvalues[eigenvalues > 1e-10])
```

**步骤 4: Φ_max 理论值**
$$\Phi_{max}(N) = \frac{N \cdot \log_2(N)}{2}$$

**步骤 5: η_IIT 计算**
$$\eta_{IIT}(N) = \frac{\Phi_{actual}(N)}{\Phi_{max}(N)}$$

### 4.2 子系统采样

**挑战:** 对于 N_total=116 脑区，穷举 C(116, N) 不可行

**解决方案:** 随机采样
```python
def sample_subsystems(mi_matrix, n_min=2, n_max=15, n_samples=100):
    n_total = mi_matrix.shape[0]
    results = []
    
    for n in range(n_min, n_max+1):
        phi_values = []
        for _ in range(n_samples):
            indices = np.random.choice(n_total, n, replace=False)
            sub_mi = mi_matrix[np.ix_(indices, indices)]
            phi = compute_phi(sub_mi)
            phi_values.append(phi)
        
        results.append({
            'N': n,
            'phi_mean': np.mean(phi_values),
            'phi_max': n * np.log2(n) / 2
        })
    return results
```

### 4.3 模型拟合

**模型:**
$$\eta_{IIT}(N) = \eta_0 \times \exp[-\varepsilon \times (N-2)]$$

**拟合:**
```python
from scipy.optimize import curve_fit

def eta_iit_model(N, eta0, epsilon):
    return eta0 * np.exp(-epsilon * (N - 2))

popt, pcov = curve_fit(eta_iit_model, N_values, eta_values, p0=[1.0, 0.15])
eta0_fit, epsilon_fit = popt
```
