## 一、研究背景与动机

### 1.1 理论背景

DC-483 完成η_IIT 指数模型第一性原理推导：
$$\eta_{IIT}(N) = \eta_0 \times \exp[-\varepsilon \times (N-2)]$$

其中：
- η_IIT(N): N 子系统系统的 IIT 积分效率
- η₀: 基准效率 (N=2 时的η_IIT)
- ε: 衰减常数 (平台依赖)

## 二、数据与方法

### 2.1 数据集

**OpenNeuro ds000001 (BART 任务)**:
- 被试数：16 (sub-01 ~ sub-16)
- Runs 每被试：3
- 总数据集：48 个功能连接矩阵
- 模态：fMRI BOLD 信号
- 任务：气球模拟风险任务 (BART)

### 2.2 分析流程 (6 步)

**步骤 1**: 数据获取与预处理
```bash
datalad get ds000001/sub-01/func/
# 输出：预处理后的 BOLD 时间序列
```

**步骤 2**: ROI 时间序列提取
- 图谱：AAL (116 ROI) 或 Schaefer (200-1000 ROI)
- 方法：平均体素信号
- 输出：N ROI × T 时间点的矩阵

**步骤 3**: 互信息计算
$$I(X;Y) = \sum_{x,y} p(x,y) \log\frac{p(x,y)}{p(x)p(y)}$$
- 方法：高斯 Copula 或 KSG 估计
- 输出：N × N 互信息矩阵

**步骤 4**: 子系统采样
- 子系统大小：n = 2, 3, 4, ..., N_max
- 采样策略：随机采样 (100 次/大小)
- 输出：Φ(n) 曲线

**步骤 5**: Φ_IIT 计算
- 方法：MIP 近似 (小 n 穷举，大 n 启发式)
- 输出：Φ_IIT(n) for each subsystem size

**步骤 6**: η_IIT 拟合
$$\eta_{IIT}(n) = \frac{\Phi_{IIT}(n)}{\Phi_{IIT}^{max}(n)}$$
$$\eta_{IIT}(n) = \eta_0 \times \exp[-\varepsilon \times (n-2)]$$
- 拟合方法：非线性最小二乘
- 输出：η₀, ε, R², 95% CI

### 2.3 代码实现 (Python)

```python
# 核心分析脚本 (DC-484 已创建模板)
# reports/DC-484_sub-01 试点分析.md 包含 6 个 Python 脚本模板

# 关键函数：
def compute_eta_iit(phi_values, n_values):
    """拟合η_IIT 指数衰减模型"""
    from scipy.optimize import curve_fit
    
    def eta_model(n, eta0, epsilon):
        return eta0 * np.exp(-epsilon * (n - 2))
    
    popt, pcov = curve_fit(eta_model, n_values, phi_values)
    return popt, pcov
```

### 3.2 统计分析

**组水平推断**:
- 均值：ε̂ = mean(ε_sub-01, ..., ε_sub-16)
- 标准差：σ_ε = std(ε_i)
- 95% CI: ε̂ ± 1.96 × σ_ε/√16
- 单样本 t 检验：H₀: ε = 0.15

**效应量**:
- Cohen's d: (ε̂ - 0.15) / σ_ε
- R²: 指数模型拟合优度

### 5.2 不支持η_IIT 模型的结果

**替代模式**:
- 幂律衰减：η_IIT(n) ∝ n^{-α}
- 平台期：η_IIT(n) 在 n>n_c 后稳定
- 线性衰减：η_IIT(n) = η₀ - k×n

**理论含义**:
- 需修正η_IIT 模型形式
- 可能反映人脑特殊架构 (模块化、层级)
- 触发 DC-486 模型修正
