#### 2. 极限行为验证 ✅

| 极限 | 行为 | 物理合理性 |
|------|------|------------|
| N→2 | η_IIT → η₀ | ✅ 基准点正确 |
| N→∞ | η_IIT → 0⁺ | ✅ 效率趋于 0 (正侧) |
| ε→0 | η_IIT → η₀ | ✅ 理想算法极限 |
| ε→∞ | η_IIT → 0 | ✅ 极端情况合理 |

**对比 DC-483 修正前的线性模型**:
- 线性模型: η_IIT = η₀×(1-ε×N) → N→∞时 η_IIT→-∞ ❌
- 指数模型: η_IIT = η₀×exp[-ε×(N-2)] → N→∞时 η_IIT→0⁺ ✅

#### 3. 与 T418 v4.1 兼容性 ✅

**联合公式**:
$$\Phi_{actual}(N) = \eta_0 \times \exp[-\varepsilon(N-2)] \times \frac{k_0(N-1)}{1+(N-1)/N_c}$$

- T418 单独预测Φ随 N 线性增长后饱和
- η_IIT 预测效率随 N 指数衰减
- **联合行为**: Φ_actual 先增长后衰减 → **存在最优系统规模 N_opt**

**新预测**: 对于给定平台，存在最优 N 使Φ_actual 最大化
