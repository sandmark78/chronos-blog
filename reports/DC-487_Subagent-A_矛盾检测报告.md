**🟡 记录问题 1: η_IIT 与ε参数符号混用风险**

```
DC-483 定义: η_IIT(N) = η₀ × exp[-ε×(N-2)]
             其中ε = 效率衰减常数

DC-487 使用: η_corr(N) = η_IIT × exp(-N/N_corr)
             其中η_IIT 有时被当作常数 (而非 N 的函数)

潜在混淆:
- DC-483: η_IIT 是 N 的函数
- DC-487 某些段落: η_IIT 被当作平台依赖常数 (如η_IIT ≈ 0.15)

建议澄清:
- 明确区分 η_IIT(N) [函数] vs η_IIT(platform) [特定 N 下的值]
- 或引入新符号: η_IIT^0(platform) = N=2 时的基准效率
```

**🟡 记录问题 2: fMRI 采样数公式的边界条件**

```
DC-487 定义:
n_samples(N) = 100           for N ≤ 20
n_samples(N) = 500 + (N-20)×50  for 20 < N ≤ 30

边界检查:
- N=20: n_samples = 100
- N=21: n_samples = 500 + 1×50 = 550

问题: N 从 20→21 时，采样数从 100 跳跃到 550 (5.5 倍突变)

建议改进:
- 使用平滑过渡: n_samples(N) = 100 × exp[(N-20)/10] for N > 20
- 或明确说明跳跃是故意的 (计算成本 vs 统计效力的权衡)
```

**结论:** DC-487 内部基本自洽，2 个🟡记录问题为符号澄清和边界条件优化，非阻塞性矛盾。
