### 2. 极限行为：T→0, T→∞, N→1, N→∞极限是否物理合理？

**验证结果:** ✅ **通过 (行为合理)**

#### N→1 极限 (单子系统)

**IIT 4.0 上限:**
```
Φ_max(N=1) = 1 · log₂(1) / 2 = 0 bits

η_IIT(N=1) = Φ_measured / 0 → 未定义
```

**物理意义:** 单子系统系统无整合信息 (IIT 基本公理)，η_IIT 在 N=1 时无定义是合理的。

**DC-491 处理:** 分析使用 N≥2 (ROI 数量≥2)，避开 N=1 奇点 ✅

#### N→∞极限 (大系统)

**理论预测 (T418 v4.2 + T429):**
```
Φ_GHPII(N→∞) = k₀ × N_c (饱和值)

η_IIT(N→∞) = Φ_measured / Φ_theoretical_max
           = (k₀ × N_c) / [N·log₂(N)/2]
           → 0 (分母~N·log N，分子~常数)
```

**DC-489 经验模型:**
```
η_corr(N) = η_IIT × exp(-N/N_corr)

N → ∞: η_corr → 0 ✅ (指数衰减)
```

**物理意义:** 大系统时理论最大Φ ~ N·log N 增长，但实际Φ受退相干限制饱和，因此η_IIT → 0。

**DC-491 使用:** 人脑有效自由度 N_eff ~ 10²-10³ (ROI 数量)，远小于 N→∞极限，经验模型适用 ✅

#### T→0 极限 (零温)

**T423 v1.1 预测:**
```
k₀(T→0) = k₀(0) × [1 - (0/T_crit)^δ] = k₀(0) ✅
N_c(T→0) = N_c(0) / [1 + (0/T_γ)^m] = N_c(0) ✅
Φ_max(T→0) = k₀(0) × N_c(0) (最大值)
```

**DC-491 应用:**
- 人脑 fMRI: T ≈ 310K (体温，固定)
- 不扫描温度，η_IIT 定义为 310K 下的有效值
- 与 T→0 极限无直接冲突 (正交关系)

**结论:** ✅ T→0 极限行为合理，DC-491 不涉及温度扫描

#### T→∞极限 (高温)

**T423 v1.1 预测:**
```
k₀(T→∞) = k₀(0) × [1 - (∞)^δ] → 负值 ❌ (T423 v1.1 问题)

修正 (KC-441 T436 v1.2):
```
