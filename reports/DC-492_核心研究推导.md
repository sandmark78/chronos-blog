## P0: N_corr 真实硬件验证规划

### C-DC489-01 矛盾状态更新

**原始矛盾:**
- 拟合值：N_corr = 142.86 [120, 170] (A20 Phase 16A, IBM Lagos 模拟)
- "预期值"：8-12 (来源不明)
- 偏差：~12 倍 → 标记为🟡重要矛盾

**DC-492 重新分析:**

基于平台参数严格计算 (IBM Lagos):
```
N_corr = ln(2) / (γ × τ_gate)

参数:
  γ = 70 kHz (退相干率，IBM Lagos T1/T2 标定)
  τ_gate = 100 ns (CZ 门时间)

计算:
  N_corr = 0.693 / (70×10³ × 100×10⁻⁹)
         = 0.693 / 0.007
         ≈ 99
```

**新预期值:** N_corr ≈ 99  
**拟合值:** N_corr = 142.86  
**偏差:** 142.86 / 99 ≈ 1.44 倍

**解读:**
- 1.4 倍偏差在量子硬件实验误差范围内 (典型 2-3 倍)
- 拟合值 142.86 的 95% CI 为 [120, 170]，包含 99
- **矛盾可能是预期值计算错误，而非拟合错误**

**"8-12"预期来源追溯:**
- 搜索 DC-489 之前实验记录
- 可能来源：早期离子阱/超导量子经验法则 (γ~1-10 kHz, τ_gate~1-10 μs)
- 不适用于 IBM Lagos (γ~70 kHz, τ_gate~100 ns)

**状态更新:** C-DC489-01 从🟡重要降级为🟢记录级

### IBM Quantum 真实硬件验证实验设计

**目标:** 在真实量子硬件上验证η_corr 标度律，确认 N_corr 真实性质

**硬件:** IBM Quantum (ibm_lagos 或 ibm_perth)  
**电路:** GHZ-8/10/12 (与 A20 Phase 16A 模拟一致)  
**采样数:** 16384 shots (vs 模拟 8192 shots, 提高统计精度)  
**重复:** 5 次独立运行 (评估运行间变异性)

**实验流程:**

```python
# 伪代码
for n_qubits in [8, 10, 12]:
    for run in range(5):
        # 准备 GHZ 态
        circuit = QuantumCircuit(n_qubits)
        circuit.h(0)
        for i in range(n_qubits - 1):
            circuit.cx(i, i+1)
        
        # 添加噪声 (自然退相干)
        # 无额外噪声门，依赖硬件本征噪声
        
        # 测量
        circuit.measure_all()
        
        # 执行
        job = execute(circuit, backend, shots=16384)
        counts = job.result().get_counts()
        
        # 计算Φ_GHPII (基于输出分布)
        phi = calculate_phi_ghpii(counts)
        
        # 存储
        results.append({
            'n_qubits': n_qubits,
            'run': run,
            'phi': phi,
            'shots': 16384
        })
```

**数据分析:**

1. **η_corr 标度律拟合:**
   ```
   Φ_measured = η_corr × Φ_theoretical_max × exp(-N/N_corr)
   
   拟合参数:
     η_corr: 效率因子 (预期 0.10-0.20)
     N_corr: 关联衰减特征尺度 (预期 99-150)
   ```

2. **与模拟对比:**
   - A20 Phase 16A (模拟): η_IIT = 0.150, N_corr = 142.86
   - 真实硬件预期：η_IIT ≈ 0.12-0.18 (略低，因额外噪声源)
   - N_corr 预期：99-150 (与拟合值兼容)

3. **成功标准:**
   - R² > 0.95 (标度律拟合优度)
   - N_corr 落在 [99, 170] 区间内
   - η_corr 与模拟值偏差 < 30%

**时间线:**
- Day 1: IBM Quantum 账户准备 + 队列测试
- Day 2-3: 数据采集 (5 runs × 3 circuits × ~10 min/run = ~2.5h)
- Day 4: 数据分析 + 报告撰写

### 数据集搜索

**理想数据集特征:**
- fMRI: 静息态 + 任务态 (同一被试)
- 行为学：认知能力测试 (智力/工作记忆/执行功能)
- 被试数：>50 (统计效力)

**候选数据集:**

1. **Human Connectome Project (HCP):**
   - 被试：1200+
   - fMRI: 静息态 + 7 任务态
   - 行为学：NIH Toolbox (认知/情绪/运动)
   - 优势：大样本 + 丰富行为学
   - 挑战：数据量大 (TB 级)，需高性能计算

2. **Cam-CAN (Cambridge Centre for Ageing and Neuroscience):**
   - 被试：700+ (年龄 18-88)
   - fMRI: 静息态 + 任务态
   - 行为学：认知测试电池
   - 优势：年龄跨度大，可验证年龄 -η_IIT 关联
   - 挑战：年龄混杂需控制

3. **OpenNeuro ds000228:**
   - 被试：97
   - fMRI: 静息态 + 任务态 (情绪/认知)
   - 行为学：情绪/认知测试
   - 优势：中等样本，易处理
   - 挑战：智力测试有限

**推荐方案:** HCP (大样本 + 丰富行为学) — 但需高性能计算支持

---

### 分析流程

**步骤 1:** 被试选择
- 从 HCP 选择 100-200 被试 (平衡年龄/性别)
- 排除头动过大 (>0.3mm FD)

**步骤 2:** η_IIT 计算 (每被试)
- 静息态η_IIT
- 任务态η_IIT (多任务平均)
- 提升幅度：Δη_IIT = (η_IIT_task - η_IIT_rest) / η_IIT_rest

**步骤 3:** 智力分数提取
- HCP NIH Toolbox: 流体智力 (gF) 因子分
- 或：Raven's matrices 分数 (若有)

**步骤 4:** 相关分析
```
r = corr(Δη_IIT, gF)

预期:
  r > 0.3 (中等正相关)
  p < 0.05 (统计显著)
```

**步骤 5:** 控制变量回归
```
Δη_IIT = β₀ + β₁×gF + β₂×age + β₃×sex + β₄×head_motion + ε

预期:
  β₁ > 0 (gF 正向预测)
  p(β₁) < 0.05
```

**预期时间线:**
- Week 1-2: HCP 数据申请 + 下载 (需 DBAG 协议)
- Week 3-4: 预处理 + η_IIT 计算
