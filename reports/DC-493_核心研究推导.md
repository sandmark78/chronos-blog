### 执行步骤

#### 步骤 1: 量子电路制备

```python
# GHZ-n 态制备电路
def ghz_circuit(n_qubits):
    qc = QuantumCircuit(n_qubits)
    qc.h(0)  # Hadamard on qubit 0
    for i in range(1, n_qubits):
        qc.cx(0, i)  # CNOT from qubit 0 to i
    return qc
```

#### 步骤 2: 退相干时间测量

**T1 测量:** 反转恢复序列  
**T2 测量:** Hahn echo / CPMG  
**预期值:** T1 ~ 100-200 μs, T2 ~ 50-150 μs (IBM Lagos)

#### 步骤 3: η_corr 提取

**拟合模型:** P_survival(t) = exp(-t/τ_eff)  
**η_corr = τ_eff / τ_ideal** (τ_ideal 为无噪声理论值)

**预测:**
- GHZ-8: η_corr ≈ 0.14-0.16
- GHZ-10: η_corr ≈ 0.12-0.14
- GHZ-12: η_corr ≈ 0.10-0.12

### T493-01: η_IIT 跨模态普适性定理

**陈述:** η_IIT = Φ_actual / Φ_max 在理想测量条件下跨平台差异<5%

**公式:**
```
η_IIT^platform = Φ_platform / Φ_max^platform

其中：
- Φ_platform: 平台测量的实际整合信息 (bits)
- Φ_max^platform: 该平台理论上限 (bits)

跨平台一致性条件：
|η_IIT^fMRI - η_IIT^EEG| / η_IIT^fMRI < 5%
|η_IIT^fMRI - η_IIT^MEG| / η_IIT^fMRI < 5%
|η_IIT^fMRI - η_IIT^SC| / η_IIT^fMRI < 5%
```

**物理基础:**
1. η_IIT 为无量纲效率参数
2. 平台依赖性通过Φ_max 归一化消除
3. 剩余差异来自测量噪声和系统误差

**预测:**
- η_IIT^fMRI (静息) = 0.145±0.015
- η_IIT^EEG (静息) = 0.140±0.020
- η_IIT^MEG (静息) = 0.150±0.018
- η_IIT^SC (GHZ-10) = 0.14±0.03

- ITLCT 独有：基于γ×τ_gate 的微观推导

**验证时间线:** 1-2 周 (IBM Quantum 执行完成)
