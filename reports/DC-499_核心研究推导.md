## 阶段 2: 核心研究推导

### 焦点：论文草稿 v0.2 — 温度依赖形式化

#### 问题陈述
DC-497 的论文 v0.1 中，温度依赖仅在第 2.3 节 (T423) 和第 3.7 节 (极限行为) 简要提及，缺乏：
1. 完整的温度依赖公式推导
2. 与 T436/T429 的统一表述
3. 实验可检验的定量预测

#### 推导目标
在论文 v0.2 中补充：
1. **N_corr(T) 完整形式:** N_corr(T) = N_corr(0) × exp[-(T/T_crit)^δ]
2. **δ指数来源:** 唯象拟合 (1.8-2.2) + 热容论证 (δ=1+α=2)
3. **适用范围:** T < T_crit (严格有效) / T > T_crit (外推，需高温唯象模型)

#### 步骤 1: 从 T423 继承 k₀(T)
```
k₀(T) = k₀(0) × exp[-(T/T_crit)^δ]
```
- T_crit = ℏω₀/(2k_B) (物理温度定义)
- δ ≈ 2 (唯象拟合 + 平均场近似)

#### 步骤 2: N_corr 与 k₀的关系
从 T495-01:
```
N_corr = Φ_GHPII / (k₀ × N)
```

在温度 T 下:
```
N_corr(T) = Φ_GHPII(T) / [k₀(T) × N]
```

#### 步骤 3: Φ_GHPII 的温度依赖
从 T429:
```
Φ(T) = k₀(T) × (N-1) / [1 + (N-1)/N_c(T)]
```

其中 N_c(T) = N_c(0) × exp[-(T/T_crit)^δ] (从 T423 继承)

#### 步骤 4: N_corr(T) 完整形式
代入得:
```
N_corr(T) = [k₀(T) × (N-1) / (1 + (N-1)/N_c(T))] / [k₀(T) × N]
          = (N-1)/N × 1/[1 + (N-1)/N_c(T)]
```

在 N ≫ 1 极限下:
```
N_corr(T) ≈ 1/[1 + N/N_c(T)]
          = N_c(T) / [N + N_c(T)]
```

#### 步骤 5: 高低温极限
- **低温 (T ≪ T_crit):** N_c(T) ≈ N_c(0), N_corr ≈ N_c(0)/N (常数)
- **高温 (T → T_crit):** N_c(T) → 0, N_corr → 0 (指数衰减)

#### 步骤 6: 与 N²标度律的兼容性
在 N ≪ N_c 条件下:
```
N_corr ≈ N_c(T) / N_c(T) × N²/ln²(2) = N²/ln²(2)
```
温度依赖通过 N_c(T) 间接体现。

**完整形式:**
```
N_corr(N, T) = N²/ln²(2) × exp[-(T/T_crit)^δ] × [1 + 𝒪(1/N)]
```

#### 步骤 7: 实验可检验预测
| 平台 | T_crit | 室温 (300K) 衰减因子 | 预测 |
|------|--------|---------------------|------|
| 超导 | ~0.1K | exp[-(300/0.1)^2] ≈ 0 | N_corr → 0 |
| 离子阱 | ~1K | exp[-(300/1)^2] ≈ 0 | N_corr → 0 |
| 中性原子 | ~0.01K | exp[-(300/0.01)^2] ≈ 0 | N_corr → 0 |
| 光子 | N/A (室温工作) | 无温度依赖 | N_corr 常数 |

**独特预测:** 物质平台 (超导/离子阱/中性原子) 在室温下 N_corr 应指数衰减至 0，而光子平台无此效应。
