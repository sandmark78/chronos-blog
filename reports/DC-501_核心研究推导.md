## P1: C-DC500-A01 修正 (N ≪ N_c 近似推导逻辑断裂)

### 问题回顾

DC-499 步骤 6 存在逻辑跳跃：
```
原文：在 N ≪ N_c 条件下:
      N_corr ≈ N_c(T) / N_c(T) × N²/ln²(2) = N²/ln²(2)

问题：N_c(T)/N_c(T) = 1，但这与 N²/ln²(2) 的来源无关
```

### DC-501 修正推导

**N_corr 的两个独立推导路径：**

#### 路径 A (T495-01 信息几何)

从 Fisher 信息矩阵第一性原理推导 (KC-495 步骤 3)：

```
定义：N_corr = (N / ln 2)² × I_F / I_F^{max}

其中：
- I_F = Fisher 信息矩阵行列式
- I_F^{max} = 最大可能 Fisher 信息 (完全相干极限)

在完全相干极限 (T→0, N ≫ 1):
I_F / I_F^{max} → 1

因此：
N_corr = N²/ln²(2) × [1 + O(1/N) + O(τ_gate/τ_dec)²]
```

**适用范围:**
- N ≥ 8 (有限 N 修正<12%)
- T ≪ T_crit (量子相干主导)
- τ_gate ≪ τ_dec (门操作快于退相干)

#### 路径 B (T429 温度依赖)

从Φ定义与 N_c(T) 关系推导：

```
定义：Φ = k₀(T) × (N-1) / [1 + (N-1)/N_c(T)]

在 N ≪ N_c 条件下：
Φ ≈ k₀(T) × (N-1) × [N/N_c(T)]
  = k₀(T) × N(N-1) / N_c(T)

对比 T495-01 的 Φ = k₀ × N_corr / N：
N_corr = N² × k₀(T) / [k₀ × N_c(T)]
```

**注意：** 路径 B 给出的是 N_corr 与 N_c(T) 的关系，而非 N_corr 的绝对值。

#### 统一形式 (DC-501 新推导)

结合路径 A 的标度律与路径 B 的温度依赖：

```
N_corr(N, T) = N²/ln²(2) × η(T) × [1 + O(1/N)]

其中温度效率因子：
η(T) = exp[-(T/T_crit)^δ]  (T < T_crit)
η(T) = (T_crit/T)^γ        (T > T_crit, 唯象)

完整形式：
N_corr(N, T) = N²/ln²(2) × exp[-(T/T_crit)^δ] × [1 + O(1/N)]
```

**适用范围澄清：**
| 条件 | 公式 | 误差 |
|------|------|------|
| N ≥ 8, T < 0.5×T_crit | N²/ln²(2) × exp[-(T/T_crit)^δ] | <10% |
| N ≥ 8, 0.5-1.5×T_crit | N²/ln²(2) × exp[-(T/T_crit)^δ] | 10-20% (外推) |
| N < 8 | 需添加 O(1/N) 修正 | 依 N 而定 |
| T > 1.5×T_crit | 需高温唯象模型 | 待研究 |

### 与 T429 的兼容性

**T429 完整形式：**
```
Φ = k₀(0) × exp[-(T/T_crit)^δ] × (N-1) / [1 + (N-1)/N_c(T)]
```

**T495-01 形式：**
```
Φ = k₀(T) × N_corr / N
  = k₀(0) × exp[-(T/T_crit)^δ] × [N²/ln²(2) × exp[-(T/T_crit)^δ]] / N
  = k₀(0) × N/ln²(2) × exp[-2(T/T_crit)^δ]
```

**表观矛盾：** T495-01 给出 exp[-2(T/T_crit)^δ]，而 T429 给出 exp[-(T/T_crit)^δ]

**DC-501 解决：**
- T429 的温度依赖来自 k₀(T) (单比特信息容量)
- T495-01 的温度依赖来自 N_corr (关联比特数)
- 两者物理机制不同，**不应直接比较指数系数**

**澄清：** N_corr 的温度依赖应通过实验确定，当前采用 exp[-(T/T_crit)^δ] 是唯象假设，待 IBM Quantum 验证。

## P1: C-DC500-A03 修正 (光子平台表述过于绝对)

### 问题回顾

KC-499 步骤 7 表格：
```
| 光子 | N/A (无质量) | 1 (无温度依赖) | N_corr 常数 |
```

**问题：** "无温度依赖"表述过于绝对，未区分 T_crit vs T_cross 机制。

### DC-501 修正

**光子平台温度依赖性澄清：**

| 机制 | T_crit | T_cross | 物理来源 |
|------|--------|---------|---------|
| 光子平台 | **不适用** (无能隙) | **~2000±1000K** | 热光子退相干 |

**修正后表格：**

```
| 平台 | T_crit | T_cross | N_corr 温度依赖 |
|------|--------|---------|----------------|
| 超导 | ~0.1K | ~1-10K | exp[-(T/T_crit)^δ] (T<T_crit), 平台 (T_crit<T<T_cross), 衰减 (T>T_cross) |
| 离子阱 | ~0.1K | ~100-500K | 类似超导，T_cross 更高 |
| 中性原子 | ~0.1K | ~300-600K | 类似超导，T_cross 中等 |
| 光子 | N/A (无能隙) | ~2000±1000K | **常数 (T<T_cross)**, 热光子衰减 (T>T_cross) |
```

**备注：**
- 光子平台无 T_crit 机制 (无能级激发阈值)
- 但存在 T_cross 机制 (热光子退相干，~2000K)
- 室温 (300K) ≪ T_cross，故 N_corr 近似常数
- 高温区 (T > T_cross) 需考虑热光子效应：N_corr(T) ≈ N_corr(0) × [1 - (T/T_cross)^γ]

### 物理机制解释

**为什么光子平台无 T_crit？**
- T_crit = ℏω₀/(2k_B) 来自能级激发阈值
- 光子无质量，连续谱，无离散能隙
- 因此 T_crit 机制不适用

**为什么光子平台有 T_cross？**
- T_cross 来自退相干机制转换 (低频噪声 → 高频热激发)
- 光子平台在 T > T_cross 时产生热光子背景
- 热光子与信号光子相互作用导致退相干
- T_cross ~2000K 来自典型光学频率 (~100THz) 的热激发阈值
