### 与 T420Φ_max 兼容性检查

**检查项：** N_corr 与Φ_max≈2 bits 的关系？

**分析：**

根据 T420 v1.6 (KC-411)：
```
Φ_max(platform) = k₀ × N_c(platform)
- 超导：Φ_max ≈ 2.1 bits
- 离子阱：Φ_max ≈ 20.8 bits
- 光量子：Φ_max ≈ 203.8 bits
```

根据 T-DC504-01 / T495-01：
```
N_corr = Φ_eff / (k₀ × N) × N/ln(2)
       = Φ_eff / k₀ × 1/ln(2)
```

**关系推导：**

在 N ≪ N_c 且 T ≪ T_crit 极限下：
```
Φ_eff ≈ k₀ × (N-1)  (T420 近似)

代入 N_corr 定义：
N_corr ≈ [k₀ × (N-1)] / k₀ × 1/ln(2)
       = (N-1) / ln(2)
```

但 T495-01 的完整推导给出：
```
N_corr = N²/ln²(2)
```

**关键澄清：**

这两个结果来自**不同的定义路径**：
1. **路径 A (T420Φ_max)：** Φ_max = k₀ × N_c (饱和值)
2. **路径 B (T495-01 N_corr)：** N_corr = N²/ln²(2) (标度律)

**两者关系：**
- N_corr 是**归一化关联数**，不直接等于Φ_max
- N_corr 与Φ_max 通过 k₀ 和 N_c 间接关联
- **无矛盾：** 两者描述不同物理量

**✅ 结论：** T-DC504-01 与 T420Φ_max 兼容，但需澄清 N_corr 与Φ_max 的定义差异。
