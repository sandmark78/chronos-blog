## 阶段 2: 核心研究推导

### T-DC516-01 定理 1: N_corr 温度依赖实验设计深化 (D-499-01)

**预测 D-499-01:**

$$\frac{N_{\text{corr}}(T_1)}{N_{\text{corr}}(T_2)} = \exp\left[-\left(\frac{T_1}{T_{\text{crit}}}\right)^\delta + \left(\frac{T_2}{T_{\text{crit}}}\right)^\delta\right]$$

**实验设计:**

#### 1. 温度范围选择

基于 T495-01 v1.2 的适用范围 T < T_crit：

| 平台 | T_crit | 实验温度范围 | 测量点数 |
|------|--------|--------------|----------|
| **超导 (IBM Quantum)** | ~0.1 K (100 mK) | 10 mK → 300 mK | 11 点 (间隔 20 mK) |
| **离子阱** | ~1 K | 100 mK → 3 K | 11 点 |
| **中性原子** | ~0.05 K | 5 mK → 150 mK | 11 点 |
| **光子** | N/A (无温度依赖) | 室温固定 | 1 点 (基准) |

**理由:**
- 下限：平台基温 (IBM Quantum 基温~10 mK)
- 上限：3×T_crit (覆盖 T < T_crit 严格区 + T ≈ T_crit 过渡区)
- 间隔：ΔT = 0.2×T_crit (保证足够分辨率捕捉指数衰减)

#### 2. 测量协议

**步骤 1: GHZ 态制备**
```
|0⟩^⊗N → Hadamard → CNOT 级联 → |GHZ_N⟩ = (|0⟩^⊗N + |1⟩^⊗N)/√2
```

**步骤 2: 量子 Fisher 信息测量**
- 方法：量子态层析 (QST) + QFI 重建
- 或：干涉对比度测量 (更快速，适用于大 N)

**步骤 3: N_corr 提取**
从测量的Φ(T,N) 拟合：
$$\Phi(T,N) = k_0(T) \cdot F_{\text{thermal}}(T) \cdot \frac{N_{\text{corr}}(N,T)}{N + N_{\text{corr}}(N,T)}$$

在 N ≪ N_c 极限下：
$$\Phi(T,N) \approx k_0(T) \cdot F_{\text{thermal}}(T) \cdot \frac{N_{\text{corr}}(N,T)}{N_c(T)}$$

**步骤 4: 温度依赖比值计算**
$$R(T_1, T_2) = \frac{N_{\text{corr}}(T_1)}{N_{\text{corr}}(T_2)} = \frac{\Phi(T_1,N) \cdot N_c(T_1)}{\Phi(T_2,N) \cdot N_c(T_2)} \cdot \frac{k_0(T_2) \cdot F_{\text{thermal}}(T_2)}{k_0(T_1) \cdot F_{\text{thermal}}(T_1)}$$

#### 3. 预期结果

**理论预测 (T495-01 v1.2):**

| T/T_crit | N_corr/N² | 相对比值 R(T, T_ref=0.1T_crit) |
|----------|-----------|--------------------------------|
| 0.1 | 0.99 | 1.00 |
| 0.3 | 0.96 | 0.97 |
| 0.5 | 0.88 | 0.89 |
| 0.7 | 0.74 | 0.75 |
| 0.9 | 0.57 | 0.58 |
| 1.0 | 0.47 | 0.48 |
| 1.5 | 0.22 | 0.22 |
| 2.0 | 0.08 | 0.08 |
| 2.5 | 0.02 | 0.02 |
| 3.0 | 0.006 | 0.006 |

**拟合参数:**
- δ (临界指数): 预期 1.8-2.2 (平均场≈2)
- T_crit: 从拟合提取，预期~100 mK (IBM Quantum)

#### 4. 误差分析

**系统误差源:**

| 误差源 | 量级 | 修正方法 |
|--------|------|----------|
| 温度测量不确定度 | ±5 mK | 原位温度计校准 |
| GHZ 态制备保真度 | 95-98% | 误差缓解 (Zero-Noise Extrapolation) |
| 读出误差 | 2-5% | 读出误差校正矩阵 |
| 退相干 (T₁, T₂) | T₁~100μs, T₂~50μs | 快速测量 (<10μs) |
| 串扰 | 1-3% | 校准 + 补偿脉冲 |

**统计误差:**
- 每点重复测量：1000 shots
- 标准误差：σ/√1000 ≈ 3% (假设σ~100%)
- 总不确定度：~5-8% (系统 + 统计)

#### 5. 验证标准

**通过标准:**
- 拟合优度：R² > 0.95
- δ指数：1.5 < δ < 2.5 (与平均场预测一致)
- T_crit：与独立测量 (退相干率γ(T)) 一致 within 20%

**失败标准:**
- R² < 0.8：模型形式可能错误
- δ < 1 或 δ > 3：需重新考虑物理机制
- T_crit 偏差 > 50%：可能遗漏其他温度依赖机制

### T-DC516-02 定理 2: IBM Quantum 线路设计细化 (D-495-01)

**预测 D-495-01:**

$$\Phi(N) \propto N^\beta, \quad \beta = 2 \text{ (马尔可夫区)}$$

**实验设计:**

#### 1. 量子线路架构

**GHZ 态制备线路 (N 量子比特):**

```
q[0]: ──H──■─────────────────────────
           │
q[1]: ─────■──■─────────────────────
              │
q[2]: ────────■──■──────────────────
                 │
...              ...
                 │
q[N-1]: ──────────...──■────────────
                        │
ancilla: ───────────────X── (可选，用于误差检测)
```

**门序列:**
1. H 作用于 q[0]
2. CNOT(q[0], q[1]), CNOT(q[1], q[2]), ..., CNOT(q[N-2], q[N-1])
3. 总门数：1 + (N-1) = N 门
4. 线路深度：N (串行 CNOT)

**优化方案 (并行 CNOT):**
```
q[0]: ──H──■─────────■─────────
           │         │
q[1]: ─────■─────────┼─────────
                     │
q[2]: ───────────────■─────────
...
```
- 深度：log₂(N) (树状结构)
- 适用于耦合图允许的平台

#### 2. N 范围选择

基于 DC-513 修正 (N=2-20) 和 N_c(IBM)≈69：

| N | N/N_c | 预期区域 | 测量优先级 |
|---|-------|----------|------------|
| 2-5 | 3-7% | N ≪ N_c (线性区) | 高 (基准) |
| 6-10 | 9-14% | N ≪ N_c (线性区) | 高 (β拟合) |
| 11-15 | 16-22% | 过渡区起始 | 中 |
| 16-20 | 23-29% | 过渡区 | 中 (饱和趋势) |

**理由:**
- N=2-10: 精确测定β指数 (线性区最敏感)
- N=11-20: 观测饱和趋势起始 (验证 N_c 量级)
- N>20: 留待后续实验 (需要更高保真度)

#### 3. Φ测量协议

**方法 A: 量子态层析 (QST)**

- 测量基：Pauli 基 {X, Y, Z}^⊗N
- 测量次数：3^N (指数增长，N≤10 可行)
- 重建：最大似然估计 (MLE)
- 计算Φ：从重建的ρ计算 QFI

**方法 B: 干涉对比度 (推荐，N>10)**

```
制备 |GHZ⟩ → 相位演化 U(θ) = exp(-iθΣZ_i/2) → 逆制备 → 测量 P(|0⟩^⊗N)
```

对比度：
$$C = \frac{P_{\text{max}} - P_{\text{min}}}{P_{\text{max}} + P_{\text{min}}}$$

Φ与对比度关系：
$$\Phi \approx \frac{C^2}{1 - C^2} \cdot N^2 \text{ (小 C 近似)}$$

**方法 C: 洛氏层析 (Shadow Tomography)**

- 测量次数：O(log(N) · ε⁻²) (远优于 QST)
- 适用于 N>15
- 实现：随机 Clifford 测量 + 经典阴影重建

#### 4. 误差预算

**目标精度:** β不确定度 < 0.2

| 误差源 | 对β的影响 | 缓解策略 |
|--------|-----------|----------|
| GHZ 制备误差 | Δβ ~ 0.1-0.3 | 误差缓解 (ZNE) |
| 读出误差 | Δβ ~ 0.05-0.1 | 读出校正矩阵 |
| 退相干 | Δβ ~ 0.1-0.2 | 快速测量 + 低温 |
| 串扰 | Δβ ~ 0.05 | 校准 + 补偿 |
| 统计涨落 | Δβ ~ 0.02 (1000 shots) | 增加 shots |

**总预算:** Δβ ~ 0.15-0.25 (满足 D-495-01 验证需求)

### T-DC516-03 定理 3: T_cross 平台依赖性文献调研 (D-501-01, 部分完成)

**预测 D-501-01:**

T_cross (黑体辐射 crossover 温度) 具有平台依赖性：

| 平台 | T_cross 预期 | 物理来源 |
|------|--------------|----------|
| **超导** | ~100-200 K | 微波光子背景 + 封装辐射 |
| **离子阱** | ~300-500 K | 黑体辐射诱导退相干 |
| **中性原子** | ~100-300 K | 光晶格光子散射 |
| **光子** | ~2000 K | 自由空间热光子 (极高) |

**文献调研结果 (快速搜索):**

1. **超导量子比特:**
   - Krantz et al. (2019): 微波背景辐射在~100 mK 可忽略，但在>1 K 显著
   - 封装热辐射：~100-200 K 等效温度 (取决于屏蔽)

2. **离子阱:**
   - Brownnutt et al. (2015): 黑体辐射在室温下诱导退相干
   - T_cross ~ 300 K (与环境温度相当)

3. **中性原子:**
   - 光晶格深度温度依赖：T_cross ~ 100-300 K (取决于晶格激光)

4. **光子:**
   - 自由空间：T_cross ~ 2000 K (热光子数~1 的温度)
   - 波导/腔：T_cross 降低 (模式限制)

**结论:** 平台依赖性确认，但精确值需实验标定。
