### P0: T495-01 文档修正

**T-DC519-02: 弱耦合近似显式声明**

**问题背景:** DC-517/DC-518 中 T495-01 使用Φ≈k₀(N-1) 近似，但未显式声明适用条件。

**修正内容:**

**原表述:**
```
Φ(N) ≈ k₀ × (N-1)  (N ≪ N_c 时)
```

**修正后表述:**
```
Φ(N) = k₀ × (N-1)/[1 + (N-1)/N_c]  (T420 完整形式)

弱耦合近似 (N ≪ N_c):
当 (N-1)/N_c ≪ 1 时，分母 [1 + (N-1)/N_c] ≈ 1
→ Φ(N) ≈ k₀ × (N-1)

适用条件:
- N ≪ N_c: 量子比特数远小于临界尺寸
- 对各平台:
  - 超导 (N_c≈69): N < 10 时误差<15%, N < 5 时误差<7%
  - 离子阱 (N_c≈6930): N < 1000 时误差<15%
  - 中性原子 (N_c≈47200): N < 7000 时误差<15%
  - 光子 (N_c→∞): 近似始终有效

本实验 (IBM Quantum, N=2-20):
- N=20 时，(N-1)/N_c = 19/69 ≈ 0.275
- 近似误差: 1/(1+0.275) ≈ 0.78 → 低估~22%
- 结论: N=2-12 区近似合理 (误差<15%), N=13-20 区需用完整形式
```

## 🔬 核心推导 (质量优先)

**T-DC519-04: β=2 预测的鲁棒性分析**

**问题:** D-495-01 核心预测β=2，但需考虑可能的系统误差来源。

**误差来源分析:**

**1. 退相干不均匀性:**
- 各量子比特γ_i 存在差异 (芯片制造公差)
- 影响: Φ 测量值系统性偏低
- 估计: γ 分布σ_γ/γ ~ 5-10%
- 对β的影响: < 0.1 (通过误差传播)

**2. 读出误差:**
- IBM Quantum 读出保真度 ~95-98%
- 影响: Φ 测量值系统性偏低
- 估计: 读出误差~2-5%
- 对β的影响: < 0.05 (标度指数对系统性偏差不敏感)

**3. 有限采样误差:**
- 4096 shots/点，统计误差~1/√4096 ≈ 1.6%
- 对β的影响: 通过拟合不确定度体现 (~0.15-0.2)

**4. 温度波动:**
- 稀释制冷机温度稳定性 ~±1 mK
- T=10 mK 时，相对波动~10%
- 对β的影响: < 0.05 (T ≪ T_crit 时温度依赖弱)

**综合不确定度估计:**
```
β_total_uncertainty = √(σ_fit² + σ_sys²)
                     = √(0.15² + 0.12²) ≈ 0.19

保守估计: β = 2.0 ± 0.2 (95% 置信区间)

验证标准:
- 强验证: β_fitted ∈ [1.8, 2.2] 且 p(β=2) > 0.1
- 弱验证: β_fitted ∈ [1.6, 2.4] 且 p(β=2) > 0.05
- 证伪: β_fitted ∉ [1.6, 2.4] 或 p(β=2) < 0.05
```

**与竞争理论对比:**

| 理论 | β预测 | 物理机制 | 可证伪性 |
|------|-------|---------|---------|
| **ITLCT (GHPII)** | **β=2** | 量子关联标度 (N² 希尔伯特空间) | **高 (定量精确)** |
| 标准神经科学 | β≈1 | 经典整合 (线性叠加) | 中 (定性) |
| IIT 3.0/4.0 | 无明确预测 | Φ 计算依赖特定网络结构 | 低 (参数可调) |
| 量子经典混合 | β∈[1,2] | 部分量子增强 | 中 (依赖混合比例) |

**结论:** β=2 是 ITLCT 的**独特定量预测**，与竞争理论有明确区分度。
