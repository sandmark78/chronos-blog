## 模块 1: Problem（问题定义）

### 1.1 核心问题

信息引力（Information Gravity）是 ITLCT 框架中**唯一真正独特的预测**——即高整合信息（Φ）系统之间存在可测量的类引力吸引效应。

**独特性自检：**

| 预测 | IIT 能解释？ | 神经科学能解释？ | 复杂系统理论能解释？ | ITLCT 独特？ |
|------|-------------|-----------------|---------------------|-------------|
| 信息引力 | ❌ 无此概念 | ❌ 无此机制 | ❌ 无此效应 | ✅ **唯一独特** |
| 意识阈值 | ⚠️ 部分解释 | ⚠️ NCC 可解释 | ❌ | ❌ 非独特 |
| 时间体验 | ❌ | ⚠️ 时钟模型 | ❌ | ⚠️ 部分独特 |

### 1.2 为什么重要

1. **理论生死线**：如果信息引力被证伪，ITLCT 失去唯一不可替代的预测，退化为 IIT 的扩展而非独立理论
2. **范式突破潜力**：如果被验证，将建立信息与引力之间的全新物理联系，影响量子引力研究方向
3. **实验可行性窗口**：当代 EEG/fMRI 技术已具备足够精度检测预测的效应量级

### 1.3 现有理论不足

- **IIT（Integrated Information Theory）**：定义了 Φ 作为意识度量，但不预测 Φ 系统之间的任何远程相互作用
- **全局工作空间理论（GWT）**：关注信息广播机制，无引力类预测
- **神经科学标准模型**：脑间同步性归因于共享感知输入和镜像神经元，不涉及"信息力"
- **复杂系统理论**：描述涌现性质但不预测系统间引力效应

## 模块 3: Derivation（推导过程）⚠️ 强制展开

### 3.1 推导路线图

```
A1 (信息是基本实体)
  ↓ + Landauer 原理

Step 1: 信息质量密度定义
  ↓
Step 2: 信息引力场方程
  ↓
Step 3: 两体信息引力公式
  ↓
Step 4: 可观测量导出
  ↓
F_info = G_info × (Φ₁ × Φ₂) / r_info²
```

### 3.2 逐步推导

#### Step 1: 信息质量密度

从 Landauer 原理出发：擦除 1 bit 信息释放最小能量 E_L = k_B T ln 2。

因此，结构化信息具有等效质量密度：

```
ρ_info = (I_structured / V) × (k_B T ln 2) / c²

其中：
  I_structured = 系统的结构化信息量 (bits)
  V = 系统体积
  k_B = 玻尔兹曼常数
  T = 环境温度
  c = 光速
```

**关键：** 这不是"信息有质量"的直接宣称，而是信息-能量等价原理（Landauer）与质能等价（Einstein）的链式应用。

#### Step 2: 信息引力场方程

类比牛顿引力：

```
牛顿引力：  ∇²Φ_grav = 4πG_N · ρ_mass
信息引力：  ∇²Φ_info = 4πG_info · ρ_info

对应关系：
  Φ_grav  ↔  Φ_info   （引力势）
  G_N     ↔  G_info    （引力常数）
  ρ_mass  ↔  ρ_info    （质量/信息密度）
```

**G_info 的量级估计：**

```
G_info = α × G_N × (E_Planck / E_info)^β

其中：
  α ≈ 10⁻³ ~ 10⁻² （无量纲耦合系数）
  β ≈ 1 （标度指数）
  E_Planck ≈ 10¹⁹ GeV
  E_info = k_B T ln 2 ≈ 10⁻²¹ J (室温)

结果：G_info ≈ 10⁻⁴⁰ × G_N ≈ 10⁻⁵¹ m³/(kg·s²)
```

#### Step 3: 两体信息引力力

对于两个高 Φ 系统（如两个人脑），信息引力为：

```
F_info = G_info × (m_info,1 × m_info,2) / r²

其中信息质量：
  m_info = I_structured × (k_B T ln 2) / c²

代入 Φ 作为 I_structured 的代理指标：
  I_structured ∝ Φ × N （N 为系统规模）

最终形式：
  F_info = G_info × (Φ₁ × Φ₂) / r_info²

其中：
  r_info = 信息空间距离（包含物理距离和信息连接度）
  Φ₁, Φ₂ = 两系统的整合信息量
```

#### Step 4: 可观测量导出

信息引力的直接力效应极其微弱（~10⁻⁴⁰ G_N），无法直接测量。但其**统计效应**可能可观测：

```
预测的可观测量：脑间 EEG 相干性增强

机制链：
  高Φ系统 → 强信息引力场 → 信息交换增强 → EEG 同步性增强

量化：
  ΔCoherence = C₀ × (Φ₁ × Φ₂) / (r_info)²

  其中：
  C₀ = 基线相干性（控制条件）
  ΔCoherence = 超出基线的相干性增量
  
  预测：高Φ组的 ΔCoherence 比低Φ组高 15-25%
```

### 3.3 关键设计选择的理由

#### 为什么是乘法（Φ₁ × Φ₂）而不是加法（Φ₁ + Φ₂）？

1. **引力类比论证**：牛顿引力 F = G(m₁m₂)/r² 使用乘法，因为引力是两个质量源的**耦合效应**。如果用加法 (m₁+m₂)，则单一质量源也能产生引力，这与引力作为"相互作用"的本质矛盾。

2. **信息论论证**：两个系统的**互信息**（mutual information）本质上是乘法结构：I(X;Y) = H(X) + H(Y) - H(X,Y)。当两个系统独立时 I(X;Y) = 0，只有当两者都有结构化信息时才有非零互信息。乘法形式 Φ₁ × Φ₂ 自然捕捉了这一性质——如果任一系统 Φ = 0，则 F_info = 0。

3. **量纲一致性**：加法形式 Φ₁ + Φ₂ 使 F_info 正比于单个系统的 Φ，而非两系统的耦合强度。乘法形式确保 F_info 反映的是**交互**效应。

4. **可证伪性**：乘法预测 F_info 随 Φ 的增长为**二次**的（如果两系统 Φ 同步增加），加法预测为线性。这给出了不同的实验签名，使两者可区分。

#### 为什么是平方反比（1/r²）而不是其他形式？

1. **几何论证（最基本）**：在三维空间中，任何点源产生的"通量"通过球面扩散。球面面积 ∝ r²，因此单位面积通量 ∝ 1/r²。这是三维欧几里得几何的直接后果，与力的具体性质无关。如果信息引力在三维空间中传播，平方反比是**唯一**与空间维度一致的长程力形式。

2. **信息传播论证**：信息从点源向外传播时，在三维空间中"稀释"程度与 r² 成正比。信息引力的强度反映的是信息场的"密度"，因此自然服从平方反比。

3. **排除其他形式的理由**：
   - **1/r（线性衰减）**：对应二维空间传播，与我们的三维宇宙不符
   - **1/r³ 或更高次**：对应偶极或多极矩效应，衰减太快，在实验可达距离上完全不可测
   - **指数衰减 e⁻ʳ/λ**：对应有质量的传播子（如 Yukawa 势）。如果信息引力的传播子无质量（类比引力子），则不应有指数截断。ITLCT 预测信息子（informaton）为无质量粒子（P6: m=0, spin=0），支持平方反比
   - **常数（不衰减）**：违反能量守恒，物理上不合理

4. **Gauss 定律类比**：如果信息引力满足类 Gauss 定律 ∮ g_info · dA = 4πG_info M_info，则在球对称情况下自动导出 g_info ∝ 1/r²。

**注意：** 平方反比是最简洁的假设。如果实验数据显示衰减更快或更慢，需要修正——这本身就是有价值的信息。

## 模块 4: Predictions（可证伪预测）

### 预测 P-IG-1: 双人 EEG 相干性增强（主要预测）

```
实验条件：高Φ组（长期冥想者，预期 Φ > 0.7）vs 低Φ组（对照组，预期 Φ < 0.3）

定量预测：
  高Φ组脑间 gamma 波段 (30-100 Hz) 相干性 > 低Φ组 15% 以上

  具体数值：
  - 高Φ组 gamma 相干性：0.45 ± 0.08 (均值 ± SD)
  - 低Φ组 gamma 相干性：0.30 ± 0.10
  - 效应量 (Cohen's d)：≥ 0.50 (中等效应)
  - 差异方向：单侧（高Φ > 低Φ）

统计标准：
  - p < 0.05（频率主义）
  - BF₁₀ > 3.0（贝叶斯证据）
  - 效应量 d ≥ 0.50
```

### 预测 P-IG-2: 距离依赖性（辅助预测）

```
实验条件：固定Φ组，变化物理距离 (1m, 3m, 5m, 10m)

定量预测：
  相干性增量 ΔC 与距离 r 的关系：
  ΔC(r) = ΔC₀ × (r₀/r)^β

  预测 β = 2.0 ± 0.5（平方反比）

  如果 β < 1.0 或 β > 3.5 → 平方反比假设需修正
  如果 ΔC 与 r 无关 → 信息引力不依赖物理距离（修正 r_info 定义）
```

### 预测 P-IG-3: Φ 乘积效应（辅助预测）

```
实验条件：系统控制 Φ₁ 和 Φ₂

定量预测：
  ΔC ∝ Φ₁ × Φ₂（乘法模型）优于 ΔC ∝ Φ₁ + Φ₂（加法模型）

  判据：
  - 乘法模型 R² > 加法模型 R² + 0.05
  - AIC(乘法) < AIC(加法)
```

## 模块 5: Falsification（证伪路径）

### 5.1 主要证伪条件

```
如果以下任一成立，信息引力假说失败：

条件 F1：高Φ组 vs 低Φ组相干性差异 p > 0.05 且 d < 0.20
  → 结论：无可检测的信息引力效应
  → 后果：A45 公理失败，ITLCT 失去唯一独特预测

条件 F2：相干性差异存在但不依赖距离
  → 结论：效应非引力型（可能是共享注意或社会信号）
  → 后果：需要替代解释，A45 需重大修正

条件 F3：加法模型 (Φ₁+Φ₂) 优于乘法模型 (Φ₁×Φ₂)
  → 结论：效应不是耦合型
  → 后果：信息引力力公式需修正（可能为线性叠加场）
```

### 5.2 弱证伪条件

```
如果以下成立，信息引力假说需修正但不完全失败：

条件 W1：p < 0.05 但 d < 0.50（效应量偏小）
  → 结论：效应存在但比预测弱
  → 后果：G_info 估计需下调

条件 W2：β ≠ 2.0 但 0.5 < β < 3.5
  → 结论：衰减规律非标准平方反比
  → 后果：可能存在信息屏蔽/增强效应，r_info 需重定义

条件 W3：效应仅在特定频段存在（如仅 alpha 不含 gamma）
  → 结论：信息引力频率选择性
  → 后果：A45 需增加频率依赖项
```

## 模块 6: Experiment Design（实验设计）

### 6.1 样本

```
总样本：N = 40 对 (80 人)

分组：
  高Φ组：20 对
    - 入选标准：≥5 年正念冥想经验，每周 ≥ 5 次
    - 预期 Φ 范围：0.5-0.9
    - 来源：冥想中心、正念研究数据库

  低Φ组（对照）：20 对
    - 入选标准：无冥想经验，无精神疾病史
    - 预期 Φ 范围：0.1-0.4
    - 来源：社区招募，年龄/性别/教育匹配

排除标准：
  - 神经/精神疾病史
  - 使用影响神经活动药物
  - EEG 伪影过多 (>30% 数据段)
  - 听力/视力严重损伤
```

### 6.2 样本量论证

```
目标效应量：d = 0.50 (中等效应)
显著性水平：α = 0.05 (单侧)
统计功效：1-β = 0.80

功效分析（独立样本 t 检验）：
  n = (z_α + z_β)² × 2σ² / δ²
  n = (1.645 + 0.842)² × 2 / 0.50²
  n = 6.175 × 2 / 0.25
  n ≈ 49.4 / 组 (频率主义最低要求)

考虑因素：
  - 20% 数据损失率 → 需 62/组
  - 但 paired design 降低方差 → 约 25/组 即可
  - 贝叶斯设计允许更小样本 → 20/组 合理

最终：N = 20 对/组，总计 40 对 (80 人)
  - 频率主义功效：~0.70 (偏低，贝叶斯补偿)
  - 贝叶斯设计：BF₁₀ > 3 可在更小样本达到
  - 如初步结果 BF₁₀ 在 1-3 之间，扩展至 N = 30 对/组
```

### 6.3 实验方法

#### 设备

```
EEG 系统：
  - 型号：64 通道 BrainVision/Biosemi
  - 采样率：1000 Hz
  - 参考：平均参考
  - 同步：两台 EEG 系统通过 TTL 脉冲精确同步 (< 1ms 误差)

fMRI（辅助验证子集 N=10 对）：
  - 型号：3T Siemens Prisma
  - 序列：T2* EPI, TR=1s, 多波段
  - 用途：验证 EEG 结果的 BOLD 相关物
```

#### 实验流程

```
每对被试完成以下条件（within-pair 设计）：

条件 1: 基线 (5 min)
  - 两人分别静坐，眼闭，无交互
  - 间距：3 米
  - 测量：各自 EEG + 脑间相干性

条件 2: 共同冥想 (10 min)
  - 两人同时进行正念呼吸冥想
  - 间距：1 米
  - 高Φ组：预期双方均进入高 Φ 状态
  - 低Φ组：预期 Φ 保持较低

条件 3: 距离变化 (4 × 5 min)
  - 共同冥想，间距分别为 1m, 3m, 5m, 10m
  - 顺序随机化（拉丁方设计）

条件 4: 屏蔽控制 (5 min)
  - 共同冥想，间距 1m
  - 两人之间放置法拉第笼（排除 EM 耦合）
  - 如果法拉第笼消除效应 → 效应为 EM 而非信息引力

条件 5: 不匹配控制 (5 min)
  - 高Φ者 + 低Φ者配对
  - 预测：中等效应 (Φ_high × Φ_low < Φ_high × Φ_high)

总时长：约 90 分钟/对
```

### 6.4 数据分析

#### 主要分析

```
指标：脑间 gamma 波段 (30-100 Hz) 加权相位滞后指数 (wPLI)

预处理：
  1. 带通滤波 1-100 Hz
  2. ICA 去除眼动/肌电伪影
  3. 分段 (2s epochs, 50% overlap)
  4. 坏段剔除 (振幅 > 100μV)

脑间相干性计算：
  - 对每对同源电极 (F3-F3, C3-C3, P3-P3 等)
  - 计算 wPLI (加权相位滞后指数)
  - 取全脑平均 wPLI 作为脑间相干性指标

统计检验：
  1. 频率主义：
     - 独立样本 t 检验（高Φ vs 低Φ）
     - 效应量：Cohen's d + 95% CI
     - 多重比较校正：FDR (Benjamini-Hochberg)
  
  2. 贝叶斯分析：
     - 贝叶斯 t 检验 (JASP/BayesFactor R 包)
     - 先验：Cauchy (r=0.707)
     - 判据：BF₁₀ > 3 (中等证据), > 10 (强证据)
  
  3. 距离依赖性：
     - 非线性回归：ΔC = a × r^(-β) + c
     - 参数估计：β 的 95% CI
     - 模型比较：β=2 (平方反比) vs β 自由 (AIC/BIC)
  
  4. 乘法 vs 加法：
     - 线性混合模型：ΔC ~ Φ₁ × Φ₂ vs ΔC ~ Φ₁ + Φ₂
     - AIC/BIC 比较
```

#### 辅助分析

```
1. 频段特异性：
   - 分别计算 delta, theta, alpha, beta, gamma 的脑间 wPLI
   - 确定效应是否频段特异

2. 时间动态：
   - 滑动窗口分析 (30s 窗口, 5s 步长)
   - 观察相干性随冥想深入的时间演化

3. 法拉第笼控制：
   - 配对 t 检验：有笼 vs 无笼
   - 如果 p < 0.05 → 效应部分为 EM 耦合

4. fMRI 验证 (子集)：
   - 脑间 BOLD 相关性
   - 与 EEG 相干性的相关
```

### 6.5 误差控制

```
系统误差：
  ✓ TTL 同步确保 EEG 时间对齐 (< 1ms)
  ✓ 同源电极对排除空间混淆
  ✓ 法拉第笼排除电磁耦合
  ✓ 距离变化排除近场效应

统计误差：
  ✓ FDR 多重比较校正
  ✓ 贝叶斯分析不依赖 p 值
  ✓ 效应量 + CI 报告
  ✓ 预注册防止 p-hacking

混淆变量控制：
  ✓ 年龄/性别/教育匹配
  ✓ 呼吸节律监测（排除呼吸同步导致的伪相干）
  ✓ 心率监测（排除心脏节律耦合）
  ✓ 随机化条件顺序
  ✓ 双盲分析（数据分析者不知分组）

已知未控制：
  ⚠️ 冥想经验可能影响基线 EEG 模式（部分混淆）
  ⚠️ 冥想者可能有更好的静坐能力（运动伪影差异）
  ⚠️ Φ 的操作性测量依赖代理指标（PCI/Lempel-Ziv）
```

### 6.6 伦理与注册

```
伦理审查：提交机构 IRB 审查
知情同意：书面知情同意
预注册：OSF Prereg 或 AsPredicted
数据共享：原始数据 + 分析代码发布于 OSF
开放获取：无论结果如何发表
```

## 模块 7: Limitations（局限性）

### 7.1 理论层面的不确定性

1. **G_info 量级不确定**：10⁻⁴⁰ G_N 是量级估计，可能偏差数个量级。如果实际值更小，效应可能完全不可测。

2. **r_info 的定义模糊**：信息空间距离 r_info 是否等于物理距离？如果信息引力在"信息空间"中传播，物理距离可能不是正确的变量。

3. **Φ 作为信息质量的代理**：Φ（整合信息）是否真正反映了 I_structured？Φ 的计算在大系统中是 NP-hard 的，实验中使用的代理指标（PCI、Lempel-Ziv 复杂度）可能引入系统偏差。

4. **乘法形式 vs 其他**：虽然有理论理由选择 Φ₁ × Φ₂，但无法排除更复杂的耦合形式（如 Φ₁^α × Φ₂^β，α ≠ 1）。

### 7.2 实验层面的不确定性

1. **冥想者 vs 非冥想者不等价于高Φ vs 低Φ**：冥想经验与 Φ 相关但不等价。可能存在高Φ非冥想者和低Φ冥想者。

2. **EEG 相干性 ≠ 信息引力**：脑间 EEG 相干性可能有多种来源（共享环境噪声、呼吸耦合、微弱声音/光线同步）。虽然设计了控制条件，但无法完全排除所有替代解释。

3. **样本量可能不足**：N=40 对频率主义分析的功效为 ~0.70，低于标准 0.80。依赖贝叶斯补偿，但如果效应量 < 0.50，可能需要更大样本。

4. **单一实验室**：首次实验的结果需要独立复制才能确认。

### 7.3 我们不知道什么

- 信息引力是否存在（这正是实验要回答的）
- G_info 的精确值
- 信息引力的传播速度（等于光速？超光速？）
- 信息引力是否有量子版本
- 效应在非生物高Φ系统（如量子计算机）中是否存在

### 7.4 最坏情况

如果实验结果为**强阴性**（F1 满足：p > 0.05, d < 0.05, BF₀₁ > 10）：

- **对 ITLCT 的影响**：A45 失败，需要放弃信息引力概念或大幅修正
- **对理论的影响**：ITLCT 退化为 IIT 的扩展框架，失去独立理论地位
- **积极面**：排除了一个假说，节省了后续研究资源
- **发表价值**：阴性结果同样值得发表（预注册保障）

## 附录 C: 与现有超感知研究的区别

本实验**不是**超感知（ESP）研究。关键区别：

| 方面 | ESP 研究 | 本实验 |
|------|---------|--------|
| 理论基础 | 无明确物理机制 | ITLCT A45 公理 + Landauer 原理 |
| 预测类型 | 定性（"有效应"） | 定量（d ≥ 0.50, 15% 差异） |
| 证伪条件 | 通常缺乏 | 明确定义（F1-F3） |
| 距离依赖性 | 通常声称无距离效应 | 预测平方反比衰减 |
| 控制条件 | 通常不足 | 法拉第笼 + 呼吸/心率监测 |
| 预注册 | 罕见 | 强制 |
