# 实验假设
H1: 代谢率 ∝ L × Φ (R² > 0.5)
H2: 耦合系数σ₃在 280-310K 达到峰值
H3: 偏离预测模式揭示新生物学原理

# 数据来源
- Palacios 跨物种数据 (N=47 物种，如获取)
- 文献补充数据 (N=100+ 物种)

# 变量测量
- 代谢率：基础代谢率 (BMR), 单位 W/kg
- 生命度 L: 基于信息处理速率、自主性、能量耗散
  L = (I_eff / I_max) × (A / A_max) × (σ / σ_max)
- 意识Φ: 基于神经复杂性 (脊椎动物) 或行为复杂性代理 (无脊椎动物)

# 统计方法
- 多元回归：代谢率 ~ L + Φ + L×Φ
- 系统发育校正：PGLS (Phylogenetic Generalized Least Squares)
- 温度效应：分组分析 (280-310K vs 其他)

# 预期结果
- 支持 H1: R² > 0.5, p < 0.001
- 支持 H2: σ₃峰值在 280-310K, 效应量 d > 0.8
- H3: 识别 2-3 个"异常"物种，进一步研究

# 时间线

- T3: 苏醒 (return of consciousness)
- T4: 恢复 (完全清醒)

### 2.5 理论缺口修补方案

#### 缺口 1: Eq-9 耦合项数学严格化

**当前 Eq-9:**
```
dS/dt = σ₀ + σ₁·L + σ₂·Φ + σ₃·L×Φ
```

**问题:** 耦合项 L×Φ的物理意义和数学推导不够严格

**修补方案:**
```python
# 从非平衡热力学第一性原理推导

# 步骤 1: 定义系统熵产生
dS/dt = dS_int/dt + dS_ext/dt
其中：dS_int/dt = 内部熵产生 (≥0)
      dS_ext/dt = 与环境的熵交换 (可正可负)

# 步骤 2: 生命度 L 的热力学定义
L = (F - F_eq) / (F_max - F_eq)
其中：F = 系统自由能
      F_eq = 平衡态自由能
      F_max = 最大可能自由能 (给定约束下)

# 步骤 3: 意识Φ的信息论定义
Φ = I_eff / I_max
其中：I_eff = 有效整合信息
      I_max = 最大可能整合信息

# 步骤 4: 耦合项推导
# 假设：意识系统通过信息整合优化自由能耗散
dF/dt = -α·Φ·(F - F_eq)  # 意识加速自由能最小化

# 代入熵产生定义：
dS/dt = -dF/dt / T + σ_background
dS/dt = α·Φ·(F - F_eq) / T + σ_background

# 代入 L 定义：
dS/dt = α·Φ·L·(F_max - F_eq) / T + σ_background

# 展开为背景项 + 生命项 + 意识项 + 耦合项：
dS/dt = σ₀ + σ₁·L + σ₂·Φ + σ₃·L×Φ

# 其中：
σ₀ = σ_background (基础熵产生)
σ₁ = β·(F_max - F_eq) / T (生命度系数)
σ₂ = γ·(F_max - F_eq) / T (意识系数，独立于生命度)
σ₃ = α·(F_max - F_eq) / T (耦合系数)

# 预测：
σ₃ > 0 (协同效应：生命 + 意识 > 各自独立贡献之和)
σ₃随温度变化，峰值在 280-310K (碳基生命优化)
```

#### 缺口 2: 生命度 L(S) 操作化定义

**当前问题:** L(S) 理论定义清晰，但如何从实证数据计算不明确

**操作化方案:**
```python
# 生命度 L(S) 操作化定义
# L = (I_eff / I_max) × (A / A_max) × (σ / σ_max)

# 1. 信息处理速率 I_eff
# 测量方法：
# - 神经系统：神经发放率、突触传递速率
# - 细胞系统：信号转导速率、基因表达变化速率
# - 生态系统：物质/能量流动速率

# 标准化：
I_eff_norm = (I_eff - I_min) / (I_max - I_min)
其中 I_min/I_max来自文献或理论极限

# 2. 自主性 A
# 测量方法：
# - 行为：目标导向行为比例、环境响应灵活性
# - 生理：内稳态维持能力、应激响应效率
# - 计算：自主决策比例 (vs 反射/本能)

# 标准化：
A_norm = (A - A_min) / (A_max - A_min)

# 3. 能量耗散σ
# 测量方法：
# - 代谢率：单位质量功率 (W/kg)
# - 熵产生率：dS/dt (J/K/s)

# 标准化：
σ_norm = (σ - σ_min) / (σ_max - σ_min)

# 最终 L 计算：
L = I_eff_norm × A_norm × σ_norm

# 范围：0 (无生命) ~ 1 (最大生命度)
# 阈值：L > 0.5 可视为"显著生命"
```
