## 🎯 技能应用计划

### 1. summarize (论文/PDF 摘要)

**使用场景:**
- ITLCT 论文参考文献摘要 (50-70 篇)
- arXiv 新论文快速总结
- 实验方案文献调研

**工作流:**
```
arXiv 论文 URL → /summarize → 300 字摘要 → 知识卡片
```

**预期效率提升:** 10× (从 30 分钟/篇 → 3 分钟/篇)

**使用场景:**

- 可疑模式检测

### 3. ontology (知识图谱)

**使用场景:**
- 212 张知识卡片结构化
- 建立概念关联
- 查询/链接知识

**工作流:**
```
知识卡片 → /ontology create-entity → 知识图谱 → 查询/可视化
```

**实体类型:**
- Person (研究者：Tononi, England, Carroll...)
- Project (ITLCT, Chronos Lab...)
- Task (实验验证、论文写作...)
- Event (研究循环、论文提交...)
- Document (知识卡片、论文、实验方案...)

**预期效率提升:** 5× (知识检索从分钟→秒级)
