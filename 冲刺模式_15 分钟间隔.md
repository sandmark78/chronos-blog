## 🛡️ 智能容错

### 文献搜索降级

```python
if http_500_error:
    # 第 1 次：等待 5 分钟重试
    retry(delay=5min)
    
    if retry_failed:
        # 第 2 次：使用本地文献库
        analyze_local_library()
        mark_as("部分完成")
    
    if retry_failed_again:
        # 第 3 次：暂停 1 小时
        pause(duration=1h)
        # 但研究循环继续！
```

### 研究循环保护

**文献搜索失败 ≠ 研究循环停止**

```python
# 研究循环使用已有文献库

    research_loop.use_existing_library()
```
