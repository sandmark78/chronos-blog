## 📋 技能详解

### 1. create_card — 创建知识卡片

**用途：** 将研究成果固化为标准化知识卡片

**参数：**
- `title`: 卡片标题
- `category`: 分类（时间/生命/宇宙/意识/复杂系统/信息）
- `content`: 内容字典（definition, key_points, theories, scientists, questions）
- `sources`: 引用来源列表
- `tags`: 标签列表
- `related`: 相关概念列表

**示例：**
```python
result = create_card(
    title="熵增原理",
    category="时间",
    content={
        "definition": "孤立系统的熵永不减少",
        "key_points": ["熵是系统无序度的度量", "熵增定义时间方向"],
        "theories": ["热力学第二定律"],
        "questions": ["问题 #3: 为什么时间只有一个方向？"]
    },
    sources=["Clausius, 1865"],
    tags=["热力学", "熵"],
    related=["时间箭头", "热力学第二定律"]
)

# 返回：
# {
#   "success": True,
#   "path": "knowledge/时间/熵增原理.md",
#   "message": "知识卡片已创建：knowledge/时间/熵增原理.md"
# }
```

---

### 2. generate_log — 生成研究日志

**用途：** 将研究循环结果固化为标准化研究日志

**参数：**
- `cycle_id`: 研究循环 ID
- `problem`: 研究问题
- `problem_id`: 问题编号
- `findings`: 关键发现列表
- `hypotheses`: 新假设列表
- `questions`: 新问题列表
- `cards_created`: 创建的知识卡片列表
- `duration_seconds`: 执行时长
- `tokens_used`: 消耗 Token 数
- `quality_rating`: 质量评分 (1-5 星)
- `next_task`: 下一步建议

**示例：**
```python
result = generate_log(
    cycle_id="cycle_001",
    problem="为什么时间只有一个方向？",
    problem_id=3,
    findings=["时间箭头来自熵增", "宇宙初始低熵是前提"],
    hypotheses=["时间是信息变化的度量"],
    questions=["宇宙初始为什么低熵？"],
    cards_created=[{"title": "Past Hypothesis", "path": "..."}],
    duration_seconds=187,
    tokens_used=462400,
    quality_rating=4,
    next_task={"problem": "宇宙初始为什么低熵？", "reason": "直接后续"}
)
```

### 3. update_status — 更新任务状态

**用途：** 更新研究队列中的任务状态

**参数：**
- `task_id`: 任务 ID
- `status`: 新状态 (pending/in_progress/completed/blocked)
- `result`: 执行结果（可选）

**示例：**
```python
result = update_status(
    task_id="task_001",
    status="completed",
    result={"findings": ["..."], "cards": 2}
)
```

**相关函数：**
- `add_task(task)` — 添加新任务
- `batch_add_tasks(tasks)` — 批量添加任务
- `get_next_task()` — 获取下一个待处理任务
- `update_progress(cycle_id, cards_created, questions_generated)` — 更新进度

---

### 4. derive_questions — 衍生新问题

**用途：** 基于研究结果自动生成新的研究问题

**参数：**
- `problem`: 原始研究问题
- `findings`: 关键发现列表
- `hypotheses`: 新假设列表
- `context`: 上下文信息（可选）

**返回：** 新问题列表（包含 id, type, question, priority, source, derivation_type, tags）

**衍生类型：**
- `deep_dive` — 深入探索（针对发现的细节）
- `validation` — 假设验证
- `cross_domain` — 跨领域连接
- `reverse` — 反向思考
- `synthesis` — 统一理论

**示例：**
```python
questions = derive_questions(
    problem="为什么时间只有一个方向？",
    findings=["时间箭头来自熵增", "因为宇宙初始低熵"],
    hypotheses=["时间是信息变化的度量"],
    context={"domain": "physics"}
)

# 返回 5-7 个新问题
```

**优先级排序：**
```python
sorted_questions = prioritize_questions(questions)
```

### 5. link_concepts — 建立知识关联

**用途：** 自动建立知识卡片之间的关联

**参数：**
- `card_path`: 卡片路径
- `auto_link`: 是否自动查找关联
- `max_links`: 最大关联数

**示例：**
```python
result = link_concepts(
    card_path="knowledge/时间/熵增与时间箭头.md",
    auto_link=True,
    max_links=5
)
```

**知识图谱：**

### 6. generate_daily_report — 生成日报

**用途：** 生成研究进度报告（日报/周报）

**参数：**
- `date`: 日期字符串 (YYYY-MM-DD)，默认为今天

**示例：**
```python
# 生成日报
report = generate_daily_report("2026-03-11")
print(report["markdown"])

# 生成周报

print(report["markdown"])

# 保存报告
result = save_report(report, output_dir="problem-database/outputs/reports/")
```

## 🔗 组合使用示例

### 完整研究循环

```python
from skills.research import (
    create_card, generate_log, update_status,
    derive_questions, link_concepts
)

# 1. 创建知识卡片
card_result = create_card(
    title="Past Hypothesis",
    category="时间",
    content={
        "definition": "宇宙初始处于极低熵状态",
        "key_points": ["是时间箭头的终极前提"],
        "questions": ["问题 #32: 宇宙初始为什么低熵？"]
    }
)

# 2. 衍生新问题
new_questions = derive_questions(
    problem="为什么时间只有一个方向？",
    findings=["宇宙初始低熵"],
    hypotheses=["时间是信息度量"]
)

# 3. 添加到队列
from skills.research import batch_add_tasks
batch_add_tasks(new_questions)

# 4. 生成研究日志
log_result = generate_log(
    cycle_id="cycle_001",
    problem="为什么时间只有一个方向？",
    findings=["..."],
    cards_created=[card_result],
    duration_seconds=187
)

# 5. 更新任务状态
update_status("task_001", "completed")

# 6. 建立知识关联
link_concepts(card_result["path"])
```
