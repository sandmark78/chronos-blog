## 三层架构

### 第 1 层：研究引擎 (Chronos Engine v1)

**核心：** 7 阶段研究循环

| 阶段 | Agent | 职责 |
|------|-------|------|
| 1. 任务选择 | Chronos | 从队列选择最高优先级任务 |
| 2. 信息搜索 | Hermes | arXiv 搜索、论文抓取 |
| 3. 文献分析 | Athena | 第一性原理分析、理论对比 |
| 4. 理论推演 | Prometheus | 提出新假设 |
| 5. 思想实验 | Prometheus | 设计思想实验验证 |
| 6. 知识固化 | Mnemosyne | 生成知识卡片 |
| 7. 新问题 | Chronos | 生成问题加入队列 |

**1M Token 上下文：**
- 当前研究问题
- 相关知识卡片 (50K)
- 论文摘要 (100K)
- 历史研究日志 (100K)
- 相关理论 (50K)

```bash
# 手动启动
openclaw cron run 55e93a2e-22af-4680-8759-729ba58a45c2

# 或等待自动执行（每日 09:00）
```

### 查看系统状态

```bash
# 研究队列
cat problem-database/queue.json | jq '.stats'

# 知识卡片数量
find knowledge/ -name "*.md" | wc -l

# Cron 任务
openclaw cron list
```

```json
// 编辑 problem-database/queue.json
{
  "id": "task_009",

}
```
