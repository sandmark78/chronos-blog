### 1. 间隔自适应

```python
# 伪代码
def adjust_interval(task_type, actual_time, expected_time):
    ratio = actual_time / expected_time
    
    if ratio < 0.5:  # 实际时间远小于预期
        return current_interval * 0.75  # 缩短 25%
    elif ratio > 1.5:  # 实际时间远大于预期
        return current_interval * 1.25  # 延长 25%
    else:
        return current_interval  # 保持不变
```

### 2. 提示词优化

```python
# 伪代码
def optimize_prompt(agent, feedback):
    if feedback["quality"] < 4:  # 质量低于 4 星
        # 分析低质量原因
        if "missing_context" in feedback["issues"]:
            agent.prompt += "\n\n请确保考虑以下上下文：..."
        elif "unclear_output" in feedback["issues"]:
            agent.prompt = agent.prompt.replace(
                "输出分析结果",
                "输出结构化的分析结果，包含：1... 2... 3..."
            )
```

**执行时机：** 每次质量评分后
