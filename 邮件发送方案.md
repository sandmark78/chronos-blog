## 方案 1: 临时邮箱 API ⭐⭐⭐

### 推荐服务

| 服务 | 免费额度 | API | 适合度 |
|------|---------|-----|--------|
| **Mail.tm** | 无限 | REST API | ⭐⭐⭐⭐ |
| **Guerrilla Mail** | 无限 | API | ⭐⭐⭐ |
| **Temp Mail** | 有限 | API | ⭐⭐ |

### Mail.tm 使用示例

```python
import requests

# 1. 创建账户
response = requests.post('https://api.mail.tm/accounts', json={
    "address": "itlct.research@mail.tm",
    "password": "secure_password_123"
})

# 2. 获取 token
token_response = requests.post('https://api.mail.tm/token', json={
    "address": "itlct.research@mail.tm",
    "password": "secure_password_123"
})
token = token_response.json()["token"]

# 3. 发送邮件
headers = {"Authorization": f"Bearer {token}"}
message_data = {
    "from": {"address": "itlct.research@mail.tm", "name": "Chronos Lab"},
    "to": [{"address": "recipient@example.com", "name": "Recipient"}],
    "subject": "ITLCT Framework for Consideration",
    "text": "Dear Professor...\n\n[学术投稿邮件正文]\n\nBest regards,\nChronos Lab Team"
}

send_response = requests.post('https://api.mail.tm/messages', 
                               headers=headers, 
                               json=message_data)
```

**优点:** 免费、API 友好、可长期使用  
**缺点:** 临时邮箱可能被视为不专业

## 方案 2: SendGrid 邮件 API ⭐⭐⭐⭐⭐

### 注册流程

1. 访问 https://sendgrid.com
2. 免费注册 (100 封/天 免费)
3. 获取 API Key
4. 使用 API 发送邮件

### Python 示例

```python
from sendgrid import SendGridAPIClient
from sendgrid.helpers.mail import Mail

message = Mail(
    from_email=('itlct.research@chronos-lab.org', 'Chronos Lab'),
    to_emails='recipient@university.edu',
    subject='ITLCT: A Unified Framework for Information-Time-Life-Consciousness',
    html_content='<strong>Dear Professor...</strong><br><br>[邮件正文]</body>'
)

try:
    sg = SendGridAPIClient('SG.xxxxx_api_key_xxxxx')
    response = sg.send(message)
    print(f"Status Code: {response.status_code}")
except Exception as e:
    print(f"Error: {e}")
```

**优点:** 专业、可靠、高送达率  
**缺点:** 需要注册账户（可用临时邮箱注册）

## ⚠️ 注意事项

### 邮件礼仪

1. **称呼:** Dear Professor [Last Name],
2. **自我介绍:** 简短说明身份/研究背景
3. **关联对方工作:** 显示你了解他们的研究
4. **简洁:** 正文<500 词，附件<5MB
5. **退路:** "We understand if you are too busy..."
6. **签名:** 完整联系信息

### 隐私保护

1. **不要:** 发送未公开的个人/机构信息
2. **不要:** 群发邮件（用 BCC）
3. **要:** 使用加密连接 (SMTPS)
4. **要:** 定期更换密码
5. **要:** 使用双因素认证
